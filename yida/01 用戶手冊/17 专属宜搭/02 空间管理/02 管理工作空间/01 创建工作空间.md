了解工作空间的基本概念后，您可以参考本文创建属于您自己的专属工作空间。

## 创建工作空间

1.  使用管理员身份，登录[宜搭工作台](https://www.aliwork.com/workPlatform)。
2.  单击右上角**平台管理**

    ![[ugxt8b6935q7znbe_1719223243015-7f46aa6b-de74-4103-847f-c216209f07ae.png]]

    按钮，进入平台管理页面。
3.  在左侧菜单栏依次选择**平台专属定制** > **空间管理**。
4.  参考以下步骤，创建空间。

1.  设置空间基础信息。

| **配置页面** | **设置项** | **说明** |
| --- | --- | --- |
|
![[ugxt8b6935q7znbe_1740622875485-9f643bfc-95dc-406b-826a-46bd9d9c5e76.png]]

 | **空间名称** | 设置空间名称，建议名字不要太长。 |
|  | **默认语言** | 默认为系统设置的语言类型。 |
|  | **空间图标** | 设置空间图标，创建空间时暂不支持自定义图标，你可以创建成功后，在空间设置页，修改自定义图标。

![[ugxt8b6935q7znbe_1719478331738-bf3ad965-59b5-493a-8309-5d2c7b9a3b9d.png]]

 |
|  | **空间描述** | 设置您的空间描述信息，最多支持 100 个字符。 |

2.  选择您空间所需要的应用，点击**下一步**。

![[ugxt8b6935q7znbe_1740623073138-87bba976-ea80-4e89-ba87-8e091dce8b34.png]]

3.  选择模版&主题，最后单击**确认**，完成创建。

![[ugxt8b6935q7znbe_1740623193514-0cabb796-e174-4669-b5c3-04a1b50fd33c.png]]

## 编辑工作空间

1.  在左侧菜单栏依次选择**平台专属定制** > **空间管理**，选择需要编辑的空间，单击**编辑按钮。**

![[ugxt8b6935q7znbe_1740121909046-7d640adf-c403-4e4c-b0ae-e493dc482d16.png]]
