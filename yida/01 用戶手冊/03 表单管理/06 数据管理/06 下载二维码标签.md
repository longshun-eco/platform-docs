## 1. 功能简介

宜搭现如今提供二维码标签的自定义设计、二维码标签的批量下载、二维码标签预览等能力，在此处可以批量下载所需的表单二维码。

二维码标签配置请移步查看文档：[二维码标签](../ckyvwt/knk7hr)

![[yida_support-wtwabe-bg8ikb-xewsyc_1650962461452-9ea30eb9-41d9-4d15-8310-0e96a604256d.png]]

## 2. 操作指南

宜搭管理员和员工均可下载二维码标签，但操作路径有区别，如下：

### 2.1 管理员下载

**编辑态：**进入宜搭 >> 我的应用 >> 点击对应应用及表单 >> 数据管理 >> 下载二维码标签

![[yida_support-wtwabe-bg8ikb-xewsyc_1650962541302-a0b87f74-b797-4c30-8341-99e5c049bd98.png]]

![[yida_support-wtwabe-bg8ikb-xewsyc_1650962604744-e37debf3-daac-4cbc-9a63-47c63c0d24be.png]]

**访问态：**宜搭 >> 应用中心 >> 找到对应应用及表单数据管理页 >> 下载二维码标签

![[yida_support-wtwabe-bg8ikb-xewsyc_1650962640993-843a4d97-c18a-473f-8e99-f0467992cb6b.png]]

### 2.2 员工下载

员工仅支持在访问态下载二维码标签。

**路径：**宜搭 >> 应用中心 >> 找到对应应用及表单数据管理页 >> 下载二维码标签

![[yida_support-wtwabe-bg8ikb-xewsyc_1650962669089-3a8991ff-ccc1-4ef8-a8da-4b097d8aa694.png]]

### 2.3 可下载尺寸

选择不同的标签样式则下载的尺寸不同，在设置标签样式的时候可以查看到可下载尺寸。

![[yida_support-wtwabe-bg8ikb-xewsyc_1650962736641-ecd2b533-bffb-4fa5-9582-148104ee57c4.png]]

![[yida_support-wtwabe-bg8ikb-xewsyc_1647498344821-a492b3c3-b201-45d3-a92b-da0eded27e8e.png]]

在使用宜搭码功能中遇到任何问题或有意见反馈，可以扫码下方二维码，加入交流群进行反馈。
