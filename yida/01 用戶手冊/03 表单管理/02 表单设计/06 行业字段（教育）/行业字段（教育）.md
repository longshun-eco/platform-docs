根据教育版宜搭企业的场景需求，宜搭提供以下 9 个教育组件打通了教育行业的基础数据与宜搭，通过教育组件的形式，使得宜搭应用天然具有与教育的基础数据互联互通的能力。

**钉钉组织行业类型属于教育行业的宜搭免费版、轻享版、专业版所有版本均可使用，若非教育行业类型的组织不支持使用。**

**查看行业类型路径：****主管理员（创建人）**在【手机钉钉】>>【通讯录】>>【管理】>> 界面底部【其他】>>【更多】>>【所在行业】

## 1. 使用场景

| **组件名称** | **场景** |
| --- | --- |
| 选学校 | 可用于 K12 业务中需要进行学校选择的应用场景，用在教育局端的应用场景较多 |
| 选校区 | 可用于 K12 校园中需要选择校区的应用场景 |
| 选科目 | 可用于 K12 校园中需要进行科目选择的应用场景，例如科目兴趣班报名 |
| 选学段 | 可用于 K12 校园中需要进行学段选择的应用场景 |
| 选学年 | 可用于 K12 校园中需要进行学年选择的应用场景 |
| 选学期 | 可用于 K12 校园中需要进行学期选择的应用场景 |
| 选年级 | 可用于 K12 校园中需要选择年级的应用场景 |
| 选班级 | 可用于 K12 校园中需要选择班级的应用场景，如班级德育评价场景 |
| 选学生 | 可用于 K12 校园中需要选择学生的应用场景，例如学生评优场景 |

## 2. 可选范围

| **组件名称** | **可选范围** |
| --- | --- |
| 选学校 | 关联组织：能选择到多个学校 非关联组织：无法选择其他学校 教育局：可选择到所有学校 当前所在学校组织：只能选择自己的学校 |
| 选学生 | 主/子管理员：所有学生信息 班主任/老师：所有学生信息 家长：家校通讯里关联的学生信息 |
| 选班级 | 主/子管理员：所有班级 班主任/老师：所有班级 |

| **报表** | 目前**新建的普通表单/流程表单**直接适配支持，历史表单数据想直接在报表侧进行数据展示，需做如下操作：

2.  旧表单的实例数据，请在数据管理页内，点击详情，进入表单编辑态，**直接点击「保存」，**此时会触发表单侧数据集、字段等元数据向报表侧的同步更新，之后再提交的实例数据会被有数侧读取到。


教育组件推送到报表目前有 2 种方式：

1.  主表字段直接推送到报表，仅做展示；
2.  独立数据集形式推送到报表：

-   需使用到多表关联，采用[**视图表**](../_11/xm7x0w/yvp9hw/oim554)进行配置，
-   如果视图表上限无法满足需求，采用[**数据准备**](../../yida_qalist/uop0b0ndpgc0gevb/dkm8ph) 功能，目前无上限。

-   **主表字段形式**

-   **独立的数据形式**

-   既可做数据展示，也可进一步做数据加工处理分析，提供较强的可扩展性和灵活性

## 4. 常见问题

### 4.1 教育组件使用交流群

教育组件使用交流和建议可扫码或搜索群号「32482084」加入社区交流群咨询。

![[yida_support-wtwabe-eb291o_1639127958908-088509ee-4f81-443b-832c-f1de9f493f98.jpeg]]

### 4.2 为什么学生组件不能选择学生名字搜索

目前只支持精确搜索 ，且不支持下拉选择学生，后续会进行优化，辛苦持续关注。

效果示例：

![[yida_support-wtwabe-eb291o_1645162827809-596dc6d9-599d-4eae-a9ac-9b8007649a1f.png]]

### 4.3 为什么消息通知选择了通知学生会自动通知到家长？

该情况是受教育行业的特殊逻辑影响，目前教育行业只有家校群（只有老师和家长）和师生群（只有老师和学生），在教育核心场景应用内无法明确确认3种角色在同一个群里，因此目前通知采用同时发布给学生和家长的形式，并且家长和学生的关系可以在钉钉家校通讯录进行维护。

### 4.4 报表中教育组件展示的数据带有 \[""\] 符号怎么去掉?

该问题是已知问题，新建表单提交的数据已正常修复不会带有\[""\]，旧表单数据可用字符串函数进行处理，需要在报表中设置公式字段，具体请参考以下截图配置：
