**GETUSERNAME**函数用于获取当前登录用户在当前组织的姓名。

## 支持产品场景

GETUSERNAME函数支持在宜搭的以下场景中使用。

-   表单（主表字段）
-   表单（业务关联规则）
-   表单（校验）
-   流程设计（校验规则）
-   流程设计（关联操作）

## 格式

```
GETUSERNAME()
```

## 返回值

| **返回值类型** | **说明** |
| --- | --- |
| 文本 | 当前登录用户在当前组织的姓名。 |
