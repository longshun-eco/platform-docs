本文为你列举了宜搭所有人员相关的函数。

## 函数概览

| **函数名称** | **说明** |
| --- | --- |
| [USER](namr73vcz90gduud/vwncwe) | 无参数模式：User() 获取当前登录人； 有参数模式：User(level) 当前登录人的主管， level 是 获取当前登录人的主管。 |
| [USERFIELD](namr73vcz90gduud/oyuexdmpaxiys2tm) | 人员搜索框需要联动带出一些基本信息。 |
| [DIRECTOR](namr73vcz90gduud/pvw37ge84kxg9x28) | 获取当前登陆人的主管名称。 |
| [EMPLOYEE](namr73vcz90gduud/sxc8gv9dkhmt8zro) | 返回对应的人员信息对象数组（即人员组件底层的值）。 |
| [GETUSERNAME](namr73vcz90gduud/zl5s2mdi2ue214n4) | 获取当前登录人昵称。 |
| [LOGINUSER](namr73vcz90gduud/iak2o5xcbksb13wz) | 登录人。 |
| [LOGINUSERWORKNO](namr73vcz90gduud/sprth45e4nh19yhw) | 获取当前登录人员工唯一ID(员工UserID)。 |
| [DEPTNAME](namr73vcz90gduud/igrvxuztr0syw9sn) | 用于根据userId获取指定用户所在的部门信息。 |
