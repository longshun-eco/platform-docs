本文列举了宜搭所有的数学函数，以及数学函数常用使用场景。

## 函数概览

| **函数名称** | **说明** |
| --- | --- |
| [COUNT](nn2gzi/ygcgwobc4tewnmxr) | 统计指定表单提交的数据总数。 |
| [ADD](nn2gzi/chzvqqza7lig19gk) | 计算多个字段值总和。 |
| [AVERAGE](nn2gzi/cmntz03rxr7lvg8i) | 获取一组数值的算术平均值。 |
| [MAX](nn2gzi/hrmegrov95qkft01) | 返回一组数字中的最大值。 |
| [MIN](nn2gzi/tzmz5prxfk1m6ooo) | 返回一组数字中的最小值。 |
| [ABS](nn2gzi/nqxuge2x337g5r7u) | 当数值为负数的时候返回他的绝对值数值。 |
| [ROUND](nn2gzi/gzx4fgcgya1g7dzv) | 数值出现小数点的时候，将小位数指定到对应的位数。 |
| [CEILING](nn2gzi/pfdukipqlw5rko8s) | 返回将参数 number 向上舍入（沿绝对值增大的方向）为最接近的指定基数的倍数。 |
| [FLOOR](nn2gzi/gkzdswxg4g18443y) | 将参数 number 向下舍入（沿绝对值减小的方向）为最接近的 significance 的倍数。 |
| [INT](nn2gzi/qht3on9d40988egd) | 将数字向下舍入到最接近的整数。 |
| [LOG](nn2gzi/bunl3zq1z68guymc) | 根据指定底数返回数字的对数。 |
| [MOD](nn2gzi/smcm971lepq2nv6o) | 返回 2 个数值组件相除的余数。 |
| [POWER](nn2gzi/biskll33337npp7d) | 计算出数值组件乘幂的值。 |
| [FIXED](nn2gzi/an1t8blpsi4aue68) | 将数字舍入到指定的小数位数，以十进制数格式对该数进行格式设置，并以数字形式返回结果。 |
| [SQRT](nn2gzi/ngysm1kw31gkmbqh) | 取数值组件值正的平方根。 |
| [SUM](nn2gzi/ab2ef6nzxozn0xfe) | 函数使所有以参数形式给出的数字相加并返回和。 |
| [PRODUCT](nn2gzi/wu3xhb5ve6v0tvci) | 返回结果为双精度浮点类型。 |
| [SUMPRODUCT](nn2gzi/zrgy7twy6d9ry6ey) | 在给定的数组中，将数组间对应的元素相乘，并返回乘积之和。 |
| [LARGE](nn2gzi/rnyume8c4kkywdpg) | 返回数据集中第k个最大值。 |
| [SMALL](nn2gzi/kruy97m10v8mt3mt) | 返回数据集中第 k 个最小值。 |

## 常用场景

### 计算多条明细总金额

-   在采购报销业务场景下，会有多条类目一起报销，此时则需要统计多条明细的总金额，可以按照以下步骤实现：

(1) 在表单拉取一个子表单组件（配置单价、数量、金额）,子表单外拉取一个数值/单行文本组件

![[yida_support-wtwabe-cnzrgo-gvtpe4-nn2gzi_1639374173083-7e55d100-8024-4171-a35f-3a897c0912a8.png]]

(2) 在子表单的金额组件配置「子表单.单价\*子表单.数量」，然后在子表单外的数值/单行文本组件配置 「SUM(子表单.金额)」

![[yida_support-wtwabe-cnzrgo-gvtpe4-nn2gzi_1639374256642-d333f0e5-abe6-46ce-afbd-01cae4ffd83f.png]]
