本文为你列举了宜搭所有的文本函数。

## 函数概览

| **函数名称** | **说明** |
| --- | --- |
| [ARRAYGET](wcguece1v2ok8b00/mggsmf) | 使用该函数，可以获取数据集中指定第几个数的值。 |
| [CONCATENATE](wcguece1v2ok8b00/mbe6045dlu6ebg33) | 使用该函数，可以将多个文本字符串合并成一个文本字符串。 |
| [LEFT](wcguece1v2ok8b00/xylna43s4gpoyz9k) | 使用该函数，可以从一个文本字符串的第一个字符开始返回指定个数的字符，如果文本长度小于自定字符个数，就将原始文本返回。 |
| [RIGHT](wcguece1v2ok8b00/ch2gvwhzh2bn9u82) | 从一个文本字符串的最后一个字符开始返回指定个数的字符。 |
| [LEN](/ytnegt00zgbwos60) | 使用该函数，可以获取文本字符串中的字符个数。 |
| [LOWER](wcguece1v2ok8b00/aire3ioygvfi7vul) | 将一个文本中的所有大写字母转换为小写字母。 |
| [UPPER](wcguece1v2ok8b00/fm0og6ccdh0n0q2k) | 将一个文本字符串中的所有小写字母转换为大写字母。 |
| [MID](wcguece1v2ok8b00/pghawu8hdwng91in) | 返回文本中从指定位置开始的指定个数的字符。 |
| [PINYINHEADCHAR](wcguece1v2ok8b00/dvfmdhg8i3ohg0c4) | 返回字符串中的每个汉字首字母，首字母为大写字母。 |
| [REPLACE](wcguece1v2ok8b00/gtz0638tp96gbnhf) | 根据指定的字符数，将部分文本字符串替换为不同的文本字符串。 |
| [REPT](wcguece1v2ok8b00/scd5m0bi74omfpyg) | 可以将文本重复一定次数。 |
| [RMBFORMAT](wcguece1v2ok8b00/ff4mghegi9kbsc7m) | 将数字格式化为人民币格式。 |
| [SEARCH](wcguece1v2ok8b00/mhwnrwy04c03o2z4) | 获取文本1在文本2中的开始位置。 |
| [SPLIT](wcguece1v2ok8b00/yf6hkwf5lzcyin93) | 将文本按指定分割符分割成数组，需配合ARRAYGET来获取指定位置字符。 |
| [TEXT](wcguece1v2ok8b00/bt0wov266h45hpi4) | 格式化成文本。 |
| [TRIM](wcguece1v2ok8b00/tzmpoqk3e4szn7ar) | 删除字符串首尾的空格，但是会保留字符串内部作为词与词之间分隔的空格。 |
| [UUID](wcguece1v2ok8b00/xb7gg8vrhik5yg5s) | 生成唯一字符串赋值给单行输入框。 |
| [VALUE](wcguece1v2ok8b00/awk6sfyg9vcqe1g1) | 将文本转换成数字。通常是用作数据已经录入需要变通计算的场景。 |
