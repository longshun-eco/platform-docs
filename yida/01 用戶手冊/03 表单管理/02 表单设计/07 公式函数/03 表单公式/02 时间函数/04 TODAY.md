**TODAY**函数用于获取今天的日期。

## 支持产品场景

TODAY函数支持在宜搭的以下场景中使用。

-   表单（主表字段）
-   表单（业务关联规则）
-   表单（校验）
-   流程设计（校验规则）
-   流程设计（关联操作）
-   集成自动化
-   流程-节点
-   连接器

## 格式

```
TODAY()
```

## 返回值

| **返回值类型** | **说明** |
| --- | --- |
| 日期对象 | 当前时间`今天`的日期对象。 |
