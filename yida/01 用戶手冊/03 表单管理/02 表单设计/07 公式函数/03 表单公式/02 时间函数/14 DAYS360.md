**DAYS360**函数按每月30天的方式，计算两个日期之间相隔的天数。使用 DAYS360 时，会通过计算两个日期之间完整月份的数量，乘以30，然后根据需要添加开始和结束月份的剩余天数来进行计算。

## 支持产品场景

DAYS360函数支持在宜搭的以下场景中使用。

-   表单（主表字段）
-   表单（业务关联规则）
-   表单（校验）
-   流程设计（校验规则）
-   流程设计（关联操作）
-   集成自动化
-   流程-节点
-   连接器

## 格式

```
DAYS360(end_date,start_date,method)
```

## 属性说明

| **属性** | **是否必填** | **说明** |
| --- | --- | --- |
| end_date | 是 | 结束时间的时间对象。 |
| start_date | 是 | 开始时间的时间对象。 |
| method | 否 | 用于标识如何处理开始日期和结束日期落在同一月份的情况。取值：
-   **true**：使用欧洲方法进行计算。
-   **false**：使用美国（NASD）方法进行计算。

如果不填，默认为false。

**说明：** 例如，计算2023-01-01至2023-12-31之间的差值。

-   使用美国方法计算后的值为360。
-   使用欧洲方法计算后的值为359。

 |

## 返回值

| **返回值类型** | **说明** |
| --- | --- |
| 数字 | 指定时间之间相差的天数。 |
