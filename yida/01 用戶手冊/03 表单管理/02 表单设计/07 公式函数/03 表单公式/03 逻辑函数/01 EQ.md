**EQ**函数用于比较两个值是否相等，支持文本、数字、日期等组件。

## 支持产品场景

EQ函数支持在宜搭的以下场景中使用。

-   表单（主表字段）
-   表单（业务关联规则）
-   表单（校验）
-   流程设计（校验规则）
-   流程设计（关联操作）
-   集成自动化
-   流程-节点
-   连接器

## 格式

```
EQ(value1,value2)
```

## 属性说明

| **属性** | **是否必填** | **说明** |
| --- | --- | --- |
| value1 | 是 | 需要判断是否相等的值，支持文本、数值、日期等。 |
| value2 | 是 |  |

## 返回值

| **返回值类型** | **说明** |
| --- | --- |
| 布尔值 | 如果两个值相等，则返回**true**，否则返回**false**。 |
