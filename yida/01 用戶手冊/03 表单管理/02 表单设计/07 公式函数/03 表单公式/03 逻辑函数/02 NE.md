**NE**函数用于比较两个值是否不相等，如果不相等，则返回true。

## 支持产品场景

NE函数支持在宜搭的以下场景中使用。

-   表单（主表字段）
-   表单（业务关联规则）
-   表单（校验）
-   流程设计（校验规则）
-   流程设计（关联操作）

## 格式

```
NE(value1,value2)
```

## 属性说明

| **属性** | **是否必填** | **说明** |
| --- | --- | --- |
| value1 | 是 | 需要进行比较的值，支持使用数值、文本、日期等组件。 |
| value2 | 是 |  |

## 返回值

| **返回值类型** | **说明** |
| --- | --- |
| 布尔值 | 两个参数比较后的结果，取值。
-   **true**：如果返回true，表示两个值不相等。
-   **false**：如果返回false，标识两个值相等。

 |
