**OR**函数表示或。函数中的所有逻辑表达式，只要有一个为true，OR函数的结果就为true。

## 支持产品场景

OR函数支持在宜搭的以下场景中使用。

- 表单（主表字段）
- 表单（业务关联规则）
- 表单（校验）
- 流程设计（校验规则）
- 流程设计（关联操作）
- 集成自动化
- 流程-节点
- 连接器

## 格式

```
OR(condition1,condition2....condition.N)
```

## 属性说明

| **属性** | **是否必填** | **说明** |
| --- | --- | --- |
| condition1 | 是 | 需要评估的逻辑表达式或条件。 用于判断两个或多个条件是否有一个为真（即是否有一项满足特定条件）。 如果所有的条件都为假（逻辑值为false），那么OR函数的返回false。 如果有任何一个条件为真（逻辑值为true），那么OR函数返回true。 |
| condition2 | 是 |  |

## 返回值

| **返回值类型** | **说明** |
| --- | --- |
| 布尔值 | 函数中的所有逻辑表达式，只要有一个为true，OR函数的结果就为true。 |
