## 功能介绍

电话号码组件可以用于识别电话号码是否符合规范，支持国家区号和拨打电话。

![[yida_support-wtwabe-up3ftx-un0gpwnpqeq3l6d8_1724815764201-86bac8e3-0357-4d6c-b070-2572788ed847.png]]

![[yida_support-wtwabe-up3ftx-un0gpwnpqeq3l6d8_1724815643317-ef2efca1-b41b-41f1-82bc-64311dbdb3ff.png]]

| **配置项** | **说明** | **属性值** |
| --- | --- | --- |
| **标题** | 用于配置组件的展示标题，支持配置多语言内容和绑定变量。 | lable |
| **占位提示** | 用于配置输入框中默认展示的内容，支持配置多语言内容和绑定变量。 | placeholder |
| **描述信息** | 组件下方的描述信息内容，通常用于补充说明。 | tips |
| **状态** | 用于配置组件状态，支持普通、禁用、只读、隐藏，四种状态。 | behavior |
| **类型** | 用于限制输入的电话号码类型，支持手机、固话、手机和固话，三种类型。 | type |
| **国家电话区号** | 用于配置是否开启国家区号。 | enableCountryCode |

## 注意事项

-   只能在表单详情页和数据管理页内使用，其他任何场景均不支持使用和展示。
-   将自动校验手机号码正确性，同时可在手机上点击拨打，由于数据敏感性，系统会进行加密存储，另外请注意数据使用安全。
