## 1. 简介

### 1.1 功能简介

表单是宜搭的基础工具，数据的填报采集都需要依赖表单来完成。因此，表单也是数据的来源。

在宜搭，有两种表单类型：

**普通表单**：可以收集业务进展中的所有数据，收集上来的数据还可以进行分权协作，将数据权限 [发布](wtwabe/ckyvwt/ulqab1) 给需要管理数据的成员或者发布给外部成员进行外部数据收集。

**流程表单**：可以让数据自下而上层层流转，逐级审批。也可以分配权限，进一步管理流程数据。

普通表单与流程表单最大的区别就是流程表单需要逐级审批，且需要设置流程 。

### 1.2 应用场景

- 普通表单：调查统计、在线报名、销售上报、会议预约、采购入库、订单录入、扫码签到等。
- 流程表单：报销审批、工作汇报、任务派单、员工请假、物品领用、维修申请、意见反馈等。

### 1.3 使用条件

表单为宜搭免费功能，所有版本的用户均可免费使用。

对于免费版用户：有 20000 条的数据条数、20G 附件容量的提交限制。详情参考：[宜搭收费政策](../yida_qalist/iif7ge "宜搭收费政策")

## 2. 表单创建方式

创建宜搭应用后，创建表单/流程表单的方式如下：

方法一：进入新创建的应用页面管理 >> 选择新建表单/新建流程表单

方法二：进入应用后 >> 点击应用左上角+号的新建表单/新建流程表单

## 3. 表单设置

创建表单后，可以开始表单设置。流程表单和普通表单在表单设置上略有区别，流程表单比普通表单多了「流程设置」的步骤。

（1）普通表单

设置步骤：进入表单设计 >> 选取需要使用的组件 >> 保存

![[wtwabe_1638933908951-70122ac1-f3f6-4f50-956f-4c45689a616b.png]]

（2）流程表单

设置步骤：进入表单设置 >> 选取需要使用的组件 >> 保存 >> 流程设计 >> 设置审批人/执行人/抄送人等节点 >> 保存 >> 发布流程

![[wtwabe_1638934031538-72f4e21e-4aae-494c-9339-00937d634072.png]]

![[wtwabe_1638934171661-848a0b3a-0e1f-4bca-9de7-4e20b340c113.png]]

按照上述步骤设置好后，可将应用点击上线，然后成员即可访问该表单填报数据。对于提交的数据，管理员可以在「数据管理」中查看

### 3.1 表单设计

表单设计分为三大块，如图所示：

![[wtwabe_1638934352097-9c1de56e-a44a-4850-abdb-748f524c1517.png]]

组件选择：选择组件，不同的组件可以存储不同类型的数据

画布区：可以布局组件排列顺序等

组件属性：包括单个组件段的属性设置以及整个表单的属性设置

### 3.2 流程设计

只有流程表单会涉及设置流程设计，普通表单无需设置

![[wtwabe_1638934435575-663f9ca9-cc4f-4a53-ab14-82064025f3d2.png]]

![[wtwabe_1638934593573-7bcc90bb-42fb-405c-bfa3-5abbe84d98eb.png]]

节点类型选择：根据场景需求选择审批人、执行人或抄送人等

节点人员以及权限配置：可审批审批人/执行人/抄送人的具体人员规则，字段权限以及审批权限等

### 3.3 表单保存

无论是普通表单还是流程表单，表单设计中将组件配置完后续点击右上角的保存才算正常的创建一个表单

![[wtwabe_1638934630125-bcc646ae-746d-4dd5-ae2b-ef139ee3123e.png]]

### 3.4 流程发布

流程表单系统默认自带流程节点，且审批人为发起人自己，避免管理员忘记设置流程节点导致无法正常运转流程，可自定义修改流程节点；若自定义修改流程节点后，需点击流程页面的发布才能生效

注:新设置的流程节点发布后只针对新提交的数据生效，已提交的历史数据还是按照之前的流程规则执行

![[wtwabe_1638934687901-d63199f0-8ca3-465a-aecf-60fc75602440.png]]

## 4. 数据管理

当员工通过表单提交数据后，拥有应用管理权限的管理员可以在 数据管理 中对提交的数据进行导入、导出、编辑、删除、查看、打印等操作，目前有2个入口可以查看到数据管理

入口一：对应表单/流程表单 >> 编辑表单/流程表单 >> 数据管理

![[wtwabe_1638934739911-17c07a56-81fd-4169-ba2b-08d89dc991cc.png]]

入口二：对应表单/流程表单 >> 数据管理

![[wtwabe_1638934792602-77689f23-e2a8-458e-97ae-9907c1b8ba5d.png]]
