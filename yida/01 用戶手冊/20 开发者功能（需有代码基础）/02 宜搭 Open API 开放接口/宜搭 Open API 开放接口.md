## 1. 前言

开放接口的主要使用场景包括如下：

1.  定时将数据拉回到自建系统中进行精加工或归档。
2.  将表单提交、流程发起操作的入口集成到现有平台上。

如果你有类似上述需求的使用场景，可以考虑使用宜搭的开放接口能力。

目前，宜搭已经将 Open API 全部上架到了钉钉开放平台, 但是仍然会保留[宜搭 Open API (旧)](rrwdug/agb8im)。通过钉钉开放平台调用宜搭 Open API 是更好的选择。

[**钉钉开放平台新版服务端 API 概述**](https://developers.dingtalk.com/document/app/dingtalk-openapi-overview)
