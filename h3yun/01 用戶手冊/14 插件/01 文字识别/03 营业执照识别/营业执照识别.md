## 应用场景

在企业资质管理、合作伙伴认证等场景，需要快速录入营业执照信息。通过 `文字识别` 插件的 `营业执照识别` 能力，只需上传执照图片，即可自动识别企业名称、统一社会信用代码等关键信息，显著提高效率和数据准确度。

例如，在表单 `供应商档案` 中上传营业执照后，自动识别其中的信息并填充到`企业名称`、`法人`、`统一信用代码`等控件中。效果如图所示。

![[2707_6b4dc6d399087de4.gif]]

## 配置说明

### 插件简介

`营业执照识别` 可识别营业执照中所有关键信息并自动填充到指定的控件。它属于付费插件 `文字识别` 的其中一种识别类型，每次调用消耗并自动扣除一次调用次数，当可用调用次数为 0 时将无法使用该插件。收费标准如下（详细 [产品价格说明](https://www.h3yun.com/site/product-price)）。

![[2707_f9e9c52e8afd78a1.png]]

### 插件安装

开通插件后，登录管理员账号，依次点击 `管理员头像` > `插件中心` > `文字识别` > `安装`。安装后即可在表单设计中配置、使用 `文字识别` 控件。

![[2707_442dfe2c04615a69.png]]

### 表单设计

新建 `供应商档案` 表单，所需控件如下：

| **控件名称** | **控件类型** | **控件说明** |
| --- | --- | --- |
| `营业执照` | `文字识别` | 上传并识别营业执照 |
| `执照存档` | `图片` | 保存营业执照图片 |
| `统一信用代码` | `单行文本` | |
| `营业名称` | `单行文本` | |
| `法人/负责人` | `单行文本` | |
| `注册日期` | `日期` | |
| `执照编号` | `单行文本` | |
| `执照期限` | `单行文本` | |

### 控件配置

#### 添加控件

在 `表单设计` 中，添加 `文字识别` 和其他所需控件。

![[2707_79d1e5d64918af.png]]

#### 选择识别类型

点击已添加的 `文字识别` 控件，选择 `识别类型` 为 `营业执照`。

![[2707_ab885b690ce8e398.png]]

#### 配置填充规则

根据`数据填充规则`，系统将自动识别到的字段值填充至指定控件中，以实现数据的自动录入。

选择已添加的 `文字识别` 控件，点击 `数据填充规则`，在配置窗口中一一指定目标控件。

 ![[2707_3ce23ad804ea1ae4.png]]

### 填充数据至子表控件

-   在“主表”配置`文字识别`控件，识别信息填充至子表：每次识别都将新增一条子表数据并填充

-   在“子表”配置`文字识别`控件，识别信息填充至子表：每次识别都是填充至`文字识别`控件所在行数据


### 图片质量

为保证识别准确性，请确保上传的图片/文件符合所配置的识别类型，确保图片朝向正确、清晰、不反光、不过曝（亮度过高）、不欠曝（亮度过低）。
