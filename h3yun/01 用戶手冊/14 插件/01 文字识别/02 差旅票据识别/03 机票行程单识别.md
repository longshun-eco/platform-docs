## 应用场景

在差旅报销、行程管理等场景，需要记录机票行程信息。通过 `文字识别` 插件的 `机票行程单识别` 能力，只需上传行程单图片，即可自动识别关键信息并填充到表单控件中，无需人工填写，提高输入效率和数据准确性。

例如，在表单 `差旅报销单` 中上传机票行程单图片后，自动识别其中的信息并填充到 `乘客姓名`、`航班号`、`票价` 等控件中。效果如图所示。

![[2712_91d435688230ee28.gif]]

### 插件简介

`机票行程单识别` 可识别行程单中所有关键信息并自动填充到指定的控件。它属于付费插件 `文字识别` 的其中一种识别类型，每次调用消耗并自动扣除一次调用次数，当可用调用次数为 0 时将无法使用该插件。收费标准如下（详细 [产品价格说明](https://www.h3yun.com/site/product-price)）。

![[2712_f63ea1183f771133.png]]

如需开通插件或购买调用次数，请联系我们。

### 插件安装

开通插件后，登录管理员账号，依次点击 `管理员头像` > `插件中心` > `文字识别` > `安装`。安装后即可在表单设计中配置、使用 `文字识别` 控件。

![[2712_8752e1879cd0f5f6.png]]

### 表单设计

新建 `差旅报销单` 表单，所需控件如下：

| **控件名称** | **控件类型** | **控件说明** |
| --- | --- | --- |
| `行程单图片` | `文字识别` | 用于上传并识别机票行程单图片。 |
| `行程单图片保存` | `图片` | 用于保存机票行程单图片。 |
| `旅客姓名` | `单行文本` | |
| `日期` | `日期` | |
| `航班号` | `单行文本` | |
| `票价` | `数字` | |
| `合计` | `数字` | |

### 控件配置

#### 添加控件

在 `表单设计` 中，添加 `文字识别` 和其他所需控件。

![[2712_4cf8764ae0717983.png]]

#### 选择识别类型

点击已添加的 `文字识别` 控件，选择 `识别类型` 为 `机票行程单`。

![[2712_5bc83f9deb6af006.png]]

#### 配置数据填充规则

根据`数据填充规则`，系统将自动识别到的字段值填充至指定控件中，以实现数据的自动录入。

选择已添加的 `文字识别` 控件，点击 `数据填充规则`，在配置窗口中一一指定目标控件。

![[2712_1ae08ef2a494ca57.png]]

### 填充数据至子表控件

-   在“主表”配置`文字识别`控件，识别信息填充至子表：每次识别都将新增一条子表数据并填充

-   在“子表”配置`文字识别`控件，识别信息填充至子表：每次识别都是填充至`文字识别`控件所在行数据


### 图片质量

为保证识别准确性，请确保上传的图片/文件符合所配置的识别类型，确保图片朝向正确、清晰、不反光、不过曝（亮度过高）、不欠曝（亮度过低）。
