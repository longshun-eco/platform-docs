## 应用场景

在差旅报销、行程记录等场景，需要记录火车票信息。通过 `文字识别` 插件的 `火车票识别` 能力，只需上传火车票图片，即可自动识别关键信息并填充到表单控件中，无需人工填写，提高输入效率和数据准确性。

例如，在表单 `交通报销单` 中上传火车票图片后，自动识别其中的信息并填充到 `车次`、`出发站`、`到达站` 等控件中。效果如图所示。

![[2716_e856c159abf8649a.gif]]

## 配置说明

### 插件简介

`火车票识别` 可识别火车票中所有关键信息并自动填充到指定的控件。它属于付费插件 `文字识别` 的其中一种识别类型，每次调用消耗并自动扣除一次调用次数，当可用调用次数为 0 时将无法使用该插件。收费标准如下（详细 [产品价格说明](https://www.h3yun.com/site/product-price)）。

![[2716_d37cfed35245cd82.png]]

如需开通插件或购买调用次数，请联系我们。

### 插件安装

开通插件后，登录管理员账号，依次点击 `管理员头像` > `插件中心` > `文字识别` > `安装`。安装后即可在表单设计中配置、使用 `文字识别` 控件。

![[2716_247d65db0a1d8df9.png]]

### 表单设计

新建 `交通报销单` 表单，所需控件如下：

| **控件名称** | **控件类型** | **控件说明** |
| --- | --- | --- |
| `火车票` | `文字识别` | 用于上传并识别火车票。 |
| `火车票图片保存` | `图片` | 用于保存火车票图片。 |
| `车次` | `单行文本` | |
| `出发站` | `单行文本` | |
| `到达站` | `单行文本` | |
| `票价` | `数字` | |
| `旅客信息` | `单行文本` | |

### 控件配置

#### 添加控件

在 `表单设计` 中，添加 `文字识别` 和其他所需控件。

![[2716_405ace86309617fd.png]]

#### 选择识别类型

点击已添加的 `文字识别` 控件，选择 `识别类型` 为 `火车票`。

![[2716_9bc707ad7adc3faa.png]]

#### 配置数据填充规则

根据`数据填充规则`，系统将自动识别到的字段值填充至指定控件中，以实现数据的自动录入。

选择已添加的 `文字识别` 控件，点击 `数据填充规则`，在配置窗口中一一指定目标控件。

![[2716_a70c3bba02544720.png]]

## 注意事项

`火车票识别` 支持以下字段的识别。配置 `数据填充规则` 时，请选择合适的控件类型，格式不匹配时无法填充。

| **可识别字段** | **可填充的控件类型** |
| --- | --- |

| `识别附件` 用于火车票识别的文件，最大 6 MB；可填充到 `附件` 控件以实现保存。 | `附件` |
| `车次` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `出发站` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |

### 填充数据至子表控件

-   在“主表”配置`文字识别`控件，识别信息填充至子表：每次识别都将新增一条子表数据并填充

-   在“子表”配置`文字识别`控件，识别信息填充至子表：每次识别都是填充至`文字识别`控件所在行数据


### 图片质量

为保证识别准确性，请确保上传的图片/文件符合所配置的识别类型，确保图片朝向正确、清晰、不反光、不过曝（亮度过高）、不欠曝（亮度过低）。
