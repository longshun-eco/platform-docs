## 应用场景

在车辆管理、保险理赔、租赁登记等场景，需要记录车辆的行驶证信息。通过 `文字识别` 插件的 `行驶证识别` 能力，只需上传行驶证图片，即可自动识别关键信息并填充到表单控件中，无需人工填写，提高输入效率和数据准确性。

例如，在表单 `车辆档案` 中上传行驶证图片后，自动识别其中的信息并填充到 `车牌号码`、`车辆类型`、`所有人` 等控件中。效果如图所示。

![[2710_2c7a9235e3138bcb.gif]]

## 配置说明

### 插件简介

`行驶证识别` 可识别行驶证中所有关键信息并自动填充到指定的控件。它属于付费插件 `文字识别` 的其中一种识别类型，每次调用消耗并自动扣除一次调用次数，当可用调用次数为 0 时将无法使用该插件。收费标准如下（详细 [产品价格说明](https://www.h3yun.com/site/product-price)）。

![[2710_d2058f7f329df346.png]]

如需开通插件或购买调用次数，请联系我们。

### 插件安装

开通插件后，登录管理员账号，依次点击 `管理员头像` > `插件中心` > `文字识别` > `安装`。安装后即可在表单设计中配置、使用 `文字识别` 控件。

![[2710_61177aac633a15dd.png]]

### 表单设计

新建 `车辆档案` 表单，所需控件如下：

| **控件名称** | **控件类型** | **控件说明** |
| --- | --- | --- |
| `行驶证图片` | `文字识别` | 用于上传并识别行驶证图片。 |
| `行驶证图片保存` | `图片` | 用于保存行驶证图片。 |
| `车牌号码` | `单行文本` | |
| `车辆类型` | `单行文本` | |
| `所有人` | `单行文本` | |
| `品牌型号` | `单行文本` | |
| `车辆识别代码` | `单行文本` | |
| `注册日期` | `日期` | |

### 控件配置

#### 添加控件

在 `表单设计` 中，添加 `文字识别` 和其他所需控件。

![[2710_adb4fcd0ce788c93.png]]

#### 选择识别类型

点击已添加的 `文字识别` 控件，选择 `识别类型` 为 `行驶证`。

![[2710_3060c1a2b355c149.png]]

#### 配置数据填充规则

根据`数据填充规则`，系统将自动识别到的字段值填充至指定控件中，以实现数据的自动录入。

选择已添加的 `文字识别` 控件，点击 `数据填充规则`，在配置窗口中一一指定目标控件。

![[2710_7a58ef8c909fe0c3.png]]

## 注意事项

### 可识别字段

`行驶证识别` 支持以下字段的识别。配置 `数据填充规则` 时，请选择合适的控件类型，格式不匹配时无法填充。

| **可识别字段** | **可填充的控件类型** |
| --- | --- |
| `识别图片` 用于行驶证识别的图片，最大 20 MB；可填充到 `图片` 控件以实现保存。 | `图片` |
| `车牌号码` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `车辆类型` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `所有人` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `住址` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `使用性质` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `品牌型号` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `车辆识别代码` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `发动机号码` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `注册日期` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框`、`日期` |
| `发证日期` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框`、`日期` |

### 填充数据至子表控件

-   在“主表”配置`文字识别`控件，识别信息填充至子表：每次识别都将新增一条子表数据并填充

-   在“子表”配置`文字识别`控件，识别信息填充至子表：每次识别都是填充至`文字识别`控件所在行数据


### 图片质量

为保证识别准确性，请确保上传的图片/文件符合所配置的识别类型，确保图片朝向正确、清晰、不反光、不过曝（亮度过高）、不欠曝（亮度过低）。
