## 应用场景

在车辆管理、驾驶员信息登记等场景，需要记录驾驶证件信息。通过 `文字识别` 插件的 `驾驶证识别` 能力，只需上传驾驶证图片，即可自动识别关键信息并填充到表单控件中，大幅提升数据录入效率和准确性。

例如，在表单 `驾驶员档案` 中上传驾驶证图片后，自动识别证件信息并填充到 `姓名`、`证号`、`有效期限` 等控件中。效果如图所示。

![[2711_e5db20817e47d174.gif]]

## 配置说明

### 插件简介

`驾驶证识别` 可识别驾驶证中所有关键信息并自动填充到指定的控件。它属于付费插件 `文字识别` 的其中一种识别类型，每次调用消耗并自动扣除一次调用次数，当可用调用次数为 0 时将无法使用该插件。收费标准如下（详细 [产品价格说明](https://www.h3yun.com/site/product-price)）。

![[2711_4593f751caae3c05.png]]

如需开通插件或购买调用次数，请联系我们。

### 插件安装

开通插件后，登录管理员账号，依次点击 `管理员头像` > `插件中心` > `文字识别` > `安装`。安装后即可在表单设计中配置、使用 `文字识别` 控件。

![[2711_5034b8b029158063.png]]

### 表单设计

新建 `驾驶员档案` 表单，所需控件如下：

| **控件名称** | **控件类型** | **控件说明** |
| --- | --- | --- |
| `驾驶证图片` | `文字识别` | 用于上传并识别驾驶证图片 |
| `驾驶证保存` | `图片` | 用于保存驾驶证图片 |
| `姓名` | `单行文本` | |
| `证号` | `单行文本` | |
| `性别` | `单选框` | |
| `出生日期` | `日期` | |
| `有效起始日期` | `日期` | |
| `住址` | `单行文本` | |
| `有效期限` | `单行文本` | |

### 控件配置

#### 添加控件

在 `表单设计` 中，添加 `文字识别` 和其他所需控件。

![[2711_c6979ffff66b9c78.png]]

#### 选择识别类型

点击已添加的 `文字识别` 控件，选择 `识别类型` 为 `驾驶证`。

![[2711_98dec3e697e79575.png]]

#### 配置数据填充规则

根据`数据填充规则`，系统将自动识别到的字段值填充至指定控件中，以实现数据的自动录入。

选择已添加的 `文字识别` 控件，点击 `数据填充规则`，在配置窗口中一一指定目标控件。

![[2711_c34f0af7fbaf8c71.png]]

## 注意事项

### 可识别字段

`驾驶证识别` 支持以下字段的识别。配置 `数据填充规则` 时，请选择合适的控件类型，格式不匹配时无法填充。

| **可识别字段** | **可填充的控件类型** |
| --- | --- |
| `识别图片` 用于驾驶证识别的图片，最大 20 MB；可填充到 `图片` 控件以实现保存。 | `图片` |
| `姓名` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `证号` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `性别` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `国籍` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `准驾车型` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `初次领证日期` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框`、`日期` |
| `出生日期` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框`、`日期` |
| `有效起始日期` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框`、`日期` |
| `有效期限` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `住址` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |
| `发证单位` | `单行文本`、`多行文本`、`单选框`、`复选框`、`下拉框` |

### 填充数据至子表控件

-   在“主表”配置`文字识别`控件，识别信息填充至子表：每次识别都将新增一条子表数据并填充

-   在“子表”配置`文字识别`控件，识别信息填充至子表：每次识别都是填充至`文字识别`控件所在行数据


### 图片质量

为保证识别准确性，请确保上传的图片/文件符合所配置的识别类型，确保图片朝向正确、清晰、不反光、不过曝（亮度过高）、不欠曝（亮度过低）。
