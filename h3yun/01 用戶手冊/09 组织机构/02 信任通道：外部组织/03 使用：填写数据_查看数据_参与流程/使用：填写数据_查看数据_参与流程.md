## 表单数据权限

| | **内部组织人员** | **外部组织人员** |
| --- | --- | --- |
| **数据权限** | **权限管理**
-   本人：拥有者控件值为当前用户的数据
-   服务部门：所属部门控件值为当前用户服务部门/所属组织的数据
-   全部：表单所有数据

**人员控件**：存在勾选允许查看表单数据的人员控件时，可查看控件值为当前人员的数据 **表单属性-数据查看权限**

-   用户根据角色权限查看数据：根据当前用户的角色数据权限，显示数据
-   所有用户可查看全部数据：显示所有表单数据

 |  |
| **字段权限** | 据当前人员的角色的字段组权限，显示字段 |  |

## 填写数据

| | **内部组织人员** | **外部组织人员** |
| --- | --- | --- |
| **人员单选** **人员多选** **拥有者** | 内部组织成员 我联系的外部组织联系人 | 外部组织 |
| **部门单选** **部门多选** **所属部门** | 内部组织部门 我联系的外部组织 | 外部组织 |

-   #### 控件配置“仅以下人员可被选择”时，只可选择人员范围内的人员，此时不受权限控制。

-   #### 控件配置“仅以下部门可被选择”时，只可选择部门范围内的部门，此时不受权限控制。

## 查看数据

外部组织人员不支持查看任务提醒、评论、操作日志

## 参与流程

外部组织联系人与内部组织人员一样，可参与流程审批/经办/抄送，流程字段和操作权限保持一致，可查看流程日志、流程详情。
