## 访问外部组织

在工作台可访问外部组织，选择外部组织后，默认跳转至第一个应用。

-   #### 仅外部组织联系人可访问，根据当前用户是否是联系人、被分配的角色权限，显示外部组织列表。

-   #### 如需返回内部组织，请先前往首页工作台。

## 访问应用

| | **内部组织人员** | **外部组织人员** |
| --- | --- | --- |
| **应用重命名** | 支持 | 不支持 |
| **复制应用** | 支持 | 不支持 |
| **删除应用** | 支持 | 不支持 |
| **发布到工作台** | 支持 | 不支持 |
| **发布给其他企业** | 支持 | 不支持 |
| **移动端首页配置** | 支持 | 不支持 |

## 访问列表

| | **内部组织人员** | **外部组织人员** |
| --- | --- | --- |
| **列表操作** 新增、导入、导出、删除、自定义按钮、打印二维码、打印模板 | 取决于角色权限 | 取决于外部组织角色权限 |
| **数据范围** | 全部：取决于角色权限、表单配置、控件配置 我部门的：所属部门为人员服务部门的数据 我的：拥有者为当前人员的数据 相关的：存在勾选“允许查看表单数据”的人员控件时，可查看控件值为当前人员的数据 | 全部：取决于外部组织角色权限、表单配置、控件配置 我部门的：所属部门为人员所属组织的数据 我的：拥有者为当前人员的数据 相关的：存在勾选“允许查看表单数据”的人员控件时，可查看控件值为当前人员的数据 |
| **统计分析** | 支持 | 不支持 |
| **列表筛选** | 支持 | 不支持以下控件筛选：人员单选、人员多选、拥有者、创建人、部门单选、部门多选、所属部门 |
