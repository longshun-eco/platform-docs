数据表名：H\_bizobjectfile

所有表单数据内的所有图片、附件信息都保存在这张表内。

| 序号 | 字段名 | 字段描述 |
| --- | --- | --- |
| 1 | objectid | 数据Id/附件Id |
| 2 | bizobjectid | 附件所在表单数据Id |
| 3 | lastversion | 版本 |
| 4 | propertyname | 附件所在表单的附件控件编码 |
| 5 | schemacode | 附件所在表单的表单编码 |
| 6 | childschemacode | 如附件控件在子表中，则有子表控件编码 |
| 7 | sourcebizobjectid | （此字段为平台底层所需，用户无需关注） |
| 8 | sourcepropertyname | （此字段为平台底层所需，用户无需关注） |
| 9 | sourcefileid | （此字段为平台底层所需，用户无需关注） |
| 10 | createdby | 上传人员Id |
| 11 | modifiedby | 最后修改人员Id |
| 12 | contenttype | 附件类型 |
| 13 | contentlength | 附件大小 |
| 14 | thumbnailurl | （此字段为平台底层所需，用户无需关注） |
| 15 | filename | 附件文件名 |
| 16 | description | 对附件的描述 |
| 17 | createdtime | 附件上传时间 |
| 18 | modifiedtime | 附件最后修改时间 |
| 19 | fileflag | （此字段为平台底层所需，用户无需关注） |
| 20 | parentobjectid | （此字段为平台底层所需，用户无需关注） |
| 21 | parentpropertyname | （此字段为平台底层所需，用户无需关注） |
| 22 | parentindex | （此字段为平台底层所需，用户无需关注） |
| 23 | sortkey | 本附件是附件控件中第几个（从0开始） |
