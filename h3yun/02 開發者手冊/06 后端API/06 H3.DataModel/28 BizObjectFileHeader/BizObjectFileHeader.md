类名 : BizObjectFileHeader
说明 : 业务对象文件的摘要信息 属性 : 

| 名称 | 说明 |
| --- | --- |
| BizObjectId | 业务对象ID |
| LastVersion | 是否是最新版本 |
| PropertyName | 属性名称 |
| SchemaCode | 业务对象模式编码 |
| ChildSchemaCode | 子业务对象模式编码，如果该记录是子表中的文件，那么该属性有效，否则为Null |
| SourceBizObjectId | 这个对象的源对象ID |
| SourcePropertyName | 这个对象的源属性名称 |
| SourceFileId | 这个对象的源文件 |
