类名 : FunctionNodeType
说明 : 菜单节点类型 
成员 : 

| 名称 | 说明 |
| --- | --- |
| AppPackage | 应用程序 |
| FormModule | 列表模块(不能发起流程) |
| WorkflowModule | 流程列表模块(可以发起流程) |
| ReportModule | 报表模块 |
| CustomListModule | 自定义列表模块（不含表单） |
| GroupModule | 节点分组 |
| SingletonModule | 表单模块(无列表，只能单独一行数据) |
