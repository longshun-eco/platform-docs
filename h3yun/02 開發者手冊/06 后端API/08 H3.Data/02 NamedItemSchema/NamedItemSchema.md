类名 : NamedItemSchema
说明 : 项目模式，这个是业务对象属性模式和业务结构项目模式的基类。 属性 : 

| 名称 | 说明 |
| --- | --- |
| Name | 项目名称 |
| DisplayName | 显示名称 |
| FullName | 项目全名称 |
| DataType | 数据类型 |
| MaxLength | 最大长度。0表示不限制长度 |
| RealType | .Net数据类型 |
| DefaultValue | 默认值 |
