类名 : TaskType
说明 : 任务类型 
成员 : 

| 名称 | 说明 |
| --- | --- |
| Task | 用户任务，可以指定一个完成时间和完成人员 |
| Reminder | 提醒，分为未提醒状态和提醒状态 |
| Calendar | 日程，指定一个开始时间和结束事件，需要再开始前某个时间进行提醒 |
