类名 : SNSComment
说明 : 用于记录用户的评论 属性 : 

| 名称 | 说明 |
| --- | --- |
| ReplyTo | 回复的评论的ID，如果不针对任何一条回复，则为Null，否则为回复的Comment的Id |
| ReplyToUserId | 回复的评论对应的用户Id，如果不针对任何一条回复，则为Null，否则为回复的Comment的Id |
| SchemaCode | 业务对象的模式编码 |
| BizObjectId | 业务对象Id |
| PostId | 帖子的Id |
| UserId | 用户ID |
| Text | 评论的文本 |
| CreatedTime | 用户ID |
| IsLike | 是否是点赞 |
