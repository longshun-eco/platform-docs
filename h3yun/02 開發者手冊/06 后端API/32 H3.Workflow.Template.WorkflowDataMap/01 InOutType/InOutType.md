类名 : InOutType
说明 : 输入输出类型 
成员 : 

| 名称 | 说明 |
| --- | --- |
| In | 仅以覆盖模式传入 |
| Out | 仅以覆盖模式传出 |
| InOut | 以覆盖模式传入并且以覆盖模式传出 |
| OutAppend | 不传入，仅以添加模式传出 |
| InOutAppend | 传入，并且以添加模式传出 |
