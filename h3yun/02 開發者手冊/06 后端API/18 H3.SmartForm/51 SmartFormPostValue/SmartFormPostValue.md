类名 : SmartFormPostValue
说明 : 表单传递到后台数据，包括表单上本身的数据、用户的指令等 属性 : 

| 名称 | 说明 |
| --- | --- |
| Command | 执行命令 |
| DestActivityCode | 获取或设置提交/驳回的节点名称 |
| Data | 表单请求传递过来的数据 |
| Comment | 用户的审批意见，用于流程 |
| ForwardTo | 转发给某人 |



方法名称 : GetObjectTrack(System.String)

| 参数 | 说明 |
| --- | --- |
| "objectName" | 对象名称 |
