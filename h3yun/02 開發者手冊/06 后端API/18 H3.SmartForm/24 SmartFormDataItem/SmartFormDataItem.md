类名 : SmartFormDataItem
说明 : 返回到前端的统一数据项格式 属性 : 

| 名称 | 说明 |
| --- | --- |
| RealType | 数据项的类型 |
| BizDataType | 数据项的逻辑类型 |
| Value | 数据项的值 |
| Visible | 在当前活动下是否可见 |
| Editable | 在当前活动下是否可写 |
| TrackVisible | 在当前活动下是否可以查看痕迹 |
| Required | 在当前活动下是否必输 |
| Printable | 在当前活动下是否可打印 |
| DisplayName | 获取显示名称

 |
