类名 : PermittedActions
说明 : 表单上可以进行的操作 属性 : 

| 名称 | 说明 |
| --- | --- |
| CancelInstance | 获取或设置是否提交前允许取消流程 |
| RetrieveInstance | 取回流程 |
| Viewed | 已阅 |
| FinishInstance | 获取或设置是否允许结束流程 |
| Reject | 获取或设置是否允许驳回 |
| Close | 获取或设置是否显示关闭按钮 |
| Print | 获取或设置是否显示打印按钮 |
| Save | 获取或设置是否显示保存按钮 |
| Submit | 获取或设置是否允许提交 |
| ViewInstance | 获取或设置是否可以查看流程状态

 |
