类名 : RecentChange
说明 : 需要Rencent更新的对象 属性 : 

| 名称 | 说明 |
| --- | --- |
| FormDataType | 获取平台模式(流程/开发平台) |
| FormMode | 打开表单所使用的模式，比如：只读模式、打印模式、工作模式等，如果URL参数中未指定，则默认为Unspecified，由表单程序自行指定 |
| SchemaDisplayName | 业务对象模式显示名称
