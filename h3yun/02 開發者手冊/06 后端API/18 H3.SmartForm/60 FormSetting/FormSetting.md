类名 : FormSetting
说明 : 表单配置 属性 : 

| 名称 | 说明 |
| --- | --- |
| DesignModeContent | 设计态的表单内容，由表单设计器保存的HTML |
| RuntimeContent | 运行态的表单内容，由表单设计器保存内容 |
| DisplayName | 表单名称 |
| EnableFormSns | 启用表单动态 |
| EnableTask | 启用任务提示 |
| EnableExternalForm | 是否启用外链表单 |
| AssociationCodes | 关联列表 |
| Valid | 验证编码是否正确

 |
