类名 : ListViewColumnSetting
说明 : 查询列显示设置 属性 : 

| 名称 | 说明 |
| --- | --- |
| PropertyName | 获取或设置属性名称 |
| Visible | 获取或设置是否显示 |
| IsChild | 是子表类型 |
| ShowMode | 显示模式 |
| ChildColumns | 获取或设置列表字段的显示设置 |



成员 : 

| 名称 | 说明 |
| --- | --- |
| TableName | 表名称 |
| PropertyName\_PropertyName | 属性名称 |
| PropertyName\_Visible | 是否显示 |
| PropertyName\_IsChild | 子表属性 |
| PropertyName\_ChildColumns | 查询的字段显示设置 |
