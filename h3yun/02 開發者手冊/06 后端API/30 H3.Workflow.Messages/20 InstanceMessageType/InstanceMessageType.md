类名 : InstanceMessageType
说明 : 流程实例的消息类型 
成员 : 

| 名称 | 说明 |
| --- | --- |
| Start | 启动流程实例 |
| Finish | 完成流程实例 |
| Cancel | 取消 |
| ActivityFinished | Activity已经完成 |
| ForwardActivity | 转发给Activity |
| ActivateInstance | 将取消/结束的流程重新激活为运行状态 |
