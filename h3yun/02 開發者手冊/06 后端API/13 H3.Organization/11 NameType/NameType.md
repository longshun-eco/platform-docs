类名 : NameType
说明 : 名称的类型 
成员 : 

| 名称 | 说明 |
| --- | --- |
| Name | 名称 |
| Path | 组织路径，/Company/OU1/OU2这种格式 |
| FullName | 全名称，如果是用户则是：用户名\[用户编码\]；如果是OU，则是：/Company/OU1/OU2这种格式 |
