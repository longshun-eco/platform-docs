类名 : IInstanceDataItem
说明 : 流程数据项要实现的接口 属性 : 

| 名称 | 说明 |
| --- | --- |
| DisplayName | 获取数据项显示名称 |
| Value | 获取数据项值 |
| Visible | 获取是否可见 |
| Editable | 获取是否可修改 |
| Required | 获取是否是必输 |
| Printable | 可打印否可见 |
| TrackVisible | 获取数据日志是否可见 |
| RealType | 获取数据项的类型 |
| BizDataType | 获取数据项的逻辑类型

 |
