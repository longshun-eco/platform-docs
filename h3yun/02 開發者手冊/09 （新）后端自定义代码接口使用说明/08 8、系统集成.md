| 序号 | 接口名称 | 说明 |

## 接口说明

### 1. Invoke(string userId, AccessPointType type, string schemaCode, string methodName, BizStructure param)

#### 1.1. 说明

以 **WebService** 形式调用第三方接口。兼容旧版本，由于氚云能配置连接上的 **WebService** 必须按照规定格式进行定义，且只对`.NET` 框架下开发的 **WebService** 兼容性友好，对于 `JAVA`、`Python` 等语言兼容性较差。因此，更推荐使用`InvokeApi`，用于氚云调用第三方 **Web API** 服务。

#### 1.2. 参数

| 参数名 | 说明 |
| --- | --- |
| userId | 用户ID |
| type | 要进行的操作的系统类型，详见[`H3.BizBus.AccessPointType`](/contents/628/710.html) |
| schemaCode | 表单编码 |
| methodName | 要执行的方法名称 |
| param | 业务结构，用来传递可自解析的数据，详见[`H3.BizBus.BizStructure`](/contents/630/717.html) |

#### 1.3. 返回值

| 类型 | 说明 |
| --- | --- |
| [H3.BizBus.InvokeResult](/contents/633/722.html) | 调用返回结果 |

#### 1.4. 示例

详见[氚云调用WebService](/contents/1126/2234.html)。

### 2. InvokeApi(string userId, AccessPointType type,  string code, string method, string contentType, Dictionary<string, string> headers, Dictionary<string, string> queries, Dictionary<string, object> bodies, BizStructureSchema structureSchema)

#### 2.1. 说明

以 **WebApi** 形式调用第三方接口。第三方Api要求：`http/https`协议，请求参数和响应数据不支持文件类型，且响应数据必须是`JSON`格式，**推荐用法**。

#### 2.2. 参数

| 参数名 | 说明 |
| --- | --- |
| userId | 用户ID |
| type | 要进行的操作的系统类型，详见[`H3.BizBus.AccessPointType`](/contents/628/710.html) |
| code | 表单编码 |
| method | 方法名称，如`GET`、`POST`等，字母必须全大写 |
| contentType | 内容类型，如`application/json` |
| headers | 请求头 |
| queries | 查询条件 |
| bodies | 请求头 |
| structureSchema | 业务结构，用来传递可自解析的数据，详见[`H3.BizBus.BizStructureSchema`](/contents/631/720.html) |

#### 2.3. 返回值

| 类型 | 说明 |
| --- | --- |
| [H3.BizBus.InvokeResult](/contents/633/722.html) | 调用返回结果 |

#### 2.4. 示例

详见[第三方连接](/contents/1125/2233.html)。

### 3. GetList(string userId, AccessPointType type, string schemaCode, Filter filter)

#### 3.1. 说明

以 **WebService** 形式获取表单数据。

#### 3.2. 参数

| 参数名 | 说明 |
| --- | --- |
| userId | 用户ID |
| type | 要进行的操作的系统类型，详见[`H3.BizBus.AccessPointType`](/contents/628/710.html) |
| schemaCode | 表单编码 |
| filter | 过滤条件，详见[`H3.Data.Filter.Filter`](/contents/458/688.html) |

#### 3.3. 返回值

| 类型 | 说明 |
| --- | --- |
| [H3.BizBus.ListResult](/contents/632/721.html) | 查询结果 |

#### 3.4. 示例

```csharp
string userId = this.Request.UserContext.UserId;
H3.Data.Filter.Filter filter = new H3.Data.Filter.Filter();
H3.BizBus.AccessPointType type = H3.BizBus.AccessPointType.Legacy;

//webservice查询数据
H3.BizBus.ListResult lstResult = this.Engine.BizBus.GetList(userId, type, "D000001ABC", filter);
if(lstResult != null && lstResult.Data.Length > 0)
{
    foreach(H3.BizBus.BizStructure structure in lstResult.Data)
    {
        // 处理相关逻辑
    }
}
```
