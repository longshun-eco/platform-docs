LoadBizObject 为加载单个数据，请勿使用该接口来循环加载数据，可以使用LoadBizObjects 来批量加载数据。

请求方式：POST（HTTPS）

请求地址：[https://www.h3yun.com/OpenApi/Invoke](https://www.h3yun.com/OpenApi/Invoke)

请求包体：

| { "ActionName":"LoadBizObject", "SchemaCode":"D000024chuangjian", "BizObjectId":"fea0ade0-505d-4ffc-bca0-6cbdf3c302c6" } |
| --- |

参数说明：

| 参数 | 参数类型 | 必须 | 说明 |
| --- | --- | --- | --- |
| ActionName | String | 是 | 调用的方法名 |
| SchemaCode | String | 是 | 需要查询的表单编码 |
| BizObjectId | String | 是 | 需要查询的数据ID，每个表单都有唯一的ObjectId |

需要注意：使用该方法读取子表附件时，需要将子表schemacode和objectid作为参数传入读取，

读取主表单数据时不会带出子表附件信息。
