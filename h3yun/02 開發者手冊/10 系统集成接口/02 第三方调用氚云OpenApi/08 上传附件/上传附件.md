通过本接口上传附件，附件大小不可超过3M。

请求方式：POST（HTTPS）

请求地址：[https://www.h3yun.com/OpenApi/UploadAttachment?SchemaCode=SchemaCode&](https://www.h3yun.com/OpenApi/UploadAttachment?SchemaCode=SchemaCode&)

FilePropertyName=FilePropertyName&BizObjectId=BizObjectId

参数说明：

| 参数 | 参数类型 | 必须 | 说明 |
| --- | --- | --- | --- |
| SchemaCode | String | 是 | 表单编码 |
| ChildSchemaCode | String | 否 | 子表单编码 |
| FilePropertyName | String | 是 | 表单控件编码(如F0000002) |
| BizObjectId | String | 是 | 表单ObjectId值 |
