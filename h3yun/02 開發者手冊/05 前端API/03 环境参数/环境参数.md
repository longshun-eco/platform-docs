环境参数($.SmartForm.ResponseContext)

| 属性名称 | 说明 |
| --- | --- |
| Actions | 表单按钮集合 |
| ActivityCode | 流程环节Code |
| DisplayName | 表单名称 |
| FormDataType | 表单数据类型 |
| FormMode | 表单模式 |
| InstanceId | 流程实例ID |
| IsCreateMode | 是否创建模式 |
| Errors | 返回的错误信息 |
| Message | 返回的其他信息 |
| Originator | 发起人用户ID |
| OriginatorCode | 发起人用户名 |
| OriginatorPatentId | 发起人所在部门ID |
| BizObjectId | 当前对象ID |
| BizObjectStatus | 当前对象状态 |
| ReturnData | 返回自定义数据项的定义与值的集合 |
| SchemaCode | 当前SchemaCode |
| Successful | 是否成功 |
| WorkItemId | 当前流程工作项ID |
| WorkItemType | 当前流程工作项类型 |
| WorkflowVersion | 当前流程模板的版本号 |
