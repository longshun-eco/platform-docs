$.SmartForm.ResponseContext 对象获取相关环境变量，重要的变量包括：

| 属性名称 | 说明 |
| --- | --- |
| Actions | 表单按钮集合 |
| ActivityCode | 流程节点编码 |
| DisplayName | 表单名称 |
| FormDataType | 表单数据类型 |
| FormMode | 表单模式 0为审批/办理  1为办理完结  2为创建  4为查阅 |
| InstanceId | 当前表单数据的流程实例ID |
| IsCreateMode | 是否创建模式 true：是创建模式 |
| Errors | 返回的错误信息 |
| Message | 返回的其他信息 |
| Originator | 发起人用户ID |
| OriginatorCode | 发起人用户名 |
| OriginatorParentId | 发起人所在部门ID |
| BizObjectId | 当前对象ID |
| BizObjectStatus | 当前对象状态 值为0表示 草稿； 值为1表示 生效/流程结束； 值为2表示 流程进行中； 值为3表示 作废。 |
| ReturnData | 返回自定义数据项的定义与值的集合 |
| SchemaCode | 当前SchemaCode |
| Successful | 是否成功 |
| WorkItemId | 当前流程工作项ID |
| WorkItemType | 当前流程工作项类型 |
| WorkflowVersion | 当前流程模板的版本号 |
| Comments | 审批意见记录 |
| IsMobile | 是否移动端 true：是在移动端打开 |
