## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 群模板被启用或停用 |
| 英文名称 | chat\_template\_change |

## 功能描述

群模板被启用或停用时，给订阅了该事件的开发者推送的数据。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
