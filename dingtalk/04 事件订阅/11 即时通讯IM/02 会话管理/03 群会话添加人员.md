## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 群会话添加人员 |
| 英文名称 | chat\_add\_member |

## 功能描述

开发者监听群回调事件可以更及时地响应群的变化，与业务集成。该文档为群会话添加人员事件字段说明。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
