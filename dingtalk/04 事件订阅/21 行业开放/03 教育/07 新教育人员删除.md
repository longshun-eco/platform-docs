## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 新教育人员删除 |
| 英文名称 | edu\_user\_delete |

## 功能描述

家校通讯录2.0，家校通讯录人员变更，主要包括两部分：人员在家校业务场景下的身份以及用户关系（目前关系只有监护人与学生的关系)。该事件表示在某个班级中人员相关身份删除事件数据。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
