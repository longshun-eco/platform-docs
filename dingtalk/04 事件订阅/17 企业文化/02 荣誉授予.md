## 事件信息

|  |  |
| --- | --- |

| 中文名称 | 荣誉授予 |
| 英文名称 | honor\_grant |

## 功能描述

授予企业成员荣誉勋章时，推送的荣誉授予事件内容。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
