## 事件信息

| 名称 | 值 |
| --- | --- |
| 中文名称 | 直播结束数据处理完成事件 |
| 英文名称 | live\_statistic\_done\_event |

## 功能描述

eventType为live\_statistic\_done\_event，表示直播结束数据处理任务完成事件。

## 支持应用类型

| 应用类型 | 是否支持 |
| --- | --- |
| 企业内部应用 | 支持 |
| 第三方企业应用 | 支持 |

## **事件体描述**

### header部分

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| eventUnifiedAppId | String | bbb381b6-f01xxxxx58daac | 统一应用身份Id。 |
| eventCorpId | String | ding9f50b15bxxxx16741 | 事件所属的corpId。 |
| eventType | String | live\_statistic\_done\_event | 事件类型。 |
| eventId | String | c7c7120f2c07419\*\*ebdba0318c8 | 事件的唯一Id。 |
| eventBornTime | Long | 1683533823336 | 事件生成时间。 |

### data部分(事件业务信息)

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| finishTime | Long | 166011401308 | 数据处理完成时间。 |
| liveId | String | 1fc2eaca-\*\*\*\*-\*\*\*\*-\*\*\*\*-b23e1bda4225 | 直播id。 |
