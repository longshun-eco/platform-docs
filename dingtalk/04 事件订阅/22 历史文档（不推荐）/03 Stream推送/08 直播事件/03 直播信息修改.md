## 事件信息

| 名称 | 值 |
| --- | --- |
| 中文名称 | 直播信息修改 |
| 英文名称 | live\_update\_event |

## 功能描述

eventType为live\_update\_event时，表示直播修改信息事件数据。

## 支持应用类型

| 应用类型 | 是否支持 |
| --- | --- |
| 企业内部应用 | 支持 |
| 第三方企业应用 | 支持 |

## **事件体描述**

### header部分

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| eventUnifiedAppId | String | bbb381b6-f01xxxxx58daac | 统一应用身份Id。 |
| eventCorpId | String | ding9f50b15bxxxx16741 | 事件所属的corpId。 |
| eventType | String | live\_update\_event | 事件类型。 |
| eventId | String | c7c7120f2c07419\*\*ebdba0318c8 | 事件的唯一Id。 |
| eventBornTime | Long | 1683533823336 | 事件生成时间。 |

### data部分(事件业务信息)

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| coverUrl | String | http://xxx.png | 直播封面。 |
| preEndTime | Long | 1660268147000 | 直播预计结束时间戳。 |
| title | String | 测试直播 | 直播标题。 |
| liveId | String | 1fc2eaca-\*\*\*\*-\*\*\*\*-\*\*\*\*-b23e1bda4225 | 直播id。 |
| preStartTime | Long | 1660266147700 | 直播预计开播时间戳。 |
| introduction | String | 测试直播简介 | 直播简介。 |
