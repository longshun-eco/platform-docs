## 事件信息

| 名称 | 值 |
| --- | --- |
| 中文名称 | 企业在应用市场购买商品对应的权益服务开通 |
| 英文名称 | market\_service\_open |

## 功能描述

eventType为market\_service\_open时，表示企业在应用市场购买商品对应的权益服务开通事件数据。目前仅续费变配服务开通时推送。

## 支持应用类型

| 应用类型 | 是否支持 |
| --- | --- |
| 企业内部应用 | 不支持 |
| 第三方企业应用 | 支持 |

## **事件体描述**

### header部分

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| eventUnifiedAppId | String | bbb381b6-f01xxxxx58daac | 统一应用身份Id。 |
| eventCorpId | String | ding9f50b15bxxxx16741 | 事件所属的corpId。 |
| eventType | String | market\_service\_open | 事件类型。 |
| eventId | String | c7c7120f2c07419\*\*ebdba0318c8 | 事件的唯一Id。 |
| eventBornTime | Long | 1683533823336 | 事件生成时间。 |

### data部分(事件业务信息)

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| suiteId | Long | 123123 | 用户购买第三方企业应用的suiteId。 |
| corpId | String | dingxxxxxxxxxxxx | 购买企业的corpId。 |
| orderId | String | 312139204444444 | 订单id。 |
| itemCode | String | xxxxxxx | 规格码。 |
| subQuantity | Integer | 3 | 购买数量。 |
| eventType | String | subscription\_open | 事件类型：   * subscription\_open：服务开通事件 |
| openType | Integer | 3 | 服务开通类型：   * 3：续费变配开通 |
| itemName | String | xxxxx | 规格名称。 |
| payFee | Integer | 1000 | 实际支付价格（单位：分）。 |
| serviceStopTime | Long | 168000031311 | 订购服务结束时间（单位：毫秒）。 |
| serviceStartTime | Long | 167000031311 | 订购服务开始时间（单位：毫秒）。 |
| suiteKey | String | xxxxx | 用户购买第三方企业应用的suiteKey。 |
| goodsCode | String | xxxxxxx | 商品码。 |
| goodsName | String | 测试规格 | 商品名称。 |
