## 事件信息

| 名称 | 值 |
| --- | --- |
| 中文名称 | 群内卸载酷应用事件 |
| 英文名称 | im\_cool\_app\_uninstall |

## 功能描述

eventType为im\_cool\_app\_uninstall时，表示群内卸载群酷应用事件数据。

## 支持应用类型

| 应用类型 | 是否支持 |
| --- | --- |
| 企业内部应用 | 支持 |
| 第三方企业应用 | 支持 |

## **事件体描述**

### header部分

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| eventUnifiedAppId | String | bbb381b6-f01xxxxx58daac | 统一应用身份Id。 |
| eventCorpId | String | ding9f50b15bxxxx16741 | 事件所属的corpId。 |
| eventType | String | im\_cool\_app\_uninstall | 事件类型。 |
| eventId | String | c7c7120f2c07419\*\*ebdba0318c8 | 事件的唯一Id。 |
| eventBornTime | Long | 1683533823336 | 事件生成时间。 |

### data部分(事件业务信息)

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| operateTime | String | 1641866135051 | 操作时间。 |
| coolAppCode | String | COOLAPP-1-1018xxxxxxxxxxxxxxxx | 群扩展code。 |
| openConversationCorpId | String | ding9bd1bfb59xxxxxxxxxxxxxxxxxxx | 群会话企业corpId。 |
| robotCode | String | rBLBXuiaA2rn3xxxxxxxxxxxxxx | 机器人code。 |
| openConversationId | String | cidT461wC7yvGJxxxxxxxxxxxxxx | 群加密会话ID。 |
| operator | String | 0213454xxxx1745 | 操作人userId。 |
