## 事件信息

| 名称 | 值 |
| --- | --- |
| 中文名称 | 智能硬件绑定 |
| 英文名称 | device\_bind\_update |

## 功能描述

eventType为device\_bind\_update，表示企业发生硬件设备绑定变更时推送的智能硬件绑定事件数据。

## 支持应用类型

| 应用类型 | 是否支持 |
| --- | --- |
| 企业内部应用 | 不支持 |
| 第三方企业应用 | 支持 |

## **事件体描述**

### header部分

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| eventUnifiedAppId | String | bbb381b6-f01xxxxx58daac | 统一应用身份Id。 |
| eventCorpId | String | ding9f50b15bxxxx16741 | 事件所属的corpId。 |
| eventType | String | device\_bind\_update | 事件类型。 |
| eventId | String | c7c7120f2c07419\*\*ebdba0318c8 | 事件的唯一Id。 |
| eventBornTime | Long | 1683533823336 | 事件生成时间。 |

### data部分(事件业务信息)

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| actionType | String | bind | * bind：绑定 * unbind：解绑 |
| corpId | String | dinga39fdxxxx | 企业的corpid。 |
| outBindStatus | String | ok | 外部的绑定状态， ok 表示成功，其他表示失败 |
| dn | String | 设备dn | 设备序列号。 |
| pk | String | 产品编码 | productKey，产品类型编码。 |
| sn | String | 设备sn | 设备序列号。 |
| nodeType | Integer | 0 | 节点类型：   * 0： 节点 * 1：网关 |
| serviceId | Integer | 12246 | 服务serviceId。 |
| userId | String | 123455 | 操作人userId。 |
| deviceId | Integer | 13344 | 设备ID。 |
| eventTimestamp | Long | 16848000097 | 操作时间。 |
