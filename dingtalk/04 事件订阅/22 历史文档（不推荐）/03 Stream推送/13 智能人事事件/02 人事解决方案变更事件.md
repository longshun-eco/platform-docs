## 事件信息

| 名称 | 值 |
| --- | --- |
| 中文名称 | 人事解决方案变更事件 |
| 英文名称 | hrm\_solution\_manage |
| 事件BizType | 175 |

## 功能描述

当biz\_type=175时，数据为人事解决方案变更事件的相关数据。该数据为人事解决方案变更事件相关的数据推送。

## 支持应用类型

| 应用类型 | 是否支持 |
| --- | --- |
| 企业内部应用 | 支持 |
| 第三方企业应用 | 支持 |

## **事件体描述**

### header部分

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| eventUnifiedAppId | String | bbb381b6-f01xxxxx58daac | 统一应用身份Id。 |
| eventCorpId | String | ding9f50b15bxxxx16741 | 事件所属的corpId。 |
| eventType | String | hrm\_solution\_manage | 事件类型 |
| eventId | String | c7c7120f2c07419\*\*ebdba0318c8 | 事件的唯一Id。 |
| eventBornTime | Long | 1683533823336 | 事件生成时间。 |

### data部分(事件业务信息)

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| corpId | String | ding57935b18bfd13e9735cxxxxxxxxxx | 企业corpId。 |
| solutionType | String | onboarding | 人事解决方案类型：   * onboarding：新人流程 |
| staffId2SolutionInstanceIdMap | String | {"16846746xxxx5028":"875f797531ef49xxxx296feaa"} | 用户userId和解决方案id的映射map。 |
| solutionStatus | String | start | 事解决方案状态：   * init：初始化解决方案 * start：发起解决方案 |
