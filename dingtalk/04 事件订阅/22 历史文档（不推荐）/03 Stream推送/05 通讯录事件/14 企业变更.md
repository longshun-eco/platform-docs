## 事件信息

| 名称 | 值 |
| --- | --- |
| 中文名称 | 企业变更 |
| 英文名称 | org\_update |

## 功能描述

该数据为在授权的第三方企业应用中，当eventType为org\_update，表示企业信息发生变更的推送信息。

## 支持应用类型

| 应用类型 | 是否支持 |
| --- | --- |
| 企业内部应用 | 不支持 |
| 第三方企业应用 | 支持 |

## **事件体描述**

### header部分

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| eventUnifiedAppId | String | bbb381b6-f01xxxxx58daac | 统一应用身份Id。 |
| eventCorpId | String | ding9f50b15bxxxx16741 | 事件所属的corpId。 |
| eventType | String | org\_update | 事件类型。 |
| eventId | String | c7c7120f2c07419\*\*ebdba0318c8 | 事件的唯一Id。 |
| eventBornTime | Long | 1683533823336 | 事件生成时间。 |

### data部分(事件业务信息)

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| errcode | Integer | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |
| corpLogoUrl | String | https://static.xxx.com | 企业图标url。 |
| corpid | String | dingxxx2cff796 | 企业corpId。 |
| industry | String | 建筑安装 | 行业。 |
| corpName | String | 企业1 | 企业名称。 |
| isAuthenticated | Boolean | true | 企业是否认证：   * true 已认证 * false 未认证 |
| authLevel | Integer | 2 | 企业认证级别：   * 0：未认证 * 1： 高级认证 * 2： 中级认证 * 3 ：初级认证 |
