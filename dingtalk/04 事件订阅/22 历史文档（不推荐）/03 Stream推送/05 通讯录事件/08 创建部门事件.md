## 事件信息

| 名称 | 值 |
| --- | --- |
| 中文名称 | 创建部门事件 |
| 英文名称 | org\_dept\_create |

## 功能描述

该数据为在授权的第三方企业应用中，当eventType为org\_dept\_create时，表示企业增加部门的推送信息，字段值来自于[获取部门详情](https://open.dingtalk.com/document/isvapp/query-department-details0-v2)接口。

## 支持应用类型

| 应用类型 | 是否支持 |
| --- | --- |
| 企业内部应用 | 不支持 |
| 第三方企业应用 | 支持 |

## **事件体描述**

### header部分

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| eventUnifiedAppId | String | bbb381b6-f01xxxxx58daac | 统一应用身份Id。 |
| eventCorpId | String | ding9f50b15bxxxx16741 | 事件所属的corpId。 |
| eventType | String | org\_dept\_create | 事件类型。 |
| eventId | String | c7c7120f2c07419\*\*ebdba0318c8 | 事件的唯一Id。 |
| eventBornTime | Long | 1683533823336 | 事件生成时间。 |

### data部分(事件业务信息)

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| brief | String | 部门介绍 | 部门简介。 |
| userPermits | String |  | 用户的权限。 |
| outerDept | Boolean | false | 是否为仅自己可见部门：   * true：是 * false ：否 |
| errcode | Integer | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |
| deptManagerUseridList | String | 123|234 | 部门管理员列表。 |
| parentid | Long | 1123 | 父部门ID。 |
| groupContainSubDept | Boolean | false | 部门群是否包含子部门。 |
| outerPermitUsers | String |  | 仅自己可见部门的用户列表。 |
| autoApproveApply | String | true | 当部门群已经创建后，是否有新人加入部门会自动加入该群：   * true：自动加入群 * false：不会自动加入群 |
| outerPermitDepts | String | 123|234 | 配置的部门员工可见部门Id列表。 |
| deptPerimits | String |  | 配置可见userId列表。 |
| createDeptGroup | Boolean | true | 是否同步创建一个关联此部门的企业群：   * true：创建 * false：不创建 |
| name | String | 测试部门 | 部门名称。 |
| id | Long | 123455 | 部门id。 |
| autoAddUser | Boolean | true | 当部门群已经创建后，是否有新人加入部门会自动加入该群：   * true：自动加入群 * false：不会自动加入群 |
| deptHiding | Boolean | false | 部门权限是否开启。 |
| order | Long | 12345 | 在父部门中的次序值。 |
