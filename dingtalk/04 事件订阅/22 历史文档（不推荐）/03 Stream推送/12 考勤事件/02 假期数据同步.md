## 事件信息

| 名称 | 值 |
| --- | --- |
| 中文名称 | 假期数据同步 |
| 英文名称 | overtime\_to\_vacation\_data |

## 功能描述

eventType为overtime\_to\_vacation\_data，表示企业发生假期相关的数据变更时，钉钉推送的假期数据同步事件数据。

## 支持应用类型

| 应用类型 | 是否支持 |
| --- | --- |
| 企业内部应用 | 不支持 |
| 第三方企业应用 | 支持 |

## **事件体描述**

### header部分

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| eventUnifiedAppId | String | bbb381b6-f01xxxxx58daac | 统一应用身份Id。 |
| eventCorpId | String | ding9f50b15bxxxx16741 | 事件所属的corpId。 |
| eventType | String | overtime\_to\_vacation\_data | 事件类型。 |
| eventId | String | c7c7120f2c07419\*\*ebdba0318c8 | 事件的唯一Id。 |
| eventBornTime | Long | 1683533823336 | 事件生成时间。 |

### data部分(事件业务信息)

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| workHoursPerDay | Integer | 8 | 一天的工作时长（单位：小时）。 |
| overtimeDayType | String | restDay | 加班类型：   * workDay：工作日加班转调休 * restDay：休息日加班转调休 * holiday：节假日加班转调休 |
| workDate | String | 20230521 | 对应的工作日，格式yyyyMMdd，如：20210312。 |
| overtimeDay | Float | 0.262 | 加班转调休时长（以天为单位），例如：0.262。 |
| corpId | String | ding33f33xxxx | 企业的corpid。 |
| delayEndTime | Long | 1640966399000 | 假期的延迟失效时间（unix时间戳）。 startTime < endTime <= delayEndTime。 |
| leaveCode | String | c1f772d2-xxxx-407e-816a-55be5c245569 | 转成的调休假，对应的假期code。 |
| overtimeHour | Float | 2.1 | 加班转调休时长（以小时为单位），例如：2.1。 与overtimeDay的换算关系：overtimeDay=overtimeHour/workHoursPerDay。 |
| vacationRate | Integer | 2 | 加班转调休的比例，即实际工作时长\*vacationRate=调休时长。 |
| startTime | Long | 1615392000000 | 该假期的开始生效时间（unix时间戳）。 |
| endTime | Long | 1640966399000 | 该假期的结束生效时间（unix时间戳）。 |
| userId | String | 3350xxxx555 | 对应的员工id。 |
