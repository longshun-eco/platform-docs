## 事件信息

| 名称 | 值 |
| --- | --- |
| 中文名称 | 文件或文件夹删除 |
| 英文名称 | storage\_dentry\_delete |

## 功能描述

eventType为storage\_dentry\_delete，表示文件或文件夹删除事件数据。

## 支持应用类型

| 应用类型 | 是否支持 |
| --- | --- |
| 企业内部应用 | 支持 |
| 第三方企业应用 | 支持 |

## **事件体描述**

### header部分

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| eventUnifiedAppId | String | bbb381b6-f01xxxxx58daac | 统一应用身份Id。 |
| eventCorpId | String | ding9f50b15bxxxx16741 | 事件所属的corpId。 |
| eventType | String | storage\_dentry\_delete | 事件类型。 |
| eventId | String | c7c7120f2c07419\*\*ebdba0318c8 | 事件的唯一Id。 |
| eventBornTime | Long | 1683533823336 | 事件生成时间。 |

### data部分(事件业务信息)

| 名称 | 类型 | 示例值 | 描述 |
| --- | --- | --- | --- |
| spaceId | String | 214xxx5919 | 空间Id。 |
| dentryId | String | 104898xx5 | 文件Id。 |
| eventScope | String | ORG | 订阅范围：   * ORG: 企业 * SPACE: 空间 |
| extension | String | jpg | 后缀信息。 |
| unionId | String | NCYJxxxxxMQjzgiEiE | 操作人unionId。 |
| eventScopeId | String | ding7a9bc1707d4cf25e35c2f4xxx | 订阅ID：   * 当eventScope为ORG时，对应当前企业corpId。 * 当eventScope为SPACE时，对应空间id。 |
| type | String | FILE | 文件类型：   * FILE：文件 * FOLDER：文件夹 |
