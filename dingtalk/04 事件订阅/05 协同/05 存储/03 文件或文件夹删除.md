## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 文件或文件夹删除 |
| 英文名称 | storage\_dentry\_delete |

## 功能描述

文件或文件夹删除事件。如果仅在开发者后台开启存储事件订阅开关，无法接收回调事件，必须与接口配合使用，接口详情参见[订阅文件变更事件](https://open.dingtalk.com/document/orgapp/subscribe-to-file-change-events)。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
