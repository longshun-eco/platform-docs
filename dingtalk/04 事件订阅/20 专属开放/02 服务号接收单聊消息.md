## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 服务号接收单聊消息 |
| 英文名称 | isw\_user\_msg\_received |

## 功能描述

服务号收到用户单聊消息的事件,钉钉服务器给开发者推送的事件内容，开发者根据收到的用户消息，结合发消息的接口，实现个性化的自动回复功能。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
