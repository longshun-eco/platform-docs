## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 假期数据同步 |
| 英文名称 | overtime\_to\_vacation\_data |

## 功能描述

该数据为企业发生假期相关的数据变更时推送。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
