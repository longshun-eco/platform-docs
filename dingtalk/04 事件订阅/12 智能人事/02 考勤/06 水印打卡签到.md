## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 水印打卡签到 |
| 英文名称 | watermark\_check\_in |

## 功能描述

该事件用于对外开放水印打卡。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
