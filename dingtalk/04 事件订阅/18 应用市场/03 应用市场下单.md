## 事件信息

|  |  |
| --- | --- |

| 中文名称 | 应用市场下单 |
| 英文名称 | market\_order |

## 功能描述

数据为企业在钉钉服务市场购买开通应用产生订单时刻，推送的订单信息事件内容。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
