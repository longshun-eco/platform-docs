## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 企业金融用户协议回调事件 |
| 英文名称 | open\_user\_agreement\_callback |

## 功能描述

用户代扣签解约事件回调。配合[用户协议签约](https://open.dingtalk.com/document/orgapp/user-agreement-signing)接口使用。首先调用该接口获取签约页面员工签约或解约代扣协议时，给对应归属的ISV应用定向推送回调。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
