## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 批量支付消息通知 |
| 英文名称 | open\_batch\_trade\_callback |

## 功能描述

批量支付完成事件回调。配合[创建批量付款单](https://open.dingtalk.com/document/isvapp/create-batch-payment-order?spm=a2q3p.21071111.0.0.554c1cfaVaJxdq)，[拉起批量支付](https://open.dingtalk.com/document/isvapp/create-batch-payment-order?spm=a2q3p.21071111.0.0.554c1cfaVaJxdq) 接口使用，创建批量付款单并支付后，当支付处理完成时，给对应归属的ISV应用定向推送回调。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
