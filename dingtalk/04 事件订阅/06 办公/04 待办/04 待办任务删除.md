## 事件信息

|  |  |

| 英文名称 | todo\_task\_delete |

## 功能描述

删除钉钉待办任务信息时，会触发待办任务删除事件。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
