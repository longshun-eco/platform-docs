## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 服务窗取关事件 |
| 英文名称 | official\_account\_unfollow |

## 功能描述

用户取消关注服务窗时,钉钉服务器会向回调服务推送的服务窗取消关注事件数据。

**说明**

服务窗事件回调，要求在服务窗自建应用中订阅，才可以正常接收回调信息。服务窗自建应用请参考[自建服务窗应用](https://open.dingtalk.com/document/orgapp/self-built-service-window-application)。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
