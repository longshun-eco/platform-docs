## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 服务号接收用户交互 |
| 英文名称 | isw\_user\_event\_received |

## 功能描述

服务号收到用户的交互事件, 目前只有菜单点击事件。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |

## 事件体描述

| eventType | String | isw\_user\_event\_received | 事件类型。 |

| eventBornTime | Long | 1683533823336 | 事件生成时间。 |
| data | Object |  | 事件体data。 |
