## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 增加角色或者角色组 |
| 英文名称 | label\_conf\_add |

## 功能描述

该数据为在授权的企业内部应用中，增加角色或者角色组的推送数据。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
