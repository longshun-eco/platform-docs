## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 通讯录用户更改 |
| 英文名称 | user\_modify\_org |

## 功能描述

该数据为在授权的企业内部应用中，通讯录用户更改事件数据推送说明文档。

> 说明：只有当前企业内的用户信息变更时才会触发此事件，用户的个人信息变更并不会触发，例如个人头像、个人昵称、钉钉号等。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
