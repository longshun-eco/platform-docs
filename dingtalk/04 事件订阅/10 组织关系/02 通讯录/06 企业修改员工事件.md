## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 企业修改员工事件 |
| 英文名称 | user\_modify\_org |

## 功能描述

企业内部用户变更事件。该文档为企业修改员工信息字段说明，字段值来自于[根据userId获取用户详情](https://open.dingtalk.com/document/isvapp/queries-the-details-of-a-dedicated-account)接口 。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
