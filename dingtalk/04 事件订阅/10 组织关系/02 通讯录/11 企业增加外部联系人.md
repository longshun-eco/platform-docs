## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 企业增加外部联系人 |
| 英文名称 | contact\_add\_org |

## 功能描述

该数据为在授权的第三方企业应用中，企业增加外部联系人的推送信息。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
