## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 企业内部应用发布 |
| 英文名称 | inner\_app\_release |

## 功能描述

当开发者对企业内部应用进行发布时，推送事件相关数据。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
