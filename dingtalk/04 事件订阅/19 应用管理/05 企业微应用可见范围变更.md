## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 企业微应用可见范围变更 |
| 英文名称 | org\_micro\_app\_scope\_update |

## 功能描述

数据为第三方企业应用的最新状态，该事件为第三方企业应用可见范围变更推送的数据。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 第三方企业应用 | 支持 | 不支持 | 支持 |
