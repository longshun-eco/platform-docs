## 事件信息

|  |  |
| --- | --- |
| 名称 | 值 |
| 中文名称 | 小程序版本发布事件 |
| 英文名称 | inner\_app\_version\_publish |

## 功能描述

当开发者在开发者后台操作或者调用开放平台接口对企业内部小程序的开发版本进行体验发布和线上发布，推送的小程序版本发布事件数据。

## 支持应用类型

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
