## 功能描述

当钉钉视频会议设备绑定、解绑钉钉会议室时，钉钉推送的设备绑定会议室变更事件内容。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | Stream模式推送 | HTTP推送 | SyncHTTP/RDS推送 |
| 企业内部应用 | 支持 | 支持 | 不支持 |
