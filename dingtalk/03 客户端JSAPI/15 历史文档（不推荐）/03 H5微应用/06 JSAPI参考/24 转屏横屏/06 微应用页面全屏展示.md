更新于 2025-09-17通过**dd\_full\_screen**参数设置微应用页面全屏展示，若在每一个路由跳转的页面上拼接该参数，能实现所有页面全屏。

## 使用说明

|  |  |  |  |
| --- | --- | --- | --- |
| **客户端** | **Android** | **iOS** | **PC** |
| 支持说明 | 支持 | 支持 | 不支持 |

在url后面拼接`dd_full_screen=true`参数，示例如下：

```
http://abc.xyz?dd_full_screen=true
```
