更新于 2025-12-26通过**dd\_orientation**参数设置微应用页面支持转屏。

## 使用说明

|  |  |  |  |
| --- | --- | --- | --- |
| **客户端** | **Android** | **iOS** | **PC** |
| 支持说明 | 支持 | 支持 | 不支持 |

**说明**

鸿蒙端支持通过URL参数`dd_orientation=landscape`强制横屏。

在url后面拼接`dd_orientation=auto`参数，示例如下：

```
http://abc.xyz?dd_orientation=auto
```
