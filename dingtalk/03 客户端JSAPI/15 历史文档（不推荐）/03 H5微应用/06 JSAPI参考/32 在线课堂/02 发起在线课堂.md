更新于 2025-09-17调用**biz.live.startClassRoom**发起在线课堂。

## 使用说明

调用本接口前，请先引入钉钉js，参考[准备工作](/document/orgapp/read-before-development)。

使用该接口，发起公开的在线课堂。

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **客户端** | **是否需要鉴权** | **Android** | **iOS** | **PC** |
| 支持说明 | 需要 | 不支持 | 不支持 | 支持（钉钉版本≥5.1.16） |

```javascript
dd.biz.live.startClassRoom({
  startParam: {
    liveUuid:"ee03cba7-xxxxx",
  },
  success:function(result) {
  },
  fail:function(err) {
  }

});
```

## 参数说明

|  |  |  |
| --- | --- | --- |
| 参数 | 类型 | 说明 |
| startParam | Object | 直播的uuid，可以调用[创建直播](/document/orgapp/create-live-streaming)接口获取。 |
| onSuccess | Function | 调用成功的回调函数。 |
| onFail | Function | 调用失败的回调函数。 |

## 返回结果

|  |  |
| --- | --- |
| 参数 | 说明 |
| 1 | 在线课堂已结束。 |
| 2 | 参数错误。 |
