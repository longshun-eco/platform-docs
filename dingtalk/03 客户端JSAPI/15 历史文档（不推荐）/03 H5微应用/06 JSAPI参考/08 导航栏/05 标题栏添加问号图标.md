更新于 2025-09-17调用**biz.navigation.setIcon**标题栏添加问号图标。

## 调试

访问[JSAPI Explorer](https://open-dev.dingtalk.com/apiExplorer#/jsapi?api=biz.navigation.setIcon)在线调试该接口。

## 使用说明

| **客户端** | **Android** | **iOS** | **PC** |
| --- | --- | --- | --- |
| 支持说明 | 支持 | 支持 | 不支持 |

此JSAPI在iOS和Android上的显示不同，如下图所示：

* 显示在导航栏标题的旁边，紧靠着标题。

  ![[development-title-bar-add-question-mark-icon_p177931.png]]
* 显示在导航栏右侧按钮组的最左边。

  ![[development-title-bar-add-question-mark-icon_p177932.png]]

```java
dd.biz.navigation.setIcon({
    showIcon : true,//是否显示icon
    iconIndex : 1,//显示的iconIndex, 如上图
    onSuccess : function(result) {
        /*结构
        {
        }*/
        //点击icon之后将会回调这个函数
    },
    onFail : function(err) {
    //jsapi调用失败将会回调此函数
    }
})
```

## 参数说明

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| showIcon | Boolean | 是否显示icon。 |
| iconIndex | Number | 显示的iconIndex，如下图。 |

![[development-title-bar-add-question-mark-icon_p177951.png]]
