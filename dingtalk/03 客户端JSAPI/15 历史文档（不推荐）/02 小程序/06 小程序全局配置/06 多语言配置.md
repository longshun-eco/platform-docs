更新于 2025-09-17钉钉小程序可以配置 native 渲染的 tabBar 和 titleBar 部分的多语言文案。多语言配置通过小程序全局配置文件和页面配置文件进行注入。

**说明**

目前支持 zh\_CN（简体中文）、zh\_TW（繁体中文台湾）、zh\_HK（繁体中文香港）、en\_US（美式英文）、ja\_JP（日文）五种语言。

## 示例代码

```
// app.json 配置 tabBar 多语言文案
{
  "tabBar": {
    "items": [
      {
        "name": "首页",
        "name_locale": {
            "zh_CN": "首页",
          "en_US": "Home"
        }
      },
      {
        "name": "关于",
        "name_locale": {
            "zh_CN": "关于",
          "en_US": "About"
        }
      },
    ]
  }
}
```

```
// page.json 配置 titleBar 多语言文案
{
    "defaultTitle": "文件",
  "defaultTitle_locale": {
    "zh_CN": "文件",
    "en_US": "File",
    "ja_JP": "ファイル"
  }
}
```
