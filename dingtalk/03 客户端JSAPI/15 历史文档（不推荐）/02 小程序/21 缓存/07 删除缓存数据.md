更新于 2025-09-17调用**dd.removeStorage**删除缓存数据。

## **示例****代码**

```javascript
dd.removeStorage({
  key: 'currentCity',
  success: function(){
    dd.alert({content: '删除成功'});
  }
});
```

## **入参**

| **参数** | **类型** | **是否必填** | **说明** |
| --- | --- | --- | --- |
| key | String | 是 | 缓存数据的key。 |
| success | Function | 否 | 调用成功的回调函数 |
| fail | Function | 否 | 调用失败的回调函数 |
| complete | Function | 否 | 调用结束的回调函数（调用成功、失败都会执行）。 |
