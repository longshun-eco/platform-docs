更新于 2025-09-17调用dd.setStorageSync同步将数据存储在本地缓存中指定的 key 中。

**重要**

同步数据IO操作可能会影响小程序流畅度，建议使用异步接口，或谨慎处理调用异常。

## **示例****代码**

```java
dd.setStorageSync({
  key: 'currentCity',
  data: {
    cityName: '杭州',
    adCode: '330100',
    spell: ' hangzhou',
  }
});
```

## **入参说明**

| **参数** | **类型** | **是否必填** | **说明** |
| --- | --- | --- | --- |
| key | String | 是 | 缓存数据的key。 |
| data | Object/String | 是 | 要缓存的数据。 |
