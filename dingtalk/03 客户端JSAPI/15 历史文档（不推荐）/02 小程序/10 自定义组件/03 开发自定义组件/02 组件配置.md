更新于 2025-09-17开发者需要在.json文件和.js文件中添加组件配置。

开发者需要在.json文件中指明自定义组件的依赖。

```json
{
  "component": true,
  "usingComponents": {
    "c1":"../x/index"
  }
}
```

| 属性 | 类型 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| component | Boolean | 是 | 指明是组件。 |
| usingComponents | Object | 否 | 指明依赖的组件所在的路径： 项目绝对路径以 / 开头，相对路径以 ./ 或者 ../ 开头，npm 路径不以 / 开头。 |

开发者需要在`.js`文件中调用Component定义组件。

```
Component({
  mixins:[{ didMount() {}, }],
  data: {y:2},
  props:{x:1},
  didUpdate(prevProps,prevData){},
  didUnmount(){},
  methods:{
    onMyClick(ev){
      dd.alert({});
      this.props.onXX({ ...ev, e2:1});
    },
  },
})
```
