更新于 2025-09-17除了 data、setData、props等属性外，组件实例上还有如下属性：

* `is`: 组件路径
* `$page`: 组件所属页面实例
* `$id`: 组件 id，在 axml 中也可直接渲染

  **说明**

  在组件中可以使用 dd 调用 api。

```
// /components/xx/index.js
Component({
  didMount(){
    console.log(this.is);
    console.log(this.$page);
    console.log(this.$id);
  }
});
```
