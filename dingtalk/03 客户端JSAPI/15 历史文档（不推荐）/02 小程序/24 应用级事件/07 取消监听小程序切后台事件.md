更新于 2025-09-17调用dd.offAppHide取消监听小程序切后台事件。

## 使用说明

|  |  |  |  |
| --- | --- | --- | --- |
| **客户端** | **Android** | **iOS** | **PC** |
| 支持说明 | 支持 | 支持 | 不支持 |

**说明**

* 开发者可以通过[dd.canIUse](/document/orgapp/dd-caniuse)函数判断端上是否支持此能力。
* 请勿使用本API监听匿名函数，否则将无法关闭监听。

## 示例代码

### .axml 示例代码

### .js 示例代码

```
Page({
onLoad() {
    dd.onAppHide(this.onAppHideHandler)
},
// 监听切换到后台方法
onAppHideHandler() {
    console.log('监听切换到后台方法')
},
// 取消监听切换到后台方法
 offAppHideHandler(){
    dd.offAppHide(this.onAppHideHandler)
  },
})
```
