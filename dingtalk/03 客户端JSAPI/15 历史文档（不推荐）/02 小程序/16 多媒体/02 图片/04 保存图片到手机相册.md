更新于 2025-09-17调用dd.saveImage保存在线、本地临时或者永久地址图片到手机相册。

## **扫码体验**

![[development-dd-saveimage_p163546.png]]

## **示例代码**

```
dd.saveImage({
    url:'https://img.alicdn.com/tps/TB1sXGYIFXXXXc5XpXXXXXXXXXX.jpg'
});
```

## **入参**

| 参数 | 类型 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| url | String | 是 | 要保存的图片地址 |
| success | Function | 否 | 调用成功的回调函数。 |
| fail | Function | 否 | 调用失败的回调函数。 |
| complete | Function | 否 | 调用结束的回调函数。 |

## **错误码**

| **error** | **描述** |
| --- | --- |
| 2 | 参数无效，没有传 url 参数。 |
| 15 | 没有开启相册权限(ios only)。 |
| 16 | 手机相册存储空间不足(ios only)。 |
| 17 | 保存图片过程中的其他错误。 |
