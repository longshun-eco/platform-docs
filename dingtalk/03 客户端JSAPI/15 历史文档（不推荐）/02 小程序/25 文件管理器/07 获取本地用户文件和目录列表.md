更新于 2025-09-17调用**FileSystemManager.readdir**，获取本地用户文件列表。

## **扫码体验**

![[development-obtains-a-list-of-local-user-files_p497732.png]]

**说明**

本接口获取的是调用[保存文件](/document/orgapp/save-file)接口保存为本地用户的文件，保存为本地缓存的文件不支持获取。

## **使用说明**

|  |  |  |  |
| --- | --- | --- | --- |
| **客户端** | **Android** | **iOS** | **PC** |
| 支持说明 | 支持(钉钉版本≥6.5.26) | 支持(钉钉版本≥6.5.25) | 不支持 |

## **示例代码**

```javascript
let fileManager = dd.getFileSystemManager();
fileManager.readdir({
  dirPath:`${dd.env.USER_DATA_PATH}/newDir`,
  success:(res)=>{
    console.log(res);
},
  fail:(err) =>{
    console.log(err)
  }
});
```

## **入参说明**

|  |  |  |  |
| --- | --- | --- | --- |
| **参数** | **类型** | **是否必填** | **描述** |
| dirPath | String | 是 | 本地用户文件目录，调用[创建本地用户目录](/document/orgapp/creat-folder)接口获取。 |
| success | Function | 否 | 调用成功的回调函数。 |
| fail | Function | 否 | 调用失败的回调函数。 |
| complete | Function | 否 | 调用结束的回调函数（调用成功、失败都会执行）。 |

## **返回结果**

### **成功**

|  |  |  |
| --- | --- | --- |
| **字段** | **类型** | **说明** |
| success | Boolean | 成功获取本地用户文件和目录列表时，返回true。 |
| files | Array<String> | 本地用户目录下的文件或目录列表。 |

### **失败**

|  |  |  |
| --- | --- | --- |
| **返回信息** | **类型** | **说明** |
| Object | Object | 错误信息。 |

## **错误码**

|  |  |  |
| --- | --- | --- |
| **errorCode** | **errorMessage** | **说明** |
| 10022 | 目录不存在 https://usr/xxx | dirPath本地用户文件目录不存在。 |
