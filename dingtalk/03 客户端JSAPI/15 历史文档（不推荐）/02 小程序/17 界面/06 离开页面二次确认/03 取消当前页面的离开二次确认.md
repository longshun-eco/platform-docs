更新于 2025-09-17调用**dd.disableLeaveConfirm**取消当前页面的离开二次确认配置。

**说明**

当前页面指调用dd.enableLeaveConfirm及dd.disableLeaveConfirm时**小程序栈顶页面**，与**该 JSAPI 在哪个 page 实例下调用无关。**

## 示例代码

```javascript
dd.disableLeaveConfirm({
    success: function(){},
  fail: function(){},
  complete: function(){}
})
```

## 入参

|  |  |  |
| --- | --- | --- |
| **参数** | **类型** | **说明** |
| success | Function | 成功回调。 |
| fail | Function | 错误回调。 |
| complete | Function | 完成回调。 |

## 兼容性

**平台**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **平台** | **iOS** | **Android** | **Mac** | **Windows** |
| **兼容平台** | 钉钉版本≥4.7.10 | 钉钉版本≥4.7.10 | 不支持 | 不支持 |

**兼容性判断**

请使用[dd.canIUse](/document/orgapp/dd-caniuse)('enableLeaveConfirm') 和[dd.canIUse](/document/orgapp/dd-caniuse)('disableLeaveConfirm') 进行兼容性判断。
