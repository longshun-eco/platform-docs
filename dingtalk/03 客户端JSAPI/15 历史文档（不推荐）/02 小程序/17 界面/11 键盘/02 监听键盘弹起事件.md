更新于 2025-09-17调用**dd.onKeyboardShow**监听键盘弹起事件，并返回键盘高度。

## **示例****代码**

```
Page({
    onKeyboardShow(res) {
        dd.alert({
            content: 'keyboard show, height = ' + res.data.height
        })
    }
})
```

## 入参

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| data.height | Number | 获取键盘高度，单位为px。  **重要**  需要在page中设置该回调。 |
