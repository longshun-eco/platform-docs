更新于 2025-09-17调用**dd.navigateTo**跳转到应用内的某个指定页面，并保留当前页面。可以使用dd.navigateBack返回到原来页面。

**重要**

页面最大深度为5，即可连续调用 5 次 navigateTo。

## 扫码体验

![[development-dd-navigateto_p163550.png]]

## 示例代码

```
dd.navigateTo({
  url: 'new_page?count=100'
})
```

```
// test.js
Page({
  onLoad(query){
    dd.alert({
      content: JSON.stringify(query),
    });
  }
})
```

## 入参

| **参数** | **类型** | **是否必填** | **说明** |
| --- | --- | --- | --- |
| url | String | 是 | 需要跳转的应用内非 tabBar 的目标页面路径，路径后可以带参数。参数规则如下：路径与参数之间使用`?`分隔，参数键与参数值用`=`相连，不同参数必须用`&`分隔；如 `path?key1=value1&key2=value2。` |
| success | Function | 否 | 调用成功的回调函数。 |
| fail | Function | 否 | 调用失败的回调函数。 |
| complete | Function | 否 | 调用结束的回调函数（调用成功、失败都会执行）。 |
