更新于 2025-09-17调用**dd.hideLoading**隐藏加载提示，可与dd.showLoading配合使用。

## **示例****代码**

```
dd.hideLoading();
```

## **入参**

| 参数 | 类型 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| page | Object | 否 | 具体指当前 page 实例，某些场景下，需要指明在哪个 page 执行 hideLoading。 |
