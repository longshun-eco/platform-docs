更新于 2025-09-17调用**dd.showLoading**显示加载提示，可与dd.hideLoading配合使用。

## 扫码体验

![[development-dd-showloading_p163559.png]]

## **示例****代码**

```
dd.showLoading({
  content: '加载中...',
});
// 设置加载时间
setTimeout(() => {
     dd.hideLoading();
}, 5000)
```

## 入参

| **参数** | **类型** | **是否必填** | **说明** |
| --- | --- | --- | --- |
| content | String | 否 | loading的文字内容。 |
| success | Function | 否 | 调用成功的回调函数。 |
| fail | Function | 否 | 调用失败的回调函数。 |
| complete | Function | 否 | 调用结束的回调函数（调用成功、失败都会执行）。 |
