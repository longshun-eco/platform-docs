更新于 2025-09-17调用**dd.previewFileInDingTal**k预览钉盘文件。

## 代码示例

```
dd.previewFileInDingTalk({
    corpId:"dingf8b3508f3073b265",
    spaceId:"13557022",
    fileId:"11452819",
    fileName:"钉盘快速入门.pdf",
    fileSize:1024,
    fileType:"pdf",
})
```

## 入参

| **参数** | **类型** | **说明** |
| --- | --- | --- |
| spaceId | String | 空间ID。 |
| fileId | String | 文件ID。 |
| fileName | String | 文件名称。 |
| fileSize | long | 文件大小，字节数。 |
| fileType | String | 文件扩展名。 |
