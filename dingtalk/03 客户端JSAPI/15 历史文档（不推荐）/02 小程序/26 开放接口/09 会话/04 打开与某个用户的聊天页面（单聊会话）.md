更新于 2025-09-17调用**dd.openChatByUserId**打开与某个用户的聊天页面。

| **客户端** | **Android** | **iOS** | **PC** |
| --- | --- | --- | --- |
| 支持说明 | 支持 | 支持 | 不支持 |

## 示例代码

```
dd.openChatByUserId({
    userId:'xxx', // 用户的userid
    success: res => {

    },
    fail: err =>{
        dd.alert({
            content:JSON.stringify(err)
        })
    }
})
```

## **入参**

| **参数** | **类型** | **说明** |
| --- | --- | --- |
| userId | String | 用户的userid。 |
