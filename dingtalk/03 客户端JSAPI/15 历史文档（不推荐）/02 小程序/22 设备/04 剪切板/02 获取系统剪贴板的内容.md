更新于 2025-09-17调用**dd.getClipboard**获取系统剪贴板的内容。

## 扫码体验

![[development-dd-getclipboard_p180594.png]]

## **示例****代码**

```java
Page({
  data: {
    text: '3.1415926',
    copy: '',
  },

  handlePaste() {
    dd.getClipboard({
      success: ({ text }) => {
        this.setData({ copy: text });
      },
    });
  },
});
```

## **入参**

| **参数** | **类型** | 是否必填 | **说明** |
| --- | --- | --- | --- |
| success | Function | 否 | 调用成功的回调函数。 |
| fail | Function | 否 | 调用失败的回调函数。 |
| complete | Function | 否 | 调用结束的回调函数（调用成功、失败都会执行）。 |

**success返回值**

| **名称** | 类型 | 描述 |
| --- | --- | --- |
| text | String | 剪切板数据。 |
