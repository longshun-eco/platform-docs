更新于 2025-09-17调用**dd.setClipboard**设置剪切板数据。

## **示例代码**

```java
Page({
  data: {
    text: '3.1415926',
    copy: '',
  },

  handleCopy() {
    dd.setClipboard({
      text: this.data.text,
    });
  },
});
```

## **入参**

| **参数** | **类型** | 是否必填 | **说明** |
| --- | --- | --- | --- |
| text | String | 是 | 剪切板数据。 |
| success | Function | 否 | 调用成功的回调函数。 |
| fail | Function | 否 | 调用失败的回调函数。 |
| complete | Function | 否 | 调用结束的回调函数（调用成功、失败都会执行）。 |
