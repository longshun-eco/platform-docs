更新于 2025-09-17调用**dd.vibrateShort**使用短振动功能。

## 扫码体验

![[development-dd-vibrateshort_p180598.png]]

## 使用限制

仅在 iPhone 7 / 7 Plus 以上及 Android 机型生效。

## **示例代码**

```
Page({
  vibrateShort() {
    dd.vibrateShort({
      success: () => {
        dd.alert({ title: '短振动'});
      }
    });
  },
})
```

## 兼容性

请使用 dd.canIUse('vibrateShort') 进行可用性判断
