更新于 2025-09-17调用dd.vibrateLong使用长振动功能。

## 扫码体验

![[development-dd-vibratelong_p180600.png]]

## 代码示例

```
Page({
  vibrateShort() {
    dd.vibrateLong({
      success: () => {
        dd.alert({ title: '长振动'});
      }
    });
  },
})
```

## 兼容性

请使用 dd.canIUse('vibrateLong') 进行可用性判断。
