更新于 2025-09-17使用**dd.corpId**接口获取当前访问用户的企业corpId。

## 示例代码

```
Page({
  onReady() {
    // 页面加载完成
    dd.alert({
      content: dd.corpId,
    })
  },
});
```

## 返回值

为String类型，表示当前访问用户的企业corpId。
