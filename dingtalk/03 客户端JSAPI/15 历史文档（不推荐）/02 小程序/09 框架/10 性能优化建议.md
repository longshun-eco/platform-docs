## 优化setData逻辑

每次 setData 数据量要小，不要直接修改 this.data 然后整个 set，这样会导致传输数据量过大影响渲染性能。

推荐指定路径设置数据：

```java
this.setData({
  'array[0]': 1,
  'obj.x':2,
});
```

而不是

```java
const array = this.data.array.concat();
array[0] = 1;
const obj={...this.data.obj};
obj.x=2;
this.setData({array,obj});
```

更不是（违反不可变数据原则）

```java
this.data.array[0]=1;
this.data.obj.x=2;
this.setData(this.data)
```
