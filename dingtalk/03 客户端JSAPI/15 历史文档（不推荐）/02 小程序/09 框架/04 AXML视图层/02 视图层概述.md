更新于 2025-09-17视图文件的后缀名是`axml`，定义了页面的标签结构。AXML是小程序框架设计的一套标签语言，用于描述小程序页面的结构。

AXML语法可分为五个部分：

* **数据绑定**

  ```json
   {{message}}
  ```

  ```
  // page.js
  Page({
    data: {
      message: 'Hello dingtalk!'
    }
  })
  ```

* **列表渲染**

  ```json
   {{item}}
  ```

  ```
  // page.js
  Page({
    data: {
      items: [1, 2, 3, 4, 5, 6, 7]
    }
  })
  ```
* **条件渲染**

  ```
   WEBVIEW
   APP
   dingtalk
  ```

  ```
  // page.js
  Page({
    data: {
      view: 'dingtalk'
    }
  })
  ```

* **模板**

  ```
      FirstName: {{firstName}}, LastName: {{lastName}}
  ```

  ```
  // page.js
  Page({
    data: {
      staffA: {firstName: 'san', lastName: 'zhang'},
      staffB: {firstName: 'si', lastName: 'li'},
      staffC: {firstName: 'wu', lastName: 'wang'},
    },
  })
  ```
* **事件**

  ```json
   {{count}}
  ```

  ```java
  Page({
    data: {
      count: 1
    },
    add(e) {
      this.setData({
        count: this.data.count + 1
      })
    }
  })
  ```
