## getRed()

获得颜色RGB值中的R值。

### 返回结果

number - RGB值中的R值。

### 示例代码

```javascript
const red = color.getRed();
```

## getGreen()

获得颜色RGB值中的G值。

### 返回结果

number - RGB值中的G值。

### 示例代码

```javascript
const green = color.getGreen();
```

## getBlue()

获得颜色RGB值中的B值。

### 返回结果

number - RGB值中的B值。

### 示例代码

```javascript
const blue = color.getBlue();
```

## getHexString()

获得颜色的十六进制字符串。

### 返回结果

string - 颜色的十六进制字符串，如 #ff0000 。

### 示例代码

```javascript
const hex = color.getHexString();
```
