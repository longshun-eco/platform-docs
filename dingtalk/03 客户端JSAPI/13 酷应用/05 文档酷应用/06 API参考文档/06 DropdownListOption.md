## API描述

下拉列表选项。用于 Range.insertDropdownLists() API中指定要插入的下拉列表属性。

## 属性说明

| 属性 | 类型 | 是否必传 | 说明 |
| --- | --- | --- | --- |
| value | string | 是 | 选项值。 |
| color | string | 否 | 选项的背景色。 |
