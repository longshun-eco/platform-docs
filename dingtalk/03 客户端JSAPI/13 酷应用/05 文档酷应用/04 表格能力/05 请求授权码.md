更新于 2025-08-27本文通过调用**Dingdocs.workbook.host.getAuthCode**请求钉钉免登授权码。

## **准备工作**

初始化脚本服务JSAPI需依赖[dingtalk-docs-cool-app](https://www.npmjs.com/package/dingtalk-docs-cool-app)，请先升级到最新版本。

```bash
npm install dingtalk-docs-cool-app --save
```

## **API使用说明**

**说明**

建议钉钉版本使用最新版本。

|  |  |  |  |
| --- | --- | --- | --- |
| **客户端** | **Android** | **IOS** | **PC** |
| 支持说明 | 不支持 | 不支持 | 支持 |

调用**Dingdocs.workbook.host.getAuthCode**请求钉钉免登授权码。

免登是指用户进入应用后，无需输入钉钉用户名和密码，酷应用可自动获取当前用户身份登录系统的流程。

```
Dingdocs.workbook.host.getAuthCode('corpId')
.then((res) => {
  console.log(res.code);
}).catch((e) => {
  console.log(e);
})
```

## **参数说明**

|  |  |  |  |
| --- | --- | --- | --- |
| **参数** | **类型** | **是否必填** | **说明** |
| corpId | String | 是 | 企业的corpId。 |

## **返回结果**

|  |  |  |
| --- | --- | --- |
| **参数** | **类型** | **说明** |
| code | String | 授权码。  **说明**  有效期5分钟，且只能使用一次，使用后会失效。 |
