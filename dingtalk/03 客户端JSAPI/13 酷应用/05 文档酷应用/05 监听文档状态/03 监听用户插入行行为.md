更新于 2025-08-27本文通过调用**Dingdocs.workbook.event.onRowInserted**监听文档是否有插入行操作。

## **准备工作**

初始化脚本服务JSAPI需依赖[dingtalk-docs-cool-app](https://www.npmjs.com/package/dingtalk-docs-cool-app)，请先升级到最新版本。

```bash
npm install dingtalk-docs-cool-app --save
```

## **API使用说明**

**说明**

建议钉钉版本使用最新版本。

|  |  |  |  |
| --- | --- | --- | --- |
| **客户端** | **Android** | **IOS** | **PC** |
| 支持说明 | 不支持 | 不支持 | 支持 |

调用**Dingdocs.workbook.event.onRowInserted**监听文档是否有插入行操作。

```
offRowInserted = Dingdocs.workbook.event.onRowInserted(({ sheetId, a1Notation }) => {
  console.log('RowInserted', sheetId, a1Notation);
})
// 注销监听
offRowInserted()
```

## **参数说明**

|  |  |  |  |
| --- | --- | --- | --- |
| **参数** | **类型** | **是否必填** | **说明** |
| handler | Function | 是 | 监听到插入行后的回调：   * **sheetId**：发生变更的表格id * **a1Notation**：发生变更的单元格位置 |
