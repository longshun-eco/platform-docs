更新于 2025-09-23调用本接口，审核词条。

针对新建或更新的词条都是需要通过审核以后才能生效，本接口操作审核通过或者拒绝。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业百科写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=pedia_1.0%23PediaWordsApprove) |
| 第三方企业应用 | 支持 | 企业百科写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=pedia_1.0%23PediaWordsApprove) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/pedia/words/approve HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "uuid" : Long,
  "userId" : "String",
  "approveStatus" : "String",
  "approveReason" : "String",
  "imHighLight" : Boolean,
  "simHighLight" : Boolean
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| uuid | Long | 是 | 当前审核的词条的主键编号，可调用[分页获取企业词条信息](/document/development/entry-search)接口获取未审核的词条。 |
| userId | String | 是 | 操作人的组织员工userId。 |
| approveStatus | String | 是 | 审核的结果：   * 1：通过 * 0：拒绝 |
| approveReason | String | 否 | 拒绝的原因。 |
| imHighLight | Boolean | 是 | 当前内部群是否高亮：   * true：高亮 * false：不高亮 |
| simHighLight | Boolean | 是 | 服务群是否高亮：   * true：高亮 * false：不高亮 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 请求是否成功，true表示成功。 |
