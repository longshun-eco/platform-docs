更新于 2025-09-23调用本接口撤销员工的荣誉勋章。

### 接口功能介绍

调用本接口，根据荣誉勋章Id，实现撤销员工所获得的荣誉勋章。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 组织文化荣誉信息写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=orgCulture_1.0%23RecallHonor) |
| 第三方企业应用 | 支持 | 组织文化荣誉信息写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=orgCulture_1.0%23RecallHonor) |
| 第三方个人应用 | 暂不支持 | 组织文化荣誉信息写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/orgCulture/honors/{honorId}/recall HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "userId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| honorId | String | 是 | 荣誉勋章id，可通过调用[查询员工已获得的组织荣誉](/document/development/check-the-honors-that-an-employee-has-received)接口，获取荣誉勋章`honorId`字段。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 撤销荣誉勋章的员工userId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 接口调用是否成功。 |
| result | Object | 撤销荣誉勋章结果。 |
| honorId | String | 被撤销荣誉勋章id。 |
