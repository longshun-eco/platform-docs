更新于 2025-09-23调用本接口实现创建荣誉模板。

### 接口功能介绍

调用本接口，根据创建荣誉勋章模板人Id、荣誉勋章名称等信息，创建企业荣誉勋章模板。

**说明**

* 创建企业荣誉勋章模板，会流入钉钉后台进行审核，一般5个工作日内审核完毕。
* 企业每周的审核次数有限制（标准版每周10次/专业版每周20次），超过后接口会报错。如有特殊原因希望提高审核次数上限，请联系我们申请并讲明原因。
* 针对同一家组织，不允许并发调用本接口创建。如需批量创建，请串行逐个执行。

### 荣誉设计规范

为了保障勋章能够顺利上架，避免安全、法务、涉敏、涉政等问题，荣誉名称和图形设计需要注意内容规范，详情参见[荣誉设计规范](https://alidocs.dingtalk.com/i/p/nb9XJor4WgY8myAp/docs/lo1YvX0prG98k244KR79JPw7xzbmLdEZ)。

### 钉钉荣誉勋章规范条款

1. 您承诺遵守《钉钉荣誉规范条款》，保证所上传的内容不违反国家法律法规，不侵犯任何第三方的合法权益，不存在违背社会公序良俗的情况。
2. 您同意，钉钉有权对您上传的内容进行审核、作出判断，对违反本规则的内容进行处理（包括停止发布、删除等），按钉钉相关规则对您的使用钉钉进行限制。
3. 您同意，钉钉有权基于正常运营的需要对本服务进行变更（包括升级、停止运营等），或以钉钉提供的其他服务进行替代。
4. 钉钉有权根据正常运营的需要对本规则进行修订，如您不接受修订后的内容，请停止使用本服务，您使用的行为将视为您同意遵守本规则。
5. 上传素材中不得包含涉黄、涉政等违规的内容（包含但不仅限于图片、文字）。

## 请求方法

```
POST /v1.0/orgCulture/honors/templates HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "userId" : "String",
  "medalName" : "String",
  "medalDesc" : "String",
  "medalMediaId" : "String",
  "avatarFrameMediaId" : "String",
  "defaultBgColor" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 创建荣誉勋章模板的管理员所在组织的userId信息。  **说明**  需要主管理员角色或者子管理员角色。 |
| medalName | String | 是 | 荣誉勋章名称。  **说明**   * 最大长度10字符 。 * 不支持表情图标等。 |
| medalDesc | String | 是 | 荣誉勋章描述。  **说明**   * 最大长度30字符 。 * 不支持表情图标等。 |
| medalMediaId | String | 是 | 荣誉勋章图片，可调用[上传媒体文件](/document/development/upload-media-files)接口获取出参`media_id`参数字段。  **说明**   * 图片尺寸 900\*900，不超过1M，支持PNG 。 * 图片请使用钉钉媒体资源标识符media\_id。 |
| avatarFrameMediaId | String | 是 | 头像挂件，可调用[上传媒体文件](/document/development/upload-media-files)接口获取出参`media_id`参数字段。  **说明**   * 图片尺寸 240\*240，不超过1M，支持PNG。 * 图片请使用钉钉媒体资源标识符media\_id。 |
| defaultBgColor | String | 是 | 背景颜色:   * #FFFBB4 * #FFE7BC * #FFDAF4 * #DAF6A8 * #E4D7FF * #BFDFFF * #B9F2D6   ![[development-create-medal-of-honor-template_颜色分类.png]]  **说明**  仅以上颜色可选。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 接口调用是否成功。 |
| result | Object | 荣誉勋章创建结果。 |
| honorId | String | 荣誉勋章模板Id。 |
