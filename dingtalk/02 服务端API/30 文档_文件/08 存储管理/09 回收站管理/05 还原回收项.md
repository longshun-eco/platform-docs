例如，某企业的云盘内，**团队文件**信息如下图所示。

调用本接口，可以将之前删除的名为**新建空间手动**的文件夹，还原到原来的位置。接口调用成功后如下图所示。

![[development-restore-recycle-items_O1CN01wR1cMX28ZJupSRijr_!!6000000007946-2-tps-2620-814.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储回收站写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_1.0%23RestoreRecycleItem) |
| 第三方企业应用 | 支持 | 企业存储回收站写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23RestoreRecycleItem) |
| 第三方个人应用 | 暂不支持 | 企业存储回收站写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/storage/recycleBins/{recycleBinId}/recycleItems/{recycleItemId}/restore?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "option" : {
    "conflictStrategy" : "String"
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| recycleBinId | String | 是 | 回收站Id，可调用[获取回收站信息](/document/development/obtain-information-about-the-recycle-bin)接口获取。 |
| recycleItemId | String | 是 | 回收项Id，可调用[获取回收项列表](/document/development/gets-the-list-of-recycle-items)接口获取id参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作者unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| option | Object | 否 | 可选参数。 |
| conflictStrategy | String | 否 | 文件名称冲突策略，还原时原路径可能已经存在同名的文件。   * **AUTO\_RENAME**：自动重命名，默认值 * **OVERWRITE**：覆盖 * **RETURN\_DENTRY\_IF\_EXISTS**：返回已存在文件 * **RETURN\_ERROR\_IF\_EXISTS**：文件已存在时报错 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| async | Boolean | 是否是异步任务。  **说明**  如果操作对象有子节点，则会异步处理。   * **true**：是 * **false**：不是 |
| taskId | String | 异步任务ID，用于查询任务执行状态。 |
| spaceId | String | 操作对应根节点还原之后的空间ID。  **说明**  非失败的情况下，同步或者异步都会返回该参数。 |
| dentryId | String | 操作对应根节点还原之后的文件ID。  **说明**  非失败的情况下，同步或者异步都会返回该参数。 |
