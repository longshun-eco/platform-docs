更新于 2025-09-23调用本接口，获取回收站内的回收项信息列表。

调用本接口，可以分页获取回收站所有回收项的详情信息，包括原来所在空间Id、原路径、大小、名称等信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储回收站读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_1.0%23ListRecycleItems) |
| 第三方企业应用 | 支持 | 企业存储回收站读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23ListRecycleItems) |
| 第三方个人应用 | 暂不支持 | 企业存储回收站读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/storage/recycleBins/{recycleBinId}/recycleItems?nextToken=String&maxResults=Integer&unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| recycleBinId | String | 是 | 回收站Id，可调用[获取回收站信息](/document/development/obtain-information-about-the-recycle-bin)接口获取id参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| nextToken | String | 否 | 分页游标。   * 如果是首次调用，该参数不传。 * 如果是非首次调用，该参数传上次调用时返回的nextToken。 |
| maxResults | Integer | 否 | 每页最大条目数，默认值50，最大值50。 |
| unionId | String | 是 | 操作者unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| recycleItems | Array | 回收项列表。 |
| id | String | 回收项id。 |
| spaceId | String | 原文件或文件夹所在的空间Id。 |
| dentryId | String | 原文件或文件夹Id。 |
| size | Long | 原文件或文件夹大小，单位Byte。 |
| type | String | 类型。   * **FILE**：文件 * **FOLDER**：文件夹 |
| originalName | String | 原文件或文件夹名称。 |
| originalPath | String | 原文件或文件夹路径。 |
| operatorId | String | 删除操作人unionId。 |
| operatorTime | String | 删除时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| nextToken | String | 分页游标。  **说明**  不为空表示有更多数据。 |
