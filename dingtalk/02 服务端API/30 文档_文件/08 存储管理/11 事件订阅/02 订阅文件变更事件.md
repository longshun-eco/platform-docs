更新于 2025-09-23调用本接口，订阅文件变更事件。

### 使用说明

1，在开发者后台应用事件订阅中，先开启订阅存储事件。

![[development-subscribe-to-file-change-events_O1CN01pz5Amw22rkBpo7M7E_!!6000000007174-0-tps-2400-1350.jpg]]

2，调用本接口，指定订阅范围是企业或者指定的存储空间。

**说明**

如果仅在开发者后台开启存储事件订阅开关，无法接收回调事件，必须与本接口配合使用。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储文件变更事件订阅写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_1.0%23SubscribeEvent) |
| 第三方企业应用 | 支持 | 企业存储文件变更事件订阅写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23SubscribeEvent) |
| 第三方个人应用 | 暂不支持 | 企业存储文件变更事件订阅写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/storage/events/subscribe?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "scopeId" : "String",
  "scope" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作者unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| scopeId | String | 是 | 订阅范围对应的ID。   * 当scope参数值为ORG时，scopeId对应的是企业corpId，请参考[基础概念-CorpId](/document/development/basic-concepts-beta#section-bbk-mv0-oxd)。 * 当scope为SPACE时，scopeId对应的是存储空间ID，可调用[添加空间](/document/development/add-space)接口获取id。 |
| scope | String | 是 | 订阅范围。   * ORG：企业 * SPACE：空间 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 本次操作是否成功，true表示成功。 |
