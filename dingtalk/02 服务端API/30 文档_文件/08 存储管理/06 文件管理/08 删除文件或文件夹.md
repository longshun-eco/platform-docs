更新于 2025-09-12调用本接口，删除文件或文件夹。

### 接口功能介绍

调用本接口，可实现以下操作：

* 删除文件。
* 删除文件夹。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_1.0%23DeleteDentry) |
| 第三方企业应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23DeleteDentry) |
| 第三方个人应用 | 暂不支持 | 企业存储文件写权限 | 暂不支持 |

## 请求方法

```sql
DELETE /v1.0/storage/spaces/{spaceId}/dentries/{dentryId}?toRecycleBin=Boolean&unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | String | 是 | 空间Id，调用[添加空间](/document/development/add-space)接口获取id参数值。 |
| dentryId | String | 是 | 需要被删除的文件或文件夹Id，调用[获取文件或文件夹列表](/document/development/obtain-the-file-list-storage)接口获取id参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| toRecycleBin | Boolean | 否 | 删除后，是否备份到回收站。   * **true**：是 * **false**：否，默认值   **说明**  支持备份到回收站的情况。目前仅[新建空间](/document/development/new-space)接口创建的云盘空间内的文件和文件夹支持备份到回收站。 |
| unionId | String | 是 | 操作者unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| async | Boolean | 是否是异步任务。   * **true**：是 * **false**：否 |
| taskId | String | 任务Id，用于查询任务执行状态。 |

## 错误码

| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | operationConcurrentControl | %s | 并发控制 |
| 400 | paramError | %s | 参数错误 |
| 400 | recycleBinNotSupportForSpace | %s | 空间不支持回收站 |
