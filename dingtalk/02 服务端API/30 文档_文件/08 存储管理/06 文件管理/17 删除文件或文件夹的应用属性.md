更新于 2025-09-12调用本接口，删除文件或文件夹的应用属性。

例如，测试文件的应用属性信息是：

```
"appProperties":{
         "dingzuws6w3mi5svgwky":[
                 {"name":"属性1","value":"属性值1","visibility":"PRIVATE"},
                 {"name":"属性2","value":"属性值2","visibility":"PRIVATE"}
          ]
}
```

调用本接口，可删除该文件的应用属性。例如，删除属性2，接口调用结果如下：

```
"appProperties":{
        "dingzuws6w3mi5svgwky":[
                 {"name":"属性1","value":"属性值1","visibility":"PRIVATE"}
          ]
}
```

**说明**

* 存储空间类型为USER时，空间拥有权限的用户都有操作权限，其他员工调用[添加权限](/document/development/add-permissions-file)接口进行授权。
* 存储空间类型为APP时，任何人操作都需要调用[添加权限](/document/development/add-permissions-file)接口进行授权。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_1.0%23DeleteDentryAppProperties) |
| 第三方企业应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23DeleteDentryAppProperties) |
| 第三方个人应用 | 暂不支持 | 企业存储文件写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/storage/spaces/{spaceId}/dentries/{dentryId}/appProperties/remove?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "propertyNames" : [ "String" ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | Long | 是 | 空间Id，调用[添加空间](/document/development/add-space)接口获取id参数值。 |
| dentryId | Long | 是 | 文件或文件夹Id，调用[获取文件或文件夹列表](/document/development/obtain-the-file-list-storage)接口获取id参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作者unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| propertyNames | Array of String | 是 | 文件或文件夹的应用属性名称列表，最大值3。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 本次删除是否成功。   * **true**：成功 * **false**：失败 |
