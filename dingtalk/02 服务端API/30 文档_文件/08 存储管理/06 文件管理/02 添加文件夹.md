更新于 2025-09-12调用本接口，在存储空间内添加文件夹。

### 接口调用说明

在添加存储空间后，可调用本接口在空间内添加文件夹。

* 存储空间类型为USER时，空间拥有权限的用户都有操作权限，其他员工调用[添加权限](/document/development/add-permissions-file)接口进行授权。
* 存储空间类型为APP时，任何人操作都需要调用[添加权限](/document/development/add-permissions-file)接口进行授权。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_1.0%23AddFolder) |
| 第三方企业应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23AddFolder) |
| 第三方个人应用 | 暂不支持 | 企业存储文件写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/storage/spaces/{spaceId}/dentries/{parentId}/folders?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "name" : "String",
  "option" : {
    "conflictStrategy" : "String",
    "appProperties" : [ {
      "name" : "String",
      "value" : "String",
      "visibility" : "String"
    } ]
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | String | 是 | 空间Id，可调用[添加空间](/document/development/add-space)接口获取id参数值。 |
| parentId | String | 是 | 父目录Id，调用[获取文件或文件夹列表](/document/development/obtain-the-file-list-storage)接口获取。根目录时，该参数是0。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作者unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| name | String | 是 | 文件夹的名称，命名有以下要求：   * 头尾不能包含空格，否则会自动去除 * 不能包含特殊字符，包括：制表符、\*、"、<、>、| * 不能以"."结尾 |
| option | Object | 否 | 可选参数。 |
| conflictStrategy | String | 否 | 文件夹名称冲突策略。   * **AUTO\_RENAME**：自动重命名，默认值 * **OVERWRITE**：覆盖 * **RETURN\_DENTRY\_IF\_EXISTS**：返回已存在文件夹 * **RETURN\_ERROR\_IF\_EXISTS**：文件夹已存在时报错 |
| appProperties | Array | 否 | 当前文件夹的应用属性列表，最大值3。 |
| name | String | 是 | 属性名称。 |
| value | String | 是 | 属性值。 |
| visibility | String | 是 | 属性可见性。   * **PUBLIC**：所有应用都可见 * **PRIVATE**：仅限当前应用可见 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| dentry | Object | 文件夹信息。 |
| id | String | 文件夹id。 |
| spaceId | String | 所在空间Id。 |
| parentId | String | 父目录Id。  **说明**  该字段为0，表示根目录。 |
| type | String | 类型，本接口返回类型是固定值。   * **FOLDER**：文件夹 |
| name | String | 文件夹的名称。 |
| size | Long | 大小, 单位Byte。  **说明**  本接口添加文件夹，不返回该字段。 |
| path | String | 文件夹在空间内的路径。 |
| version | Long | 版本。  **说明**  本接口添加文件夹，不返回该字段。 |
| status | String | 状态。   * **NORMAL**：正常 * **DELETED**：已删除 |
| extension | String | 后缀。  **说明**  本接口添加文件夹，不返回该字段。 |
| creatorId | String | 创建者unionId。 |
| modifierId | String | 修改者unionId。 |
| createTime | String | 创建时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| modifiedTime | String | 修改时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| properties | Object | 文件夹属性。 |
| readOnly | Boolean | 是否只读。   * **true**：是 * **false**：否 |
| appProperties | Map<String, Array> | 在特定应用上的属性。key是空间的ownerId，value是属性列表。 |
|  | Array | 属性列表。 |
| name | String | 属性名。 |
| value | String | 属性值。 |
| visibility | String | 属性可见性。   * **PUBLIC**：所有应用都可见 * **PRIVATE**：仅限当前应用可见 |
| uuid | String | 标识字段。 |
| partitionType | String | 存储分区。   * **PUBLIC\_OSS\_PARTITION**：公有云OSS存储分区 * **MINI\_OSS\_PARTITION**：专属MiniOSS存储分区 |
| storageDriver | String | 驱动类型。   * **DINGTALK**：钉钉统一存储驱动 * **ALIDOC**：钉钉文档存储驱动 * **UNKNOWN**：未知驱动 |
