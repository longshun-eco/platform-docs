更新于 2025-09-12调用本接口，批量移动文件或文件夹。

### 接口功能介绍

调用本接口，可实现以下操作：

* 批量移动文件至其他任意位置。
* 批量移动文件夹至其他任意位置。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_1.0%23MoveDentries) |
| 第三方企业应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23MoveDentries) |
| 第三方个人应用 | 暂不支持 | 企业存储文件写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/storage/spaces/{spaceId}/dentries/batchMove?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "targetSpaceId" : "String",
  "targetFolderId" : "String",
  "dentryIds" : [ "String" ],
  "option" : {
    "conflictStrategy" : "String",
    "preservePermissions" : Boolean
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | String | 是 | 源文件或文件夹空间ID，调用[添加空间](/document/development/add-space)接口获取id参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作人的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| targetSpaceId | String | 是 | 目标空间ID，调用[添加空间](/document/development/add-space)接口获取id参数值。 |
| targetFolderId | String | 是 | 目标文件夹id，调用[获取文件或文件夹列表](/document/development/obtain-the-file-list-storage)接口获取id参数值。 根目录id值为0。 |
| dentryIds | Array of String | 是 | 源文件或文件夹的ID列表，最大值30，调用[获取文件或文件夹列表](/document/development/obtain-the-file-list-storage)接口获取id参数值。 |
| option | Object | 否 | 可选参数。 |
| conflictStrategy | String | 否 | 文件或文件夹的名称冲突策略。   * **AUTO\_RENAME**：自动重命名，默认值 * **OVERWRITE**：覆盖 * **RETURN\_DENTRY\_IF\_EXISTS**：返回已存在文件 * **RETURN\_ERROR\_IF\_EXISTS**：文件已存在时报错 |
| preservePermissions | Boolean | 否 | 移动后，是否保留权限。   * **true**：保留 * **false**（默认）：不保留 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| resultItems | Array | 批量移动文件或文件夹的结果列表。 |
| spaceId | String | 源文件或文件夹所在的空间ID。 |
| dentryId | String | 源文件或文件夹的ID。 |
| async | Boolean | 是否是异步任务。   * **true**：是 * **false**：否   **说明**  如果操作对象有子节点，则会异步处理。 |
| success | Boolean | 是否成功。   * **true**：成功 * **false**：失败   **说明**  异步任务该字段不返回。 |
| errorCode | String | 错误原因。  **说明**  异步任务该字段不返回。 |
| taskId | String | 异步任务ID，可用于查询任务执行状态。 |
| targetSpaceId | String | 移动之后所属的空间ID。  **说明**  非失败情况下，同步或异步都会返回。 |
| targetDentryId | String | 移动之后文件或文件夹的ID。  **说明**  非失败情况下，同步或异步都会返回。 |
