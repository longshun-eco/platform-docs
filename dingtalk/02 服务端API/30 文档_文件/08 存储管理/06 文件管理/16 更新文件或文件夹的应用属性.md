更新于 2025-09-12调用本接口，更新文件或文件夹的应用属性。

例如，测试文件的应用属性信息是：

```
"appProperties":{
        "dingzuws6w3mi5svgwky":[
                 {"name":"属性1","value":"属性值1","visibility":"PRIVATE"}
          ]
}
```

调用本接口，可更新该文件的应用属性。例如，添加一个属性2，接口调用结果如下：

```
"appProperties":{
         "dingzuws6w3mi5svgwky":[
                 {"name":"属性1","value":"属性值1","visibility":"PRIVATE"},
                 {"name":"属性2","value":"属性值2","visibility":"PRIVATE"}
          ]
}
```

**说明**

接口传入的应用属性信息，会根据属性名称校验。

* 如果文件或文件夹原有属性名与接口参数传入的属性名相同，接口执行更新属性。
* 如果文件或文件夹原有属性名与接口参数传入的属性名不相同，接口执行添加属性。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_1.0%23UpdateDentryAppProperties) |
| 第三方企业应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23UpdateDentryAppProperties) |
| 第三方个人应用 | 暂不支持 | 企业存储文件写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/storage/spaces/{spaceId}/dentries/{dentryId}/appProperties?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "appProperties" : [ {
    "name" : "String",
    "value" : "String",
    "visibility" : "String"
  } ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | Long | 是 | 空间Id，调用[添加空间](/document/development/add-space)接口获取id参数值。 |
| dentryId | Long | 是 | 文件或文件夹的id，调用[获取文件或文件夹列表](/document/development/obtain-the-file-list-storage)接口获取id参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作者unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| appProperties | Array | 是 | 应用属性列表，最大值3。 |
| name | String | 是 | 属性名称。  **说明**   * 如果属性名称已存在，会更新原有的属性信息。 * 如果属性名称不存在，会添加该属性信息。 |
| value | String | 是 | 属性值。 |
| visibility | String | 是 | 属性可见范围。   * **PUBLIC**：该属性所有App可见 * **PRIVATE**：该属性仅其归属App可见，默认值 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 本次更新是否成功。   * **true**：成功 * **false**：失败 |
