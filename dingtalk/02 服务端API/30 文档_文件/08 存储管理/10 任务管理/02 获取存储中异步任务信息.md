更新于 2025-09-23调用本接口，获取存储中异步任务信息。

### 接口调用说明

存储内的异步操作，可调用本接口查询异步任务的信息，异步任务是否已完成、是否成功等。

例如，批量移动文件或文件夹、批量拷贝文件或文件夹、批量删除文件或文件夹等操作，如果被操作对象存在子节点，对应的操作就是异步操作。通过返回的异步任务taskId参数值，可调用本接口查询任务信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储任务读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_1.0%23GetTask) |
| 第三方企业应用 | 支持 | 企业存储任务读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23GetTask) |
| 第三方个人应用 | 暂不支持 | 企业存储任务读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/storage/tasks/{taskId}?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| taskId | String | 是 | 存储中的异步任务ID，可调用[批量移动文件或文件夹](/document/development/bulk-move-files-or-folders)接口获取taskId参数值。  例如，批量移动文件或文件夹的异步任务ID。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作人的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| task | Object | 任务信息。 |
| id | String | 任务ID。 |
| status | String | 任务状态。   * **INIT**：初始化 * **IN\_PROGRESS**：进行中 * **SUCCESS**：成功 * **FAIL**：失败 |
| totalCount | Long | 子任务总数。 |
| successCount | Long | 子任务成功总数。 |
| failCount | Long | 子任务失败总数。 |
| failMessage | String | 任务失败原因。 |
| beginTime | String | 任务开始时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| endTime | String | 任务结束时间，iso8601格式，例如：2022-07-29T14:55Z。 |
