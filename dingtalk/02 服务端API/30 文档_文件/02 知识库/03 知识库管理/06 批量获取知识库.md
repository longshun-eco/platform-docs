更新于 2025-09-12批量获取知识库。

### 接口功能介绍

调用本接口，根据操作人unionId和知识库ID，查询知识库信息。示例如下：

![[development-batch-acquisition-of-knowledge-base_知识库列表.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 知识库读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=wiki_2.0%23GetWorkSpaces) |
| 第三方企业应用 | 支持 | 知识库读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=wiki_2.0%23GetWorkSpaces) |
| 第三方个人应用 | 暂不支持 | 知识库读权限 | 暂不支持 |

## 请求方法

```
POST /v2.0/wiki/workspaces/batchQuery?operatorId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "workspaceIds" : [ "String" ],
  "option" : {
    "withPermissionRole" : Boolean
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorId | String | 是 | 操作人unionId。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| workspaceIds | Array of String | 是 | 知识库id，可通过调用[获取知识库列表](/document/development/get-knowledge-base-list)接口，获取返回参数`workspaceId`字段。  **说明**  最大size20。 |
| option | Object | 否 | 可选参数 |
| withPermissionRole | Boolean | 否 | 是否获取权限信息：   * **true**：获取 * **false**：不获取   **说明**  默认值false。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| workspaces | Array | 知识库列表。 |
| workspaceId | String | 知识库id(spaceUuid)。 |
| corpId | String | 知识库归属企业的corpId。 |
| teamId | String | 知识库归属知识小组的id。  **说明**  不归属任何知识小组的时候, 返回空。 |
| rootNodeId | String | 根节点id(根节点dentryUuid)。 |
| name | String | 知识库名称。 |
| type | String | 知识库类型，枚举值:   * **TEAM**：知识库 * **PERSONAL**：我的文档 |
| description | String | 知识库简介。 |
| url | String | 知识库访问url。 |
| icon | Object | 知识库图标。 |
| type | String | 图标类型，枚举值:   * **URL**：链接 * **EMOJI**：emoji图标 |
| value | String | 图标值：   * 当 type 为 URL 时，返回图标链接。 * 当 type 为 EMOJI 时，返回 emoji 图标的 unicode 值。 |
| cover | String | 知识库封面url。 |
| creatorId | String | 知识库创建者userId。 |
| modifierId | String | 知识库修改者userId。 |
| createTime | String | 知识库创建时间。 |
| modifiedTime | String | 知识库修改时间。 |
| permissionRole | String | 查询人对知识库的权限角色，枚举值:   * **OWNER**：拥有者 * **MANAGER**：管理者 * **EDITOR**：编辑者 * **DOWNLOADER**：可查看下载者 * **READER**：仅可查看者 * **NONE**：无权限者 |
