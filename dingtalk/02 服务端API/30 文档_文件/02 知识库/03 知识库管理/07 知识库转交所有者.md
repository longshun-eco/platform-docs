## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | SNS知识库文档写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_2.0%23HandoveryWorkspace) |
| 第三方企业应用 | 暂不支持 | SNS知识库文档写权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | SNS知识库文档写权限 | 暂不支持 |

## 请求方法

```
POST /v2.0/doc/dentries/workspace/handover HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "param" : {
    "resourceId" : "String",
    "receiverUnionId" : "String",
    "leave" : Boolean
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| param | Object | 是 | 必选参数。 |
| resourceId | String | 是 | 待转交的知识库的rootNodeUUid，可通过[批量获取知识库](/document/development/batch-acquisition-of-knowledge-base)接口获取。 |
| receiverUnionId | String | 是 | 接收人unionId。 |
| leave | Boolean | 否 | 转交者转交后是否退出此知识库，可不填，默认是false。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 执行结果。 |
