更新于 2025-09-12通过链接获取节点

### 接口功能介绍

调用本接口，根据操作者unionId和文档链接，获取节点信息。示例如下：

![[development-get-node-by-link_根据url获取节点信息.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 知识库节点读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=wiki_2.0%23GetNodeByUrl) |
| 第三方企业应用 | 支持 | 知识库节点读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=wiki_2.0%23GetNodeByUrl) |
| 第三方个人应用 | 暂不支持 | 知识库节点读权限 | 暂不支持 |

## 请求方法

```
POST /v2.0/wiki/nodes/queryByUrl?operatorId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "url" : "String",
  "option" : {
    "withStatisticalInfo" : Boolean,
    "withPermissionRole" : Boolean
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorId | String | 是 | 操作人unionId。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| url | String | 是 | 文档链接 |
| option | Object | 否 | 可选参数 |
| withStatisticalInfo | Boolean | 否 | 是否获取统计信息：   * true：获取 * false：不获取   **说明**  默认值false。 |
| withPermissionRole | Boolean | 否 | 是否获取权限信息：   * true：获取 * false：不获取   **说明**  默认值false。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| node | Object | 节点。 |
| nodeId | String | 节点id(dentryUuid)。 |
| workspaceId | String | 知识库id(spaceUuid)。 |
| name | String | 名称。 |
| size | Long | 大小。 |
| type | String | 类型，枚举值：   * **FILE**：文件 * **FOLDER**：文件夹 |
| category | String | 类别，枚举值:   * **ALIDOC**：钉钉文档 * **DOCUMENT**：本地文档 * **IMAGE**：图片 * **VIDEO**：视频 * **AUDIO**：音频 * **ARCHIVE**：归档文件 * **OTHER**：其他类型 |
| extension | String | 后缀。 |
| url | String | 访问url。 |
| creatorId | String | 创建者userId。 |
| modifierId | String | 修改者userId。 |
| createTime | String | 创建时间。 |
| modifiedTime | String | 修改时间。 |
| hasChildren | Boolean | 是否有子节点。 |
| statisticalInfo | Object | 统计信息。 |
| wordCount | Long | 字数。 |
| permissionRole | String | 当前用户对知识库节点的权限角色，枚举值:   * **OWNER**：拥有者 * **MANAGER**：管理者 * **EDITOR**：编辑者 * **DOWNLOADER**：可查看下载者 * **READER**：仅可查看者 * **NONE**：无权限者 |
