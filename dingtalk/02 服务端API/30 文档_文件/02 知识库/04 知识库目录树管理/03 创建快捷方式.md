更新于 2025-09-24创建知识库或文件的快捷方式。

调用本接口可在指定位置创建快捷方式，功能与下图相同。

![[development-api-createshortcut_O1CN01TytzRB26AvtLZqX8d_!!6000000007622-2-tps-3024-1654.png]]

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 调用SNS API时需要具备的基本权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_2.0%23CreateShortcut) |
| 第三方企业应用 | 暂不支持 | 调用SNS API时需要具备的基本权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 调用SNS API时需要具备的基本权限 | 暂不支持 |

## 请求方法

```
POST /v2.0/doc/resource/shortcut/create HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "param" : {
    "sourceResourceId" : "String",
    "sourceResourceType" : "String",
    "targetResourceId" : "String",
    "targetResourceType" : "String",
    "targetResourceName" : "String"
  }
}
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，调用[获取用户token](https://open.dingtalk.com/document/development/obtain-user-token)接口获取。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| param | Object | 是 | 必选参数。 |
| sourceResourceId | String | 是 | 源资源位置的id，可以是dentryUUid或rootNodeUUid，通过调用[搜索文件](https://open.dingtalk.com/document/development/search-for-files)接口获取的dentryUUid字段值。 |
| sourceResourceType | String | 是 | 源资源的类型：   * **DENTRY**：文件 * **WORKSPACE**：知识库 |
| targetResourceId | String | 是 | 目标位置的id，可以是dentryUUid或rootNodeUUid，通过调用[搜索文件](https://open.dingtalk.com/document/development/search-for-files)接口获取的dentryUUid字段值。 |
| targetResourceType | String | 是 | 目标资源的类型：   * **DENTRY**：文件 * **WORKSPACE**：知识库 |
| targetResourceName | String | 否 | 快捷方式名字，可不传入，不传入默认使用源资源名字。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| openDentryInfo | Object | 快捷方式的信息。 |
| dentryUuid | String | 文件唯一ID。 |
| driveSpaceId | String | 存储层空间ID。 |
| driveDentryId | String | 存储层文件ID。 |
| extension | String | 文件后缀名。 |
| name | String | 节点名称。 |
| url | String | 节点访问链接。 |
| spaceInfo | Object | 节点所在空间信息。 |
| sceneType | String | 资源所属场景。 |
