更新于 2025-09-24复制知识库或钉盘文档。

该接口功能与图中添加副本到功能一直，调用后可将节点复制到其他节点下。

![[development-api-copydoc_O1CN01kGBKTV29IfQaggJB5_!!6000000008045-2-tps-3024-1648.png]]

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | SNS知识库文档写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_2.0%23CopyDoc) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v2.0/doc/dentries/copy HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "param" : {
    "sourceDentryUuid" : "String",
    "targetParentDentryUuid" : "String",
    "targetPreDentryUuid" : "String"
  }
}
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 接口调用凭证，通过调用[获取用户token](https://open.dingtalk.com/document/development/obtain-user-token)接口获取。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| param | Object | 是 | 必选参数。 |
| sourceDentryUuid | String | 是 | 源文件唯一标识，调用[搜索文件](https://open.dingtalk.com/document/development/search-for-files)接口获取的dentryUUid字段值。 |
| targetParentDentryUuid | String | 是 | 目标文件唯一标识，调用[搜索文件](https://open.dingtalk.com/document/development/search-for-files)接口获取的dentryUUid字段值。 |
| targetPreDentryUuid | String | 否 | 目标的上一个文件唯一标识（可选），调用[搜索文件](https://open.dingtalk.com/document/development/search-for-files)接口获取的dentryUUid字段值。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| isAsync | Boolean | 本次操作是否为异步，若为异步则需要通过 taskId 去轮询异步任务的结果。 |
| taskId | String | 任务id，异步时才有该字段。 |
| syncCopyResult | Object | 同步操作的结果，需要注意当 isAsync 为 true 时，该字段仍会返回值。 |
| dentryUuid | String | 文件唯一ID。 |
| driveSpaceId | String | 存储层空间ID。 |
| driveDentryId | String | 存储层文件ID。 |
| extension | String | 文件后缀名。 |
| name | String | 节点名称。 |
| url | String | 节点访问链接。 |
| spaceInfo | Object | 节点所在空间信息。 |
| sceneType | String | 资源所属场景。 |
