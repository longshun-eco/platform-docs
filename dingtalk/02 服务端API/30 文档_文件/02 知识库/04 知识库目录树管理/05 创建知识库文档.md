更新于 2025-09-12调用本接口，在知识库内创建文档或者文件夹。

例如，企业员工小明在测试知识库内，单击**新建**按钮创建文档，与调用本接口实现效果一致，如下图：

![[development-create-team-space-document_O1CN01vO3UkN1fl7GOxXxqs_!!6000000004046-2-tps-2732-1582.png]]

调用成功后，在测试知识库的目录内会新增一个文档，如下图所示。

![[development-create-team-space-document_O1CN012wbIyu27sj5MF6ybj_!!6000000007853-2-tps-2732-1582.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 知识库文档写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_1.0%23CreateWorkspaceDoc) |
| 第三方企业应用 | 支持 | 知识库文档写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=doc_1.0%23CreateWorkspaceDoc) |
| 第三方个人应用 | 暂不支持 | 知识库文档写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/doc/workspaces/{workspaceId}/docs HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "name" : "String",
  "docType" : "String",
  "operatorId" : "String",
  "parentNodeId" : "String",
  "templateId" : "String",
  "templateType" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| workspaceId | String | 是 | 知识库ID，调用[新建知识库](/document/development/new-knowledge-base)接口或者[获取知识库列表](/document/development/get-knowledge-base-list)接口获取的workspaceId字段值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| name | String | 是 | 文档名称。 |
| docType | String | 是 | 文档类型，取值：   * **DOC**：文字 * **WORKBOOK**：表格 * **MIND**：脑图 * **FOLDER**：文件夹 |
| operatorId | String | 是 | 用户的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |
| parentNodeId | String | 否 | 父节点Id，调用[获取节点列表](/document/development/get-node-list)接口获取DentryModel参数值。如果不传，默认在根目录创建。  **说明**  父节点可以是文件夹，也可以是普通的文档。 |
| templateId | String | 否 | 文档模板id。 |
| templateType | String | 否 | 文档模板类型，支持以下值：   * **public\_template**：推荐模板 * **team\_template**：团队模板 * **user\_template**：个人模板 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| workspaceId | String | 知识库ID。 |
| nodeId | String | 知识库文档ID。 |
| docKey | String | 文档docKey。 |
| url | String | 文档URL。 |
| dentryUuid | String | 当前返回的文件Uuid |

## 错误码

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | invalidRequest.workspaceNode.nameConflict | 知识库文档名称冲突 | 知识库文档名称冲突 |
| 400 | invalidRequest.inputArgs.invalid | 方法入参校验失败 | 方法入参校验失败，检查是否有必填参数未填，或者unionId是否合法等 |
| 400 | invalidRequest.workspaceNode.parentNotFound | 父节点不存在 | 父节点不存在 |
| 400 | invalidRequest.template.notFound | 文档模板不存在 | 文档模板不存在 |
| 403 | forbidden.user.notInOrg | 操作用户不在组织内 | 操作用户不在组织内 |
| 403 | forbidden.accessDenied | 用户无操作权限 | 用户无操作权限 |
| 404 | invalidRequest.resource.notFound | 资源找不到 | 资源找不到 |
| 404 | invalidRequest.workspace.deleted | 知识库已被删除 | 知识库已被删除 |
| 500 | internalError | 系统内部错误 | 系统内部错误 |
