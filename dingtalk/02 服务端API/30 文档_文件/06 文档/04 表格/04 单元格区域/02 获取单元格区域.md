更新于 2025-09-12调用本接口，获取单元格属性。目前可以获取单元格内的值、公式、显示值、背景色、字体大小、水平对齐方式、垂直对齐方式。

例如，工作表内的内容如下图所示。

调用本接口，可获取A3:C3的单元格属性信息，能获取到以下信息：

* A3:C3每一个单元格内的显示值，如[[11,4,7]]。
* A3:C3每一个单元格内的公式，如[["=B3+C3","",""]]。
* A3:C3每一个单元格内带格式的显示值，如[["1100%","400%","7"]]。
* A3:C3每一个单元格的背景色信息，如[[{"blue":0,"green":0,"hexString":"#000000","red":0},{"blue":0,"green":0,"hexString":"#000000","red":0},{"blue":0,"green":0,"hexString":"#000000","red":0}]]。
* A3:C3每一个单元格的字体大小，如[[14, 14, 14]]。
* A3:C3每一个单元格的水平对齐方式，如[["right", "center", "center"]]。
* A3:C3每一个单元格的垂直对齐方式，如[["middle", "middle", "middle"]]。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉表格读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_1.0%23GetRange) |
| 第三方企业应用 | 暂不支持 | 钉钉表格读权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 钉钉表格读权限 | 暂不支持 |

## 请求方法

```sql
GET /v1.0/doc/workbooks/{workbookId}/sheets/{sheetId}/ranges/{rangeAddress}?select=String&operatorId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| workbookId | String | 是 | 表格文件 ID ，[知识库 API](/document/development/knowledge-base-overview) 返回的`nodeId(dentryUuid)`即是表格`workbookId`，可通过调用[获取节点](/document/development/get-node)和[创建知识库文档](/document/development/create-team-space-document)接口获取。 |
| sheetId | String | 是 | 工作表ID或名称，可调用[获取所有工作表](/document/development/obtain-all-worksheets)接口获取id或name参数值。 |
| rangeAddress | String | 是 | Range地址。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| select | String | 否 | 筛选要返回的字段，该参数不传则返回所有字段。   * 如`select=values`，该接口只返回values字段。 * 返回多个字段时，使用逗号分隔，例如`select=values,formulas`，该接口只返回values和formulas字段。   **说明**  建议指定此参数以提高性能，并避免后续增加返回值时造成API超时。 |
| operatorId | String | 是 | 操作人的unionId。   * 可通过[通过免登码获取用户信息](/document/development/obtain-the-userid-of-a-user-by-using-the-log-free)接口获取unionid参数值。 * 可调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。   **说明**  若操作人无权限，接口会报错`The operator has no permission`。 |
