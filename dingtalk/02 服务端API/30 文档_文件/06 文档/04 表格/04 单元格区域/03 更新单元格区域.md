更新于 2025-09-12调用本接口，更新单元格信息，包括单元格中的值、背景色、超链接、字体大小、水平对齐方式、垂直对齐方式等。

例如，**测试统计表**信息如下图所示。

![[development-update-cell-properties_O1CN01L0PxDP1CXipND0GMk_!!6000000000091-2-tps-2728-672.png]]

调用本接口，可更新单元格的背景色、值、超链接、字体大小、水平对齐方式、垂直对齐方式信息。接口调用成功后，效果如下图所示。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉表格写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_1.0%23UpdateRange) |
| 第三方企业应用 | 支持 | 钉钉表格写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=doc_1.0%23UpdateRange) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
PUT /v1.0/doc/workbooks/{workbookId}/sheets/{sheetId}/ranges/{rangeAddress}?operatorId=String HTTP/1.1
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "horizontalAlignments" : [ [ "String" ] ],
  "numberFormat" : "String",
  "values" : [ [ "String" ] ],
  "backgroundColors" : [ [ "String" ] ],
  "fontSizes" : [ [ Integer ] ],
  "hyperlinks" : [ [ {
    "type" : "String",
    "link" : "String",
    "text" : "String"
  } ] ],
  "verticalAlignments" : [ [ "String" ] ]
}
```

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |

| sheetId | String | 是 | 工作表ID或名称。   * 企业内部应用，可调用[获取所有工作表](/document/development/obtain-all-worksheets)接口获取id或name参数值。 * 第三方企业应用，可调用[创建工作表](/document/development/create-a-worksheet)接口获取id和name。 |
| rangeAddress | String | 是 | Range地址。 |
