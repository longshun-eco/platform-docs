更新于 2025-09-12调用本接口，在表格文档中创建一个新的工作表。

例如，测试统计表中只有默认的名为**Sheet1**的工作表，如下图所示。

![[development-create-a-worksheet_O1CN01oqNAoJ1WggtX3wO7Y_!!6000000002818-2-tps-2704-1582.png]]

调用本接口，可新增一个名为**统计表1**的工作表。接口调用成功后，效果如下图所示。

![[development-create-a-worksheet_O1CN01UTQffs1xSUErrLmN7_!!6000000006442-2-tps-2718-1582.png]]

**说明**

当指定的工作表名称和已有的工作表名称重复时，将自动重命名为合法值。

![[development-create-a-worksheet_O1CN01mLV1M81n0sCbm0AHS_!!6000000005028-2-tps-1848-194.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉表格写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_1.0%23CreateSheet) |
| 第三方企业应用 | 支持 | 钉钉表格写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=doc_1.0%23CreateSheet) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/doc/workbooks/{workbookId}/sheets?operatorId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "name" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| workbookId | String | 是 | 表格文件 ID ，[知识库 API](/document/development/knowledge-base-overview) 返回的`nodeId(dentryUuid)`即是表格`workbookId`，可通过调用[获取节点](/document/development/get-node)和[创建知识库文档](/document/development/create-team-space-document)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorId | String | 是 | 操作人的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。  **说明**  若操作人无权限，接口会报错`The operator has no permission`。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| name | String | 是 | 工作表的名称。  **说明**  当指定的工作表名称和已有的工作表名称重复时，将自动重命名为合法值。 ![[development-create-a-worksheet_O1CN01mLV1M81n0sCbm0AHS_!!6000000005028-2-tps-1848-194.png]] |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| visibility | String | 工作表可见性，固定值为visible。 |
| name | String | 工作表的名称。 |
| id | String | 工作表的ID。 |
