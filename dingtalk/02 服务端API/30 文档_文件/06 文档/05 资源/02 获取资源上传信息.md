更新于 2025-09-12调用该接口可以查询文档指定资源的上传地址。

资源指文档中的图片/文件等附件。该接口一般配合其它产品的上传附件接口使用。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储文件读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_1.0%23GetResourceUploadInfo) |
| 第三方企业应用 | 暂不支持 | 企业存储文件读权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 企业存储文件读权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/doc/docs/resources/{docId}/uploadInfos/query?operatorId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "size" : Long,
  "mediaType" : "String",
  "resourceName" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| docId | String | 是 | 文档ID (dentryUuid/documentId/workbookId/baseId)，可通过以下两种方式获取：   * 调用[创建知识库文档](/document/development/create-team-space-document)接口获取。 * 调用[获取节点列表](/document/development/get-node-list)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorId | String | 是 | 操作人的unionId，可通过调用[查询用户详情](/document/development/query-user-details)接口或[通过免登码获取用户信息](/document/development/obtain-the-userid-of-a-user-by-using-the-log-free)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| size | Long | 是 | 资源大小，单位：字节。 |
| mediaType | String | 是 | 资源类型，具体值请参考 [MIME Types](https://developer.mozilla.org/zh-CN/docs/Web/HTTP/Guides/MIME_types/Common_types)。 |
| resourceName | String | 是 | 资源名称。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 是否成功。 |
| result | Object | 结果。 |
| uploadUrl | String | 上传url。 |
| resourceId | String | 资源ID。 |
| resourceUrl | String | 资源路径。 |
