更新于 2025-10-29调用本接口，通过Markdown的形式覆写文档。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 文档应用中文档写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_2.0%23DocUpdateContentWithDelegatedPermission) |
| 第三方企业应用 | 支持 | 文档应用中文档写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=doc_2.0%23DocUpdateContentWithDelegatedPermission) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v2.0/doc/me/suites/documents/{docKey}/overwriteContent HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "dataType" : "String",
  "content" : "String"
}
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过 [获取用户token](https://open.dingtalk.com/document/development/obtain-user-token) 接口获取 |

## Path参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| docKey | String | 是 | 文档id（doc\_key or dentryUuid）。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| dataType | String | 否 | 覆盖内容的数据类型。 |
| content | String | 是 | 内容正文。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| data | Map<String, Any> | 返回结果。 |
|  | Any | 返回结果。 |
