更新于 2025-09-12搜索知识库。

### 接口功能介绍

调用本接口，根据操作人unionId和关键词信息，搜索知识库信息。示例如下：

![[development-search-knowledge-base_获取单个知识库信息.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 知识库搜索权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_2.0%23SearchWorkspaces) |
| 第三方企业应用 | 支持 | 知识库搜索权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_2.0%23SearchWorkspaces) |
| 第三方个人应用 | 暂不支持 | 知识库搜索权限 | 暂不支持 |

## 请求方法

```
POST /v2.0/storage/workspaces/search?operatorId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "keyword" : "String",
  "option" : {
    "nextToken" : "String",
    "maxResults" : Integer
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorId | String | 是 | 操作人unionId。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| keyword | String | 是 | 搜索关键词。 |
| option | Object | 否 | 可选参数。 |
| nextToken | String | 否 | 分页游标。  **说明**  首次拉取不用传。 |
| maxResults | Integer | 否 | 分页大小，默认值20。  **说明**  最大值20。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| items | Array | 搜索结果列表，最大size20。 |
| workspaceId | String | 知识库id(spaceUuid)。 |
| name | String | 知识库名称。 |
| url | String | 知识库访问url。 |
| nextToken | String | 分页游标。  **说明**  不为空表示有更多数据。 |
