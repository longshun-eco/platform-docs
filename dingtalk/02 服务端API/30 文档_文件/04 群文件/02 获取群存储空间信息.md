更新于 2025-09-12调用本接口，获取群存储空间信息。

### 接口功能介绍

调用本接口，可获取某个指定群的存储空间信息，包括群存储空间的ID、群存储空间创建的时间和修改时间等。群存储空间如下图所示。

![[development-obtain-group-storage-space-information_O1CN013x1cJs1pJlmswCg09_!!6000000005340-2-tps-2612-1306.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 群文件空间读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=convFile_1.0%23GetSpace) |
| 第三方企业应用 | 暂不支持 | 群文件空间读权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 群文件空间读权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/convFile/conversations/spaces/query?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "openConversationId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作人的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| openConversationId | String | 是 | 会话openConversationId，可调用[创建群](/document/development/create-group-session)接口获取openConversationId参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| space | Object | 会话存储空间信息。 |
| spaceId | String | 会话存储空间ID。 |
| corpId | String | 空间归属企业的corpId。 |
| createTime | String | 群存储空间的创建时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| modifiedTime | String | 群存储空间的修改时间，iso8601格式，例如：2022-07-29T14:55Z。 |
