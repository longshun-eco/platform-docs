更新于 2025-09-12调用本接口，发送文件到指定会话。

### 接口功能介绍

调用本接口，可实现将存储空间内的文件发送到指定群内。接口调用效果如下图所示。

![[development-send-file-to-specified-session_O1CN01aggpcP1QLj5eIG3ZO_!!6000000001960-2-tps-1838-1068.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 群文件发送权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=convFile_1.0%23Send) |
| 第三方企业应用 | 暂不支持 | 群文件发送权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 群文件发送权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/convFile/conversations/files/send?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "spaceId" : "String",
  "dentryId" : "String",
  "openConversationId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作人的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | String | 是 | 空间ID，调用[添加空间](/document/development/add-space)接口获取id参数值。 |
| dentryId | String | 是 | 文件ID，调用[获取文件或文件夹列表](/document/development/obtain-the-file-list-storage)接口获取id参数值。 |
| openConversationId | String | 是 | 目标会话的openConversationId，调用[创建群](/document/development/create-group-session)接口获取openConversationId参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| file | Object | 发送到目标会话的文件信息。 |
| id | String | 文件ID。 |
| conversationId | String | 文件所在的会话ID。 |
| spaceId | String | 文件所在空间ID。 |
| parentId | String | 文件的父目录ID，根目录ID值为0。 |
| type | String | 文件类型。   * **FILE**：文件 * **FOLDER**：文件夹 |
| name | String | 文件名称。 |
| size | Long | 文件大小，单位Byte。 |
| path | String | 文件路径。 |
| version | Long | 文件版本号。 |
| status | String | 文件状态。   * **NORMAL**：正常 * **DELETED**：已删除 * **EXPIRED**：已过期 |
| extension | String | 文件后缀。 |
| creatorId | String | 文件的创建者unionId。 |
| modifierId | String | 文件的修改者unionId。 |
| createTime | String | 文件的创建时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| modifiedTime | String | 文件的修改时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| uuid | String | 标识字段。 |

## 错误码

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | paramError | %s | 参数错误 |
| 400 | paramError.spaceId | %s | 参数错误-spaceId |
| 400 | paramError.dentryId | %s | 参数错误-dentryId |
| 400 | paramError.conversationId | %s | 参数错误-conversationId |
| 400 | folderNotSupport | %s | 不支持发送文件夹 |
| 403 | permissionDenied | %s | 用户缺少待发送文件的拷贝权限或者用户缺少目标会话存储空间添加文件的权限 |
| 404 | conversationNotExist | %s | 目标会话不存在 |
| 500 | systemError | %s | 系统错误 |
| 500 | unknownError | Unknown Error | 未知错误 |
