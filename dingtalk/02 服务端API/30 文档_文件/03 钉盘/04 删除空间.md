更新于 2025-09-12调用本接口删除企业共享空间。

**说明**

暂不支持删除全员文件夹。

调用本接口，删除企业共享空间，具体示例如下图所示。

![[development-delete-a-space_O1CN01eSzePG1pepvocUe8T_!!6000000005386-2-tps-335-656.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉盘应用盘空间写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=drive_1.0%23DeleteSpace) |
| 第三方企业应用 | 支持 | 钉盘应用盘空间写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=drive_1.0%23DeleteSpace) |
| 第三方个人应用 | 暂不支持 | 钉盘应用盘空间写权限 | 暂不支持 |

## 请求方法

```sql
DELETE /v1.0/drive/spaces/{spaceId}?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | String | 是 | 钉盘空间ID，调用[获取空间列表](/document/development/queries-a-space-list)接口获取spaceId参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 用户unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |
