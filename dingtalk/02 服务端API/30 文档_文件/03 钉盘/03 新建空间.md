## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉盘应用盘空间写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=drive_1.0%23AddSpace) |
| 第三方企业应用 | 支持 | 钉盘应用盘空间写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=drive_1.0%23AddSpace) |
| 第三方个人应用 | 暂不支持 | 钉盘应用盘空间写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/drive/spaces HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "name" : "String",
  "unionId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| name | String | 是 | 空间名称（不能为空）。 |
| unionId | String | 是 | 用户unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| spaceId | String | 空间ID。 |
| spaceName | String | 空间名称。 |
| spaceType | String | 空间类型。 |
| quota | Long | 空间总额度。 |
| usedQuota | Long | 空间已使用额度。 |
| permissionMode | String | 授权模式，取值：   * **acl**：acl授权 * **custom**：自定义授权 |
| createTime | String | 创建时间。 |
| modifyTime | String | 修改时间。 |
