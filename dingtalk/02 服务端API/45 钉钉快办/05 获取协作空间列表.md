更新于 2025-10-09调用本接口，可以拉取钉钉快办协作空间列表。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉快办管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=teamSphere_1.0%23SearchProjectsV3) |
| 第三方企业应用 | 支持 | 钉钉快办管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=teamSphere_1.0%23SearchProjectsV3) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/teamSphere/projects?userId=String&projectIds=String&name=String&maxResults=Integer&nextToken=String&sourceId=String&includeTemplate=Boolean HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作者userId。 |
| projectIds | String | 否 | 协作空间ID集合，逗号分隔。 |
| name | String | 否 | 协作空间名字(模糊匹配)。 |
| maxResults | Integer | 否 | 每页返回最大数量。默认10，最大300。 |
| nextToken | String | 否 | 分页标，从上一次请求结果中获取。 |
| sourceId | String | 否 | 协作空间ID。 |
| includeTemplate | Boolean | 否 | 是否包含模板协作空间，默认 false。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array | 协作空间列表。 |
| id | String | 协作空间ID。 |
| name | String | 协作空间名称。 |
| logo | String | 协作空间LOGO。 |
| description | String | 协作空间描述。 |
| organizationId | String | 企业ID。 |
| isTemplate | Boolean | 是模板协作空间。 |
| creatorId | String | 创建人ID。 |
| created | String | 创建时间。 |
| updated | String | 更新时间。 |
| sourceId | String | 来源协作空间 ID。 |
| requestId | String | 请求 ID，请求异常时可提供此 ID，进行问题排查。 |
| nextToken | String | 分页标，供分页使用，下一页token。 |
