更新于 2025-10-09调用本接口，创建一个钉钉快办自由任务（非协作空间的任务）。

### 什么是自由任务

自由任务是在钉钉快办中直接创建，但是不关联任何协作空间的任务。

![[development-api-createorganizationtask_O1CN010Hf2qH1seKcESciFx_!!6000000005791-0-tps-1530-445.jpg]]

### 接口功能介绍

调用本接口，给员工小钉创建一个名为**测试任务**的任务。接口调用成功后，员工小钉在我的任务列表查看到的信息如下图所示，有一个名为**测试任务**的任务。

![[development-api-createorganizationtask_O1CN01mJuwTe1nqdAeG4rWT_!!6000000005141-0-tps-1676-465.jpg]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉快办管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=teamSphere_1.0%23CreateOrganizationTask) |
| 第三方企业应用 | 支持 | 钉钉快办管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=teamSphere_1.0%23CreateOrganizationTask) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/teamSphere/organizations/users/{userId}/tasks HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "content" : "String",
  "note" : "String",
  "involveMembers" : [ "String" ],
  "executorId" : "String",
  "dueDate" : "String",
  "visible" : "String",
  "disableNotification" : Boolean,
  "disableActivity" : Boolean
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作人userId。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| content | String | 是 | 任务标题。 |
| note | String | 否 | 任务备注。 |
| involveMembers | Array of String | 否 | 参与者id。 |
| executorId | String | 否 | 执行者id。 |

| visible | String | 是 | 任务可见性。involves：仅参与者可见。 |

| disableActivity | Boolean | 否 | 是否禁止动态：   * true：禁止 * false：不禁止 |
