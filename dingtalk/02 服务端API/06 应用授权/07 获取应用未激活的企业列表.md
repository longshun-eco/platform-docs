更新于 2026-02-27通过此接口，第三方服务商可以获取尚未激活指定应用的企业（Corp）列表，用于后续的激活引导、数据统计或运营分析。

### **使用场景**

该接口适用于第三方服务商定期检查已授权但未激活应用的企业列表。例如：

* 分析潜在客户使用情况，针对长时间未激活的企业进行定向推广或技术支持；
* 搭配事件订阅机制，监控企业授权后的行为路径，优化产品引导流程；
* 构建自动化运营系统，对未激活企业批量发送提醒消息或触发客服介入。

## 权限

服务端API是以应用维度授权的，在调用接口前，确保对应用已经添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否可调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 否 | — | — |
| 第三方企业应用 | 是 | 无需申请 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.service.get_unactive_corp) |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/service/get_unactive_corp`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| suite\_access\_token | String | 是 | 6ed1bxxxx | 第三方企业应用的suite\_access\_token，可调用[获取第三方企业应用的suiteAccessToken](/document/development/obtains-the-suite-acess-token-of-third-party-enterprise-applications)接口获取凭证。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| app\_id | Number | 否 | 123 | 套件下的微应用ID，可以在[钉钉开发者后台](https://open-dev.dingtalk.com/#/appMgr/provider/eapp/57453/1)查看。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| app\_id | Number | ding123 | 微应用ID。 |
| corp\_list | String[] |  | 授权未激活应用的CorpId列表。 |
| has\_more | Boolean | false | 是否还有更多数据。 |
| errmsg | String | ok | 调用失败时返回的错误信息。 |
| errcode | Number | 0 | 返回码。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/service/get_unactive_corp?suite_access_token=SUITE_ACCESS_TOKEN
```

请求正文

```json
{
        "app_id": 123
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient(
"https://oapi.dingtalk.com/service/get_unactive_corp?suite_access_token=SUITE_ACCESS_TOKEN");
OapiServiceGetUnactiveCorpRequest req = new OapiServiceGetUnactiveCorpRequest();
req.setAppId(123L);
OapiServiceGetUnactiveCorpResponse rsp = client.execute(req);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
    "errcode": 0,
    "corp_list": [
        "ding123",
        "ding456"
    ],
    "errmsg": "ok",
    "has_more": false,
    "app_id": 57453
}
```
