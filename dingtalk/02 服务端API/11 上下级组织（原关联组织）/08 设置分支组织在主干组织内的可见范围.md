更新于 2025-10-21调用本接口设置分支组织在主干组织内的可见范围，也可以设置分支组织在主干组织内可以被哪些部门可见。

### 接口功能介绍

例如，主干组织**测试企业**下有个分支组织为**关联组织，**调用本接口可以实现：

* 设置**关联组织**在主干组织内可以查看到哪些部门。
* 设置主干组织内哪些部门可以查看到**关联组织**。

如图。

![[development-sets-the-visible-range-of-branch-organizations-within-the-group_p494069.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录合作空间写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23UpdateBranchVisibleSettingInCooperate) |
| 第三方企业应用 | 支持 | 通讯录合作空间写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=contact_1.0%23UpdateBranchVisibleSettingInCooperate) |
| 第三方个人应用 | 暂不支持 | 通讯录合作空间写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/contact/cooperateCorps/branchVisibleSettings HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

[ {
  "branchCorpId" : "String",
  "type" : Long,
  "open" : Boolean,
  "visibleBranchCorpIds" : [ "String" ],
  "visibleDeptIds" : [ Long ]
} ]
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。   **说明**  调用本接口，需要使用**主干组织**的访问凭证，不能使用分支组织的访问凭证。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
|  | Array | 是 | 请求参数。 |
| branchCorpId | String | 是 | 伙伴组织的企业corpId，详情参见[CorpId](/document/development/development-basic-concepts#section-bbk-mv0-oxd)。  ![[development-sets-the-visible-range-of-branch-organizations-within-the-group_O1CN01X0RGo41GFBFOwwpZD_!!6000000000592-2-tps-2198-389.png]] |
| type | Long | 是 | 设置可见性类型，取值：   * **0**：在主干组织通讯录隐藏分支组织。 * **1**：仅可见分支组织自己。 |
| open | Boolean | 是 | 是否开启，取值：   * **true**：开启 * **false**：关闭 |
| visibleBranchCorpIds | Array of String | 否 | 设置例外的加入主干组织的分支企业corpId列表，详情参见[CorpId](/document/development/development-basic-concepts#section-bbk-mv0-oxd)。  ![[development-sets-the-visible-range-of-branch-organizations-within-the-group_O1CN01X0RGo41GFBFOwwpZD_!!6000000000592-2-tps-2198-389.png]] |
| visibleDeptIds | Array of Long | 否 | 设置例外的部门deptId列表，可调用[获取部门列表](/document/development/obtain-the-department-list-v2)接口获取dept\_id参数值。 |
