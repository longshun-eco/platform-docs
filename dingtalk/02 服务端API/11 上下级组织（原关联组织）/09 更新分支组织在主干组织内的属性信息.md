更新于 2025-09-10调用本接口更新分支组织在主干组织内的属性信息，包括分支组织的别名，分支组织在主干组织内的通讯录位置等。

**说明**

如果更新分支组织在通讯录的位置是一级目录，调用本接口的挂载部门参数linkDeptId，需要传-1。

例如，**关联组织**是**测试组织**的分支组织，调用本接口可以更新关联组织的别名和在通讯录内的位置，如图。

![[development-updates-the-property-information-of-a-branch-organization-in-a_p494075.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录合作空间写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23UpdateBranchAttributesInCooperate) |
| 第三方企业应用 | 支持 | 通讯录合作空间写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=contact_1.0%23UpdateBranchAttributesInCooperate) |
| 第三方个人应用 | 暂不支持 | 通讯录合作空间写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/contact/cooperateCorps/branchAttributes HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

[ {
  "branchCorpId" : "String",
  "unionRootName" : "String",
  "linkDeptId" : Long
} ]
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。   **说明**  调用本接口，需要使用**主干组织**的访问凭证，不能使用所属组织的访问凭证。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
|  | Array | 是 | 请求参数。 |
| branchCorpId | String | 是 | 分支组织的企业ID，可调用[获取分支组织列表](/document/development/obtains-the-branch-organization-list)接口获取union\_corpid参数值。 |
| unionRootName | String | 是 | 分支组织在主干组织内的别名。 |
| linkDeptId | Long | 是 | 挂载节点部门ID，如果是根部门，需要传-1，主干组织调用[获取部门列表](/document/development/obtain-the-department-list-v2)接口获取dept\_id参数值。 |
