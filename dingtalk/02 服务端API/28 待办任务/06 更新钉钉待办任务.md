更新于 2025-09-12调用本接口，更新钉钉待办任务信息及状态。

### 接口调用说明

调用本接口，更新钉钉待办任务信息及状态，ISV在自身应用待办数据被更新后，需调用该接口将钉钉侧待办数据同步更新。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 待办应用中待办写权限,待办应用中待办读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=todo_1.0%23UpdateTodoTask) |
| 第三方企业应用 | 支持 | 待办应用中待办写权限,待办应用中待办读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=todo_1.0%23UpdateTodoTask) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
PUT /v1.0/todo/users/{unionId}/tasks/{taskId}?operatorId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "subject" : "String",
  "description" : "String",
  "dueTime" : Long,
  "done" : Boolean,
  "executorIds" : [ "String" ],
  "participantIds" : [ "String" ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 当前访问资源所归属用户的unionId，和操作者的unionId保持一致，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |
| taskId | String | 是 | 待办ID。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorId | String | 否 | 当前操作者的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| subject | String | 否 | 待办标题，最大长度1024。 |
| description | String | 否 | 待办描述，最大长度4096。 |
| dueTime | Long | 否 | 截止时间，Unix时间戳，单位毫秒。 |
| done | Boolean | 否 | 完成状态。   * **true**：已完成 * **false**：未完成 |
| executorIds | Array of String | 否 | 执行者的unionId列表，可调用[查询用户详情](/document/development/query-user-details)接口获取，最大数量1000。 |
| participantIds | Array of String | 否 | 参与者的unionId列表，可调用[查询用户详情](/document/development/query-user-details)接口获取，最大数量1000。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 更新结果。true表示成功。 |
