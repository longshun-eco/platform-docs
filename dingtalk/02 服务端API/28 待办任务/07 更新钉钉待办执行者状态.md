更新于 2025-09-12调用本接口，当待办存在多个执行者时，可更新部分执行者的完成状态。

调用本接口，实现多个待办任务执行者的完成状态，具体示例如下图所示。

![[development-update-dingtalk-to-do-status_O1CN01QlxpxI1NCcMT5OpbZ_!!6000000001534-2-tps-1378-584.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 待办应用中待办写权限,待办应用中待办读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=todo_1.0%23UpdateTodoTaskExecutorStatus) |
| 第三方企业应用 | 支持 | 待办应用中待办写权限,待办应用中待办读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=todo_1.0%23UpdateTodoTaskExecutorStatus) |
| 第三方个人应用 | 暂不支持 | 待办应用中待办写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/todo/users/{unionId}/tasks/{taskId}/executorStatus?operatorId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "executorStatusList" : [ {
    "id" : "String",
    "isDone" : Boolean
  } ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 当前访问的资源所归属用户的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |
| taskId | String | 是 | 待办ID。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorId | String | 否 | 当前操作者的用户的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| executorStatusList | Array | 否 | 执行者状态列表，id需传用户的unionId，调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值，最大数量1000。 |
| id | String | 否 | 执行者的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |
| isDone | Boolean | 否 | 执行者完成状态。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 更新结果。true表示成功。 |
