更新于 2025-10-14本文档介绍使用AppLink协议打开创建群功能的相关说明。

## **使用场景**

打开创建钉钉群功能。

快速体验，请点击<https://applink.dingtalk.com/action/creategroup>。

## 版本支持

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **钉钉客户端** | **Android** | **iOS** | **macOS** | **Windows** |
| 版本 | ≥6.5.45 | ≥6.5.45 | ≥6.5.50 | ≥6.5.50 |

## **协议**

```
https://applink.dingtalk.com/action/creategroup
```

## **字段说明**

**说明**

本协议无字段。
