更新于 2026-01-22 本文档说明如何通过AppLink协议唤起钉钉客户端并跳转至“加入团队”页面，适用于需要在外部H5或第三方应用中引导用户快速进入钉钉组织加入流程的场景。

## **使用场景**

适用于第三方应用或H5页面希望引导用户快速进入钉钉“加入团队”界面的场景，例如在推广页中提供“立即加入组织”入口，点击后唤起钉钉并跳转至对应页面。

以下为典型业务使用情境：

* **教育机构邀请教师加入组织**：在线教育平台生成专属推广页，教师点击“立即加入学校组织”按钮后，自动唤起已安装的钉钉客户端，并跳转至“加入团队”页面，简化注册与组织关联流程。
* **企业合作伙伴协作接入**：供应链管理系统为合作企业提供接入入口，合作伙伴员工点击链接后可直接打开钉钉并进入组织邀请页面，提升跨企业协作效率。
* **SaaS服务商客户引导**：第三方OA系统在完成账号开通后，提供“一键加入企业钉钉组织”功能，减少用户手动操作步骤，优化落地体验。

## **协议**

```
https://applink.dingtalk.com/page/findMoreTeam
```

## **字段说明**

* 当前协议不支持携带上下文参数，所有跳转均为标准“加入团队”入口。
* 调用时需确保设备已安装钉钉客户端且版本满足要求。

**注意事项：**

* **权限要求**：无需特殊API权限，但需确保用户具有打开钉钉客户端的权限（如管理员身份非必需）。
* **调用限制**：该协议为通用跳转链接，不可用于深度定制化场景（如自动填写企业编号）。
* **兼容性建议**：对于未安装钉钉的设备，建议增加引导下载提示。
