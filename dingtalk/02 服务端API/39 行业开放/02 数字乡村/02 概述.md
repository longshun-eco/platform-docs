## 什么是数字区县

数字区县是一种解决方案，助力区县政府数字型基础建设，全面提升区县基层组织的治理能力，实现乡村振兴。

![[development-overview-of-digital-districts-and-counties_p384958.png]]

## 如何开通数字区县

查看[数字区县解决方案](https://page.dingtalk.com/wow/dingtalk/act/szxy?spm=a213l2.13146415.4929779444.72.39671d5aAMOhHV)了解如何开通数字区县，可以根据实际需求提交申请。
