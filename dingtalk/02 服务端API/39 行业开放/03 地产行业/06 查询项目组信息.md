更新于 2025-09-23调用本接口，查询指定项目组信息。

例如，开通地产行业的钉钉组织的企业管理后台内，**项目通讯录**内的信息如下图所示。

![[development-query-a-project-group-in-the-specified-park_O1CN01AWh0AJ1PYGHjf0Yhs_!!6000000001852-2-tps-2776-890.png]]

调用本接口可获取**测试项目组**的信息，包括测试项目组的名称和测试项目组的扩展信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 行业化园区管理读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=industry_1.0%23CampusGetCampusGroup) |
| 第三方企业应用 | 支持 | 行业化园区管理读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=industry_1.0%23CampusGetCampusGroup) |
| 第三方个人应用 | 暂不支持 | 行业化园区管理读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/industry/campuses/projects/groupInfos?groupId=Long HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| groupId | Long | 是 | 项目组ID，可调用[创建项目组](/document/development/create-a-project-group)接口获取groupId参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| projectGroupName | String | 项目组的名称。 |
| extend | String | 项目组的扩展信息。 |
