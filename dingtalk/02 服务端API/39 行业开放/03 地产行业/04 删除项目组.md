更新于 2025-12-05调用本接口，删除指定项目组。

例如，开通地产行业的钉钉组织的企业管理后台内，**项目通讯录**内的信息如下图所示。

![[development-delete-the-project-group-team_O1CN0150O9JU1eWAi5lWqpe_!!6000000003878-2-tps-2768-1038.png]]

调用本接口可删除**测试项目组1**，接口调用成功后效果如下图所示。

![[development-delete-the-project-group-team_O1CN012HwcdZ25Un92l5Y3p_!!6000000007530-2-tps-2770-970.png]]

**说明**

如果项目组内存在项目或者员工，不支持删除操作。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 行业化园区管理写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=industry_1.0%23CampusDeleteCampusGroup) |
| 第三方企业应用 | 支持 | 行业化园区管理写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=industry_1.0%23CampusDeleteCampusGroup) |
| 第三方个人应用 | 暂不支持 | 行业化园区管理写权限 | 暂不支持 |

## 请求方法

```sql
DELETE /v1.0/industry/campuses/projects/groups?campusProjectGroupId=Long HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| campusProjectGroupId | Long | 是 | 项目组ID，可调用[创建项目组](/document/development/create-a-project-group)接口获取groupId参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 是否删除成功。   * **true**：成功 * **false**：失败 |
