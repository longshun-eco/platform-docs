更新于 2025-09-23调用本接口，查询指定园区项目的信息。

例如，开通地产行业的钉钉组织的企业管理后台内，**项目通讯录**内的信息如下图所示。

![[development-query-a-project-in-a-specified-campus_O1CN01Ogl1Ey1YY6gw2C8e3_!!6000000003070-2-tps-2772-972.png]]

调用本接口可获取**测试项目**的信息，包括测试项目的名称、所在地信息、容纳的人数等。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 行业化园区管理读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=industry_1.0%23CampusGetCampus) |
| 第三方企业应用 | 支持 | 行业化园区管理读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=industry_1.0%23CampusGetCampus) |
| 第三方个人应用 | 暂不支持 | 行业化园区管理读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/industry/campuses/projectInfos?campusDeptId=Long HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| campusDeptId | Long | 是 | 园区项目的部门ID，可调用[创建园区项目](/document/development/create-a-campus-project)接口获取campusDeptId参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| campusName | String | 园区项目的名称。 |
| campusCorpId | String | 园区项目的组织corpId。 |
| campusDeptId | Long | 园区项目的部门ID。 |
| belongProjectGroupId | String | 所在的项目组ID。 |
| telephone | String | 电话。 |
| description | String | 园区项目的描述。 |
| area | double | 园区项目的面积。 |
| country | String | 园区项目所在国家。 |
| provId | Integer | 园区项目所在的省行政编码。 |
| cityId | Integer | 园区项目所在的市行政编码。 |
| countyId | Integer | 园区项目所在的区行政编码。 |
| address | String | 园区项目的详细地址。 |
| location | String | 园区项目所在的经纬度。 |
| capacity | String | 园区项目容纳的人数。 |
| orderStartTime | Long | 园区项目的订购时间。 |
| orderEndTime | Long | 园区项目的过期时间。 |
| orderInfo | String | 园区项目的购买信息。 |
| extend | String | 园区项目的扩展属性。 |
