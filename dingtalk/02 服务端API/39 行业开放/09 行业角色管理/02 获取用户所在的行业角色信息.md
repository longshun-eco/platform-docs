更新于 2025-09-23根据用户userId，获取员工所在的角色信息，如角色名称、角色编码和角色ID。

### 接口功能介绍

例如， 企业员工小钉的角色是**主管理员**、**品质部员工**和**自定义角色**，如下图所示。

调用本接口，可根据员工小钉的userId，获取小钉在企业的角色信息如下：

**说明**

角色类型不同，接口获取的信息不同。

* 角色是主管理员时，属于管理角色，可获取角色名称和角色编码。
* 角色是品质部员工时，属于预置角色，可获取角色名称、角色编码和角色ID。
* 角色是自定义角色时，属于企业自定义角色，可获取角色名称和角色ID。

![[development-obtain-the-industry-role-information-of-the-user_O1CN01JajxAb1SK0nUM8hz7_!!6000000002227-2-tps-2850-1188.png]]

### 角色类型说明

* **管理角色**：包含创建者、主管理员和子管理员。
* **自定义角色**：通过**创建角色**按钮手动添加的角色都属于自定义角色。

  ![[development-obtain-the-industry-role-information-of-the-user_O1CN01PhAGlR1JAY4XO19CJ_!!6000000000988-2-tps-2802-850.png]]
* **预置角色**：通讯录内自带的角色，**不同行业通讯录内的预置角色不同**。
  如下图所示为物业行业预置角色。

  ![[development-obtain-the-industry-role-information-of-the-user_O1CN01o9pcIa1i8amBtLeb0_!!6000000004368-2-tps-2700-1338.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 数字区县居民通讯录读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=resident_1.0%23ListUserIndustryRoles) |
| 第三方企业应用 | 支持 | 数字区县居民通讯录读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=resident_1.0%23ListUserIndustryRoles) |
| 第三方个人应用 | 暂不支持 | 数字区县居民通讯录读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/resident/users/industryRoles?userId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 用户userId，可通过调用[获取部门用户userid列表](/document/development/query-the-list-of-department-userids)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| roleList | Array | 角色列表。 |
| roleId | Long | 角色ID。  **说明**  以下管理角色没有角色ID，即该参数为空。   * super-admin：创建者 * main-admin：主管理员 * sub-admin：子管理员 |
| roleName | String | 角色名称。 |
| tagCode | String | 角色编码。  **说明**  如果是自定义角色，该参数值为空。 |
