更新于 2025-09-23本文主要介绍如何申请新教育通用接口的接口调用权限。

## 名词解释

* 元数据

  钉钉定义的教育行业标准主数据，提供了一个行业标准，供合作伙伴参考。
* 实例数据

  合作伙伴基于钉钉定义的元数据，自己做个性化定义，沉淀的数据实例。

## 权限申请

在调用教育接口前，除获取企业授权的access\_token外，还需要申请接口权限。

1. 登录[开发者后台](https://open-dev.dingtalk.com/#/index)。
2. 在企业应用类别中找到已创建的应用，单击目标应用打开详情页。
3. 在**权限管理**页面，搜索“教育”，单击**新教育**。

   ![[development-overview-comment_p525243.png]]
4. 选择教育元数据读/写权限包，然后单击**申请权限**。

   ![[development-overview-comment_p525244.png]]
5. 在申请特殊权限时，需尽可能详细地描述权限接口及使用场景。

   **说明**

   如果接口申请被拒绝，可将鼠标移动至**审核未通过**处查看原因。

   ![[development-overview-comment_p525245.png]]
