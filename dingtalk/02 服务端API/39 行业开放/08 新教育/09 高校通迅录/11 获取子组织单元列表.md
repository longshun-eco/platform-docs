更新于 2025-09-23调用本接口，获取组织单元下的所有直属子组织单元列表。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉教育高校通讯录读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=edu_1.0%23ListCollegeContactSubDepts) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/edu/collegeContact/subDepts?deptId=Long&language=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| deptId | Long | 是 | 父组织单元ID，根组织单元ID为1,只支持查询下一级子组织单元，不支持查询多级子组织单元。 |
| language | String | 否 | 通讯录语言（默认zh\_CN）。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 是否成功。 |
| result | Array | 子组织单元列表。 |
| deptId | Long | 组织单元ID。 |
| name | String | 组织单元名称。 |
| struId | Long | 所属架构部门ID。 |
| parentId | Long | 父组织单元ID。 |
| sourceIdentifier | String | 基础通讯录部门标识。 |
| deptType | String | 高校组织单元类型。 |
| deptCode | String | 高校组织单元编码，由开发者确定，高校组织内唯一标识的组织单元编码。 |
| createDeptGroup | Boolean | 是否同步创建一个关联此组织单元的群, true表示是, false表示不是。 |
| autoAddUser | Boolean | 当群已经创建后，是否有新人加入组织单元会自动加入该群。 |
| tags | String | 家校通讯录部门类型，包括campus,period,grade,class；如果不是家校通讯录则忽略。 |
| fromUnionOrg | Boolean | 组织单元是否来自关联组织。 |
| extension | String | 扩展字段。 |
