更新于 2025-09-23调用本接口，根据组织单元ID获取指定组织单元（即部门）详情。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉教育高校通讯录读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=edu_1.0%23GetCollegeContactDeptDetail) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/edu/collegeContact/depts?deptId=Long&language=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| deptId | Long | 是 | 组织单元ID，根组织单元ID为1。 |
| language | String | 否 | 通讯录语言（默认zh\_CN）。 |
