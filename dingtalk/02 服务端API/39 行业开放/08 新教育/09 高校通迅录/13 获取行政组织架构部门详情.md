更新于 2025-09-23通过本接口，获取行政组织架构部门详情。

**说明**

行政组织架构及其下属的学生，教职工部门，是高校通讯录中的固定部门，请勿删除。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉教育高校通讯录读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=edu_1.0%23GetCollegeContactStandardStruDeptDetail) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/edu/collegeContact/depts/structures/standards?language=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| language | String | 否 | 通讯录语言，默认zh\_CN。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 是否成功。 |
| result | Object | 返回结果。 |
| struId | Long | 行政组织架构部门ID。 |
| teacherDeptId | Long | 教职工节点组织单元ID。 |
| studentDeptId | Long | 学生节点组织单元ID。 |
