更新于 2025-09-23调用本接口，获取授课回放链接。

调用本接口，可获取课程回放链接，用于进行在线课堂授课的内容回放。

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否可调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 否 | — | — |
| 第三方企业应用 | 是 | 钉钉教育在线课堂数据读权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.edu.course.replay) |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/edu/course/replay`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | Be3xxxx | 调用该接口的应用凭证，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| course\_code | String | 是 | nRFRa5001 | 需要回放的课程编码，调用[创建课程](/document/development/create-course)接口获取course\_code参数值。 |
| op\_user\_id | String | 是 | manager7078 | 操作用户的userId。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| result | ReplayCourseResponse |  | 返回结果。 |
| replayable | Boolean | true | 是否可回放。   * **true**：可以回放 * **false**：不能回放 |
| replay\_urls | String[] | ["https://h5.dingtalk.com/live/video\_lesson.htm?feedId=xxxx&mcnId=xxxx&feedProperty=1&itemId=xxxx&dd\_nav\_bgcolor=FF2C2D2F#/live"] | 回放url列表。 |
| success | Boolean | true | 请求是否成功。   * **true**：成功 * **false**：失败 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/edu/course/replay?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "op_user_id":"manager7078",
        "course_code":"nRFRa5001"
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/edu/course/replay");
OapiEduCourseReplayRequest req = new OapiEduCourseReplayRequest();
req.setCourseCode("nRFRa5001");
req.setOpUserId("manager7078");
OapiEduCourseReplayResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
    "result": {
        "replayable": true,
        "replay_urls": [
            "https://h5.dingtalk.com/live/video_lesson.htm?feedId=xxxx&mcnId=xxxx&feedProperty=1&itemId=xxxxⅆ_nav_bgcolor=FF2C2D2F#/live"
        ]
    },
    "errcode": 0,
    "success": true,
    "errmsg": "ok"
}
```
