更新于 2025-09-23调用本接口，移除课程参与方。

调用本接口，可移除课程参与方，参与方角色包括学生、监护人和老师。

**重要**

授课老师只支持通过[创建课程](/document/development/create-course)和[修改课程](/document/development/modify-course)接口，进行添加和修改。

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 否 | — | — |
| 第三方企业应用 | 是 | 钉钉教育在线课堂数据写权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.edu.course.participant.remove) |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/edu/course/participant/remove`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | Be3xxxx | 调用该接口的应用凭证，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| op\_userid | String | 是 | manager1 | 当前用户的userId。 |
| participant\_corpid | String | 是 | ding4220d8e5128d0edd | 参与方的组织cropId。 ![[development-remove-course-participants_p526067.png]]  **重要**  **必须和当前组织相同或者存在关联关系。**  第三方企业应用请参考[关联关系](/document/development/associated-organizations-overview)。 |
| course\_code | String | 是 | GJKI49001 | 课程唯一编码，调用[创建课程](/document/development/create-course)接口获取course\_code参数值。 |
| participant\_type | Number | 是 | 1 | 参与方类型。   * **1**：用户 * **2**：部门，对应家校通讯录中的班级、年级 * **3**：组织 |
| participant\_id | String | 是 | user01 | 参与方ID。   * participant\_type=1时，participant\_id为用户的userid * participant\_type=2时，participant\_id为部门ID * participant\_type=3时，participant\_id为组织的corpid |
| role | String | 是 | student | 参与方角色。   * **student**：学生 * **guardian**: 监护人 * **teacher**：老师   **重要**  授课老师只支持通过[创建课程](/document/development/create-course)和[修改课程](/document/development/modify-course)接口进行添加和修改。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/edu/course/participant/remove?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "participant_type":1,
        "role":"student",
        "course_code":"GJKI49001",
        "participant_corpid":"ding4220d8e5128d0edd",
        "participant_id":"user01",
        "op_userid":"manager1"
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/edu/course/participant/remove");
OapiEduCourseParticipantRemoveRequest req = new OapiEduCourseParticipantRemoveRequest();
req.setOpUserid("manager1");
req.setParticipantCorpid("ding4220d8e5128d0edd");
req.setCourseCode("GJKI49001");
req.setParticipantId("user01");
req.setRole("student");
req.setParticipantType(1L);
OapiEduCourseParticipantRemoveResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
        "result":true,
        "errcode":0,
        "success":true,
        "errmsg":"ok",
        "request_id": "2zw2h7s074d1"
}
```
