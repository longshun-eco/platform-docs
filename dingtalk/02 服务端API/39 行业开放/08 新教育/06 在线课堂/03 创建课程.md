更新于 2025-09-23调用本接口，创建课程。

调用本接口，可创建课程。接口调用成功后，获取课程唯一编码。

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 否 | — | — |
| 第三方企业应用 | 是 | 钉钉教育在线课堂数据写权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.edu.course.create) |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/edu/course/create`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | bE3xxxx | 调用该接口的应用凭证，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| op\_userid | String | 是 | manager1 | 当前用户的userId。 |
| teacher\_corpid | String | 是 | ding4220xxxx | 老师的组织的corpId。 ![[development-create-course_p526067.png]] |
| teacher\_userid | String | 是 | teacher1 | 老师的userId。 |
| introduce | String | 是 | 数字管理师 | 课程介绍。 |
| biz\_key | String | 是 | uk\_1 | 业务唯一键，用于保证课程的唯一性，防止重复创建。 |
| name | String | 是 | 数字化管理 | 课程名称。 |
| start\_time | Number | 否 | 1596506100000 | 课程的开始时间，Unix毫秒时间戳。 |
| end\_time | Number | 否 | 1596506200000 | 课程的结束时间，Unix毫秒时间戳。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| course\_code | String | FBnCv218004 | 课程唯一编码。 |
| success | Boolean | true | 请求是否成功。   * **true**：成功 * **false**：失败 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |
| request\_id | String | f033gckmia | 请求ID。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/edu/course/create?access_token=ACCESS_TOKEN
```

请求正文

```json
{
 "biz_key":"uk_1",
 "introduce":"数字化管理师",
 "name":"数字化管理",
 "teacher_userid":"manager1102",
 "teacher_corpid":"ding99b58ff12396b07d24f2f5cc6abecb85",
 "op_userid":"manager1102",
 "start_time":1596506100000,
 "end_time":1596506200000
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/edu/course/create");
OapiEduCourseCreateRequest req = new OapiEduCourseCreateRequest();
req.setOpUserid("manager1");
req.setTeacherCorpid("ding4220d8e5128d0edd");
req.setTeacherUserid("teacher1");
req.setIntroduce("我是课程介绍");
req.setBizKey("uk_1");
req.setName("我是课程名称");
req.setStartTime(1596506100000L);
req.setEndTime(1596506200000L);
OapiEduCourseCreateResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
    "course_code": "FBnCv218004",
    "errcode": 0,
    "errmsg": "ok",
    "success": true,
    "request_id": "f033gf5ckmia"
}
```
