更新于 2025-12-08调用本接口，根据班级ID和学生ID获取学生信息，接口会校验学生或者家长是否已授权。

调用本接口，可获取学生信息，包括学号、姓名、家长姓名等。

![[development-obtain-student-information_p526870.png]]

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方企业应用 | 是 | 钉钉教育学生信息读权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.edu.class.studentinfo.get) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/edu/class/studentinfo/get`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | Be3xxxx | 调用该API的应用凭证，通过[服务商获取第三方应用授权企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| class\_id | Number | 是 | 26347 | 班级ID，调用[获取部门列表](/document/development/obtains-the-department-node-list)接口获取dept\_type为class时的dept\_id参数值。 |
| app\_id | Number | 是 | 13708001 | 应用ID，可在[开发者后台](https://open-dev.dingtalk.com/#/)的**应用信息**页面查看。  ![[development-obtain-student-information_p768234.png]] |
| userid | String | 是 | student1 | 学生的userId。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| result | OpenStudentSelectDto |  | 返回结果。 |
| userid | String | student1 | 学生的userId。 |
| student\_num | String | web001 | 学生的学号。 |
| name | String | 杨xx | 学生姓名。 |
| class\_id | Number | 26347 | 班级ID。 |
| avatar | String | @mediaId | 学生头像ID。 |
| guardians | OpenPatriarchSelectDto[] |  | 家长列表。 |
| userid | String | user01 | 家长ID。 |

| relation\_name | String | 爸爸 | 关系名称。 |
| name | String | 杨xx | 家长姓名。 |

| avatar | String | @asdf12zcv | 家长头像ID。 |
| success | Boolean | true | 请求是否成功。   * **true**：成功 * **false**：失败 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/edu/class/studentinfo/get?access_token=ACCESS_TOKEN
```

请求正文

```json
{
    "class_id": 26347,
    "app_id": 13708001,
    "userid": "student1"
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/edu/class/studentinfo/get");
OapiEduClassStudentinfoGetRequest req = new OapiEduClassStudentinfoGetRequest();
req.setAppId(13708001L);
req.setClassId(26347L);
req.setUserid("student1");
OapiEduClassStudentinfoGetResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
    "errcode": 0,
    "errmsg":"ok",
    "result": {
        "userid": "student1",
        "student_num": "web001",
        "name": "杨xx",
        "class_id": 26347,
        "avatar": "@mediaId",
        "guardians": {
            "userid": "user01",
            "relation": "F",
            "relation_name": "爸爸",
            "name": "杨xx",
            "is_active": true,
            "avatar": "@asdf12zcv"
        }
    },
    "success": true
}
```
