## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉教育家校通讯录写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=edu_1.0%23MoveStudent) |
| 第三方企业应用 | 支持 | 钉钉教育家校通讯录写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=edu_1.0%23MoveStudent) |
| 第三方个人应用 | 暂不支持 | 钉钉教育家校通讯录写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/edu/students/move HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "operator" : "String",
  "userId" : "String",
  "originClassId" : Long,
  "targetClassId" : Long
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operator | String | 是 | 操作者的userId，可调用[通过免登码获取用户信息](/document/development/obtain-the-userid-of-a-user-by-using-the-log-free)接口获取userid参数值。 |
| userId | String | 是 | 学生的userId，可调用[获取人员列表](/document/development/obtains-a-list-of-home-school-user-identities)接口获取userid参数值。 |
| originClassId | Long | 是 | 原班级ID，可调用[获取部门列表](/document/development/obtains-the-department-node-list)接口获取dept\_type为class时的dept\_id参数值。 |
| targetClassId | Long | 是 | 目标班级ID，可调用[获取部门列表](/document/development/obtains-the-department-node-list)接口获取dept\_type为class时的dept\_id参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 调班是否成功，true表示调班成功。 |
