更新于 2025-09-23调用本接口，可在指定的班级下更新学生信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉教育家校通讯录写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=edu_1.0%23UpdateStudent) |
| 第三方企业应用 | 支持 | 钉钉教育家校通讯录写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=edu_1.0%23UpdateStudent) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
PUT /v1.0/edu/students/infos HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "userId" : "String",
  "operator" : "String",
  "classId" : Long,
  "name" : "String",
  "bizId" : "String",
  "studentNo" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 学生ID，可调用[获取人员列表](/document/development/obtains-a-list-of-home-school-user-identities)接口获取userid参数值。 |
| operator | String | 是 | 钉钉企业管理员的userId。 |
| classId | Long | 是 | 班级ID，可调用[获取部门列表](/document/development/obtains-the-department-node-list)接口获取dept\_type为class时的dept\_id参数值。 |
| name | String | 是 | 学生姓名。 |
| bizId | String | 是 | 业务的唯一ID，自定义值，每次调用保持唯一。 |
| studentNo | String | 是 | 学生学号，可调用[获取人员列表](/document/development/obtains-a-list-of-home-school-user-identities)接口获取student\_no参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 接口调用是否成功。 |
