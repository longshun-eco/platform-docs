更新于 2025-09-23调用本接口，可在指定的班级下更新班级信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉教育家校通讯录写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=edu_1.0%23UpdateClass) |
| 第三方企业应用 | 支持 | 钉钉教育家校通讯录写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=edu_1.0%23UpdateClass) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
PUT /v1.0/edu/classes/infos HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "deptId" : Long,
  "gradeLevel" : Integer,
  "operator" : "String",
  "openClass" : {
    "nick" : "String",
    "onlyUseNick" : "String",
    "classLevel" : Integer
  },
  "superId" : Long
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| deptId | Long | 是 | 班级ID。 |
| gradeLevel | Integer | 是 | 年级Level。 |
| operator | String | 是 | 钉钉企业管理员userId，即有家校通讯录管理范围的管理员userId。 |
| openClass | Object | 是 | 班级信息。 |
| nick | String | 是 | 班级昵称。 |
| onlyUseNick | String | 是 | 是否只展现nick。 |
| classLevel | Integer | 是 | 每个年级下班级级数，1班为1，2班为2。 |
| superId | Long | 是 | 年级ID。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 接口调用是否成功。 |
| result | Object | 调用结果。 |
| deptId | Long | 班级ID。 |
