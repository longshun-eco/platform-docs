更新于 2025-09-23在调用家校通讯录接口前，你需要先申请该接口的调用权限。本文主要介绍如何申请家校通讯录2.0接口调用权限和完整的数据链路展示。

## 支持类型

家校通讯录2.0接口，支持“**基础教育通讯录**”和“**自定义家校通讯录**”两种类型的通讯录。

![[development-call-instructions-home-school-address-book-2-0_p148961.png]]

## 权限申请

1. 登录[开发者后台](https://open-dev.dingtalk.com/#/index)，单击目标应用打开详情页。
2. 单击**权限管理**，然后选择**新教育**。

   ![[development-call-instructions-home-school-address-book-2-0_p526329.png]]
3. 添加家校通讯录相关接口权限，然后单击**申请权限**。

   ![[development-call-instructions-home-school-address-book-2-0_p526331.png]]

## 家校通讯录回调

回调是开发者在钉钉开放平台注册一个HTTP接口，并订阅相关的事件，当家校通讯录事件发生变更时，钉钉会主动调用开发者注册的HTTP接口，推送对应的事件信息。

## 部门完整示例

当前一个测试组织，其下包含基础教育通讯录和自定义通讯录两种类型。以下通过获取部门列表，完整的展示整条链路的数据。

   顶层节点校区ID：4240004

   1. 其中一个校区下的所有学段列表：

      ```json

        "result": {
          "details": [
            {

              "nick": "",

        "errcode": 0,
        "request_id":"63h2j929f9oj"

   2. 其中一个学段下的所有年级列表

      ```
