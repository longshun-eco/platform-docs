更新于 2025-09-23调用本接口，根据userId和业务code，判断用户是否是认证组织的语文老师接口。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方企业应用 | 支持 | 成员信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=edu_1.0%23IsYuwenCertifiedTeacher) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/edu/paas/certifiedTeachers/chineseTeachers/check HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "userId" : "String",
  "bizCode" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该API的应用凭证，通过[服务商获取第三方应用授权企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 可通过根据手机号获取userid接口获取userid；长度限制1～50个字符。 |
| bizCode | String | 是 | 三方ISV接入的业务编码，长度限制1～50个字符。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 判断接口调用是否正常返回。 |
| result | Boolean | 是否为认证组织的语文老师。 |
