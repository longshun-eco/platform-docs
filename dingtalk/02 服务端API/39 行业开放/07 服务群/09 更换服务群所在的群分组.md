更新于 2025-09-23调用本接口，更换服务群所在的群分组。

调用本接口可更换服务群所在的群分组。接口调用效果与下图操作实现效果一致。

![[development-modify-a-service-group_O1CN01jChBv61UVa4O9y5Kk_!!6000000002523-2-tps-2176-738.png]]

**说明**

内部群只能更换到内部服务群组，外部群只能更换到外部服务群组。

例如，服务群A当前所在分组为**内部服务群组**，更换服务群A所在的分组时，只能更换到另一个**内部服务群组**内。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 场景服务群读写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=serviceGroup_1.0%23UpdateGroupSet) |
| 第三方企业应用 | 暂不支持 | 场景服务群读写权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 场景服务群读写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/serviceGroup/groups/configurations HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "openTeamId" : "String",
  "openConversationId" : "String",
  "openGroupSetId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| openTeamId | String | 否 | 开放团队ID。  ![[development-modify-a-service-group_O1CN01AtncAQ1Q2wqXNGPxX_!!6000000001919-2-tps-2336-1280.png]] |
| openConversationId | String | 否 | 开放群ID，可调用[创建场景服务群](/document/development/create-a-scenario-service-group)接口获取openConversationId参数值。 |
| openGroupSetId | String | 否 | 开放群组ID。  ![[development-modify-a-service-group_O1CN01AtncAQ1Q2wqXNGPxX_!!6000000001919-2-tps-2336-1280.png]] |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 更新是否成功，true表示更新成功。 |
