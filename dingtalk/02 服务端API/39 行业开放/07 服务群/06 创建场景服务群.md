## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 场景服务群读写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=serviceGroup_1.0%23CreateGroup) |
| 第三方企业应用 | 暂不支持 | 场景服务群读写权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 场景服务群读写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/serviceGroup/groups HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "groupBizId" : "String",
  "openTeamId" : "String",
  "openGroupSetId" : "String",
  "groupName" : "String",
  "ownerStaffId" : "String",
  "memberStaffIds" : [ "String" ],
  "groupTagNames" : [ "String" ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| groupBizId | String | 否 | 业务关联ID，自定义参数值。 |
| openTeamId | String | 是 | 开放团队ID。 ![[development-create-a-scenario-service-group_O1CN017qV2Yi1NQoZjuGx7D_!!6000000001565-2-tps-956-254.png]] |
| openGroupSetId | String | 是 | 开放群组ID。 ![[development-create-a-scenario-service-group_O1CN017qV2Yi1NQoZjuGx7D_!!6000000001565-2-tps-956-254.png]] |
| groupName | String | 是 | 群名称。 |
| ownerStaffId | String | 是 | 群主员工userid。 |
| memberStaffIds | Array of String | 否 | 群成员员工ID列表，最大值20。 |
| groupTagNames | Array of String | 否 | 群标签。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| openConversationId | String | 开放群ID。 |
| groupUrl | String | 加群的url。 |
