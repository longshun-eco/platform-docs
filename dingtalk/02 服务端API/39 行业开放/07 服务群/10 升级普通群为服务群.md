更新于 2025-09-23调用本接口将企业内部群、普通群，升级为智能服务群中的服务群。

接口调用时如果不指定团队id和目标群组id，则必须指定服务群模板id，此时群会升级到企业默认团队下、该模板对应的默认群分组中

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 场景服务群读写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=serviceGroup_1.0%23UpgradeNormalGroup) |
| 第三方企业应用 | 暂不支持 | 场景服务群读写权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 场景服务群读写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/serviceGroup/normalGroups/upgrade HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "openGroupSetId" : "String",
  "templateId" : "String",
  "openConversationId" : "String",
  "openTeamId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| openGroupSetId | String | 否 | 升级的目标群组ID。 |
| templateId | String | 否 | 升级的目标群模板ID。 |
| openConversationId | String | 是 | 群ID。 |
| openTeamId | String | 否 | 升级的目标团队ID。 |
