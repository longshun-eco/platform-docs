更新于 2025-09-23调用本接口，分页查询已经注册的设备信息，可以通过设备类型、分组标识和设备编号进行查询。

### 接口功能介绍

**说明**

本接口可以通过设备类型、分组标识或设备编号进行查询。

以通过**设备编号**查询为例，企业设备上钉应用内，有个设备的设备编号是**device\_1234**，如下图所示。

![[development-query-information-about-a-registered-device_O1CN01rhoVgG1GXxTLau1sW_!!6000000000633-2-tps-2342-1414.png]]

调用本接口，根据设备编号**device\_1234**，可获取设备的详情信息。

接口调用成功后，设备编号为**device\_1234**的设备信息，如下图所示，设备名称为**测试设备1**，设备型号为**测试类型**，设备组为**车间**。

![[development-query-information-about-a-registered-device_O1CN017hqUdb1IXA0vK2nDa_!!6000000000902-2-tps-1986-1362.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 制造业设备信息写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=devicemng_1.0%23ListActivateDevices) |
| 第三方企业应用 | 支持 | 制造业设备信息写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=devicemng_1.0%23ListActivateDevices) |
| 第三方个人应用 | 暂不支持 | 制造业设备信息写权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/devicemng/customers/devices/activations/infos?deviceTypeId=String&pageNumber=Integer&groupId=String&pageSize=Integer&deviceCode=String&deviceCategory=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| deviceTypeId | String | 否 | 设备型号。  **说明**  该参数需线下提供，请通过[技术支持-在线答疑](/document/services/ngliko)咨询。 |
| pageNumber | Integer | 否 | 当前页码，从1开始。 |
| groupId | String | 否 | 分组标识。  **说明**  该参数需线下提供，请通过[技术支持-在线答疑](/document/services/ngliko)咨询。 |
| pageSize | Integer | 否 | 每页大小，最大值50。 |
| deviceCode | String | 否 | 设备编号，可调用查询已经注册的设备信息接口获取。 |
| deviceCategory | Integer | 否 | 设备分类。   * 0：设备 * 1：助手 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| totalCount | Long | 返回总数。 |
| success | Boolean | 接口调用是否成功。   * **true**：成功 * **false**：失败 |
| result | Array | 返回的设备列表。 |
| bizExt | String | 拓展信息。  **说明**  该字段暂时返回为空。 |
| deviceCallbackUrl | String | 设备回调链接。 |
| deviceCode | String | 设备编号。 |
| deviceDetailUrl | String | 设备详情链接。 |
| deviceName | String | 设备名称。 |
| groupUuid | String | 设备分组标识。 |
| icon | String | 设备标题。  **说明**  该字段暂无使用场景。 |
| introduction | String | 设备的简介。 |
| typeUuid | String | 设备型号。 |
| uuid | String | 钉钉侧设备标识。 |
| deviceCategory | Integer | 设备分类。   * 0：设备 * 1：助手 |
