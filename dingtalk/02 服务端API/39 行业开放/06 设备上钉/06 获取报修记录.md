更新于 2025-09-23调用该接口，获取设备上钉中的历史报修记录。

调用本接口，获取设备上钉应用内的保修记录信息，如下图所示。

![[development-obtain-the-repair-report-record_O1CN01PNPZ2R1wwsvvu8CCc_!!6000000006373-2-tps-2472-946.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 制造业设备信息写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=devicemng_1.0%23ListMaintainInfo) |
| 第三方企业应用 | 支持 | 制造业设备信息写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=devicemng_1.0%23ListMaintainInfo) |
| 第三方个人应用 | 暂不支持 | 制造业设备信息写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/devicemng/customers/devices/maintainInfos/query HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "pageNumber" : Integer,
  "pageSize" : Integer,
  "deviceUuid" : [ "String" ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| pageNumber | Integer | 否 | 页码，从1开始。 |
| pageSize | Integer | 否 | 页面大小，最大值20。 |
| deviceUuid | Array of String | 否 | 设备uuIi列表，调用[查询已经注册的设备信息](/document/development/query-information-about-a-registered-device)接口获取的uuid，最大值10。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| totalCount | Long | 总共的数量。 |
| success | Boolean | 是否成功，true表示成功。 |
| result | Array | 返回的结果列表。 |
| gmtCreate | String | 报修时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| deviceCode | String | 报修设备码。 |
| deviceName | String | 设备名称。 |
| remark | String | 报修单内填写的异常描述。 |
| maintenanceStaff | Array of String | 报修人姓名列表。 |
| processState | Integer | 处理结果。   * 0：同意 * 1：拒绝 * 2：终止 * 3：删除 * 4：进行中 |
| handleTime | String | 处理时间，iso8601格式，例如：2022-07-29T14:55Z。 |
