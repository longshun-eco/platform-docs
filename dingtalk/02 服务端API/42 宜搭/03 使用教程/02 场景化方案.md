## 制造业销售下单搬上钉钉

[制造业销售下单搬上钉钉](https://open.dingtalk.com/abilities/168)能力包，介绍制造业企业下达销售订单是企业营销业务当中最具分量的一环，通过结合钉钉业务移动化、在线化的能力，在钉钉宜搭低代码上实现了销售订单的下达，驱动了企业各项业务的行动执行，从而完成公司获利的目标。
