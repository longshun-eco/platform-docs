更新于 2025-12-26调用本接口，根据矩阵ID获取指定权限矩阵的明细数据，返回信息包含矩阵的表头定义数据。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 宜搭平台资源读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=yida_2.0%23GetMatrixDetailById) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v2.0/yida/forms/resources/matrices?pageSize=Integer&pageNumber=Integer&matrixId=String&corpId=String&userId=String&token=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，可调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 |

## Query参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| pageSize | Integer | 否 | 分页大小。 |
| pageNumber | Integer | 否 | 分页页码。 |
| matrixId | String | 是 | 矩阵唯一标识，获取方式：平台管理-权限矩阵管理-权限矩阵ID。  ![[development-api-getmatrixdetailbyid_O1CN01YtkRx21Ykw51UCssh_!!6000000003098-2-tps-2970-920.png]] |
| corpId | String | 是 | 组织的corpId。 ![[development-api-getmatrixdetailbyid_O1CN01eZOoRQ1W63Un9lANu_!!6000000002738-2-tps-2984-1370.png]] |
| userId | String | 是 | 用户的userid。 |
| token | String | 是 | 验权token。  校验方式如下：`md5(corpId + userId + code)`。md5取32位大写值。  **说明**  每个企业有自己的唯一code。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| result | Object | 返回结果数据。 |
| matrixId | String | 矩阵唯一标识。 |
| name | Object | 矩阵名称。 |
| en\_US | String | 矩阵名称英文。 |
| zh\_CN | String | 矩阵名称中文。 |
| type | String | 矩阵名称类型，i18n代表多语言。 |
| description | Object | 矩阵自定义描述。 |
| en\_US | String | 矩阵描述英文。 |
| zh\_CN | String | 矩阵描述中文。 |
| type | String | 矩阵名称类型，i18n代表多语言。 |
| matrixTable | Object | 矩阵表头定义的详细数据。 |
| conditionColumn | Array | 矩阵表条件列定义信息。 |
| columnId | String | 列唯一标识。 |
| name | String | 矩阵列的定义名称。 |
| componentType | String | 矩阵条件列数据类型。   * **TextField**：自定义文本 * **DepartmentSelectField**：部门 |
| resultColumn | Array | 矩阵表角色列定义信息。 |
| columnId | String | 列唯一标识。 |
| name | String | 矩阵列的定义名称。 |
| componentType | String | 矩阵角色列数据类型。   * **EmployeeField**：人员 |
| matrixData | Object | 矩阵行明细数据。 |
| currentPage | Integer | 行数据分页页码。 |
| totalCount | Integer | 矩阵行数据总数。 |
| data | Any | 行数据明细。 |
| rowTotalCount | Integer | 矩阵行数据总数。 |
| success | Boolean | 是否成功，true/false。 |
