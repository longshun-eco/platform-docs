更新于 2025-12-26调用本接口，根据矩阵ID批量修改或新增指定矩阵的明细数据。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 宜搭平台资源写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=yida_2.0%23SaveAndUpdateMatrixData) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v2.0/yida/forms/resources/matrices/upsert HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "matrixId" : "String",
  "corpId" : "String",
  "userId" : "String",
  "token" : "String",
  "dataJson" : "String"
}
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，可调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| matrixId | String | 是 | 矩阵唯一标识，获取方式：平台管理-权限矩阵管理-权限矩阵ID。  ![[development-api-saveandupdatematrixdata_O1CN01YtkRx21Ykw51UCssh_!!6000000003098-2-tps-2970-920.png]] |
| corpId | String | 是 | 组织的corpId。 ![[development-api-saveandupdatematrixdata_O1CN01eZOoRQ1W63Un9lANu_!!6000000002738-2-tps-2984-1370.png]] |
| userId | String | 是 | 用户的userid。 |
| token | String | 是 | 验权token，校验方式如下：`md5(corpId + userId + code)`。md5取32位大写值。  **说明**  每个企业有自己的唯一code。 |
| dataJson | String | 是 | 矩阵数据详情。 参数说明：   * column\_xx为列唯一标识，rowId为行唯一标识。 * 传rowId代表按rowId更新已有行数据。如果 rowid 不存在，则按新增行处理。 * "isNew":1可指定新增行数据，新增行时不需要传rowId。如果isNew和rowId同时传值，优先按照 rowId 处理。   `[{ "column_xx": "deptId", "column_yy": ["userId"], "column_zz": "项目一", "rowId": "row_1748398062718" }, { "isNew":1 "column_xx": "deptId", "column_yy": ["userId", "userId"], "column_zz": "项目二" }]` |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| success | Boolean | 接口调用是否成功。 |
