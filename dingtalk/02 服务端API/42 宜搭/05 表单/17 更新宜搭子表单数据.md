更新于 2025-09-24调用本接口更新宜搭表单的子表数据。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 宜搭表单数据写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=yida_2.0%23UpdateSubTable) |
| 第三方企业应用 | 支持 | 宜搭表单数据写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=yida_2.0%23UpdateSubTable) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v2.0/yida/forms/updateSubTable HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "updateFormDataJson" : "String",
  "systemToken" : "String",
  "formInstanceId" : "String",
  "userId" : "String",
  "appType" : "String",
  "useLatestFormSchemaVersion" : Boolean,
  "tableFieldIds" : "String",
  "useAlias" : Boolean,
  "language" : "String",
  "noExecuteExpression" : Boolean
}
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| updateFormDataJson | String | 是 | 用于更新表单实例的数据：   * **key**：组件标识，宜搭表单编辑页面，高级设置中查看。 * **value**：组件内的值。 ![[development-api-updatesubtable_O1CN01yTqqI51eViHNpsmb2_!!6000000003877-2-tps-2662-1014.png]] |
| systemToken | String | 是 | 宜搭应用秘钥。 ![[development-api-updatesubtable_O1CN01jA8xVe1ZStrN1fmGj_!!6000000003194-2-tps-2698-1144.png]] |
| formInstanceId | String | 是 | 表单实例id。 |
| userId | String | 是 | 用户的userId，可通过调用[获取部门用户基础信息](https://open.dingtalk.com/document/development/queries-the-simple-information-of-a-department-user)接口获取用户userId。 |
| appType | String | 是 | 宜搭应用ID。 ![[development-api-updatesubtable_O1CN01auMafb1gfuQBoESlW_!!6000000004170-2-tps-2694-1114.png]] |
| useLatestFormSchemaVersion | Boolean | 否 | 是否使用最新的表单版本。  **说明**  默认为不使用 |
| tableFieldIds | String | 是 | 指定更新的子表字段id。 |
| useAlias | Boolean | 否 | 是否使用组件别名。  **说明**  开启之后，入参`updateFormDataJson`中组件id支持以别名形式传入。 |
| language | String | 否 | 语言，取值：   * **zh\_CN**：中文（默认值） * **en\_US**：英文 |
| noExecuteExpression | Boolean | 否 | 是否不触发表单绑定的校验规则、关联业务规则和第三方服务回调。   * **true**：不触发 * **false**：触发 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| result | Boolean | 更新结果。   * **true**：更新成功 * **false**：更新失败 |
