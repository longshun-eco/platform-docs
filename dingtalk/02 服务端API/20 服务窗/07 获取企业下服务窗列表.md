更新于 2025-09-11调用本接口，获取企业下创建的服务窗列表。

### 接口功能介绍

企业申请并开通服务窗后，可调用本接口查询企业下的服务窗信息列表，包含服务窗的accountId和服务窗名称。

**说明**

若未申请开通服务窗，请参考[如何申请开通服务窗](/document/development/customer-service-overview)。

![[development-queries-the-list-of-services-under-an-enterprise_O1CN01tYBMfz1RtuFqt6gnr_!!6000000002170-2-tps-1940-1026.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 服务窗帐号信息读取权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=link_1.0%23ListAccount) |
| 第三方企业应用 | 暂不支持 | 服务窗帐号信息读取权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 服务窗帐号信息读取权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/link/accounts HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array | 服务窗账号列表。 |
| accountId | String | 服务窗账号ID。 |
| accountName | String | 服务窗账号名称。 |
