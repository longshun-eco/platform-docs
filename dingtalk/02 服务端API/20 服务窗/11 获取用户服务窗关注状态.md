更新于 2025-09-11调用本接口获取用户服务窗关注状态。

### 接口功能介绍

通过本接口，获取指定的用户是否关注了组织内某一服务窗。

对于已关注用户，可以通过服务窗后台用户管理功能查看详情。具体如下图所示：

![[development-obtain-the-attention-status-of-the-user-service-window_服务窗关注用户.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 服务窗联系人信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=link_1.0%23GetUserFollowStatus) |
| 第三方企业应用 | 暂不支持 | 服务窗联系人信息读权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 服务窗联系人信息读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/link/followers/status?userId=String&unionId=String&accountId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用本接口的访问凭证，通过调用[获取第三方企业应用的suiteAccessToken](/document/development/obtains-the-suite-acess-token-of-third-party-enterprise-applications)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 否 | 待查询的服务窗关注者userId，可通过调用[通过免登码获取用户信息](/document/development/obtain-the-userid-of-a-user-by-using-the-log-free)接口，获取返回参数`userid`字段值。  **说明**   * 需使用[自建服务窗应用](/document/development/self-built-service-window-application)，免登获取的`userId`，否则接口无法正确调用。 * `unionId`和`userId`二选一填写。 |
| unionId | String | 否 | 待查询的服务窗关注者unionId，可通过调用[通过免登码获取用户信息](/document/development/obtain-the-userid-of-a-user-by-using-the-log-free)接口，获取返回参数`unionid`字段值。  **说明**   * 需使用企业内部应用，免登获取的`unionid`，否则接口无法正确调用。 * `unionId`和`userId`二选一填写。 |
| accountId | String | 否 | 服务窗账号ID，通过[获取企业下服务窗列表](/document/development/queries-the-list-of-services-under-an-enterprise)接口获得。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 响应结果。 |
| status | String | 用户关注服务窗的状态：   * **FOLLOWED**：已关注。 * **UNFOLLOW**：未关注。 |
