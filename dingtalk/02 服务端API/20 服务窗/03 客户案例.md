更新于 2025-09-10本文介绍了钉钉服务窗的使用案例。

钉钉服务窗能够快速连接与客户之间的业务往来，企业可通过服务窗向自己的客户提供业务服务，客户在新增的服务窗中建立与企业间的联系，实现简单与快捷的下单和订货。

## 案例一：经销商管理

**杭州百事可乐饮料有限公司**

产品价值：通过钉钉服务窗服务经销商，让物流送货流程全面数字化。

* **过去**

  经销商下了订单后，公司通知承运商送货，承运商安排司机，司机送到货后，拿着五联单找经销商的经理签字后再带回公司。

  遇到的问题：

  + 签收身份不明：五联单是谁签的，身份不确定。
  + 效率低：司机送到货，联系不上经销商经理，不能及时在五联单上签字确认。签字后不能及时向公司回传。
  + 风险高：安排其他人代签收时，经常出现事后扯皮，严重的还涉及到法律风险以及经济损失。
  + 追溯难：物流送没送、送多少、客户拒收了多少等行为没人知道。

  ![[development-service-window-use-case_p260762.png]]
* **现在**

  百事可乐可以和经销商、承运商在服务窗通过表单对该业务流程进行高效协同，建立高效清晰可追溯的业务闭环，避免纠纷风险。

  ![[development-service-window-use-case_p260605.png]]

  ![[development-service-window-use-case_p260611.png]]

## 案例二：供应商管理

**深圳市正君餐饮管理顾问有限公司**

产品价值：通过服务窗服务供应商，让供应商对账流程数字化。

* **过去**

  + 财务经常出现漏审核漏打款现象，给部分供应商留下拖欠贷款体感，透支企业信用。
  + 财务有大量录入工作，工作效率低下。
  + 供应商无法看到审批进度，经常催款，体验差。

  ![[development-service-window-use-case_p260614.png]]
* **现在**

  供应商通过服务窗自主提交对账申请，财务确认对账单审批通过后，直接对接到钉钉的付款系统付款。

  + 对账申请直接接入钉钉审批，财务在钉钉审批待办中即可操作，避免漏批。
  + 供应商可实时查看审批状态，体验好。
  + 审批通过后付款信息自动导入至付款系统，财务可直接付款，免除无谓的录入工作。

  ![[development-service-window-use-case_p260617.png]]
