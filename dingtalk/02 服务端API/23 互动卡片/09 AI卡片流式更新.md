更新于 2025-09-23AI卡片流式更新接口旨在为AIGC产生的内容提供一种持续更新的能力。通过AI流式更新接口持续更新的内容，在客户端会呈现一种打字机效果。

#### 最佳实践/教程

机器人实现 AI 卡片流式效果详情参考：AI卡片流式更新。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | AI卡片流式更新权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=card_1.0%23StreamingUpdate) |
| 第三方企业应用 | 支持 | AI卡片流式更新权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=card_1.0%23StreamingUpdate) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
PUT /v1.0/card/streaming HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "outTrackId" : "String",
  "guid" : "String",
  "key" : "String",
  "content" : "String",
  "isFull" : Boolean,
  "isFinalize" : Boolean,
  "isError" : Boolean
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| outTrackId | String | 是 | 外部卡片实例Id，与[创建卡片](/document/development/interface-for-creating-a-card-instance)/[创建并投放卡片](/document/development/create-and-deliver-cards)中的 outTrackId 保持一致。 |
| guid | String | 是 | 请求调用的唯一标志，系统内部用于幂等判断。 |
| key | String | 是 | 需要进行流式更新的变量。 |
| content | String | 是 | 此更新的流式内容。  **说明**   * 由于 markdown 需要服务端进行格式转换，出于语法完整性考虑，所以如果内容是markdown，必须是全量的内容，且需要保证 markdown 语法的完整性。 * 内容 size 单次不要超过 1 K，总大小建议不要超过 3 K。 |
| isFull | Boolean | 否 | 是否全量：   * **true**：全量 * **false**：非全量（默认）   **说明**   * 如果是true，则内部以覆盖的方式进行更新，如果是 false，则内部以增量的方式进行更新。 * 如果流式变量绑定的是 markdown，该参数必须设置为 true，否则会报错。 |
| isFinalize | Boolean | 否 | 是否是最后一帧：   * **true**：最后一帧 * **false**：不是最后一帧（默认）   **说明**  如果设置为 true，AI卡片将从「输入中」状态切换为「完成」状态。卡片状态信息请参考文档[AI 卡片模板](/document/development/ai-card-template)。 |
| isError | Boolean | 否 | 是否出错：   * **true**：出错 * **false**：没有出错（默认）   **说明**  如果设置为 true，AI卡片将从「输入中」状态切换为「出错」状态，卡片状态信息请参考文档[AI 卡片模板](/document/development/ai-card-template)。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 接口调用是否成功。 |
| result | Boolean | 更新结果。 |
