更新于 2025-09-23调用本接口可关闭通过卡片投放接口投放的吊顶卡片。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 互动卡片实例写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=card_1.0%23CloseTopCard) |
| 第三方企业应用 | 支持 | 互动卡片实例写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=card_1.0%23CloseTopCard) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/card/tops/close?outTrackId=String&openConversationId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| outTrackId | String | 是 | 外部卡片实例Id。  **说明**  由开发者自己生成并作为入参传递给钉钉的，钉钉只在对应使用到outTrackId的场景，帮助开发者对TrackId进行记录。 |
| openConversationId | String | 是 | 会话 id。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 接口是否调用成功。 |
| result | Boolean | 是否成功关闭吊顶。 |
