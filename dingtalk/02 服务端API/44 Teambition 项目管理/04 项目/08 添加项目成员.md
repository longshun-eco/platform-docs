例如，在项目应用中，有一个项目名为测试项目，该项目内的成员只有小钉一人，如下图所示。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用项目写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23AddProjectMember) |
| 第三方企业应用 | 支持 | 项目应用项目写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23AddProjectMember) |
| 第三方个人应用 | 暂不支持 | 项目应用项目写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/project/users/{userId}/projects/{projectId}/members HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "userIds" : [ "String" ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作者userId。 |
| projectId | String | 是 | 项目Id，调用[根据项目模板创建项目](/document/development/create-a-project-from-a-project-template)接口获取id参数值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userIds | Array of String | 是 | 被添加的用户userId列表，建议一次不超过10个。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array | 添加的项目成员信息列表。 |
| nickname | String | 项目成员昵称。 |
| joined | String | 项目成员加入项目时间，iso8601格式，例如：2022-07-29T14:55Z。 |
