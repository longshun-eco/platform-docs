更新于 2025-10-09调用本接口，按模板名字搜索项目模板信息。

在项目应用的模板列表内，企业可创建自定义模板，与下图步骤实现效果一致。

![[development-search-for-enterprise-custom-templates-by-project-template-name_O1CN01J8Priq1uEnXqiNVDZ_!!6000000006006-2-tps-2626-1498.png]]

创建自定义模板后，调用本接口可获取自定义模板的详情信息，包括模板id、模板名称、模板可见性等。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用项目读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23SearchProjectTemplate) |
| 第三方企业应用 | 支持 | 项目应用项目读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23SearchProjectTemplate) |
| 第三方个人应用 | 暂不支持 | 项目应用项目读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/project/organizations/users/{userId}/templates?keyword=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作者userId。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| keyword | String | 否 | 项目模板名称关键词。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array | 返回结果对象。 |
| id | String | 模板id。 |
| description | String | 模板的描述信息。 |
| visible | String | 模板可见性。   * **organization**：企业内公开。 * **involves**：私有，仅模板成员可见。 |
| isDemo | Boolean | 是否为demo模板。   * **true**：是 * **false**：否 |
| isDeleted | Boolean | 是否已删除。   * **true**：是 * **false**：否 |
| name | String | 模板名称。 |
| logo | String | 模板图标地址。 |
| created | String | 创建时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| updated | String | 更新时间，iso8601格式，例如：2022-07-29T14:55Z。 |
