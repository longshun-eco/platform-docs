更新于 2025-10-09调用本接口获取用户加入的项目。

### 接口功能介绍

调用本接口，根据用户userId信息，获取用户加入的项目Id列表。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用项目读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23GetUserJoinedProject) |
| 第三方企业应用 | 支持 | 项目应用项目读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23GetUserJoinedProject) |
| 第三方个人应用 | 暂不支持 | 项目应用项目读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/project/users/{userId}/joinProjects?maxResults=Integer&nextToken=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作者userId。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| maxResults | Integer | 否 | 分页大小。  **说明**  不传则显示全部项目。 |
| nextToken | String | 否 | 分页标。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array of String | 项目id。 |
| nextToken | String | 分页标。 |
