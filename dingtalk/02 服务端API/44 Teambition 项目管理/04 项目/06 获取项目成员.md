更新于 2025-10-09调用本接口获取项目成员。

### 接口功能介绍

调用本接口，根据操作者ID和项目ID，查询项目中的成员。示例如下：

![[development-get-project-members_查询项目成员.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用项目读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23GetProjectMemebers) |
| 第三方企业应用 | 支持 | 项目应用项目读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23GetProjectMemebers) |
| 第三方个人应用 | 暂不支持 | 项目应用项目读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/project/users/{userId}/projects/{projectId}/members?userIds=String&projectRoleId=String&maxResults=Integer&skip=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作者userId。 |
| projectId | String | 是 | 项目ID，可通过调用[查询项目](/document/development/query-enterprise-all-projects)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userIds | String | 否 | 用户ID。  **说明**  多个ID，使用英文逗号分隔。 |
| projectRoleId | String | 否 | 项目角色ID。  **说明**  仅查询拥有该角色的成员，并且仅支持单个角色查询。 |
| maxResults | Integer | 否 | 每页返回最大数量。默认10，最大300。 |
| skip | Integer | 否 | 跳过的数据数量。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array | 项目成员列表。 |
| memberId | String | 项目成员ID。 |
| userId | String | 用户ID。 |
| role | Integer | 项目角色：   * **0**：成员 * **1**：管理员 * **2**：拥有者 |
| roleIds | Array of String | 项目角色ID。 |
