更新于 2025-10-09调用本接口，查询员工可见的项目分组列表。

例如，员工小钉在项目应用中可查看的项目分组有两个，分别是**测试项目分组1**和**测试项目分组2**，如下图所示。

![[development-query-available-project-groups_O1CN01wz6iif1h7O82rkEuK_!!6000000004230-2-tps-2624-1340.png]]

调用本接口，可获取小钉可见的这两个项目分组的信息，包括项目分组名称、项目分组id、项目分组可见性、创建时间等。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用项目读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23GetProjectGroup) |
| 第三方企业应用 | 支持 | 项目应用项目读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23GetProjectGroup) |
| 第三方个人应用 | 暂不支持 | 项目应用项目读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/project/organizations/users/{userId}/groups?viewerId=String&pageSize=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 否 | 操作者userId。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| viewerId | String | 否 | 查看者userId，即查询该员工可见的项目分组。 |
| pageSize | Integer | 否 | 分页大小。从1开始，默认值10，最大值1000。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array | 返回结果对象。 |
| id | String | 项目分组id。 |
| visible | String | 项目分组可见性。   * **organization**：企业项目分组 * **involves**：私有项目分组 |
| name | String | 项目分组名称。 |
| created | String | 创建时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| updated | String | 更新时间，iso8601格式，例如：2022-07-29T14:55Z。 |
