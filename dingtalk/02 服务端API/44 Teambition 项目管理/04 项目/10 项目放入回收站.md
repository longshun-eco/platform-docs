更新于 2025-10-09调用本接口将项目放入回收站

### 接口功能介绍

调用本接口，根据操作者ID和项目ID，实现将项目放入回收站。示例如下：

![[development-items-in-recycle-bin_移动回收站.png]]

**说明**

放入回收站后所有与项目有关的内容不会被统计和搜索收录，需要去回收站恢复后才能继续使用。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用项目写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23ArchiveProject) |
| 第三方企业应用 | 支持 | 项目应用项目写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23ArchiveProject) |
| 第三方个人应用 | 暂不支持 | 项目应用项目写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/project/users/{userId}/projects/{projectId}/archive HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作者userId。 |
| projectId | String | 是 | 项目ID，可通过调用[查询项目](/document/development/query-enterprise-all-projects)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果。 |
| isArchived | Boolean | 是否已放入回收站。 |
| updated | String | 更新时间，格式：YYYY-MM-DDTHH:mm:ssZ（ISO 8601/RFC 3339）。  **说明**  转换成北京时间，需要在参数时间基础上加8小时。例如参数值为2022-06-20T00:00:00Z，表示的北京时间为2022-06-20 08:00:00。 |
