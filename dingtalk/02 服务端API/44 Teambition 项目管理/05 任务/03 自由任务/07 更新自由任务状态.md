更新于 2025-10-09调用本接口，更新自由任务的状态。

例如，员工小钉有个自由任务名为**测试任务1**，任务的状态为**未完成**，如下图所示。

![[development-change-free-task-status_O1CN013h6fES1tijoJxSdMj_!!6000000005936-2-tps-2880-1800.png]]

调用本接口，可以更新该任务的状态，将任务状态**未完成**更新为**已完成**。接口调用成功后的效果如下图所示。

**说明**

任务状态更新为**已完成**后，该任务会从任务列表移除。

![[development-change-free-task-status_O1CN01bzrje81DJoOGydjVL_!!6000000000196-2-tps-2880-1800.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用中任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23UpdateOrganizationTaskStatus) |
| 第三方企业应用 | 支持 | 项目应用中任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23UpdateOrganizationTaskStatus) |
| 第三方个人应用 | 暂不支持 | 项目应用中任务写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/project/organizations/users/{userId}/tasks/{taskId}/states HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "isDone" : Boolean,
  "disableActivity" : Boolean,
  "disableNotification" : Boolean
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| taskId | String | 是 | 任务id，调用[创建自由任务](/document/development/create-a-free-task)接口获取的id值。 |
| userId | String | 是 | 操作者userId。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| isDone | Boolean | 是 | 自由任务状态。   * **true**：完成 * **false**： 未完成 |
| disableActivity | Boolean | 否 | 是否禁止动态。   * **true**：禁止，不显示该任务动态信息。 * **false**：不禁止，显示该任务动态信息。 ![[development-change-free-task-status_O1CN01qomZVt22RdYnGUSD4_!!6000000007117-2-tps-2258-1416.png]] |
| disableNotification | Boolean | 否 | 是否禁止通知。   * **true**：禁止，不显示通知。 * **false**：不禁止，显示通知。 ![[development-change-free-task-status_O1CN01zO16bz28KCpg6HBHk_!!6000000007913-2-tps-1940-1268.png]] |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果。 |
| isDone | Boolean | 自由任务状态。   * **true**：完成 * **false**： 未完成 |
| updateTime | String | 更新任务状态的时间，格式：YYYY-MM-DDTHH:mm:ssZ（ISO 8601/RFC 3339）。  **说明**  转换成北京时间，需要在参数时间基础上加8小时。例如参数值为2022-06-20T00:00:00Z，表示的北京时间为2022-06-20 08:00:00。 |
