更新于 2025-10-09调用本接口，更新自由任务的优先级。

例如，员工小钉有个自由任务名为**测试任务1**，任务的优先级为**普通**，如下图所示。

![[development-change-free-task-priority_O1CN0100TQbC1mYw1CoTpKP_!!6000000004967-2-tps-2880-1800.png]]

调用本接口可更新该任务的优先级，将任务的优先级**普通**更新为**非常紧急**。接口调用成功后的效果如下图所示。

![[development-change-free-task-priority_O1CN01jOgzcz1sweDx6K4m0_!!6000000005831-2-tps-2880-1800.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用中任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23UpdateOrganizationTaskPriority) |
| 第三方企业应用 | 支持 | 项目应用中任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23UpdateOrganizationTaskPriority) |
| 第三方个人应用 | 暂不支持 | 项目应用中任务写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/project/organizations/users/{userId}/tasks/{taskId}/priorities HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "priority" : Integer,
  "disableActivity" : Boolean,
  "disableNotification" : Boolean
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| taskId | String | 是 | 任务id，调用[创建自由任务](/document/development/create-a-free-task)接口获取的id值。 |
| userId | String | 是 | 操作者userId。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| priority | Integer | 是 | 自由任务优先级。   * 默认优先级有以下4个：    + **-10**：较低，默认值。   + **0**：普通   + **1**：紧急   + **2**：非常紧急 * 新增优先级，调用[查询优先级列表](/document/development/query-a-priority-list)接口获取priorityId值。   **说明**   * 优先级数值越大，优先级越高。 * 自定义优先级需要开通企业版或者旗舰版项目。 |
| disableActivity | Boolean | 否 | 是否禁止动态。   * **true**：禁止，不显示该任务动态信息。 * **false**：不禁止，显示该任务动态信息。 ![[development-change-free-task-priority_O1CN01qomZVt22RdYnGUSD4_!!6000000007117-2-tps-2258-1416.png]] |
| disableNotification | Boolean | 否 | 是否禁止通知。   * **true**：禁止，不显示通知。 * **false**：不禁止，显示通知。 ![[development-change-free-task-priority_O1CN01zO16bz28KCpg6HBHk_!!6000000007913-2-tps-1940-1268.png]] |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果。 |
| priority | Integer | 更新后的自由任务优先级。   * 优先级有以下4个默认值。    + **-10**：较低，默认值。   + **0**：普通   + **1**：紧急   + **2**：非常紧急 * 自定义优先级，以接口实际调用结果为准，优先级越高，数值越大。 |
| updated | String | 更新优先级的时间，格式：YYYY-MM-DDTHH:mm:ssZ（ISO 8601/RFC 3339）。  **说明**  转换成北京时间，需要在参数时间基础上加8小时。例如参数值为2022-06-20T00:00:00Z，表示的北京时间为2022-06-20 08:00:00。 |
