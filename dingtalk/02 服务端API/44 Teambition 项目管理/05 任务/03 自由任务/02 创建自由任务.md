更新于 2025-10-09调用本接口，创建一个钉钉自由任务（非加入项目的任务）。

### 什么是自由任务

自由任务是在项目应用中创建，但是不关联任何项目的任务。

![[development-create-a-free-task_O1CN01Jl3B8K1slCK16UX9E_!!6000000005806-2-tps-2308-1434.png]]

### 接口功能介绍

例如，在员工小钉的项目中，**我的任务**列表如下图所示，没有需要执行的任务。

![[development-create-a-free-task_O1CN01t2cMjT1hdRqNa465p_!!6000000004300-2-tps-2042-1266.png]]

调用本接口，给员工小钉创建一个名为**测试任务**的任务。接口调用成功后，员工小钉在**我的任务**列表查看到的信息如下图所示，有一个名为**测试任务**的任务。

![[development-create-a-free-task_O1CN01t8CSeW1yYjEOKn8bN_!!6000000006591-2-tps-2034-1270.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23CreateOrganizationTask) |
| 第三方企业应用 | 支持 | 项目应用任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23CreateOrganizationTask) |
| 第三方个人应用 | 暂不支持 | 项目应用任务写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/project/organizations/users/{userId}/tasks HTTP/1.1
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "content" : "String",
  "note" : "String",
  "priority" : Integer,
  "involveMembers" : [ "String" ],
  "executorId" : "String",
  "dueDate" : "String",
  "createTime" : "String",
  "visible" : "String",
  "disableNotification" : Boolean,
  "disableActivity" : Boolean
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |

| createTime | String | 否 | 任务创建日期，格式：YYYY-MM-DDTHH:mm:ssZ（ISO 8601/RFC 3339）。  **说明**  转换成北京时间，需要在参数时间基础上加8小时。例如参数值为2022-06-20T00:00:00Z，表示的北京时间为2022-06-20 08:00:00。 |
