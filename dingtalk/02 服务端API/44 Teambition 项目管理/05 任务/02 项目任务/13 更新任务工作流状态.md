更新于 2025-10-09调用本接口更新任务工作流状态。

可以通过SearchTaskflowStatus接口获取项目下的工作流

注意该接口调用后是更新任务的任务状态，而不是更新任务状态本身字段内的内容。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23UpdateTaskTaskflowstatus) |
| 第三方企业应用 | 支持 | 项目应用任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23UpdateTaskTaskflowstatus) |
| 第三方个人应用 | 暂不支持 | 项目应用任务写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/project/users/{userId}/tasks/{taskId}/taskflowStatuses HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "taskflowStatusId" : "String",
  "tfsUpdateNote" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作者userId。 |
| taskId | String | 是 | 任务ID，可通过调用[查询项目中的任务](/document/development/query-tasks-in-a-project)接口，获取返回参数`taskId`字段值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| taskflowStatusId | String | 否 | 任务状态ID，调用服务端API-[搜索任务工作流状态](/document/development/search-task-workflow-status)接口，获取返回参数`taskflowStatusId`字段值。 |
| tfsUpdateNote | String | 否 | 任务流转说明。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 结果。 |
| updated | String | 更新时间。 |
