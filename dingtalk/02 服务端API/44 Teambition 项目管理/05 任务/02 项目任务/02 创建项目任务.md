更新于 2025-10-09调用本接口，创建一个钉钉项目任务。

### 什么是项目任务

项目任务是在关联指定项目中创建的任务。

![[development-create-a-project-task_O1CN01oj9R5x1UaA04bTdw7_!!6000000002533-2-tps-2564-908.png]]

### 接口功能介绍

例如，有个项目名为**文档跟进**，如下图所示，目前没有任何任务。

![[development-create-a-project-task_O1CN01v3oUCu28geF9zxZT1_!!6000000007962-2-tps-2726-978.png]]

调用本接口可以给员工小钉3创建一个**文档跟进**项目的任务。

![[development-create-a-project-task_O1CN01MvLDHY1X1IhdYJTTj_!!6000000002863-2-tps-2726-1190.png]]

接口调用成功后，**文档跟进**项目内显示的任务信息如下图所示。

![[development-create-a-project-task_O1CN01z70MUN1hgCDvPL8ck_!!6000000004306-2-tps-2874-668.png]]

### 自定义字段

项目创建者，在当前项目内任意任务中添加自定义字段，该项目内创建任务可使用自定义字段。

![[development-create-a-project-task_O1CN01ADr95Z23x4N5wWYbG_!!6000000007321-2-tps-2618-1056.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23CreateTask) |
| 第三方企业应用 | 支持 | 项目应用任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23CreateTask) |
| 第三方个人应用 | 暂不支持 | 项目应用任务写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/project/users/{userId}/tasks HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "projectId" : "String",
  "content" : "String",
  "executorId" : "String",
  "dueDate" : "String",
  "note" : "String",
  "priority" : Integer,
  "customfields" : [ {
    "customfieldName" : "String",
    "customfieldId" : "String",
    "value" : [ {
      "title" : "String"
    } ]
  } ],
  "stageId" : "String",
  "parentTaskId" : "String",
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作者userId。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| projectId | String | 是 | 项目id。  **说明**  目前需要从项目链接中获取该参数值，获取步骤：进入**项目** > 右上角单击**菜单** > **复制链接** 得到的项目链接示例：https://www.teambition.com/project/62c794xxxxx，project下一级路径的值就是项目id。 ![[development-create-a-project-task_O1CN01gt1F8b1M9a5Y64uC6_!!6000000001392-2-tps-2726-1144.png]] |
| content | String | 是 | 任务标题。 |
| executorId | String | 否 | 任务执行者userId。 |
| dueDate | String | 否 | 任务截止时间，格式：YYYY-MM-DDTHH:mm:ssZ（ISO 8601/RFC 3339）。  **说明**  转换成北京时间，需要在参数时间基础上加8小时。例如参数值为2022-06-20T00:00:00Z，表示的北京时间为2022-06-20 08:00:00。 |
| note | String | 否 | 任务备注。 |
| priority | Integer | 否 | 任务优先级，调用[查询优先级列表](/document/development/query-a-priority-list)接口获取priorityId值。 |
| customfields | Array | 否 | 自定义字段列表。  **说明**  自定义字段目前只支持单选、多选、数字、文本和日期，其他类型暂不支持。 |
| customfieldName | String | 否 | 自定义字段名称。  **说明**  该字段值需要与自定义字段名保持完全一致。 ![[development-create-a-project-task_O1CN01syqg0L1f3bdMFp5mv_!!6000000003951-2-tps-2602-1366.png]] |
| customfieldId | String | 否 | 自定义字段ID。  **说明**  自定义字段ID，暂未透出，目前该参数字段值不填即可。 |
| value | Array | 否 | 自定义字段值列表。 |
| title | String | 否 | 自定义字段值。   * 如果自定义字段为单选或多选类型，该参数值需要与设置的选项值保持完全一致。 * 如果自定义字段为日期类型，该参数值格式为yyyy-MM-dd HH:mm:ss。 |
| id | String | 否 | 字段值id，如果该字段是成员字段，那么此处应该填写成员的在Teambition系统中的userId，获取Teambition用户id请参阅文档[根据userId获取Teambition项目用户ID](/document/development/obtain-dingtalk-teambition-user-id-based-on-userid)。 |
| thumbUrl | String | 否 | 如果该字段是成员字段，那么此处应该填写成员的头像地址。 |
| stageId | String | 否 | 任务列表ID，可调用[查询项目中的任务](/document/development/query-tasks-in-a-project)接口获取stageId。 |
| parentTaskId | String | 否 | 父任务id，可调用[查询项目中的任务](/document/development/query-tasks-in-a-project)接口获取taskId。 |
| scenariofieldconfigId | String | 否 | 任务类型id，任务类型比如：缺陷、需求，可调用[查询项目中的任务](/document/development/query-tasks-in-a-project)接口获取scenariofieldconfigId。 |
| startDate | String | 否 | 任务开始时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| visible | String | 否 | 任务的可见性规则。   * involves：仅任务参与者可见 * members：项目成员可见 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果对象。 |
| taskId | String | 任务id。 |
| content | String | 任务标题。 |
| involveMembers | Array of String | 参与者userId列表。 |
| projectId | String | 项目id。 |
| executorId | String | 任务执行者userId。 |
| creatorId | String | 任务创建者userId。 |
| created | String | 任务创建时间，格式：YYYY-MM-DDTHH:mm:ssZ（ISO 8601/RFC 3339）。  **说明**  转换成北京时间，需要在参数时间基础上加8小时。例如参数值为2022-06-20T00:00:00Z，表示的北京时间为2022-06-20 08:00:00。 |
| updated | String | 任务更新时间，格式：YYYY-MM-DDTHH:mm:ssZ（ISO 8601/RFC 3339）。  **说明**  转换成北京时间，需要在参数时间基础上加8小时。例如参数值为2022-06-20T00:00:00Z，表示的北京时间为2022-06-20 08:00:00。 |
| note | String | 任务备注。 |
| dueDate | String | 任务截止时间，格式：YYYY-MM-DDTHH:mm:ssZ（ISO 8601/RFC 3339）。  **说明**  转换成北京时间，需要在参数时间基础上加8小时。例如参数值为2022-06-20T00:00:00Z，表示的北京时间为2022-06-20 08:00:00。 |
| priority | Integer | 任务优先级。   * 使用项目默认的优先级，如下图所示。优先级默认为以下。    + **-10**：较低，默认值。   + **0**：普通   + **1**：紧急   + **2**：非常紧急   ![[development-create-a-project-task_O1CN01hNuSPz25juCzgxhmW_!!6000000007563-2-tps-2682-1304.png]]   * 用户自定义优先级，如下图所示，新增一般紧急并调整优先级顺序等。该参数值以接口实际调用结果为准。优先级越高，数值越大。   ![[development-create-a-project-task_O1CN01V67b3i1mkNvJiW8D1_!!6000000004992-2-tps-2128-1126.png]] |
| customfields | Array | 自定义字段列表。 |
| customfieldId | String | 自定义字段ID。  **说明**  该参数暂未透出，目前不返回。 |
| value | Array | 自定义字段值列表。 |
| title | String | 自定义字段值。 |
