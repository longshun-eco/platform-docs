更新于 2025-10-09调用本接口搜索任务工作流状态。

什么是任务工作流状态？如下图所示，是指每个工作流下的任务状态项

![[development-search-task-workflow-status_O1CN018hLfOm29cobeNZojT_!!6000000008089-0-tps-320-186.jpg]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用项目读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23SearchTaskflowStatus) |
| 第三方企业应用 | 支持 | 项目应用项目读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23SearchTaskflowStatus) |
| 第三方个人应用 | 暂不支持 | 项目应用项目读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/project/users/{userId}/projects/{projectId}/taskflowStatuses/search?query=String&maxResults=Integer&nextToken=String&tfIds=String&tfsIds=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作者userId。 |
| projectId | String | 是 | 项目ID，可通过调用[根据项目模板创建项目](/document/development/create-a-project-from-a-project-template)接口，获取返回参数`id`字段值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| query | String | 否 | 模糊查询工作流状态名字。 |
| maxResults | Integer | 否 | 每页返回最大数量。  **说明**  默认10，最大300。 |
| nextToken | String | 否 | 分页标，从上一次请求结果中获取。 |
| tfIds | String | 否 | 工作流ID集合。  **说明**  多个ID用英文逗号隔开。 |
| tfsIds | String | 否 | 工作流状态ID集合。  **说明**  多个ID用英文逗号隔开。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array | 工作流状态列表。 |
| taskflowStatusId | String | 任务工作流状态ID。 |
| name | String | 工作流状态名字。 |
| pos | Integer | 工作流状态位置。 |
| taskflowId | String | 工作流状态ID。 |
| rejectStatusIds | Array of String | 拒绝的工作流状态Id。 |
| kind | String | 任务工作流状态类型：   * **start**: 开始 * **end**: 结束 * **unset**: 未设置 |
| creatorId | String | 创建者ID。 |
| isDeleted | Boolean | 是否已删除。 |
| created | String | 创建时间。 |
| updated | String | 更新时间。 |
| isTaskflowstatusruleexector | Boolean | 是否特定任务角色才能流转该工作流状态：   * **true**：是 * **false**：否 |
