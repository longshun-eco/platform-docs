更新于 2025-10-09调用本接口，更新项目任务的自定义字段值。

例如，创建的项目任务信息中，自定义字段如下图所示。

![[development-update-task-custom-field-value_O1CN01jZRGYS1XkdvP601ej_!!6000000002962-2-tps-2446-1298.png]]

调用本接口，可更新项目任务的自定义字段值。例如，将**自定义字段文本**的**测试值**修改为**修改后的测试值**，接口调用成功后效果如下图所示。

![[development-update-task-custom-field-value_O1CN01TouEbM1Jr8vPTT4ou_!!6000000001081-2-tps-2432-1302.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 项目应用任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=project_1.0%23UpdateCustomfieldValue) |
| 第三方企业应用 | 支持 | 项目应用任务写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=project_1.0%23UpdateCustomfieldValue) |
| 第三方个人应用 | 暂不支持 | 项目应用任务写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/project/users/{userId}/tasks/{taskId}/customFields HTTP/1.1
Host:api.dingtalkcom
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "customfieldName" : "String",
  "value" : [ {
    "title" : "String"
  } ],
  "customfieldId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 操作者userId。 |
| taskId | String | 是 | 任务id，调用[创建项目任务](/document/development/create-a-project-task)接口获取的taskId。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| customFieldName | String | 否 | 自定义字段名。 |
| value | Array | 是 | 自定义字段值列表，最多10个。 |
| title | String | 是 | 自定义字段修改后的值。 |
| customFieldId | String | 否 | 自定义字段id。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果。 |
| customFields | Array | 自定义字段数组。 |
| customFieldId | String | 自定义字段id |
| value | Array | 自定义字段值数组。 |
| title | String | 自定义字段显示值。 |
