## 什么是钉钉客联

钉钉客联是钉钉推出的跨组织沟通PaaS解决方案，专为企业构建内外部统一沟通平台而设计，实现钉钉组织内外用户间的无缝消息互通与业务协同。

![[development-dingtalk-customer-contact-product-overview-document_p624009.png]]

## **核心价值**

钉钉客联打通了钉内账号和钉外账号的消息互通桥梁，构建了企业**对内办公(2B)**和**对外服务/连接(2C)**的统一的沟通平台，通过钉钉实现让企业员工不再频繁切换各种沟通工具完成沟通。

![[development-dingtalk-customer-contact-product-overview-document_p624159.png]]

钉钉客联解决了企业内外部沟通割裂的痛点，通过技术赋能实现：

* **打破沟通壁垒**

  消除组织边界，建立企业员工与外部用户间的直接沟通渠道。
* **提升服务效率**

  减少工具切换，将沟通深度融入业务流程，提高响应速度。
* **降低集成成本**

  提供标准化API，缩短系统对接周期。
* **统一用户体验**

  跨平台消息一致性，确保用户在不同终端获得相同体验。

## **核心能力矩阵**

### **1. 业务系统无缝集成**

* 提供API服务体系，支持快速对接客服、物流、招聘等核心业务系统。
* 开放消息收发、群管理、用户关联等关键能力，实现业务与沟通的深度融合。
* 支持生态伙伴基于开放能力开发行业定制化解决方案。

### **2. 跨平台消息协同**

* 一套接口适配多终端（APP、小程序、网页、钉钉）。
* 消息实时同步，确保跨平台沟通体验一致性。
* 完整消息留痕与数据沉淀，支持后续分析与服务优化。

### **3. 场景化服务构建**

* 支持构建客户服务、用户运营、协作沟通等标准化场景。
* 提供二级会话管理，将分散的外部沟通聚合为统一服务入口。
* 灵活的渠道配置能力，满足不同业务场景的个性化需求。

### **4. 互通群闭环管理**

* 完整的钉钉客联互通群生命周期管理（创建、成员管理、信息修改、解散等）。
* 群模板化配置，快速部署标准化群组服务。
* 支持钉钉客联普通群与钉钉客联两人群两种互通群类型，满足多样沟通需求。

  钉钉客联普通群是正常的群聊，钉钉客联两人群是钉钉客联基于单聊场景特殊实现，钉钉客联普通群与钉钉客联两人群对比如下：

  |  |  |  | | --- | --- | --- | |  | 钉钉客联普通群 | 钉钉客联两人群 | | 说明 | 正常的群聊，可支持两人以上的沟通。 | 通过两人群的方式虚拟出单聊的场景，只支持两人沟通的群聊。 | | 使用场景 | 任何场景。 | 用户需要实现单聊场景时。 | | 特点 | 正常的普通群。 | 只对钉外用户的显示有差异，模拟了单聊的效果，在钉外用户显示的是群内另一个人的昵称和头像。 | | 使用限制 | 2人<=人数<=20人 | 人数 = 2人    **说明**  当群内人数发生变动时，会自动从两人群降级为普通群。 | | 功能限制 | 购买钉钉客联后使用。 | 购买钉钉客联后使用。 |

## **客户案例**

钉钉客联已助力多个行业头部企业实现业务场景创新。

* 某大型车企通过钉钉客联在自有APP构建“客户服务平台”提升用户全生命周期服务效率。
* 某物流集团通过集成钉钉客联打破员工、快递员、用户沟通壁垒，实现服务全流程在线化。

## 名词解释

|  |  |
| --- | --- |
| 名词 | 释义 |
| 钉外用户/客户/C端用户 | 指非钉钉企业组织内的用户，通过钉钉客联与其建立关联。例如：某企业自有系统中的用户“小钉1”，虽不在钉钉通讯录中，但可通过API将其与钉钉内的“小钉2”绑定，生成唯一的钉外用户标识，从而成为该企业的钉外用户。 |
| 钉内用户/B端用户 | 特指当前开通客联的钉钉组织通讯录内的成员。 |
| 二级会话 | 指折叠来自钉外用户的所有单聊、群聊，只在消息列表界面显示一个入口，可预先设置统一的logo和名称。 |
| 渠道（channelCode） | 调用钉钉客联API时必须传入的参数，开发者可创建多个渠道以适配不同业务场景，每个渠道支持独立配置UI样式和行为逻辑，更多信息可查看[渠道配置](/document/development/interconnections-channel)。 |
| 互通群 | 指互通钉外用户、钉内用户的群聊，类型包括普通群、跨钉两人群、钉外两人群。 |
| 钉钉客联两人群 | 是钉钉客联群的一种，群内包括钉内账号和钉外账号。 |
| 钉钉客联普通群 | 是钉钉客联群的一种，群内只包含钉外账号，不包含钉内账号。 |
| 钉钉客联群模板id | 创建客联互通群时需要的模板id，可通过[群模板配置](/document/development/interconnections-model)中获取。 |
