## **页面配置-移动端**

可全局配置头像、颜色；也可点击相应的页面设置布局及功能，**保存**后生效。

### **全局配置**

### ![[development-interconnections-page_p1011193.png]]

* **人员头像：**支持方形、圆形。
* **头像背景：**支持颜色选取与Hex颜色代码，支持透明度。

  **说明**

  如调用接口时传入的头像图片地址，则优先显示头像图片。
* **消息背景**：支持颜色选取与Hex颜色代码，支持透明度。

### **会话列表配置**

![[development-interconnections-page_p1011194.png]]

* **页面名称**：如果开启头部导航，则显示在头部中间位置。
* **返回按钮自定义**

  + **标准功能：**返回到上一个页面。
  + **普通链接：**可自定义跳转地址，填写返回按钮跳转链接。
* **头部导航**：默认关闭，可以手动开启。
* **头部导航右侧扩展按钮（新建应用）：最多支持两个入口。**

  + **图标：**支持默认图标和本地上传图片（点击“+”，可上传本地图标）。
  + **名称：**按钮名称。
  + **类型：**移动端目前支持链接类型。
  + **链接：输入跳转的完整链接，支持三种跳转方式**

    - http/https的协议：直接填对应的地址就好，如：https://www.taobao.com
    - 微信小程序地址：直接填对应的小程序的页面，如：/pages/index
    - Android/IOS原生页面：Scheme自有协议，如：dingtalk://dingtalkclient/page/conversation?cid=21165
* **搜索配置**

  + **标准功能：**支持搜索联系人和群会话名称。
  + **普通链接**：支持配置链接地址，开发者可自定义开发搜索页面。
