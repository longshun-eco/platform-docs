更新于 2025-09-24本教程介绍了如何创建钉钉客联账号，并建立与钉内用户建立关联关系。

## **接入流程简介**

本文档展示了，创建一个企业内部应用，实现创建钉钉客联账号（即钉外用户），建立与钉内用户建立关联关系等流程：

前提条件：完成[创建应用](/document/dingstart/create-application)的流程且已经开通了钉钉客联。

步骤一：获取应用凭证信息，获取应用 Client ID 和 Client Secret。

步骤二：申请接口权限，申请钉钉客联相关接口的权限。

步骤三：获取应用访问凭证[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)。调用接口时，通过accessToken鉴权调用者身份。

步骤四：调用钉钉客联相关API：

调用服务端API-[创建钉钉客联钉外账号](/document/development/create-bc-account-association)接口。

## **前提条件**

完成[创建应用](/document/dingstart/create-application)的流程。

## **步骤一：获取应用凭证**

1. 选择目标应用，进入应用详情页，单击**基础信息** > **凭证与基础信息**。
2. 获取应用 Client ID 和 Client Secret。

## **步骤二：添加接口权限**

单击**开发配置** > **权限管理**，在权限搜索框中输入`InterConnect.Common.ReadWrite`，并申请权限。

## **步骤三：获取应用访问凭证accessToken**

**重要**

服务端API差异详情参见[旧版服务端API、新版服务端API的区别](/document/development/how-to-call-apis#section-8lr-id4-rbz)。以下接口均使用服务端API接口，SDK下载详情参见[服务端SDK下载](/document/development/download-the-server-side-sdk)。

根据步骤一中的 Client ID 和 Client Secret，获取应用访问凭证[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)。

```java
 public void getAccessToken() throws Exception {
        Config config = new Config();
        config.protocol = "https";
        config.regionId = "central";
        com.aliyun.dingtalkoauth2_1_0.Client client = new com.aliyun.dingtalkoauth2_1_0.Client(config);
        GetAccessTokenRequest accessTokenRequest = new GetAccessTokenRequest()
                .setAppKey("ding********09p")
                .setAppSecret("LEyp8R**********hbYb146msEF");
        GetAccessTokenResponse accessToken = client.getAccessToken(accessTokenRequest);
        System.out.println(JSON.toJSONString(accessToken.getBody()));
    }
```

## **步骤四：调用服务端相关API**

调用服务端API-[创建钉钉客联钉外账号](/document/development/create-bc-account-association)接口，该接口负责创建钉外用户，并建立与钉内用户的关联关系。

```java
public void createUser() throws Exception {
  Config config = new Config();
  config.protocol = "https";
  config.regionId = "central";
  com.aliyun.dingtalkim_1_0.Client client = new com.aliyun.dingtalkim_1_0.Client(config);
  CreateInterconnectionHeaders createInterconnectionHeaders = new CreateInterconnectionHeaders();
  createInterconnectionHeaders.xAcsDingtalkAccessToken = "";
  CreateInterconnectionRequest.CreateInterconnectionRequestInterconnections interconnections = new CreateInterconnectionRequest.CreateInterconnectionRequestInterconnections()
    .setAppUserId("1107****2120")
    .setAppUserName("Foo")
    .setAppUserAvatarMediaType(1)
    .setAppUserAvatar("http://****.png")
    .setAppUserDynamics("认真工作,快乐生活")
    .setAppUserMobile("188****8655")
    .setUserId("1745****8777")
    .setChannelCode("M0U+****8Ep=");
  CreateInterconnectionRequest createInterconnectionRequest = new CreateInterconnectionRequest()
    .setInterconnections(java.util.Arrays.asList(
      interconnections
    ));
```
