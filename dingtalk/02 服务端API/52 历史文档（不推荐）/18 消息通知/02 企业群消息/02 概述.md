更新于 2025-09-08调用接口前，需要添加企业会话接口权限。

发送消息到企业群会话是指可以调用接口创建企业群会话，再以系统名义向企业群会话里推送消息。目前钉钉群共有5类，发送消息到企业群支持的钉钉群类型如下：

| 钉钉群类型 | 是否支持发送系统群消息 |
| --- | --- |
| 内部群 | 是 |
| 全员群 | 是 |
| 部门群 | 是 |
| 普通群 | 否 |
| 合作群 | 否 |

在调用该接口前，确保已开通了**企业会话**接口调用权限。

可参考以下操作进行接口授权：

1. 登录[开发者后台](https://open-dev.dingtalk.com/#/index)。
2. 在企业应用类别中找到已创建的应用，单击目标应用打开详情页。
3. 在**权限管理**页面，搜索**企业群**，然后选择**chat相关接口的管理权限**单击**申请权限**。
