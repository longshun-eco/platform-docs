更新于 2025-09-08调用本接口查询应用信息列表。

**重要**

为提升接口的使用体验，宜搭接口计划升级，后续完善更多功能，重新开放时间请关注文档更新日志。

* 宜搭接口相关文档，已于**2021年12月15日**迁移至**历史文档（不推荐）**目录。
* 不再支持新应用接入，已接入的应用可以正常调用。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | **重要**  暂不支持新增申请。 | 暂不支持 |
| 第三方企业应用 | 支持 | **重要**  暂不支持新增申请。 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/yida/apps/infos/{instanceId}?accessKey=String&pageSize=Integer&callerUid=String&pageNumber=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用调用[获取企业内部应用的accessToken](/document/orgapp/obtain-the-access_token-of-an-internal-app)获取。 * 第三方企业应用调用[获取第三方应用授权企业的accessToken](/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| instanceId | String | 否 | 实例ID。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| accessKey | String | 否 | 访问秘钥。 |
| pageSize | Integer | 否 | 分页大小。 |
| callerUid | String | 否 | 调用者的unionId。 |
| pageNumber | Integer | 否 | 分页页码。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| pageSize | Integer | 分页大小。 |
| applicationInformation | Array | 应用信息列表。 |
| usagePlugins | Array | 插件列表。 |
| pluginName | String | 插件名称。 |
| iconUrl | String | 图标的URL。 |
| appName | String | 应用名称。 |
| appType | String | 应用ID。 |
| instanceUsageAmount | Long | 实例使用数量。 |
| attachmentUsageAmount | Long | 附件使用数量。 |
| pageNumber | Integer | 当前第几页。 |
| totalCount | Integer | 总数量。 |
