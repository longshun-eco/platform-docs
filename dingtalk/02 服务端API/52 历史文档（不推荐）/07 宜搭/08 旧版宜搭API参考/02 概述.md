## 宜搭开放接口介绍

宜搭开放接口是宜搭提供的系统集成的能力之一，主要用于宜搭侧主动发起请求外部第三方系统实现参数校验或者数据同步功能。

开放接口的主要使用场景包括：

* 定时将数据拉回到自建系统中进行精加工或归档。
* 将表单提交、流程发起操作的入口集成到现有平台上。

## 如何使用宜搭开放接口

1. 阿里集团的工作人员请登录[AECP服务开放平台](https://epaas.alibaba-inc.com/acsc/appIndex)自行注册应用，获取应用的accessKey和secretKey。

   非阿里集团工作人员请先[填报申请](https://www.aliwork.com/o/openplatform#/)，申请通过之后会有技术支持同学联系。
2. 能自行注册应用的用户请根据需求在应用中订阅需要访问的接口并申请。需要帮助注册应用的用户请整理需要访问的接口列表发给群内相应的答疑同学统一申请。
3. 下载样例项目 ，获取的应用accessKey和secretKey填入aecpgateway.properties配置文件中。样例项目是独立的Spring boot项目，用户根据需求做取舍。
4. 下载[SDK](https://www.aliwork.com/support/helpDocs?pageName=index&docId=1269097471628280231)。
5. 调用接口，请参考接口说明
