更新于 2026-03-06开发者可以通过钉钉 AI 助理用自定义智能消息回复用户。本文档介绍的接口是回复接口，用户先与聊天对象（比如 AI 助理）发起对话，然后开发者获取到钉钉颁发的会话凭证，然后开发者可以使用会话凭证调用回复接口，回复用户的话。

**重要**

本文档已于 2026年 03 月 05 日迁移至历史文档（不推荐）目录，且本接口仅保持现有功能，不再新增支持其他能力。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | **重要**  暂不支持新增申请 | — |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/aiInteraction/reply HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "conversationToken" : "String",
  "contentType" : "String",
  "content" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| conversationToken | String | 是 | 会话凭证。  **说明**  凭证中保存了会话的上下文信息，包括：聊天场景（如单聊、群聊等）、聊天对象（如 AI 助理、机器人等）、会话发起人信息等。 |
| contentType | String | 是 | 消息体类型：   * **ai\_card：**AI 卡片 |
| content | String | 否 | 消息体内容，与消息体类型对应，必须是 **JSON** 格式。详见[消息体内容填写指南](/document/development/message-content-filling-guidance)。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 请求结果。 |
| success | Boolean | 接口调用是否成功。 |

## 错误码

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | invalid.conversation.token | invalid conversation token | 不合法的会话凭证。 |
| 400 | invalid.content.type | invalid content type | 消息体类型不合法。 |
| 400 | invalid.content | card template not exist | 消息体内容不合法，卡片模板不存在。 |
| 400 | invalid.content | emplateId cannot be null or empty | 消息体内容不合法，卡片模板不能为空。 |
| 400 | invalid.content | templateId in content is not equal to templateId in context | 消息体内容不合法，卡片模板需要全流程一致。 |
| 400 | invalid.content | card streaming key invalid | 消息体内容不合法，流式组件的 Key 不合法。 |
| 400 | invalid.content | invalid componentTag | 消息体内容不合法，卡片组件类型不合法。 |
| 500 | system.error | system error | 未知的系统内部错误。 |
