## **概述**

不同类型的云产品在开通方式上存在差异。根据资源属性，云产品主要分为**资源类**和**服务类**两大类型，其开通使用方法如下所示：

|  |  |  |
| --- | --- | --- |
| **资源类型** | **云产品** | **开通使用方法** |
| **资源类** | 资源类，如ECS，RDS | 无需开通，购买后可直接使用 |
| **服务类** | 经常使用的服务类云产品，见本文档-默认开通列表 | 入驻钉钉云时，默认开通 |
| 开发者后台列出的云产品列表，见本文档-自助开通云产品列表 | 自助开通 |
| 不在默认开通和自助开通列表中的云产品 | 开发者后台申请，需审批通过 |

## 开通资源类的云产品

资源类云产品（如 ECS、RDS）无需进行额外开通操作，购买后即可直接使用。

**重要提示：**当前钉钉云仅支持**华东1地域**。为避免因地域选择错误导致无法接入，请务必在购买资源时仅选择**“华东1”区域**。ESC、RDS、SLB、NAT、EIP的购买可参考：[资源的购买](/document/isvapp/purchase-resources)。

## 开通服务类的云产品

服务类云产品需完成开通流程后方可使用。根据开通机制的不同，可分为三类：**默认开通**、**自助开通** 和 **提交申请开通**。

对于高频使用的常用服务类云产品，钉钉云会提前进行开通，无需手动操作，提前开通的云产品列表如下：

| --- | --- |
| **云产品名称** | **简称** |
| 访问控制 | RAM |
| 专有网络 | VPC |
| 容器服务 | CS |
| 资源编排 | ROS |
| 日志服务 | SLS |
| 云监控 | CMS |
| 云效 | RDC |
| 堡垒机 | BASTION |
| 云防火墙 | CFW |
| 企业级分布式应用服务 | EDAS |

#### **开通步骤**

**说明**

* 请按需开通，避免产生不必要的费用。
* 若开通失败或使用过程中出现权限不足等问题，请点击页面上的 **“联系管理员”** 按钮，通过钉钉与平台管理员即时沟通解决。

1. 登录 [开发者后台](https://open-dev.dingtalk.com)，进入 **入驻钉钉云 > 应用开发 > 管理控制台。**
2. 切换至 **“云产品开通” Tab。**

   ![[development-activate-alibaba-cloud-products_p163356.png]]
3. 在列表中选择需要开通的产品，阅读并同意相关协议。

   ![[development-activate-alibaba-cloud-products_p163357.png]]
4. 点击 **“确定”** 完成开通。

### **方式三：提交申请开通**

若所需使用的云产品**未包含在默认开通或自助开通行列**，您需通过技术支持渠道提交开通申请。

**说明**

为提升审批效率，建议将当前及未来计划使用的云产品**一次性汇总提交**，避免重复申请。

* **申请方式**：通过[技术支持](https://ding-doc.dingtalk.com/document/contactus?pnamespace=app)提交申请。
* **主题**：钉钉云产品开通
* **正文**：

  + **钉钉云账号信息**：dingclouduser@XXXXXXXXXXXXXXX
  + **申请钉钉云的企业名称**：你的企业名称
  + **云产品列表**：希望开通的云产品列表（例如：消息服务MQ、短信服务）
