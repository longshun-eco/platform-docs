**阿里云账号托管为钉钉云账号**

如果应用服务商有阿里云账号，可以将阿里云账号托管为钉钉云账号。托管后的阿里云账号和钉钉云账号保持一致，主账号无法登陆，应用服务商需要使用托管后生成的子账号dingclouduser进行操作，其它安全限制与钉钉云账号的限制相同。满足以下情况建议进行托管：

1. 应用服务商在阿里云上有低于85折的折扣。钉钉云解决方案一般是85折优惠，如果应用服务商已有的折扣更优惠，建议进行托管；如果高于85折优惠，建议购买钉钉云解决方案，不进行托管。
2. 应用服务商已经在阿里云上部署了应用，希望复用阿里云已有部署。
3. 应用服务商希望通过关联已有阿里云账号进行财务结算。
