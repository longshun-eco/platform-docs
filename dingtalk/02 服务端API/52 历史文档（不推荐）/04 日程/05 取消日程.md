жӣҙж–°дәҺ 2025-09-08и°ғз”ЁиҜҘжҺҘеҸЈеҸ–ж¶Ҳж—ҘзЁӢпјҢеҸӘиғҪеҸ–ж¶ҲйҖҡиҝҮ**еҲӣе»әж—ҘзЁӢ**жҺҘеҸЈеҲӣе»әзҡ„ж—ҘзЁӢгҖӮ

**йҮҚиҰҒ**

дёәжҸҗеҚҮжҺҘеҸЈзҡ„дҪҝз”ЁдҪ“йӘҢпјҢжҸҗдҫӣжӣҙеҠ и§„иҢғзҡ„жҺҘеҸЈпјҢй’үй’үй’ҲеҜ№ж—ҘзЁӢзӣёе…іжҺҘеҸЈиҝӣиЎҢдәҶеҚҮзә§гҖӮжң¬жҺҘеҸЈж–ҮжЎЈе·ІдәҺ2021е№ҙ11жңҲ09ж—ҘиҝҒз§»иҮі**еҺҶеҸІж–ҮжЎЈпјҲдёҚжҺЁиҚҗпјү**зӣ®еҪ•дёӢпјҢжҺҘеҸЈдёҚдјҡдёӢзәҝгҖӮ

* ж–°зүҲе’Ңж—§зүҲж—ҘзЁӢAPIзҡ„IDдҪ“зі»дёҚеҗҢпјҢдёҚиғҪж··еҗҲдҪҝз”ЁпјҢжҺЁиҚҗдҪҝз”Ёж–°зүҲж—ҘзЁӢжҺҘеҸЈгҖӮ
* е·ІжҺҘе…Ҙзҡ„з”ЁжҲ·е»әи®®жӮЁж №жҚ®иҮӘиә«е®һйҷ…жғ…еҶөиҜ„дј°жҳҜеҗҰеҲҮжҚўиҮіж–°жҺҘеҸЈгҖӮ
* ж–°жҺҘе…Ҙзҡ„з”ЁжҲ·е»әи®®зӣҙжҺҘдҪҝз”Ёж–°жҺҘеҸЈ[еҲ йҷӨж—ҘзЁӢ](/document/orgapp/delete-event)гҖӮ

## зӨәдҫӢ

**иҜ·жұӮзӨәдҫӢпјҲHTTPпјү**

```http
POST https://oapi.dingtalk.com/topapi/calendar/v2/event/cancel?access_token=ACCESS_TOKEN
```

иҜ·жұӮжӯЈж–Ү

```json
{
    "agentid": 923680251,
    "event_id": "053ExxxxA1FE",
    "calendar_id": "primary"
}
```

**иҜ·жұӮзӨәдҫӢпјҲJAVA SDKпјү**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/calendar/v2/event/cancel");
OapiCalendarV2EventCancelRequest req = new OapiCalendarV2EventCancelRequest();
req.setCalendarId("primary");
req.setEventId("053ExxxxA1FE");
req.setAgentid(923680251L);
OapiCalendarV2EventCancelResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**иҝ”еӣһзӨәдҫӢ**

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "success": true,
    "request_id": "3wiymoeilfb6"
}
```
