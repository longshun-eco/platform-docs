更新于 2025-09-08调用本接口使用小蜜客服机器人的能力进行智能问答。

当用户发送问题给到小蜜客服机器人，小蜜客服机器人会根据已配置的知识库内容和绑定的知识类目回复对应的智能问答内容，包括知识库知识内容、推荐问题、闲聊等。

![[development-alimebot-intelligent-q-a-interface_p238609.png]]

**说明**

* 确保在调用本接口前，已经充分了解小蜜客服产品的收费方式和[价格](https://www.yuque.com/dingdingxiaomikefu/cdvg5o/ptd0qr) 。
* 只有开通并购买过小蜜客服专业版才支持调用本接口，如有需要请先加入小蜜客服服务群联系相关客服。
* 确保在调用本接口前已经配置了知识库，配置方法请参考[知识库配置](https://www.yuque.com/dingdingxiaomikefu/cdvg5o/crgq92?spm=ding_open_doc.document.0.0.72525df4IKXlCO)。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 小蜜客服商业化数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingmi_1.0%23AskRobot) |
| 第三方企业应用 | 暂不支持 | 小蜜客服商业化数据管理权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 小蜜客服商业化数据管理权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/dingmi/robots/ask HTTP/1.1
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "question" : "String",
  "robotAppKey" : "String",
  "sessionUuid" : "String",
  "dingUserId" : "String"
}
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| question | String | 是 | 问题内容。 |
| robotAppKey | String | 是 | 机器人唯一标识。 ![[development-alimebot-intelligent-q-a-interface_O1CN01jLG42m1FocEKmz5yv_!!6000000000534-2-tps-946-264.png]] |
| sessionUuid | String | 否 | 会话ID。 |
| dingUserId | String | 否 | 钉钉用户的userid。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| result | String | 回复的答案，需将示例值转成json。 |
