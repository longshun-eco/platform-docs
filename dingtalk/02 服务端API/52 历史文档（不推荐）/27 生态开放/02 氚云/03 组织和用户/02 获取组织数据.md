更新于 2025-09-08调用本接口获取组织部门数据信息。

**重要**

为了更进一步提升接口质量以及用户体验，我们对本接口文档做出如下调整：

* 自 2024 年 8 月 1 日起，本接口文档将会被迁移至历史文档目录。
* 氚云接口不再支持新应用接入，已接入应用可继续使用，后续若需要接入氚云接口，请使用[氚云开发者手册](https://help.h3yun.com/channels/899.html)。

**说明**

本接口可递归获取指定部门及其下属的子部门等信息。 如果在调用时没有填写**departmentId**那么会默认获取全量组织架构数据。

![[development-queries-organization-data_O1CN01G7msLh1aeB8TjKz6C_!!6000000003354-2-tps-200-358.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 氚云数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=h3yun_1.0%23GetOrganizations) |
| 第三方企业应用 | 支持 | 氚云数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=h3yun_1.0%23GetOrganizations) |
| 第三方个人应用 | 暂不支持 | 氚云数据管理权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/h3yun/departments?departmentId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| departmentId | String | 否 | 部门ID。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| code | String | 状态码。 |
| message | String | 状态码描述。 |
| data | Array | 组织部门信息。 |
| id | String | 部门ID。 |
| parentId | String | 父级部门ID。 |
| name | String | 部门名称。 |
| code | String | 部门编码。 |
| unitType | String | 组织类型，取值：   * **Company**：公司 * **OrganizationUnit**：组织单元 |
| sortKey | Long | 排序值。 |
| description | String | 描述。 |
