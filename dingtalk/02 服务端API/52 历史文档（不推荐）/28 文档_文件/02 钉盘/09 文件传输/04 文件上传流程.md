更新于 2025-09-08你可以通过以下步骤，完成文件上传。

**重要**

为提升接口的使用体验，提供更加规范的接口，钉钉针对**钉盘**产品进行了升级，目前已升级为**存储**产品。本流程文档已于2023年01月17日迁移至**历史文档（不推荐）**目录下，且本流程仅保持现有功能，不再新增支持其他能力。

* 如果未使用本流程内的接口，推荐使用存储产品[获取文件上传信息](/document/orgapp/obtain-storage-upload-information)接口内的文件上传流程。
* 如果已使用本流程内的接口，建议您根据自身实际情况切换至推荐接口。

## 操作步骤

1. 调用[获取文件上传信息](/document/orgapp/obtain-upload-information)接口，获取文件上传临时凭证。
2. 根据返回的文件上传凭证，上传文件到钉盘空间。

   Java

   ```java
   public static void main(String[] args) {

       // 以下参数为步骤1返回的上传凭证
       // 阿里云账号的临时accessKeyId。
       String accessKeyId = "";
       // 阿里云账号的临时accessKeySecret。
       String accessKeySecret = "";
       // 临时访问密钥。
       String securityToken = "";
       // OSS访问域名。
       String endpoint = "";
       // OSS存储空间。
       String bucket = "";
       // 对应OSS Object Key，可用于刷新token以及调用添加文件（夹）接口添加文件记录。
       String ossKey = "";

       CredentialsProvider credentialsProvider = new DefaultCredentialProvider(accessKeyId, accessKeySecret, securityToken);
       ClientConfiguration clientConfiguration = new ClientConfiguration();
       clientConfiguration.setProtocol(Protocol.HTTPS); // 注意, 需要是HTTPS
       OSSClient ossClient = new OSSClient(endpoint, credentialsProvider, clientConfiguration);
       PutObjectRequest putObjectRequest = new PutObjectRequest(bucket, ossKey, new File(""));
   ```

   Python

   ```
   import oss2
   ```

   Go

   ```go
   package main
   ```

   C#

   ```
   // 以下参数为步骤1返回的上传凭证
   ```

   JavaScript

   ```javascript
   const OSS = require('ali-oss')
   ```

   **说明**

   此Demo示例为本地文件上传，更多文件上传方式请参考[OSS文件上传](https://help.aliyun.com/document_detail/31848.html)。
3. 调用[添加文件（夹）](/document/orgapp/add-file)接口添加钉盘文件源信息。
