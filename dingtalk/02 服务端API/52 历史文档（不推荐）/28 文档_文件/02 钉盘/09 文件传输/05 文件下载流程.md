**重要**

为提升接口的使用体验，提供更加规范的接口，钉钉针对**钉盘**产品进行了升级，目前已升级为**存储**产品。本流程文档已于2023年01月17日迁移至**历史文档（不推荐）**目录下，且本流程仅保持现有功能，不再新增支持其他能力。

* 如果未使用本流程内的接口，推荐使用存储产品[获取文件下载信息](/document/orgapp/obtains-the-download-information-about-a-file)接口内的步骤下载文件。
* 如果已使用本流程内的接口，建议您根据自身实际情况切换至推荐接口。
