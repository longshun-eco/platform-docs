更新于 2025-09-08调用本接口，提交文件完成文件上传。

**重要**

为了提升接口的使用体验, 存储管理接口已升级。本文档已于2023年8月6日迁移至历史文档(不推荐)目录，接口不再支持新应用接入, 已接入的应用可继续调用。

* 如果未使用本接口，推荐使用[提交文件](https://open.dingtalk.com/document/orgapp/submittal-file)接口。
* 如果已使用本接口，建议您根据自身实际情况评估是否切换至推荐接口。

### 接口调用步骤

步骤一：调用获取上传信息接口，获取上传文件需要的信息。

* 企业内部应用，调用[获取文件上传信息](https://open.dingtalk.com/document/orgapp/obtain-storage-upload-information)接口。
* 第三方企业应用，调用[获取文件上传信息](https://open.dingtalk.com/document/isvapp/obtain-storage-upload-information)接口。

步骤三：调用本接口提交文件，完成文件上传。

**说明**

* 存储空间类型为USER时，只有空间拥有者和管理者有操作权限，其他员工均需要授权。

  + 企业内部应用，调用[添加权限](https://open.dingtalk.com/document/orgapp/add-storage-permissions)接口进行授权。
  + 第三方企业应用，调用[添加权限](https://open.dingtalk.com/document/isvapp/add-permissions-storage)接口进行授权。
* 存储空间类型为APP时，任何人操作都需要先进行授权。

  + 企业内部应用，调用[添加权限](https://open.dingtalk.com/document/orgapp/add-storage-permissions)接口进行授权。
  + 第三方企业应用，调用[添加权限](https://open.dingtalk.com/document/isvapp/add-permissions-storage)接口进行授权。

| 第三方企业应用 | 支持 | 企业存储文件写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23CommitFile) |
| 第三方个人应用 | 暂不支持 | 企业存储文件写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/storage/spaces/{spaceId}/files/commit?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "uploadKey" : "String",
  "name" : "String",
  "parentId" : "String",
  "option" : {
    "size" : Long,
    "conflictStrategy" : "String",
    "appProperties" : [ {
      "name" : "String",
      "value" : "String",
      "visibility" : "String"
    } ]
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | String | 是 | 空间Id。   * 企业内部应用，调用[添加空间](https://open.dingtalk.com/document/orgapp/add-space)接口获取id参数值。 * 第三方企业应用，调用[添加空间](https://open.dingtalk.com/document/isvapp/add-space)接口获取id参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作者unionId。   * 企业内部应用，调用[查询用户详情](https://open.dingtalk.com/document/orgapp/query-user-details)接口获取unionid参数值。 * 第三方企业应用，调用[查询用户详情](https://open.dingtalk.com/document/isvapp/query-user-details)接口获取unionid参数值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| uploadKey | String | 是 | 添加文件唯一标识。   * 企业内部应用，调用[获取文件上传信息](https://open.dingtalk.com/document/orgapp/obtain-storage-upload-information)接口获取uploadKey参数值。 * 第三方企业应用，调用[获取文件上传信息](https://open.dingtalk.com/document/isvapp/obtain-storage-upload-information)接口获取uploadKey参数值。 |
| name | String | 是 | 文件的名称，带后缀。命名有以下要求：   * 头尾不能包含空格，否则会自动去除 * 不能包含特殊字符，包括：制表符、\*、"、<、>、| * 不能以"."结尾 |
| parentId | String | 是 | 父目录Id。根目录时，该参数是0。   * 企业内部应用，调用[获取文件或文件夹列表](https://open.dingtalk.com/document/orgapp/obtain-the-file-list-storage)接口获取parentId参数值。 * 第三方企业应用，调用[获取文件或文件夹列表](https://open.dingtalk.com/document/isvapp/obtain-the-file-list-storage)接口获取parentId参数值。 |
| option | Object | 否 | 可选参数。 |
| size | Long | 否 | 文件大小，单位Byte。 |
| conflictStrategy | String | 否 | 文件名称冲突策略。   * **AUTO\_RENAME**：自动重命名，默认值 * **OVERWRITE**：覆盖 * **RETURN\_DENTRY\_IF\_EXISTS**：返回已存在文件 * **RETURN\_ERROR\_IF\_EXISTS**：文件已存在时报错 |
| appProperties | Array | 否 | 当前文件的应用属性列表，最大值3。 |
| name | String | 是 | 属性名。 |
| value | String | 是 | 属性值。 |
| visibility | String | 是 | 属性可见性。   * PUBLIC：所有应用都可见 * PRIVATE：仅限当前应用可见 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| dentry | Object | 文件信息。 |
| id | String | 文件Id。 |
| spaceId | String | 所在空间Id。 |
| parentId | String | 父目录Id。根目录时，该参数是0。 |
| type | String | 类型，本接口返回类型只有FILE。   * **FILE**：文件 |
| name | String | 文件名称。 |
| size | Long | 文件大小，单位Byte。 |
| path | String | 文件在空间内的路径。 |
| version | Long | 版本。 |
| status | String | 状态。   * **NORMAL**：正常 * **DELETED**：已删除 * **EXPIRED**：已过期 |
| extension | String | 文件后缀。 |
| creatorId | String | 创建者unionId。 |
| modifierId | String | 修改者unionId。 |
| createTime | String | 创建时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| modifiedTime | String | 修改时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| properties | Object | 文件属性。 |
| readOnly | Boolean | 文件是否只读。   * **true**：是 * **false**：否 |
| appProperties | Map<String, Array> | 在特定应用上的属性。key是所属空间的ownerId，value是属性列表。 |
|  | Array | 属性列表。 |
| name | String | 属性名。 |
| value | String | 属性值。 |
| visibility | String | 属性可见性。   * PUBLIC：所有应用都可见 * PRIVATE：仅限当前应用可见 |
| uuid | String | 唯一标识。 |
| partitionType | String | 存储分区。   * **PUBLIC\_OSS\_PARTITION**：公有云OSS存储分区 * **MINI\_OSS\_PARTITION**：专属MiniOSS存储分区 |
| storageDriver | String | 驱动类型。   * **DINGTALK**：钉钉统一存储驱动 * **ALIDOC**：钉钉文档存储驱动 * **UNKNOWN**：未知驱动 |
