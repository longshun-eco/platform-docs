更新于 2025-09-08调用本接口，获取文件上传信息。

**重要**

为了提升接口的使用体验, 存储接口已升级。本文档已于2023年8月6日迁移至历史文档目录。

* 如果使用入参为dentryUuid或者Uuid字段，推荐使用[获取文件上传信息](https://open.dingtalk.com/document/orgapp/obtain-file-upload-informations)接口。
* 如果使用入参为spaceId字段，推荐使用本接口。
* 如果已使用本接口，建议您根据自身实际情况评估是否切换至推荐接口。

### 接口调用流程

步骤一：调用本接口，获取上传文件需要的`resourceUrls`和`headers`参数值。

步骤二：使用OSS的header加签方式上传文件，参考示例：

                Map headers = 第一步接口返回的headers;
                URL url = new URL(resourceUrl);
                HttpURLConnection connection = (HttpURLConnection)url.openConnection();

                for (Map.Entry entry : headers.entrySet()) {
                    connection.setRequestProperty(entry.getKey(), entry.getValue());
                }

                connection.setDoOutput(true);
                connection.setRequestMethod("PUT");
                connection.setUseCaches(false);
                connection.setReadTimeout(10000);
                connection.setConnectTimeout(10000);
                connection.connect();

步骤三：调用提交文件接口，完成文件上传。

* 企业内部应用，调用[提交文件](https://open.dingtalk.com/document/orgapp-server/submit-documents)接口。
* 第三方企业应用，调用[提交文件](https://open.dingtalk.com/document/isvapp-server/submit-documents)接口。

**说明**

* 存储空间类型为USER时，只有空间拥有者和管理者有操作权限，其他员工均需要授权。

  + 企业内部应用，调用[添加权限](https://open.dingtalk.com/document/orgapp/add-storage-permissions)接口进行授权。
  + 第三方企业应用，调用[添加权限](https://open.dingtalk.com/document/isvapp/add-permissions-storage)接口进行授权。
* 存储空间类型为APP时，任何人操作都需要先进行授权。

  + 企业内部应用，调用[添加权限](https://open.dingtalk.com/document/orgapp/add-storage-permissions)接口进行授权。
  + 第三方企业应用，调用[添加权限](https://open.dingtalk.com/document/isvapp/add-permissions-storage)接口进行授权。

## 请求方法

```
POST /v1.0/storage/spaces/{spaceId}/files/uploadInfos/query?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "protocol" : "String",
  "multipart" : Boolean,
  "option" : {
    "storageDriver" : "String",
    "preCheckParam" : {
      "md5" : "String",
      "size" : Long,
      "parentId" : "String",
      "name" : "String"
    },
    "preferRegion" : "String",
    "preferIntranet" : Boolean
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | String | 是 | 空间Id。   * 企业内部应用，调用[添加空间](https://open.dingtalk.com/document/orgapp/add-space)接口获取id参数值。 * 第三方企业应用，调用[添加空间](https://open.dingtalk.com/document/isvapp/add-space)接口获取id参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作者unionId。   * 企业内部应用，调用[查询用户详情](https://open.dingtalk.com/document/orgapp/query-user-details)接口获取unionid参数值。 * 第三方企业应用，调用[查询用户详情](https://open.dingtalk.com/document/isvapp/query-user-details)接口获取unionid参数值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| protocol | String | 是 | 通过指定上传协议返回不同协议上传所需要的信息。   * **HEADER\_SIGNATURE**：Header加签 |
| multipart | Boolean | 是 | 是否需要分片上传。   * **true**：需要 * **false**：不需要   **说明**  * 5G以下文件，设为false，简化上传步骤。 * 5G以上文件，必须设为true，否则会上传失败。 |
| option | Object | 否 | 可选参数。 |
| storageDriver | String | 否 | 文件存储驱动类型。   * **DINGTALK**：钉钉统一存储驱动，默认值 * **ALIDOC**：钉钉文档存储驱动 * **UNKNOWN**：未知驱动 |
| preCheckParam | Object | 否 | 预检查的字段。  **说明**  可实现对文件名称，文件完整性，容量的校验。 |
| md5 | String | 否 | 文件md5值，`DigestUtils.md5Hex()`。  **说明**  文件完整性校验，不传则不做校验。 |
| size | Long | 否 | 文件大小，单位Byte。  **说明**  容量相关校验，不传则不做校验。 |
| parentId | String | 否 | 父目录Id。根目录时，该参数是0。  **说明**  用于同目录文件名冲突校验。 |
| name | String | 否 | 文件名称，命名有以下要求：   * 头尾不能包含空格，否则会自动去除 * 不能包含特殊字符，包括：制表符、\*、"、<、>、| * 不能以"."结尾   > 文件名称合法性和文件名称冲突校验。 |
| preferRegion | String | 否 | 优先地域。   * **ZHANGJIAKOU**：张家口 * **SHENZHEN**：深圳 * **SHANGHAI**：上海 * **SINGAPORE**：新加坡 * **UNKNOWN**：未知   **说明**  倾向于将资源存到哪个地域，可实现就近上传等功能。 |
| preferIntranet | Boolean | 否 | 是否优先使用内网传输，使用该参数前提是配置了专属存储内网传输。   * true：是，默认值 * false：否 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| uploadKey | String | 上传唯一标识。 |
| storageDriver | String | 文件存储类型。   * **DINGTALK**：钉钉统一存储驱动 * **ALIDOC**：钉钉文档存储驱动 * **UNKNOWN**：未知驱动 |
| protocol | String | 上传协议。   * **HEADER\_SIGNATURE**：Header加签 |
| headerSignatureInfo | Object | Header加签上传信息。  **说明**  当protocol参数传HEADER\_SIGNATURE时，返回该字段。 |
| resourceUrls | Array of String | 多个上传下载URL列表。 |
| headers | Map<String, String> | 请求头信息。 |
| expirationSeconds | Integer | 过期时间，单位秒。 |
| region | String | 地域。   * **ZHANGJIAKOU**：张家口 * **SHENZHEN**：深圳 * **SHANGHAI**：上海 * **SINGAPORE**：新加坡 * **UNKNOWN**：未知 |
| internalResourceUrls | Array of String | 内网访问地址。  **说明**  该字段目前无使用场景，请忽略。 |
