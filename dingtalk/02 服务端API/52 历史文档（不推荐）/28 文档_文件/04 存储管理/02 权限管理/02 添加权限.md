更新于 2025-09-08调用本接口，添加存储空间的权限。

**重要**

1. 为了提升接口的使用体验, 存储接口已升级。本文档已于2023年7月6日迁移至历史文档(不推荐)目录，接口不再支持新应用接入, 已接入的应用可继续调用。

   * 如果未使用本接口，推荐使用[添加权限](https://open.dingtalk.com/document/orgapp/add-permissions-file)接口。
   * 如果已使用本接口，建议您根据自身实际情况评估是否切换至推荐接口。
2. 新老接口中的ID不兼容, 不支持混用。

### 接口功能介绍

调用本接口，实现将对应员工添加某个存储空间的权限。

* 如果是添加USER类型的存储空间权限，授权时长是永久有效，接口参数duration不生效。
* 如果是添加APP类型的存储空间权限，授权时长是接口参数duration的值。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业存储授权信息写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=storage_1.0%23AddPermission) |
| 第三方企业应用 | 支持 | 企业存储授权信息写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=storage_1.0%23AddPermission) |
| 第三方个人应用 | 暂不支持 | 企业存储授权信息写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/storage/spaces/{spaceId}/dentries/{dentryId}/permissions?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "roleId" : "String",
  "members" : [ {
    "type" : "String",
    "id" : "String",
    "corpId" : "String"
  } ],
  "option" : {
    "duration" : Long
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | String | 是 | 空间Id。   * 企业内部应用，调用[添加空间](https://open.dingtalk.com/document/orgapp/add-space)接口获取id参数值。 * 第三方企业应用，调用[添加空间](https://open.dingtalk.com/document/isvapp/add-space)接口获取id参数值。 |
| dentryId | String | 是 | 文件或者文件夹Id。  **说明**  如果对整个空间授权，该参数值传0。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作用户的unionId。   * 企业内部应用，调用[查询用户详情](https://open.dingtalk.com/document/orgapp/query-user-details)接口获取unionid参数值。 * 第三方企业应用，调用[查询用户详情](https://open.dingtalk.com/document/isvapp/query-user-details)接口获取unionid参数值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| roleId | String | 是 | 权限角色Id。   * **OWNER**：拥有者 * **MANAGER**：管理者 * **EDITOR**：编辑者 * **DOWNLOADER**：下载者 * **READER**：查看者 |
| members | Array | 是 | 权限成员列表，最大值30。 |
| type | String | 是 | 权限成员类型：   * **ORG**：企业 * **DEPT**：部门 * **TAG**：自定义tag * **CONVERSATION**：会话 * **USER**：用户 |
| id | String | 是 | 权限成员id。   * 如果type参数值为**ORG**，该参数值传企业**corpId**。 ![[development-add-storage-permissions_O1CN01uOhCBe1DpPjVsRHYT_!!6000000000265-2-tps-2794-1066.png]] * 如果type参数值为**DEPT**，该参数值传部门**deptId**。    + 企业内部应用，调用[获取部门列表](https://open.dingtalk.com/document/orgapp/obtain-the-department-list-v2)接口获取dept\_id参数值。   + 第三方企业应用，调用[获取部门列表](https://open.dingtalk.com/document/isvapp/get-department-list)接口获取dept\_id参数值。 * 如果type参数值为**TAG**，该参数值传**tag名称**。 * 如果type参数值为**CONVERSATION**，该参数值传会话**openConversationId**。    + 企业内部应用：      - 通过[创建群会话](https://open.dingtalk.com/document/orgapp/create-group-session)接口获取openConversationId参数值。     - 通过[创建场景群](https://open.dingtalk.com/document/orgapp/create-a-scene-group-v2)接口获取open\_conversation\_id参数值。   + 第三方企业应用，通过[创建场景群](https://open.dingtalk.com/document/isvapp/create-a-scene-group-v2)接口获取open\_conversation\_id参数值。 * 如果type参数值为**USER**，该参数传用户**unionId**。    + 企业内部应用，调用[查询用户详情](https://open.dingtalk.com/document/orgapp/query-user-details)接口获取unionid参数值。   + 第三方企业应用，调用[查询用户详情](https://open.dingtalk.com/document/isvapp/query-user-details)接口获取unionid参数值。 |
| corpId | String | 否 | 权限归属的企业corpId。   * type为DEPT或TAG时，该参数必填。 * type为USER时，该参数选填。   **说明**  传入该参数，表示对应type下的员工在离职时，会自动清理权限。  ![[development-add-storage-permissions_O1CN01uOhCBe1DpPjVsRHYT_!!6000000000265-2-tps-2794-1066.png]] |
| option | Object | 否 | 可选参数。 |
| duration | Long | 否 | 授权有效时长，单位秒，仅支持APP空间类型。   * 被添加权限的空间类型为**APP**时，该字段有效，最大3600。 * 被添加权限的空间类型为**USER**时，该字段无效，默认永久有效。 |
