更新于 2025-09-08调用本接口，创建组织下的知识库。

**重要**

1. 为了提升接口的使用体验, 存储接口已升级。本文档已于2023年7月6日迁移至历史文档(不推荐)目录，接口不再支持新应用接入, 已接入的应用可继续调用。

   * 如果未使用本接口，推荐使用[新建知识库](https://open.dingtalk.com/document/orgapp/new-knowledge-base)接口
   * 如果已使用本接口，建议您根据自身实际情况评估是否切换至推荐接口。
2. 新老接口中的ID不兼容, 不支持混用。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 知识库写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_1.0%23CreateWorkspace) |
| 第三方企业应用 | 支持 | 知识库写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=doc_1.0%23CreateWorkspace) |
| 第三方个人应用 | 暂不支持 | 知识库写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/doc/workspaces HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "name" : "String",
  "description" : "String",
  "operatorId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| name | String | 是 | 知识库名称。 |
| description | String | 否 | 知识库描述。 |
| operatorId | String | 是 | 发起添加权限操作用户的unionId。   * 企业内部应用，调用[查询用户详情](https://open.dingtalk.com/document/orgapp/query-user-details)接口获取unionid参数值。 * 第三方企业应用，调用[查询用户详情](https://open.dingtalk.com/document/isvapp/query-user-details)接口获取unionid参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| workspaceId | String | 工作空间ID。 |
| name | String | 工作空间名称。 |
| description | String | 工作空间描述。 |
| url | String | 工作空间打开URL。 |

## 错误码

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | invalidRequest.workspace.nameConflict | 知识库名称冲突 | 知识库名称冲突 |
| 400 | invalidRequest.inputArgs.invalid | 方法入参校验失败 | 方法入参校验失败，检查是否有必填参数未填，或者unionId是否合法等 |
| 403 | forbidden.user.notInOrg | 用户不在组织内 | 操作用户不在组织内 |
| 403 | forbidden.accessDenied | 当前用户无此操作权限 | 当前用户无此操作权限 |
| 500 | internalError | 系统内部错误 | 系统内部错误 |
