更新于 2025-09-08调用本接口，查询知识库内的节点信息，包括节点类型、节点名称等信息。

**重要**

1. 为了提升接口的使用体验, 存储接口已升级。本文档已于2023年7月6日迁移至历史文档(不推荐)目录，接口不再支持新应用接入, 已接入的应用可继续调用。

   * 如果未使用本接口，推荐使用[获取节点](https://open.dingtalk.com/document/orgapp/get-node)接口。
   * 如果已使用本接口，建议您根据自身实际情况评估是否切换至推荐接口。
2. 新老接口中的ID不兼容, 不支持混用。

### 接口功能介绍

例如，测试知识库的目录结构如下图所示。调用本接口，可获取该节点的信息，包括节点类型、节点ID、节点名称等。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 团队空间文档读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_2.0%23QueryDentry) |
| 第三方企业应用 | 支持 | 团队空间文档读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=doc_2.0%23QueryDentry) |
| 第三方个人应用 | 暂不支持 | 团队空间文档读权限 | 暂不支持 |

## 请求方法

```
GET /v2.0/doc/spaces/{spaceId}/dentries/{dentryId}?operatorId=String&includeSpace=Boolean HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| spaceId | String | 是 | 知识库ID。   * 企业内部应用，调用[新建知识库](https://open.dingtalk.com/document/orgapp/create-a-team-space)接口或者[查询用户有权限的知识库列表](https://open.dingtalk.com/document/orgapp/querying-the-list-of-user-team-spaces)接口获取的workspaceId字段值。 * 第三方企业应用，调用[新建知识库](https://open.dingtalk.com/document/isvapp/create-a-team-space)接口或者[查询用户有权限的知识库列表](https://open.dingtalk.com/document/isvapp/querying-the-list-of-user-team-spaces)接口获取的workspaceId字段值。 |
| dentryId | String | 是 | 节点ID。   * 企业内部应用，调用[查询知识库下的目录结构](https://open.dingtalk.com/document/orgapp/query-the-directory-tree-in-the-knowledge-base)接口获取DentryModel参数值。 * 第三方企业应用，调用[查询知识库下的目录结构](https://open.dingtalk.com/document/isvapp/query-the-directory-structure-in-the-knowledge-base)接口获取DentryModel参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorId | String | 是 | 操作人unionId。   * 企业内部应用，调用[查询用户详情](https://open.dingtalk.com/document/orgapp/query-user-details)接口获取unionid参数值。 * 第三方企业应用，调用[查询用户详情](https://open.dingtalk.com/document/isvapp/query-user-details)接口获取unionid参数值。 |
| includeSpace | Boolean | 否 | 是否查询知识库信息，默认不查询。   * **true**：查询 * **false**：不查询   **说明**  如果该参数指定为true，返回结果会包含space字段，表示当前知识库的基础信息。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
|  | DentryVO | 节点信息。   * 企业内部应用，请参考[知识库通用数据格式-DentryVO](https://open.dingtalk.com/document/orgapp/common-data-structure)。 * 第三方企业应用，请参考[知识库通用数据格式-DentryVO](https://open.dingtalk.com/document/isvapp/common-data-structure)。 |
