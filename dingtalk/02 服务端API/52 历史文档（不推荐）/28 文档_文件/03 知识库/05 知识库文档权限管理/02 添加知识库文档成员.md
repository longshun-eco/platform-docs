更新于 2025-09-08调用本接口，可以添加用户对知识库文档的权限。

**重要**

1. 为了提升接口的使用体验, 知识库接口已升级。本文档已于2023年7月6日迁移至历史文档(不推荐)目录，接口不再支持新应用接入, 已接入的应用可继续调用。

   * 如果未使用本接口，推荐使用[添加权限](https://open.dingtalk.com/document/orgapp/add-permissions-file)接口。
   * 如果已使用本接口，建议您根据自身实际情况评估是否切换至推荐接口。
2. 新老接口中的ID不兼容, 不支持混用。

![[development-add-workspace-document-user-permissions_O1CN01uS0saX1DwHg0xMAex_!!6000000000280-2-tps-2794-1582.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 知识库文档写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=doc_1.0%23AddWorkspaceDocMembers) |
| 第三方企业应用 | 支持 | 知识库文档写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=doc_1.0%23AddWorkspaceDocMembers) |
| 第三方个人应用 | 暂不支持 | 知识库文档写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/doc/workspaces/{workspaceId}/docs/{nodeId}/members HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "operatorId" : "String",
  "members" : [ {
    "memberId" : "String",
    "memberType" : "String",
    "roleType" : "String"
  } ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| workspaceId | String | 是 | 知识库ID。   * 企业内部应用，调用[新建知识库](https://open.dingtalk.com/document/orgapp/create-a-team-space)接口或者[查询用户有权限的知识库列表](https://open.dingtalk.com/document/orgapp/querying-the-list-of-user-team-spaces)接口获取的workspaceId字段值。 * 第三方企业应用，调用[新建知识库](https://open.dingtalk.com/document/isvapp/create-a-team-space)接口或者[查询用户有权限的知识库列表](https://open.dingtalk.com/document/isvapp/querying-the-list-of-user-team-spaces)接口获取的workspaceId字段值。 |
| nodeId | String | 是 | 知识库文档ID。   * 企业内部应用，通过[创建知识库文档](https://open.dingtalk.com/document/orgapp/create-team-space-document)接口获取nodeId参数值。 * 第三方企业应用，通过[创建知识库文档](https://open.dingtalk.com/document/isvapp/create-team-space-document)接口获取nodeId参数值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorId | String | 是 | 发起添加权限操作用户的unionId。   * 企业内部应用，调用[查询用户详情](https://open.dingtalk.com/document/orgapp/query-user-details)接口获取unionid参数值。 * 第三方企业应用，调用[查询用户详情](https://open.dingtalk.com/document/isvapp/query-user-details)接口获取unionid参数值。 |
| members | Array | 是 | 被操作用户组。  **说明**  可一次操作多个用户，最大数量限制50，超过数量限制会直接报错。 |
| memberId | String | 是 | 被添加的文档成员。   * memberType如果是**USER**，该参数值传员工的unionId。    + 企业内部应用，调用[查询用户详情](https://open.dingtalk.com/document/orgapp/query-user-details)接口获取unionid参数值。   + 第三方企业应用，调用[查询用户详情](https://open.dingtalk.com/document/isvapp/query-user-details)接口获取unionid参数值。 * memberType如果是**DEPT**，该参数值是传部门ID。    + 企业内部应用，调用[获取部门列表](https://open.dingtalk.com/document/orgapp/obtain-the-department-list-v2)接口获取dept\_id参数值。   + 第三方企业应用，调用[获取部门列表](https://open.dingtalk.com/document/isvapp/get-department-list)接口获取dept\_id参数值。 |
| memberType | String | 是 | 用户类型，取值：   * **USER**：用户 * **DEPT**：部门 |
| roleType | String | 是 | 用户权限，取值：   * **ONLY\_VIEWER**：只读 * **VIEWER**：可查看和下载 * **EDITOR**：可编辑 |
