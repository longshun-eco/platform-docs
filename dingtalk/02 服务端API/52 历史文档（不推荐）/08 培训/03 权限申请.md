## 权限申请

在调用培训接口前，添加培训相关接口权限：

1. 登录[开发者后台](https://open-dev.dingtalk.com/#/appMgr/provider/eapp/53642/1)。
2. 选择**应用开发**，然后单击**第三方企业应用**，最后找到对应的应用，单击进入应用详情页
3. 在**权限管理**页面，搜索**培训。**

   ![[development-apply-for-permissions-training_p361703.png]]
4. 然后单击申请权限，提供申请理由，点击申请，申请通过后方可调用培训接口。

**说明**

如果接口申请被拒绝，找到对应的接口权限，然后将鼠标移至**审核未通过**上查看拒绝原因。
