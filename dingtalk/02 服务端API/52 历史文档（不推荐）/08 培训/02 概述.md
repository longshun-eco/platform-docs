## 什么是培训

针对企业培训场景，钉钉提供多群联播、直播的能力以及相应直播数据查询的能力。企业可以通过直播数据了解培训效果。

钉钉直播具有直播预约、权限自由设置、多群联播、实时互动、直播录制、直播数据统计等完善功能，打破时间和空间上的限制，方便的完成一场直播活动，高效上传下达，改变传统培训、管理模式。对于企业培训、企业招聘、各类活动会议、校园课堂等多种场景，钉钉直播能够轻松搞定，开启企业直播新方式，同时提供专业的重保服务，确保现场执行团队顺利部署及调试设备，保障直播线上到线下的整体质量和体验。

## 接入流程

1. 登录[开发者后台](https://open-dev.dingtalk.com/)，创建企业三方企业应用。
2. 进入应用详情，点击权限管理，搜索**培训**，点击申请对应权限并提交审批单。审批单审核通过后即可调用。

   ![[development-live-overview_p361702.png]]
3. 调用[创建培训课程](/document/isvapp/create-live-courses)接口，联播到绑定的群，如下图所示：

   ![[development-live-overview_b5c67fce7e8b4316b2fd9760f03c6a632460.png]]

   例如在云课堂日常预发开发企业组织下的内部群：培训群中要开一场直播培训课。

   创建好直播后可以得到唯一的课程ID，然后通过构造客户端链接的方式，钉钉用户可以访问链接进入该直播，示例如下

   ```
   dingtalk://dingtalkclient/action/start_uniform_live?liveUuid=xxx
   ```

   **说明**

   liveUuid通过[创建培训课程](/document/isvapp/create-live-courses)接口获取。

   开始直播后对应联播群的所有群成员就会收到开播提醒，如下图所示：

   ![[development-live-overview_6f879c7e114a44d9a0930d14b66b157c2460.png]]
4. 群成员点击进入观看直播，如下图所示：

   ![[development-live-overview_p413335.png]]
5. 直播结束后可以用课程id、主播id、群id等参数调用[获取培训观看数据](/document/isvapp/obtains-the-playback-data-of-a-live-stream)和[获取培训课程的基本信息](/document/isvapp/get-basic-information-about-the-course)接口查看直播的具体信息，以及当前群中观众的观看信息等。
