更新于 2025-11-14本教程介绍了如何创建快捷入口，并跳转到实现免登流程的企业内部应用。本文以企业内部应用-H5微应用为例说明详细流程。

## 接入流程

1. 创建企业内部应用。

   （1）登录[钉钉开发者后台](https://open-dev.dingtalk.com/#/index)。

   （2）在开发者后台页面，选择企业内部开发，然后单击创建应用。

   （3）创建应用页面填写基本信息，确定创建。

   （4）在**基础信息**中获取AppKey、AppSecret。

   （5）在**开发管理**中设置开发信息

   （6）在**权限管理**中申请对应权限。
2. 企业内部应用实现免登。

   （1）调用[客户端-H5微应用免登](/document/orgapp/logon-free-process)JSAPI，获取免登授权码。

   （2）调用[服务端-获取应用访问凭证ccess\_token](/document/orgapp/obtain-orgapp-token)接口。调用接口时，通过accessToken鉴权调用者身份。

   （3）调用[服务端-通过免登码获取用户信息](/document/orgapp/obtain-the-userid-of-a-user-by-using-the-log-free)接口，获取用户的userId。

   （4）调用[服务端-查询用户详情](/document/orgapp/query-user-details)接口，获取登录用户的详细信息。
3. 创建快捷入口（群插件）。

   （1）登录[钉钉开发者后台](https://open-dev.dingtalk.com/#/index)。

   （2）在开发者后台页面，选择场景群，点击创建群插件。

   （3）在开发者后台页面，新建群插件并点击新建，等待审核通过。
4. 创建群模板。

   （1）登录[钉钉开发者后台](https://open-dev.dingtalk.com/#/index)。

   （2）在开发者后台页面，选择场景群，点击创建群模板。

   （3）在开发者后台页面，创建群模板，获取模板ID参数**template\_id**。

   （4）在开发者后台页面，进入对应群模板，提交审核并发布。
5. 创建群聊会话（场景群）。

   （1）调用[服务端-获取应用访问凭证ccess\_token](/document/orgapp/obtain-orgapp-token)，通过accessToken鉴权调用者身份。

   （2）根据**template\_id**调用[服务端-创建群](/document/orgapp/create-a-scene-group-v2)接口，实现群模板及快捷入口的嵌入。
6. 使用快捷入口。

## 准备工作

在开始本教程前，确保你已经完成了以下准备工作：

* 需要成为钉钉开发者，详情请参考[成为钉钉开发者](/document/orgapp/become-a-dingtalk-developer)。
* 确保您已经安装了Java开发环境（安装JDK）以及Java项目构建工具Maven。

## 步骤一：创建企业内部应用

在本部分，你需要在开发者后台创建一个企业内部应用-H5微应用，并完成场景群权限的申请，用于场景群的创建。

1. 登录[钉钉开发者后台](https://open-dev.dingtalk.com/#/index)。

   **说明**

   只有管理员和子管理员可登录开发者后台，添加管理员流程请查看[成为钉钉开发者](/document/orgapp/become-a-dingtalk-developer)。
2. 在**开发者后台**页面，选择**企业内部开发**，然后单击**创建应用**。

   ![[development-the-group-plug-in-implements-the-logon-free-process_p344004.png]]
3. 在弹出的创建应用页面中填写基本信息，单击确定创建。

   * 应用类型：选择H5微应用。
   * 开发方式：选择企业自主开发。

     ![[development-the-group-plug-in-implements-the-logon-free-process_p348879.png]]
4. 应用创建完成后，在**基础信息**页面，复制应用的**AppKey**和**AppSecret**备用。

   ![[development-the-group-plug-in-implements-the-logon-free-process_p344327.png]]
5. 单击**开发管理**进入开发管理页面，单击**修改**，并根据以下内容配置开发信息。

   * **开发模式**：选择**开发应用**。
   * **服务器出口IP**：输入调用钉钉服务端API时使用的IP即企业服务器的公网IP，多个IP请以英文逗号","隔开，支持带一个\*号通配符的IP格式。

     本教程设置为`127.0.0.1`。
   * **应用首页地址**：输入应用首页URL，在移动端工作台点击应用图标会跳转到此页面。可输入后端服务部署的服务器的IP或域名。例如：`http://公网IP:8080`。

     ![[development-the-group-plug-in-implements-the-logon-free-process_p377713.png]]
6. 单击**权限管理**进入权限管理页面，参考以下配置添加接口调用权限。

   1. 权限范围选择**全部员工**。
   2. 选择**通讯录管理**，申请**成员信息读权限**。

      ![[development-the-group-plug-in-implements-the-logon-free-process_p377766.png]]
   3. 选择**场景群**，申请**chat相关接口的管理权限**和**chat相关接口的读取权限。**

      ![[development-the-group-plug-in-implements-the-logon-free-process_p367765.png]]

## 步骤二：实现免登

**说明**

实现免登需在钉钉容器内进行。

1. 调用[客户端-H5微应用免登](/document/orgapp/logon-free-process)JSAPI，获取免登授权码。
2. 调用[服务端-获取应用访问凭证ccess\_token](/document/orgapp/obtain-orgapp-token)接口。调用接口时，通过accessToken鉴权调用者身份。
3. 调用[服务端-通过免登码获取用户信息](/document/orgapp/obtain-the-userid-of-a-user-by-using-the-log-free)接口，获取用户的userId。
4. 调用[服务端-查询用户详情](/document/orgapp/query-user-details)接口，获取登录用户的详细信息。

## 步骤三：创建快捷入口（群插件）

## 步骤四：创建群模板

1. 登录[钉钉开发者后台](https://open-dev.dingtalk.com/#/index)。
2. 在**开发者后台**页面，选择**场景群**，点击创建**群模板**。

   ![[development-the-group-plug-in-implements-the-logon-free-process_p367780.png]]
3. 创建群模板，获取模板ID参数**template\_id**。

   （1）选择模板。

   ![[development-the-group-plug-in-implements-the-logon-free-process_p368537.png]]

   （2）填写模板信息，保存编辑，返回群场景列表。

   ![[development-the-group-plug-in-implements-the-logon-free-process_p368032.png]]
4. 进入对应群模板，提交审核并发布。

   **说明**

   审核自动通过，无需审批。

   ![[development-the-group-plug-in-implements-the-logon-free-process_p367977.png]]

## 步骤五：创建群聊会话（场景群）

1. 调用接口[服务端-获取应用访问凭证ccess\_token](/document/orgapp/obtain-orgapp-token)获取应用访问凭证。调用快捷入口相关接口时，通过accessToken鉴权调用者身份。
2. 根据**template\_id**调用[服务端-创建群](/document/orgapp/create-a-scene-group-v2)接口，实现群模板及快捷入口的嵌入。

   **说明**

   调用[启用群模板](/document/orgapp/enable-a-group-template)接口前，需要设置灰度企业。
