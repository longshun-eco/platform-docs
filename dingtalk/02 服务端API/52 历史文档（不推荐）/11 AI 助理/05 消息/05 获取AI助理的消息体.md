更新于 2026-03-06根据线程id和消息id，返回对应的消息。访问本接口请申请应用权限进行访问。

**重要**

本文档已于 2026年 03 月 05 日迁移至历史文档（不推荐）目录，且本接口仅保持现有功能，不再新增支持其他能力。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | **重要**  暂不支持新增申请 | — |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/assistant/threads/{threadId}/messages/{messageId} HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| threadId | String | 是 | 消息关联的线程id。 |
| messageId | String | 是 | 消息id。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| id | String | 消息id。 |
| threadId | String | 消息关联的线程id。 |
| createdAt | Long | 消息创建时间，毫秒时间戳。 |
| role | String | 消息角色，枚举值：   * user：用户 * assistant：助理 * tool: 技能 |
| content | Array of Any | 返回的消息内容。  **说明**  如果消息格式是text，则其中map的key值就是text，其中的value对应的是user或者assistant、tool角色的文本内容，annotations为消息的扩展字段，其中的tool表示运行过程中会调用的工具名称。 |
| assisantId | String | 消息关联的助理id。 |
| runId | String | 消息关联的运行任务id。 |
| object | String | 对象类型，默认为thread.message。 |
| metadata | Map | 保留字段，用于扩展使用，暂无实际意义。 |
