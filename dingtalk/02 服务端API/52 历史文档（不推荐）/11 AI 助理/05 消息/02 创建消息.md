更新于 2026-03-06在Thread内创建消息，提供给大模型进行推理使用。

**重要**

本文档已于 2026年 03 月 05 日迁移至历史文档（不推荐）目录，且本接口仅保持现有功能，不再新增支持其他能力。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | **重要**  暂不支持新增申请 | — |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/assistant/threads/{threadId}/messages HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "role" : "String",
  "content" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| threadId | String | 是 | 关联的线程id。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| role | String | 是 | 角色，枚举值：user 。  **说明**  用户输入必须要是 user 角色属性。 |
| content | String | 是 | 用户的输入内容，最多可输入4000个字符。 |
| metadata | Map | 否 | 保留字段，用于扩展使用，暂不支持使用，如果传入值，不会保留相关数据。 |
| extension | Map<String, String> | 否 | 保留字段，用于扩展使用，暂不支持使用，如果传入值，不会保留相关数据。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| id | String | 消息id。 |
| threadId | String | 消息关联的线程id。 |
| createdAt | Long | 消息创建时间，单位：毫秒时间戳。 |
| role | String | 消息角色：   * **user**：用户 * **assistant**：助理 * **tool**：技能 |
| content | Array of Any | 返回的消息内容。  **说明**  如果消息格式是 text，则其中 map 的 key 值就是 text，其中的 value 对应的是 user 或者 assistant、tool 角色的文本内容，annotations 为消息的扩展字段，其中的 tool 表示运行过程中会调用的工具名称。 |
| assistantId | String | 消息关联的助理id。 |
| runId | String | 消息关联的运行任务id。 |
| object | String | 对象类型，一直为 thread.message。 |
| metadata | Map | 保留字段，用于扩展使用，暂无内容。 |
