更新于 2026-03-06创建AI助理的运行任务，并获取线程下的消息，进行大模型的推理。

**重要**

本文档已于 2026年 03 月 05 日迁移至历史文档（不推荐）目录，且本接口仅保持现有功能，不再新增支持其他能力。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | **重要**  暂不支持新增申请 | — |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/assistant/threads/{threadId}/runs HTTP/1.1
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "assistantId" : "String",
  "instructions" : "String",
  "stream" : Boolean
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| threadId | String | 是 | 运行任务关联的线程id。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| assistantId | String | 是 | 运行任务关联的助理id。 |
| instructions | String | 否 | 开发者显式指定的角色设定，如果当前角色设定有值且有效，则会在大模型推理时使用当前传入的角色设定，否则默认使用助理创建时填写的角色设定。 |
| metadata | Map | 否 | 保留字段，用于扩展使用，暂无实际意义。 |
| stream | Boolean | 否 | 是否是流式输出：   * true：流式输出 * false：非流式输出   **说明**  默认为非流式。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| id | String | 运行任务id。 |
| createdAt | Long | 运行任务的创建时间，毫秒时间戳。 |
| threadId | String | 运行任务关联的线程id。 |
| assistantId | String | 运行任务关联的助理id。 |
| status | String | 运行任务的状态，枚举值：   * queued：运行任务在排队等待 * in\_progress：运行任务正在执行中 * requires\_action：运行任务正在执行中，等待技能执行返回结果（一般情况下，开发者可以将其视作in\_progress来处理后续业务流程） * failed：运行任务失败 * completed：运行任务完成 |
| lastErrorMsg | String | 运行任务最新一次失败的失败消息。 |
| expiresAt | Long | 运行任务的过期时间，毫秒时间戳。 |
| startedAt | Long | 运行任务的开始时间，毫秒时间戳。 |
| cancelledAt | Long | 运行任务的取消时间，毫秒时间戳。 |
| failedAt | Long | 运行任务的失败时间，毫秒时间戳。 |
| completedAt | Long | 运行任务的完成时间，毫秒时间戳。 |
| metadata | Map | 保留字段，用于扩展使用，暂无实际意义。 |
| object | String | 对象类型，默认为 thread.run。 |
