更新于 2026-03-06根据线程id，获取当前线程下的所有运行任务。访问本接口请申请应用权限进行访问。

**重要**

本文档已于 2026年 03 月 05 日迁移至历史文档（不推荐）目录，且本接口仅保持现有功能，不再新增支持其他能力。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | **重要**  暂不支持新增申请 | — |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/assistant/threads/{threadId}/runs?limit=Integerℴ=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| threadId | String | 是 | 运行任务关联的线程 ID。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| limit | Integer | 否 | 获取运行任务的限制数量，范围在1-100，不传默认为20。 |
| order | String | 否 | 获取运行任务列表的排序方式，枚举值：   * **asc**：升序排序 * **desc**；降序排序   **说明**  不传默认降序排序。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| data | Array | 运行任务列表。 |
| id | String | 运行任务 ID。 |
| createdAt | Long | 运行任务的创建时间，毫秒时间戳。 |
| threadId | String | 运行任务关联的线程 ID。 |
| assistantId | String | 运行任务关联的助理 ID。 |
| status | String | 运行任务的状态，枚举值有：   * **queued**：运行任务在排队等待 * **in\_progress**：运行任务正在执行中 * **failed**：运行任务失败 * **completed**：运行任务完成 |
| lastErrorMsg | String | 运行任务最新一次失败的失败消息。 |
| expiresAt | Long | 运行任务的过期时间，毫秒时间戳。 |
| startedAt | Long | 运行任务的开始时间，毫秒时间戳。 |
| cancelledAt | Long | 运行任务的取消时间，毫秒时间戳。 |
| failedAt | Long | 运行任务的失败时间，毫秒时间戳。 |
| completedAt | Long | 运行任务的完成时间，毫秒时间戳。 |
| metadata | Map | 保留字段，用于扩展使用，暂无实际意义。 |
| object | String | 对象类型，默认为 thread.run。 |
| object | String | 对象类型，默认为 list。 |

## 错误码

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 500 | run.token.error | %s | 校验AccessToken失败，请检查AccessToken是否有效，并检查“AI.Assistant.Run”该权限点已经授权 |
| 500 | run.list.error | %s | 系统繁忙，请稍后重试(E000021) |
| 500 | run.limit.error | %s | 查询Run实例数目的配置值不合法， 单次查询不允许超过100个 |
| 500 | run.order.error | %s | 传入的order参数不合法，只允许“asc”和”desc“，请检查后重试 |
| 500 | run.list.exception | %s | 系统繁忙，请稍后重试(E000122) |
