更新于 2025-09-23调用本接口检查用户是否完成所有数字化管理师的实操任务。

![[development-docking-of-provincial-practical-exercises-for-digital-managers_p314383.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 暂不支持 | 数字化管理师任务状态读权限 | 暂不支持 |
| 第三方企业应用 | 暂不支持 | 数字化管理师任务状态读权限 | 暂不支持 |
| 第三方个人应用 | 支持 | 数字化管理师任务状态读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=occupationauth_1.0%23CheckUserTasksStatus) |

## 请求方法

```
POST /v1.0/occupationauth/userTasks/check?provinceCode=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 接口调用凭证，调用[获取用户token](/document/development/obtain-user-token)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| provinceCode | String | 是 | 省级任务对接入。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| status | Boolean | 是否完成所有实操题任务。 |
