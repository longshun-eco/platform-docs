更新于 2025-09-11调用本接口，查询会议室列表。

调用本接口，实现的效果与下图操作一致，查看钉钉智能会议室应用内的会议室列表，并获取会议室的名称、可容纳人数、位置信息等。

![[development-check-the-meeting-room-list_O1CN01FfNS0k1oDzA9A8t3K_!!6000000005192-2-tps-1236-972.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=rooms_1.0%23QueryMeetingRoomList) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=rooms_1.0%23QueryMeetingRoomList) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/rooms/meetingRoomLists?nextToken=Long&maxResults=Integer&unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| nextToken | Long | 否 | 分页游标。   * 如果是首次调用，该参数不传。 * 如果是非首次调用，该参数传上传调用时返回的nextToken。 |
| maxResults | Integer | 否 | 请求分页大小，默认值20，最大值100。 |
| unionId | String | 是 | 操作人的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| hasMore | Boolean | 是否有更多。   * **true**：是 * **false**：否 |
| nextToken | Long | 分页游标。 |
| result | Array | 会议室列表。 |
| roomId | String | 会议室ID。 |
| roomStaffId | String | 会议室的企业员工Id。  **说明**  预留字段，暂无使用场景。 |
| corpId | String | 会议室所在的企业CorpId。 |
| roomName | String | 会议室名称。 |
| roomStatus | Integer | 会议室状态。   * **0**：全员可见 * **1**：仅管理员可见 * **2**：部分成员可见 |
| roomLabels | Array | 会议室标签列表。 |
| labelId | Long | 标签ID。   * **1**：电视 * **2**：电话 * **3**：投影仪 * **4**：白板 * **5**：视频会议 |
| labelName | String | 标签名称。 |
| roomCapacity | Integer | 会议室可容纳人数。 |
| roomLocation | Object | 会议室位置信息。 |
| title | String | 会议室位置名称。 |
| desc | String | 会议室位置详细信息。 |
| roomPicture | String | 会议室图片。 |
| isvRoomId | String | 调用方外部会议室ID。 |
| roomGroup | Object | 会议室分组信息。 |
| groupId | Long | 分组ID。 |
| groupName | String | 分组名称。 |
| parentId | Long | 父分组ID。 |
