更新于 2025-11-10调用本接口，查询会议室详情。

调用本接口，实现的效果与下图操作一致，查看钉钉智能会议室详情信息，包括会议室名称、可容纳人数、位置信息等。

![[development-check-meeting-room-details_O1CN01o1q2Rm237m1lZRbrw_!!6000000007209-2-tps-1160-968.png]]

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=rooms_1.0%23QueryMeetingRoom) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=rooms_1.0%23QueryMeetingRoom) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/rooms/meetingRooms/{roomId}?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：  企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。  第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| roomId | String | 是 | 会议室ID，可调用[查询会议室列表](https://open.dingtalk.com/document/development/check-the-meeting-room-list)接口获取。 |

## Query参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| unionId | String | 是 | 操作人的unionId，可调用[查询用户详情](https://open.dingtalk.com/document/development/query-user-details)接口获取获取。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| result | Object | 会议室详情。 |
| roomId | String | 会议室ID。 |
| roomStaffId | String | 会议室的企业员工Id。  **说明**  预留字段，暂无使用场景。 |
| corpId | String | 会议室所在的企业CorpId。 |
| roomName | String | 会议室名称。 |
| roomStatus | Integer | 会议室状态：   * **0**：全员可用 * **1**：仅管理员可用 * **2**：部分有权限 |
| roomLabels | Array | 会议室标签列表。 |
| labelId | Long | 标签ID：   * **1**：电视 * **2**：电话 * **3**：投影仪 * **4**：白板 * **5**：视频会议 |
| labelName | String | 标签名称。 |
| roomCapacity | Integer | 会议室可容纳人数。 |
| roomLocation | Object | 会议室位置信息。 |
| title | String | 会议室位置名称。 |
| desc | String | 会议室位置详细信息。 |
| roomPicture | String | 会议室图片。 |
| isvRoomId | String | 调用方外部会议室ID。 |
| roomGroup | Object | 会议室分组信息。 |
| groupId | Long | 分组ID。 |
| groupName | String | 分组名称。 |
| parentId | Long | 父分组ID。 |
| deviceUnionIds | Array of String | UnionId列表。 |
| reservationAuthority | Object | roomStatus部分权限时，设置的会议室预定权限。 |
| authorizedMembers | Array | 有权限的成员列表。 |
| memberId | String | 成员的id，memberType设置  人员：成员unionId，可调用[查询用户详情](https://open.dingtalk.com/document/development/query-user-details)接口获取获取。  部门：部门id，dept\_id，可调用[获取部门列表](https://open.dingtalk.com/document/development/obtain-the-department-list-v2)接口获取。 |
| memberType | String | 成员的类型，类型：   * **人员**：user * **部门**：dept |
| memberName | String | 成员的名称：   * **人员**：人员昵称 * **部门**：部门名称 |
| enableCycleReservation | Boolean | 会议室是否支持循环周期预定：   * **true**：是 * **false**：否 |
| roomUnionId | String | 会议室unionId。 |
| extensionConfig | Object | 会议室更多配置。 |
| openReservation | Boolean | 会议室是否开放预定:   * **true**：是 * **false**：否 |
| maxReservationTimeInterval | Integer | 单次可预定最长时段，单位分钟  **说明**  值为0表示无限制。 |
| minReservationTimeInterval | Integer | 单次可预定最小时段，单位分钟。 |
| advanceReservation | Object | 最早可提前预定时间。 |
| advanceReservationTime | Integer | 提前可预订时间具体值，单位参考advanceReservationTimeUnit字段。 |
| advanceReservationTimeUnit | String | 时间单位：   * **days**：表示天 * **weeks**：表示周 * **months**：表示月 * **years**：表示年 |
| advanceBookTimeFormat | String | 精确到几点钟。 |
| reservationCloseDetail | Object | 关闭预定具体信息。 |
| taskStartTime | Long | 会议室关闭预定的开始时间，时间戳，单位毫秒。 |
| taskEndTime | Long | 会议室关闭预定的结束时间。时间戳，单位毫秒。  **说明**  值为空表示永久关闭。 |
| closeReason | String | 关闭预定的原因。 |
| contactUnionId | String | 联系人的unionId。 |
| contactNick | String | 联系人昵称。 |
| sendNotify | Boolean | 是否发送通知给已预订会议室的员工。 |
| approvalSwitch | Boolean | 是否开启审批开关   * **true**：已开启 * **false**：未开启 |
| approvalType | Integer | 审批类型：   * **1**：标准审批 * **2**：自定义审批 |
| roomDescription | String | 会议室描述信息。 |
