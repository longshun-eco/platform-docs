更新于 2025-09-11调用本接口，查询视频会议设备属性信息。

调用本接口之前，需要先接入[事件订阅](/document/development/event-subscription-overview)并订阅[设备绑定会议室变更](/document/development/org-event-overview)事件。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=rooms_1.0%23QueryDeviceProperties) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/rooms/devices/properties/query?operatorUnionId=String&deviceId=String&deviceUnionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "propertyNames" : [ "String" ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorUnionId | String | 是 | 操作查询的人员unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取获取。 |
| deviceId | String | 否 | 需要查询的设备ID，该参数从订阅[设备绑定会议室变更](/document/development/org-event-overview)事件中获取。 |
| deviceUnionId | String | 否 | 需要查询的设备unionId，该参数从订阅[设备绑定会议室变更](/document/development/org-event-overview)事件中获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| propertyNames | Array of String | 否 | 设备属性名称列表，最大值10。   * dev\_code：投屏码 * dev\_model：设备型号 * dev\_app\_status：设备状态 * dev\_net\_ip：设备ip * dev\_net\_type：设备网络类型 * dev\_wifi\_mac：设备无线mac地址 * dev\_wire\_mac：设备有线mac地址 * dev\_firmware\_v：设备固件版本 * dev\_software\_v：设备软件版本 * dev\_hdmi：设备外接显示器 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array | 响应结果信息列表。 |
| propertyName | String | 设备属性名称。   * dev\_code：投屏码 * dev\_model：设备型号 * dev\_app\_status：设备状态 * dev\_net\_ip：设备ip * dev\_net\_type：设备网络类型，该属性有以下值：    + net\_wired：有线网络   + net\_offline：离线   + 具体的ssid：Wi-Fi * dev\_wifi\_mac：设备无线mac地址 * dev\_wire\_mac：设备有线mac地址 * dev\_firmware\_v：设备固件版本 * dev\_software\_v：设备软件版本 * dev\_hdmi：设备外接显示器 |
| propertyValue | String | 设备属性值。 |
