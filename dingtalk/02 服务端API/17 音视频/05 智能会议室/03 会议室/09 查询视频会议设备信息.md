更新于 2025-09-11调用本接口，查询视频会议设备信息。

调用本接口之前，需要先接入[事件订阅](/document/development/event-subscription-overview)并订阅[设备绑定会议室变更](/document/development/org-event-overview)事件。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=rooms_1.0%23QueryMeetingRoomDevice) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/rooms/devices?operatorUnionId=String&deviceId=String&deviceUnionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorUnionId | String | 是 | 操作查询的人员unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取获取。 |
| deviceId | String | 否 | 需要查询的设备ID，该参数从订阅[设备绑定会议室变更](/document/development/org-event-overview)事件中获取。  **说明**  `deviceId`字段或`deviceUnionId`字段，必须二选一。 |
| deviceUnionId | String | 否 | 需要查询的设备unionId，该参数从订阅[设备绑定会议室变更](/document/development/org-event-overview)事件中获取。  **说明**  `deviceId`字段或`deviceUnionId`字段，必须二选一。 |
