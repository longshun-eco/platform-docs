更新于 2025-09-11调用本接口，可以创建会议室预定黑名单

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=rooms_1.0%23CreateBookingBlacklist) |
| 第三方企业应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=rooms_1.0%23CreateBookingBlacklist) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/rooms/bookings/blacklist HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "unionId" : "String",
  "blacklistUnionId" : "String",
  "startTime" : Long,
  "endTime" : Long,
  "memo" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作人unionId。 |
| blacklistUnionId | String | 是 | 黑名单用户的unionId。 |
| startTime | Long | 是 | 封禁开始时间，毫秒级UTC时间戳。 |
| endTime | Long | 否 | 封禁结束时间，毫秒级UTC时间戳。不传表示永久封禁。 |
| memo | String | 否 | 备注。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 接口调用是否成功。 |
