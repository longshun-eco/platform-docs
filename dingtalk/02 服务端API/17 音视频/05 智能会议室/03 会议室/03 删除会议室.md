更新于 2025-09-11调用本接口，删除会议室。

调用本接口，实现的效果与下图操作一致，在钉钉智能会议室应用内进入某个会议室编辑页面，单击删除会议室。

![[development-delete-a-meeting-room_O1CN01QNuGtw1bgIdzecsUG_!!6000000003494-2-tps-1166-990.png]]

**说明**

删除会议室，只允许有智能会议室应用管理权限的管理员操作。

![[development-delete-a-meeting-room_O1CN01v90IEz204A3gSD44X_!!6000000006795-2-tps-2498-1632.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=rooms_1.0%23DeleteMeetingRoom) |
| 第三方企业应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=rooms_1.0%23DeleteMeetingRoom) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```sql
DELETE /v1.0/rooms/meetingRooms/{roomId}?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| roomId | String | 是 | 会议室ID，可调用[查询会议室列表](/document/development/check-the-meeting-room-list)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作人的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 是否成功删除会议室，true表示删除成功。 |
