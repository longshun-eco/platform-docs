更新于 2025-09-11调用本接口，查询会议室分组列表。

调用本接口，实现效果与下图操作一致，可查看智能会议室应用内的分组列表，并获取分组的名称、分组ID和父分组ID信息。

![[development-query-meeting-rooms-groups_O1CN0156WxJ51GpowklBy0f_!!6000000000672-2-tps-1214-962.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=rooms_1.0%23QueryMeetingRoomGroupList) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=rooms_1.0%23QueryMeetingRoomGroupList) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/rooms/groupLists?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 操作人的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array | 结果列表。 |
| groupId | Long | 分组ID。 |
| groupName | String | 分组名称。 |
| parentId | Long | 父分组ID。 |
