更新于 2025-09-11调用本接口，根据模板ID，删除自定义屏幕模板。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉会议Rooms系统自定义屏幕写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=rooms_1.0%23DeleteDeviceCustomTemplate) |
| 第三方企业应用 | 支持 | 钉钉会议Rooms系统自定义屏幕写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=rooms_1.0%23DeleteDeviceCustomTemplate) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/rooms/devices/screens/templates/remove HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "templateId" : Long
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 接口调用凭证，调用[获取用户token](/document/development/obtain-user-token)接口获取。  **说明**  获取用户token的请求中，使用的入参`code`参数，在通过构造链接获取的过程中，scope范围必须为用户id和组织id，即`scope=openid corpid`。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| templateId | Long | 是 | 模板id，可调用[查询自定义屏幕模板列表](/document/development/api-querydevicecustomtemplatelist)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 删除是否成功。 |
