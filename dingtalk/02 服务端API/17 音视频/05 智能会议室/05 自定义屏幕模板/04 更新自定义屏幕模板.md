更新于 2025-09-11根据自定义屏幕模板ID，更新自定义屏幕模板信息内容。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉会议Rooms系统自定义屏幕写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=rooms_1.0%23UpdateDeviceCustomTemplate) |
| 第三方企业应用 | 支持 | 钉钉会议Rooms系统自定义屏幕写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=rooms_1.0%23UpdateDeviceCustomTemplate) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
PUT /v1.0/rooms/devices/screens/templates HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "templateName" : "String",
  "logo" : "String",
  "orgName" : "String",
  "customDoc" : "String",
  "bgUrl" : "String",
  "bgImgList" : [ "String" ],
  "instruction" : Boolean,
  "bgType" : Integer,
  "isPicTop" : Integer,
  "hideServerCodeWhenProjecting" : Boolean,
  "showCalendarTitle" : Boolean,
  "desensitizeUserName" : Boolean,
  "picturePlayInterval" : Integer,
  "showCalendarCard" : Boolean,
  "showFunctionCard" : Boolean,
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 接口调用凭证，调用[获取用户token](/document/development/obtain-user-token)接口获取。  **说明**  获取用户token的请求中，使用的入参`code`参数，在通过构造链接获取的过程中，scope范围必须为用户id和组织id，即`scope=openid corpid`。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| templateName | String | 是 | 模板名称。 |
| logo | String | 否 | 模板logo，可调用[上传媒体文件](/document/development/upload-media-files)接口获取meidaId，图片尺寸：240\*240，单个图片大小不超过1M。 |
| orgName | String | 否 | 模板企业名称。 |
| customDoc | String | 否 | 模板自定义文字内容。 |
| bgUrl | String | 否 | 模板自定义背景网址。 |
| bgImgList | Array of String | 否 | 图片mediaId，可调用[上传媒体文件](/document/development/upload-media-files)接口获取出参`media_id`参数，最多可上传9张图片，单个图片大小不超过5M。 |
| instruction | Boolean | 否 | 显示使用说明，默认false。取值：   * true：显示 * false：不显示 |
| bgType | Integer | 否 | 模板自定义背景类型，默认1。取值：   * 1：图片 * 2：网址 |
| isPicTop | Integer | 否 | 展示首页信息栏，默认 1。取值：   * 0：展示 * 1：不展示 |
| hideServerCodeWhenProjecting | Boolean | 否 | 投屏中隐藏投屏码，默认false。取值：   * true：隐藏 * false：不隐藏 |
| showCalendarTitle | Boolean | 否 | 日程卡片展示日程标题，默认false。取值：   * true：展示 * false：不展示 |
| desensitizeUserName | Boolean | 否 | 日程卡片脱敏预订人昵称，默认false。取值：   * true：脱敏 * false：不脱敏 |
| picturePlayInterval | Integer | 否 | 自定义背景图轮播间隔秒。 |
| showCalendarCard | Boolean | 否 | 展示日程卡片，默认false。取值：   * true：展示 * false：不展示 |
| showFunctionCard | Boolean | 否 | 展示功能卡片，默认false。取值：   * true：展示 * false：不展示 |
| deviceUnionIds | Array of String | 否 | 设备unionId。该参数从订阅[设备绑定会议室变更](/document/development/org-event-overview)事件中获取。 |
| groupIds | Array of Long | 否 | 分组id，可调用[查询会议室分组列表](/document/development/query-meeting-rooms-groups)接口获取。 |
| roomIds | Array of String | 否 | 会议室id，可调用[查询会议室列表](/document/development/check-the-meeting-room-list)接口获取。 |
| templateId | Long | 是 | 模板id，可调用[查询自定义屏幕模板列表](/document/development/api-querydevicecustomtemplatelist)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 更新是否成功。 |
