更新于 2025-09-11调用本接口，查询会议录制的详情信息，包括录制人、录制开始时间、录制持续时长等。

例如，会议主持人开启了会议录制功能，会议结束后，主持人钉钉收到的推送消息如下图所示。

调用本接口，可获取会议录制的详情信息，包括录制人、录制开始时间、录制持续时长等。

![[development-query-recording-information_O1CN01h8OaRY1Vcjq6npDap_!!6000000002674-2-tps-2306-1118.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23QueryCloudRecordVideo) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23QueryCloudRecordVideo) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/conference/videoConferences/{conferenceId}/cloudRecords/getVideos?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| conferenceId | String | 是 | 会议ID，可调用[创建视频会议](/document/development/create-a-video-conference)接口获取conferenceId参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 用户unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| videoList | Array | 会议录制的视频列表。 |
| recordId | String | 录制ID。  **说明**  同一个视频会议，如果多次进行录制，产生的多份会议录制视频的recordId相同。 |
| unionId | String | 录制人unionId。 |
| startTime | Long | 录制开始时间戳，单位毫秒。 |
| recordType | Long | 记录类型，取值：   * **0**：普通录制 * **1**：合成的文件 |
| duration | Long | 录制持续时长的1000倍。  **说明**  例如，该字段值为20000，表示本次会议录制的时长为20秒。 |
| fileSize | Long | 文件大小，单位**byte**。 |
| endTime | Long | 录制结束时间戳，单位毫秒。 |
| mediaId | String | 唯一媒体文件ID。 |
| regionId | String | 媒体文件所在地域ID。 |
