更新于 2025-09-11查询用户进行中会议列表，返回进行中会议id列表及该用户在会中对应的成员信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23QueryUserOnGoingConference) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23QueryUserOnGoingConference) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/conference/users/lists?unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 要查询的用户unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| onGoingConfIdList | Array of String | 会议id。 |
| memberModelMap | Map<String, Object> | 查询用户在会中的成员信息映射。 |
|  | Object | 成员信息。 |
| unionId | String | 成员的unionId。 |
| conferenceId | String | 视频会议ID。 |
| userNick | String | 成员的昵称。 |
| joinTime | Long | 成员入会时间戳，单位毫秒。 |
| leaveTime | Long | 成员离会时间戳，单位毫秒。 |
| duration | Long | 在会时长，单位毫秒。 |
| attendStatus | Integer | 成员状态：   * 1：初始化 * 2：呼叫中 * 3：活跃（在会） * 4：入会失败（拒接等） * 5：被踢 * 6：离会 |
| host | Boolean | 是否是主持人：   * true：是 * false：否 |
| coHost | Boolean | 是否为联席主持人：   * true：是 * false：否 |
| outerOrgMember | Boolean | 是否非会议所属企业内成员：   * true：是 * false：否 |
| pstnJoin | Boolean | 是否为电话号码入会：   * true：是 * false：否 |
| deviceType | String | 入会的设备类型：   * Win * Mac * iOS * Android |
