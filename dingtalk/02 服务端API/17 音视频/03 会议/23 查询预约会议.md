更新于 2025-09-11根据预约会议id查询预约会议标题等信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23QueryScheduleConference) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23QueryScheduleConference) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/conference/scheduleConferences/{scheduleConferenceId}/infos?requestUnionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| scheduleConferenceId | String | 是 | 预约会议id，可通过[创建预约会议](/document/development/create-appointment-meeting)接口获取返回参数`scheduleConferenceId`字段。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| requestUnionId | String | 是 | 请求者unionId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| requestId | String | 请求id。 |
| scheduleConferenceId | String | 预约会议id。 |
| title | String | 预约会议标题。 |
| startTime | Long | 预约会议开始时间，毫秒级UTC时间戳。 |
| endTime | Long | 预约会议结束时间，毫秒级UTC时间戳。 |
| roomCode | String | 会议号。 |
| url | String | 预约会议分享链接。 |
| phones | Array of String | 呼入电话号码。 |
