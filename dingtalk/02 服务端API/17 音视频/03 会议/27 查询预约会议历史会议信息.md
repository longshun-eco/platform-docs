更新于 2026-03-16查询预约会议已开启过的历史会议信息，当前返回最后一次会议信息。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23QueryScheduleConferenceInfo) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23QueryScheduleConferenceInfo) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/conference/videoConferences/scheduleConferences/{scheduleConferenceId}?nextToken=String&maxResults=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证：   * 企业内部应用可调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用可调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| scheduleConferenceId | String | 是 | 预约会议id，可通过[创建预约会议接口](https://open.dingtalk.com/document/development/create-appointment-meeting)获取返回参数`scheduleConferenceId`字段。 |

## Query参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| nextToken | String | 否 | 标记当前开始读取的位置，置空表示从头开始。 |
| maxResults | Integer | 否 | 本次读取的最大数据记录数量。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| totalCount | Integer | TotalCount本次请求条件下的数据总量，此参数为可选参数，默认可不返回。 |
| nextToken | String | 表示当前调用返回读取到的位置，空代表数据已经读取完毕。 |
| conferenceList | Array | 会议列表。 |
| conferenceId | String | 会议id。 |
| title | String | 会议标题。 |
| roomCode | String | 会议号。 |
| status | Integer | 会议状态：   * **0**: 待开始 * **1**：已结束 * **2**：进行中 |
| startTime | Long | 会议开始时间，utc时间戳，单位毫秒。 |
| endTime | Long | 会议结束时间，utc时间戳，单位毫秒。 |
