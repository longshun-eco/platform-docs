更新于 2025-09-11根据用户信息，创建用户专属短链。用户通过专属短链入会后，可通过客户端jsapi查询该用户的扩展应用业务自定义信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23CreateCustomShortLink) |
| 第三方企业应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23CreateCustomShortLink) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/conference/customShortLinks HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "creatorUnionId" : "String",
  "scheduleConferenceId" : "String",
  "coolAppCode" : "String",
  "extensionAppBizData" : "String",
  "useExtensionApp" : Boolean
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| creatorUnionId | String | 是 | 预约会议创建人 unionId。 |
| scheduleConferenceId | String | 是 | 预约会议 id，可通过[创建预约会议](/document/development/create-appointment-meeting)接口获取返回参数`scheduleConferenceId`字段。 |
| coolAppCode | String | 是 | 酷应用 Code。可通过开发者后台，微应用中查看酷应用Code。 |
| extensionAppBizData | String | 否 | 用户维度扩展应用自定义信息。 |
| useExtensionApp | Boolean | 否 | 是否使用扩展应用：   * true：是 * false：否 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 响应结果。 |
| customShortLink | String | 专属短链。 |
