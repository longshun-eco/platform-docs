更新于 2025-09-11调用本接口，查询视频会议录制的语音转文字信息，包括文本内容、文本记录开始时间、文本记录结束时间等。

钉钉视频会议如果开启了云录制（闪记），在开启云录制期间，参会人员发表讲话的内容会自动被转换为文本生成记录。

例如，会议主持人开启了**会议录制-云录制**功能，会议结束后，主持人收到会议录制的推送消息。记录的语音转文字记录中的内容如下图所示。

调用本接口，可获取云录制过程中记录的文本信息，包括文本内容、文本记录开始时间、文本记录结束时间等。

![[development-queries-the-text-information-about-cloud-recording_O1CN019tvQQy1pM3jNl0JLt_!!6000000005345-2-tps-2662-1298.png]]

**说明**

主持人开启会议录制时，需要先选择**云录制（闪记）**，并且在录制期间，参会人员有发言的行为，才可调用本接口查询会议录制中的文本信息。

![[development-queries-the-text-information-about-cloud-recording_O1CN01i4Lml31i0o2RDtPBE_!!6000000004351-2-tps-2202-1568.png]]

## 请求方法

```
GET /v1.0/conference/videoConferences/{conferenceId}/cloudRecords/getTexts?unionId=String&startTime=Long&direction=String&maxResults=Long&nextToken=Long HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| conferenceId | String | 否 | 会议ID，可调用[创建视频会议](/document/development/create-a-video-conference)接口获取conferenceId参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 否 | 用户unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 |
| startTime | Long | 否 | 开始时间的千分之一秒，单位毫秒。  **说明**  例如，该参数值传2000，表示从录制开始的第2秒开始查询。 |
| direction | String | 否 | 查询方式：   * **0**：表示此次查询按照时间由小到大的顺序进行，默认方式。 * **1**：表示此次查询按照时间由大到小的顺序进行。 |
| maxResults | Long | 否 | 单词查询条数，最大2000。 |
| nextToken | Long | 否 | 分页游标。   * 如果是首次查询，该参数可不传。 * 如果是非首次查询，该参数传上次调用本接口时返回的nextToken。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| hasMore | Boolean | 是否有更多数据。   * true：是 * false：否 |
| paragraphList | Array | 段落列表。 |
| nextTtoken | Long | 下一次请求的分页游标。 |
| status | Long | 状态。  **说明**  该字段暂无使用场景。 |
| unionId | String | 发言人的unionId。 |
| nickName | String | 发言人昵称。 |
| recordId | Long | 录制ID。 |
| startTime | Long | 段落记录产生时，在会议录制期间内相对时间的千分之一秒，单位毫秒。  **说明**  例如，该参数值是2000，表示在会议录制开始的第2秒，开始了本次语音转文字的记录。 |
| endTime | Long | 段落记录结束时，在会议录制期间内相对时间的千分之一秒，单位毫秒。  **说明**  例如，该参数值是8000，表示在会议录制开始的第8秒，结束了本次语音转文字的记录。 |
| paragraph | String | 段落内容。 |
| sentenceList | Array | 句子列表。 |
| unionId | String | 用户的unionId。 |
| sentence | String | 句子。 |
| startTime | Long | 句子记录产生时，在会议录制期间内相对时间的千分之一秒，单位毫秒。  **说明**  例如，该参数值是2000，表示在会议录制开始的第2秒，开始了本次语音转文字的记录。 |
| endTime | Long | 句子记录结束时，在会议录制期间内相对时间的千分之一秒，单位毫秒。  **说明**  例如，该参数值是2000，表示在会议录制开始的第2秒，结束了本次语音转文字的记录。 |
| wordList | Array | 单词列表。 |
| word | String | 单词。 |
| startTime | Long | 单词记录产生时，在会议录制期间内相对时间的千分之一秒，单位毫秒。  **说明**  例如，该参数值是2000，表示在会议录制开始的第2秒，开始了本次语音转文字的记录。 |
| endTime | Long | 单词记录结束时，在会议录制期间内相对时间的千分之一秒，单位毫秒。  **说明**  例如，该参数值是2000，表示在会议录制开始的第2秒，结束了本次语音转文字的记录。 |
| wordId | String | 单词ID。 |
