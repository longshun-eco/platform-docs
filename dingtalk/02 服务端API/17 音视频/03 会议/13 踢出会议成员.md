更新于 2025-09-11会议过程中，踢出成员，调用该接口后，被踢用户会离会并提示已被移出会议。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23KickMembers) |
| 第三方企业应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23KickMembers) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/conference/videoConferences/{conferenceId}/members/kick HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "userList" : [ {
    "unionId" : "String",
    "participantId" : "String"
  } ],
  "forbiddenRejoin" : Boolean
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| conferenceId | String | 是 | 会议id，可调用[创建视频会议](/document/development/create-a-video-conference)接口获取返回参数`conferenceId`字段。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userList | Array | 是 | 被操作用户列表。 |
| unionId | String | 是 | 用户unionId。 |
| participantId | String | 否 | 用户参会者id。  **说明**  不指定时控制用户所有设备。 |
| forbiddenRejoin | Boolean | 否 | 是否禁止再次加入：   * **true**：禁止再次加入 * **false**：可再次加入 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 是否成功。 |
