更新于 2025-09-11调用本接口，查询视频会议的基础信息。

例如，钉钉会议如下图所示。调用本接口可获取会议的基本信息，包括会议名称、会议时长、会议开始时间、会议结束时间、累计参会人数等信息。

![[development-querying-video-conference-information_O1CN015pLq9a1ldE8c16JHl_!!6000000004841-2-tps-2214-1590.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23QueryConferenceInfo) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23QueryConferenceInfo) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/conference/videoConferences/{conferenceId} HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| conferenceId | String | 是 | 会议conferenceId，可调用[创建视频会议](/document/development/create-a-video-conference)接口获取conferenceId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| confInfo | Object | 会议信息。 |
| activeNum | Integer | 当前在会人数。 |
| attendNum | Integer | 累积入会人数。 |
| confDuration | Long | 会议时长，单位毫秒。  **说明**  会议结束后，返回该字段。 |
| conferenceId | String | 会议ID。 |
| creatorId | String | 会议创建人的unionId。 |
| creatorNick | String | 会议创建人的昵称。 |
| externalLinkUrl | String | 会议的入会链接。 |
| invitedNum | Integer | 邀请人数。 |
| startTime | Long | 会议开始时间戳，单位毫秒。 |
| status | Integer | 会议状态。   * 0：初始化 * 1：会议结束 * 2：会议开始 |
| title | String | 会议标题。 |
| roomCode | String | 会议码。 |
| endTime | Long | 会议结束时间戳，单位毫秒。  **说明**  会议结束后，返回该字段。 |
| scheduleConferenceId | String | 预约会议id。 |
| bizType | String | 会议业务类型：   * **0**：即时视频会议 * **3**：预约会议 * **audio\_conf**：语音会议 |
| cloudRecordStatus | Integer | 会议云录制状态：   * **0**：未录制 * **1**：录制中 * **2**：录制暂停 * **3**：录制已结束 |
| cloudRecordOwnerUnionId | String | 云录制归属者unionId。 |
| minutesStatus | Integer | 会议闪记状态：   * **0**：未开启 * **1**：录制中 * **2**：暂停 * **3**：已结束   **说明**  会中闪记能力迁移，该字段不再返回 |
| minutesOwnerUnionId | String | 闪记归属者unionId。  **说明**  会中闪记能力迁移，该字段不再返回 |
| extensionAppSettings | Array | 会议维度扩展应用自定义信息列表。 |
| appId | String | 会议酷应用所属微应用AgentId。 |
| appCode | String | 酷应用 appCode。 |
| autoOpenMode | Integer | 自动打开模式：   * **0**：不自动打开 * **1**：仅主持人/联席主持人自动打开 * **2**：全员自动打开 |
| extensionAppBizData | String | 会议维度扩展应用自定义信息。 |
