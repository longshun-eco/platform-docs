更新于 2025-12-08调用本接口，分页查询视频会议成员信息。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23QueryConferenceMembers) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23QueryConferenceMembers) |

## 请求方法

```
GET /v1.0/conference/videoConferences/{conferenceId}/members?nextToken=String&maxResults=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)接口获取。 |

## Path参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| conferenceId | String | 是 | 会议conferenceId。   * 企业内部应用，调用[创建视频会议](https://open.dingtalk.com/document/orgapp/create-a-video-conference)接口获取conferenceId。 * 第三方企业应用，调用[创建视频会议](https://open.dingtalk.com/document/isvapp/create-a-video-conference)接口获取conferenceId。 |

## Query参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| nextToken | String | 否 | 分页游标。   * 如果是首次调用本接口，该参数不需要传或者传0。 * 如果是非首次调用本接口，该参数值传上次调用本接口返回的nextToken。 |
| maxResults | Integer | 否 | 每页最大条目数，默认值128，无最大值限制。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| memberModels | Array | 成员列表。 |
| unionId | String | 成员的unionId。 |
| conferenceId | String | 视频会议ID。 |
| userNick | String | 成员的昵称。 |
| joinTime | Long | 成员入会时间戳，单位毫秒。 |
| leaveTime | Long | 成员离会时间戳，单位毫秒。 |
| duration | Long | 在会时长，单位毫秒。 |
| host | Boolean | 是否是主持人。   * true：是 * false：否 |
| attendStatus | Integer | 成员状态 。   * 1：初始化 * 2：呼叫中 * 3：活跃（在会） * 4：入会失败（拒接等） * 5：被踢 * 6：离会 |
| outerOrgMember | Boolean | 是否非会议所属企业内成员。   * true：是 * false：否   **说明**  如果当前参会人是所属企业内成员，该字段不返回。 |
| pstnJoin | Boolean | 是否为电话号码入会。   * true：是 * false：否 |
| coHost | Boolean | 是否为联席主持人。   * true：是 * false：否 |
| nextToken | String | 分页游标。 |
| totalCount | Integer | 总数量。 |

## 错误码

| HttpCode | 错误码 | 错误信息 | 说明 |
| --- | --- | --- | --- |
| 400 | missingRoomId | MISSING ROOM ID | 会议id参数缺失 |
| 400 | paramError | PARAM ERROR | 参数错误 |
| 403 | noPermission | NO PERMISSION | 无权查询会议 |
| 404 | roomNotFound | ROOM NOT FOUND | 查不到该会议 |
| 500 | systemError | SYSTEM ERROR | 系统内部错误 |
