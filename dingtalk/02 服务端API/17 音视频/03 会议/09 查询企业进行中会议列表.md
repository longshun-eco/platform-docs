更新于 2025-09-11调用本接口，可根据分页参数查询企业进行中会议列表。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23QueryOrgConferenceList) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23QueryOrgConferenceList) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/conference/orgConferences?nextToken=String&maxResults=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| nextToken | String | 否 | 分页游标。 |
| maxResults | Integer | 否 | 分页大小。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| nextToken | String | 分页游标。 |
| totalCount | Integer | 结果总数。 |
| hasMore | Boolean | 是否更多。 |
| onGoingConfList | Array | 进行中会议列表。 |
| conferenceId | String | 会议id。 |
| title | String | 会议标题。 |
| roomCode | String | 会议号。 |
| status | Integer | 会议状态：   * 2：进行中 |
| startTime | Long | 会议开始时间。 |
| endTime | Long | 会议结束时间。 |
| creatorId | String | 创建人unionId。 |
| creatorNick | String | 创建人昵称。 |
| bizType | String | 会议业务类型。 |
