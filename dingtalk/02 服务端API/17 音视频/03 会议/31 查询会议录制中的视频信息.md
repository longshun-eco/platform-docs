更新于 2025-09-11调用本接口，查询会议录制中的视频信息，包括视频下载地址、视频文件大小和视频时长等。

例如，会议主持人开启了会议录制功能，会议结束后，主持人钉钉收到的推送消息如下图所示。

调用本接口，可获取会议录制中的视频信息，包括视频下载地址、视频文件大小和视频时长等。

![[development-queries-the-playback-information-about-a-recorded-cloud-video_O1CN01WlRHgE1pbAnYXnTbw_!!6000000005378-2-tps-2666-1314.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23QueryCloudRecordVideoPlayInfo) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23QueryCloudRecordVideoPlayInfo) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/conference/videoConferences/{conferenceId}/cloudRecords/videos/getPlayInfos?unionId=String&mediaId=String&regionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| conferenceId | String | 是 | 会议ID，可调用[创建视频会议](/document/development/create-a-video-conference)接口获取conferenceId参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 用户unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 |
| mediaId | String | 是 | 媒体文件ID，可调用[查询会议录制的详情信息](/document/development/query-recording-information)接口获取mediaId参数值。 |
| regionId | String | 是 | 地域ID，可调用[查询会议录制的详情信息](/document/development/query-recording-information)接口获取regionId参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| playUrl | String | 录制视频的在线播放链接。 |
| mp4FileUrl | String | 录制视频的MP4格式下载链接。 |
| fileSize | Long | 录制视频的文件大小，单位**byte**。 |
| duration | Long | 录制视频的时长，单位毫秒。 |
| status | Long | 状态。  **说明**  该字段暂无使用场景。 |
