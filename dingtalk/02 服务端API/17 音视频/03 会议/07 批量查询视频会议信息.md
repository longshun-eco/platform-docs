更新于 2025-09-11调用本接口查询会议信息以及参会人员。

**说明**

目前只支持查询正在进行的会议。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23QueryConferenceInfoBatch) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23QueryConferenceInfoBatch) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/conference/videoConferences/query HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "conferenceIdList" : [ "String" ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| conferenceIdList | Array of String | 是 | 会议ID列表。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| infos | Array | 会议详情列表。 |
| conferenceId | String | 实时会议ID。 |
| title | String | 会议名称。 |
| startTime | Long | 会议开始时间。 |
| status | Long | 会议状态。   * **0**：初始化 * **1**：会议结束 * **2**：会议开始 |
| mediaStatus | Long | 媒体状态。   * **0**：正常 * **1**：麦克风静音 * **2**：摄像头关闭 * **4**：强制全员静音 |
| userList | Array | 参会用户列表。 |
| userId | String | 用户userid。 |
| nick | String | 用户昵称。 |
| attendStatus | Long | 在会状态。   * **0**：未定义 * **1**：初始化 * **2**：加入中 * **3**：在会 * **4**：加入失败 * **5**：被踢出 * **6**：离开 |
| cameraStatus | Long | 摄像头状态。   * **0**：初始化 * **1**：关闭 * **2**：打开 |
| micStatus | Long | 麦克风状态。   * **0**：初始化 * **1**：关闭 * **2**：打开 |
| rejectDescription | String | 拒绝原因。 |
