更新于 2025-09-11调用本接口，根据会议 roomCode 和分页参数会议信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23QueryConferenceInfoByRoomCode) |
| 第三方企业应用 | 支持 | 视频会议信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23QueryConferenceInfoByRoomCode) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/conference/roomCodes/{roomCode}/infos?nextToken=String&maxResults=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| roomCode | String | 是 | 会议码。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| nextToken | String | 否 | 分页游标。 |
| maxResults | Integer | 否 | 分页大小。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| nextToken | String | 分页游标。 |
| hasMore | Boolean | 是否更多。 |
| totalCount | Integer | 结果总数。 |
| conferenceList | Array | 会议信息列表，如果会议码对应的会议为预约会议、周期会议，则会议码可能对应多个会议实例信息。 |
| conferenceId | String | 会议ID。 |
| title | String | 会议标题。 |
| roomCode | String | 会议码。 |
| externalLinkUrl | String | 会议的入会链接。 |
| status | Integer | 会议状态：   * 0：初始化 * 1：会议结束 * 2：会议开始 |
| startTime | Long | 会议开始时间戳，单位毫秒。 |
| endTime | Long | 会议结束时间戳，单位毫秒，会议结束后，返回该字段。 |
| confDuration | Long | 会议时长，单位毫秒，会议结束后，返回该字段。 |
| creatorId | String | 会议创建人的unionId。 |
| creatorNick | String | 会议创建人的昵称。 |
| scheduleConferenceId | String | 预约会议id。 |
| bizType | String | 会议业务类型：   * 0：即时视频会议 * 3：预约会议 * audio\_conf：语音会议 |
