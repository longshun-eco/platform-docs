更新于 2025-09-11调用本接口停止视频会议云录制。

例如，企业内正在进行中的会议如下图所示，已开启了会议录制。

![[development-video-conferencing-stops-cloud-recording_O1CN01BslemT26nOqB2OlkM_!!6000000007706-2-tps-2196-1564.png]]

调用本接口结束会议录制。视频会议正常进行，不会被结束。

![[development-video-conferencing-stops-cloud-recording_O1CN01r59Cm11ZCPXCwhSPB_!!6000000003158-2-tps-2202-1558.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23StopCloudRecord) |
| 第三方企业应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23StopCloudRecord) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/conference/videoConferences/{conferenceId}/cloudRecords/stop HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "unionId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| conferenceId | String | 是 | 会议ID，可调用[创建视频会议](/document/development/create-a-video-conference)接口获取conferenceId参数值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 用户unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。  **说明**  只有当用户是会议主持人时，才有权限关闭云录制。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| code | String | 返回码。  **说明**  如果视频会议云录制关闭成功，且接口调用成功，该字段值为200。 |
