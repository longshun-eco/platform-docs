更新于 2025-09-11会议过程中，新增联席主持人或者删除联席主持人。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=conference_1.0%23Cohosts) |
| 第三方企业应用 | 支持 | 视频会议信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=conference_1.0%23Cohosts) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/conference/videoConferences/{conferenceId}/coHosts/set HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "action" : "String",
  "userList" : [ {
    "unionId" : "String"
  } ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| conferenceId | String | 是 | 会议id，可调用[创建视频会议](/document/development/create-a-video-conference)接口获取返回参数`conferenceId`字段。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| action | String | 是 | 行为动作：   * **add**:添加联席主持人 * **remove**：移除联系主持人 |
| userList | Array | 是 | 被操作用户列表。 |
| unionId | String | 是 | 用户unionId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 是否成功。 |
