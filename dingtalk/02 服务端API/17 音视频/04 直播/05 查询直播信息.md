调用本接口，根据直播ID，查询直播的详情信息，获取的部分直播信息如下图所示。

![[development-queries-the-live-streaming-information_O1CN01GzkINF1yPZK8lS5ln_!!6000000006571-2-tps-1362-906.png]]

**说明**

本接口适用于直播的任意状态。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉直播获取数据读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=live_1.0%23QueryLiveInfo) |
| 第三方企业应用 | 支持 | 钉钉直播获取数据读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=live_1.0%23QueryLiveInfo) |
| 第三方个人应用 | 暂不支持 | 钉钉直播获取数据读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/live/lives?liveId=String&unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| liveId | String | 是 | 直播ID，可调用[创建直播](/document/development/create-live-streaming)接口获取liveId参数值。 |
| unionId | String | 是 | 操作者的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果。 |
| liveInfo | Object | 直播信息。 |
| liveId | String | 直播ID。 |
| unionId | String | 主播的unionId。 |
| title | String | 直播标题。 |
| coverUrl | String | 直播的封面地址。 |
| startTime | Long | 直播实际的开始时间戳，单位毫秒。 |
| endTime | Long | 直播实际的结束时间戳，单位毫秒。 |
| introduction | String | 直播简介。 |
| liveStatus | Integer | 直播状态。   * **0**：预告态 * **1**：直播态 * **3**：结束态 * **4**：直播处罚态 * **5**：删除态 |
| duration | Long | 直播时长，单位毫秒。 |
| subscribeCount | Integer | 预约人数。 |
| uv | Integer | 观看人数。 |
| livePlayUrl | String | 直播观看地址。 |
| playbackDuration | Long | 直播回放时长，单位毫秒。 |
