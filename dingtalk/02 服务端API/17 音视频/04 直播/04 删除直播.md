更新于 2025-09-11调用本接口，根据直播ID删除直播。

调用本接口，删除直播。

* 如果直播未发起，调用本接口删除后，直播不能正常发起。
* 如果直播已结束，调用本接口删除后，删除全部直播数据，包括回放视频也会被删除。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉直播获取数据写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=live_1.0%23DeleteLive) |
| 第三方企业应用 | 支持 | 钉钉直播获取数据写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=live_1.0%23DeleteLive) |
| 第三方个人应用 | 暂不支持 | 钉钉直播获取数据写权限 | 暂不支持 |

## 请求方法

```sql
DELETE /v1.0/live/lives?liveId=String&unionId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| liveId | String | 是 | 直播ID，可调用[创建直播](/document/development/create-live-streaming)接口获取liveId参数值。 |
| unionId | String | 是 | 用户unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果。 |
| success | Boolean | 删除直播是否成功。   * **true**：成功 * **false**：失败 |
