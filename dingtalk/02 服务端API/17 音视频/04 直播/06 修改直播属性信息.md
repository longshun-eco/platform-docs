更新于 2025-09-11调用本接口，修改直播属性信息。

例如，小钉创建了一个直播，直播标题为**测试直播**，如下图所示。

![[development-modify-live-streaming_O1CN01Y50fbq1HgwsEYJDeg_!!6000000000788-2-tps-2586-1098.png]]

调用本接口，根据直播ID，修改直播的属性信息。修改标题**测试直播**为**测试直播1**，修改直播封面等信息。接口调用成功后，效果如下图所示。

![[development-modify-live-streaming_O1CN01HAH2W622d5TfNzFiQ_!!6000000007142-2-tps-2790-1368.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉直播获取数据写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=live_1.0%23UpdateLive) |
| 第三方企业应用 | 支持 | 钉钉直播获取数据写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=live_1.0%23UpdateLive) |
| 第三方个人应用 | 暂不支持 | 钉钉直播获取数据写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/live/lives HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "liveId" : "String",
  "unionId" : "String",
  "title" : "String",
  "introduction" : "String",
  "coverUrl" : "String",
  "preStartTime" : Long,
  "preEndTime" : Long
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| liveId | String | 是 | 直播ID，可调用[创建直播](/document/development/create-live-streaming)接口liveId参数值。 |
| unionId | String | 是 | 主播的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 |
| title | String | 否 | 直播标题。 |
| introduction | String | 否 | 直播简介。 |
| coverUrl | String | 否 | 直播的封面地址。 |
| preStartTime | Long | 否 | 直播的预计开播时间戳，单位毫秒。 |
| preEndTime | Long | 否 | 直播的预计结束时间戳，单位毫秒。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果。 |
| success | Boolean | 操作是否成功。   * **true**：成功 * **false**：失败 |

## 错误码

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | invalidParam | lippi\_live.stream.paramError | 参数错误 |
| 400 | contentNotCompliant | lippi\_live.validate.contentNotCompliant | 内容不合规 |
| 400 | liveNotExist | this live not exist, please check liveId | 直播不存在，请检查直播ID |
| 400 | liveStatusNotMatch | this live status is not match, only forecast can update | 直播状态检查失败，预告态直播才允许修改 |
| 404 | noPermission | lippi\_live.stream.noPermission | 鉴权失败 |
| 500 | serviceError | systemError | 系统服务错误 |
