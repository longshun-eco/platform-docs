## 什么是AI接口

AI接口是由阿里巴巴达摩院提供的功能服务，用于提升企业办公效率，主要有以下三个使用场景。

### 文本翻译

输入一段文本，得到指定语言翻译后的译文，支持多种语言的互译，详情请参考[钉钉文本翻译](/document/development/dingtalk-translation)。

### 文字识别

上传一个带有文字的图片，调用本接口可以识别图片中的文字并输出，详情请参考[OCR文字识别](/document/development/structured-image-recognition-api)。

## 如何使用AI接口

钉钉企业组织在[企业开发者后台](https://open-dev.dingtalk.com/)，创建钉钉应用并开通AI接口权限，即可调用接口使用。

功能使用效果可以在钉钉手机客户端-我的-发现-小工具中查看。

## AI接口能力

AI提供了丰富的接口开放能力，开发者通过API接口可以实现AI和企业业务系统打通。

|  |  |  |  |
| --- | --- | --- | --- |
| API | 说明 | 新版规范（新版服务端API） | 旧版规范（服务端API） |
| 文本翻译 | 翻译一段文本。 | - | [钉钉文本翻译](/document/development/dingtalk-translation) |
| OCR文字识别 | 识别一张图片上的文字。 | - | [OCR文字识别](/document/development/structured-image-recognition-api) |

## 如何接入AI接口能力

钉钉提供了AI接口接入流程示例，请参见[实现文本内容翻译](/document/development/translation-of-text-content)。
