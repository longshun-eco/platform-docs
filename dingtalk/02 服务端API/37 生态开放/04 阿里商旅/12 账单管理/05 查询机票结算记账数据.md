更新于 2026-01-29通过此接口查询阿里商旅中的机票结算记账数据，支持按企业、时间范围、分页等条件筛选，适用于财务对账、自动化记账系统对接等场景。

本接口主要用于以下业务场景：

* **企业财务月度对账**：财务系统定期拉取上月机票消费明细，用于生成财务报表和成本分析。
* **自动化记账系统对接**：与企业ERP或财务软件集成，自动同步商旅消费数据，减少人工录入。
* **费用审计与合规检查**：结合审批单号（applyId）和订单信息，验证差旅行为是否符合公司政策。
* **成本中心与项目核算**：根据部门、项目代码、成本中心等字段进行多维度费用归集。

## 请求方法

```
GET /v1.0/alitrip/billSettlements/flights?corpId=String&category=Long&pageSize=Long&periodStart=String&pageNumber=Long&periodEnd=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```
