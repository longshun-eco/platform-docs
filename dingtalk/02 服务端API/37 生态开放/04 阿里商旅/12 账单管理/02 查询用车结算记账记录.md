更新于 2026-01-29通过本接口，企业可查询员工在商旅出行前的机票、酒店或火车行程预估费用，用于预算控制、差标合规性判断及审批流程中的费用决策支持。

本接口适用于企业内部差旅管理系统与钉钉集成的场景。当员工提交商旅申请时，系统可通过调用此接口获取当前行程的预估费用（如机票最低/最高价、酒店差标等），结合企业报销政策进行前置费用管控。

## 请求方法

```
GET /v1.0/alitrip/billSettlements/cars?corpId=String&category=Long&pageSize=Long&periodStart=String&periodEnd=String&pageNumber=Long HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```
