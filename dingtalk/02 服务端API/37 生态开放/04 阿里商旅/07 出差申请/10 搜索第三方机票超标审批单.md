更新于 2026-01-29通过此接口查询第三方机票的超标审批单详情，支持企业差旅管理系统与阿里商旅系统的集成场景。

## **使用场景**

本接口用于企业差旅管理系统中，查询第三方机票超标审批单的状态与详细信息，适用于已接入阿里商旅服务的企业或ISV服务商。典型场景包括：员工提交超标出行申请后，系统需实时获取审批进度、出行意图及原始差旅标准等数据，用于后续流程处理或展示。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 阿里商旅专用权限点 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=alitrip_1.0%23GetFlightExceedApply) |
| 第三方企业应用 | 支持 | 阿里商旅专用权限点 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=alitrip_1.0%23GetFlightExceedApply) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/alitrip/exceedapply/getFlight?corpId=String&applyId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| corpId | String | 是 | 第三方企业的corpId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| corpId | String | 第三方企业的corpId。 |
| applyId | Long | 商旅超标审批单ID。 |
| status | Integer | 审批单状态，取值：   * **0**：审批中 * **1**：已同意 * **2**：已拒绝 |
| btripCause | String | 出差原因。 |
| exceedType | Integer | 超标类型，取值：   * **1**：折扣 * **2**、**8**、**10**：时间 * **3**、**9**、**11**：折扣和时间 |
| exceedReason | String | 超标原因。 |
| originStandard | String | 原差旅标准。 |
| submitTime | String | 审批单提交时间。 |
| userId | String | 第三方用户的userid。 |
| applyIntentionInfoDO | Object | 意向出行信息。 |
| arrCity | String | 到达城市三字码。 |
| arrCityName | String | 到达城市名称。 |
| arrTime | String | 到达时间。 |
| cabin | String | 超标的舱位，取值：   * **F**：头等舱 * **C**：商务舱 * **Y**：经济舱 * **P**：超值经济舱 |
| cabinClass | Integer | 申请超标的舱等，取值：   * **0**：头等舱 * **1**：商务舱 * **2**：经济舱 * **3**：超值经济舱 |
| cabinClassStr | String | 舱等描述。 |
| depCity | String | 出发城市三字码。 |
| depCityName | String | 出发城市名称。 |
| depTime | String | 出发时间。 |
| discount | double | 折扣。 |
| flightNo | String | 航班号。 |
| price | Long | 意向航班价格（元）。 |
| type | Integer | 超标类型，取值：   * **1**：折扣 * **2、8、10**：时间 * **3、9、11**：折扣和时间 |
| thirdpartApplyId | String | 第三方出差审批单号。 |
