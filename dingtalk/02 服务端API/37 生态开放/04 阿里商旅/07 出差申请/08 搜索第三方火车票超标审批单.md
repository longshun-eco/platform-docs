更新于 2026-01-29通过此接口查询第三方火车票超标审批单的详细信息，支持根据企业`corpId`和商旅审批单`applyId`获取审批状态、出行意向、超标原因等关键数据。

## 使用场景

该接口适用于企业差旅管理系统中对员工提交的火车票超标审批申请进行查询和处理。典型业务场景包括：

* 审批流系统获取超标申请详情，用于后续人工或自动审批决策。
* 数据分析平台拉取超标申请记录，进行差旅合规性统计与报表生成。
* 财务报销系统对接超标审批结果，作为费用核销的依据之一。
* 与内部OA系统集成，实现审批状态同步和消息通知。

## 请求方法

```
GET /v1.0/alitrip/exceedapply/getTrain?corpId=String&applyId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
