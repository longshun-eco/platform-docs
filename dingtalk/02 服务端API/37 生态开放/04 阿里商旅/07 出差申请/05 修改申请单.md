更新于 2026-01-29通过此接口可修改已创建的出差申请单，支持对申请人信息、行程详情、审批状态等核心字段进行更新，适用于企业差旅管理系统与钉钉审批流程的集成场景。

## 使用场景

本接口主要用于企业员工出差申请单的动态调整，典型业务场景包括：

* 员工在提交出差申请后需修改出行时间、交通方式或出差事由；
* 与第三方商旅平台对接时，同步更新行程变更信息至钉钉审批系统；

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 是 | 阿里商旅专用写入权限点 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingtalk.oapi.alitrip.btrip.approval.modify) |
| 第三方企业应用 | 是 | 阿里商旅专用写入权限点 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.alitrip.btrip.approval.modify) |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/alitrip/btrip/approval/modify`
