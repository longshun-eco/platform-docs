更新于 2026-01-29通过此接口设置成本中心的人员信息。

## 使用场景

本接口适用于以下业务场景：

* 同步外部差旅管理系统（如阿里商旅）中的成本中心人员信息至钉钉。
* 批量更新某成本中心下的员工归属关系，实现精细化费用管控。

![[development-set-up-cost-center-personnel-information_p375411.png]]

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |

| 第三方企业应用 | 是 | 阿里商旅专用写入权限点 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.alitrip.btrip.cost.center.entity.set) |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/alitrip/btrip/cost/center/entity/set`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | 6d1bxxxx | 调用该API的应用凭证。   * 企业内部应用，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 * 第三方企业应用，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| rq | OpenCostCenterSetEntityRq | 是 |  | 请求对象，包含成本中心及人员列表信息。 |
| thirdpart\_id | String | 是 | cost1 | 第三方成本中心id。 |
| entity\_list | OpenOrgEntityDo[] | 是 |  | 人员信息列表。 |
| entity\_id | String | 是 | 12345 | 员工/部门/角色id。 |
| entity\_type | String | 是 | 1 | 人员类型：   * **1**：员工 * **2**：部门 * **3**：角色 |
| corpid | String | 是 | corp1 | 企业的corpid。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| success | Boolean | true | 成本标识。 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | 成功 | 返回码描述。 |
| result | OpenCostCenterSetEntityRs | result | 结果对象。 |
| add\_num | Number | 1 | 增加的人员信息条数。 |
| remove\_num | Number | 2 | 删除的人员信息条数。 |
| selected\_user\_num | Number | 5 | 该成本中心下员工总数。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/alitrip/btrip/cost/center/entity/set?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "rq":{
                "thirdpart_id":"cost1",
                "corpid":"corp1",
                "entity_list":{
                        "entity_type":"1",
                        "entity_id":"12345"
                }
        }
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/alitrip/btrip/cost/center/entity/set");
OapiAlitripBtripCostCenterEntitySetRequest req = new OapiAlitripBtripCostCenterEntitySetRequest();
OpenCostCenterSetEntityRq obj1 = new OpenCostCenterSetEntityRq();
obj1.setThirdpartId("cost1");
List list3 = new ArrayList();
OpenOrgEntityDo obj4 = new OpenOrgEntityDo();
list3.add(obj4);
obj4.setEntityId("12345");
obj4.setEntityType("1");
obj1.setEntityList(list3);
obj1.setCorpid("corp1");
req.setRq(obj1);
OapiAlitripBtripCostCenterEntitySetResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
        "errcode":"0",
        "result":{
                "remove_num":"2",
                "add_num":"1",
                "selected_user_num":"5"
        },
        "success":"true",
        "errmsg":"成功"
}
```
