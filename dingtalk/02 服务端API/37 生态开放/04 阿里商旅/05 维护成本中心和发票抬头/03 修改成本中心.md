更新于 2026-01-29通过此接口修改成本中心的基本信息，支持更新成本中心名称、适用范围、绑定支付宝账号等字段，适用于企业财务系统与外部商旅平台的数据同步场景。

## 使用场景

本接口主要用于企业差旅管理系统中成本中心信息的维护与同步。典型业务场景包括：

* 企业财务系统批量更新成本中心名称或编号
* 同步第三方商旅平台的成本中心变更至钉钉组织架构
* 根据组织调整动态修改成本中心的适用范围（全员/部分员工）
* 绑定或更换用于报销结算的支付宝账户

**调用时机建议**：

1. 在完成成本中心创建后，需更新初始配置时调用
2. 当企业组织架构发生变动，需同步更新对应成本中心信息时
3. 系统对接过程中，作为数据一致性校准的补充接口使用

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/alitrip/btrip/cost/center/modify`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | 6d1bxxxx | 调用该API的应用凭证。   * 企业内部应用，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 * 第三方企业应用，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| rq | OpenCostCenterModifyRq | 是 |  | 请求对象，封装成本中心修改所需的所有参数。 |
| alipay\_no | String | 否 | a@alipay.com | 绑定的支付宝账号，用于后续费用结算。 |
| title | String | 是 | 阿里商旅 | 成本中心名称。 |
| scope | Number | 是 | 1 | 适用范围：   * **1**：全员 * **2**：部分员工 |
| thirdpart\_id | String | 是 | cost1 | 第三方成本中心id。 |
| number | String | 否 | 12345 | 成本中心编号。 |
| corpid | String | 是 | corp1 | 企业的corpid。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| success | Boolean | true | 调用是否成功，true表示请求处理成功。 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/alitrip/btrip/cost/center/modify?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "rq":{
                "number":"12345",
                "thirdpart_id":"cost1",
                "corpid":"corp1",
                "scope":"1",
                "title":"阿里商旅",
                "alipay_no":"a@alipay.com"
        }
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/alitrip/btrip/cost/center/modify");
OapiAlitripBtripCostCenterModifyRequest req = new OapiAlitripBtripCostCenterModifyRequest();
OpenCostCenterModifyRq obj1 = new OpenCostCenterModifyRq();
obj1.setAlipayNo("a@alipay.com");
obj1.setTitle("阿里商旅");
obj1.setScope(1L);
obj1.setThirdpartId("cost1");
obj1.setNumber("12345");
obj1.setCorpid("corp1");
req.setRq(obj1);
OapiAlitripBtripCostCenterModifyResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
        "errcode":"0",
        "success":"true",
        "errmsg":"成功"
}
```
