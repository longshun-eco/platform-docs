更新于 2026-01-29调用本接口可查询企业用户可用的发票列表，适用于员工报销、商旅预订等场景中核验可使用发票信息。

## 使用场景

该接口主要用于以下业务场景：

* 企业员工在提交报销申请时，查询个人名下可使用的发票列表。
* 与阿里商旅系统集成，在预订机票、酒店时自动匹配并核销符合条件的发票。
* 财务管理系统中批量获取发票数据，用于对账和审计。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |

| 第三方企业应用 | 是 | 阿里商旅专用权限点 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.alitrip.btrip.invoice.search) |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/alitrip/btrip/invoice/search`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | dc73axxxx | 调用该API的应用凭证。   * 企业内部应用，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 * 第三方企业应用，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/alitrip/btrip/invoice/search?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "rq":{
                "corpid":"corp1",
                "title":"发票1",
                "userid":"user1"
        }
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/alitrip/btrip/invoice/search");
OapiAlitripBtripInvoiceSearchRequest req = new OapiAlitripBtripInvoiceSearchRequest();
OpenInvoiceRq obj1 = new OpenInvoiceRq();
obj1.setTitle("发票1");
obj1.setUserid("user1");
obj1.setCorpid("corp1");
req.setRq(obj1);
OapiAlitripBtripInvoiceSearchResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
        "errcode":0,
        "success":"true",
        "errmsg":"成功",
        "invoice":{
                "id":"123",
                "title":"阿里巴巴"
        }
}
```
