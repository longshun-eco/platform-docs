更新于 2025-09-23调用本接口获取企业实名地址。当企业未实名时，可通过该接口获取实名地址，在应用内展示给企业实名入口。

e签宝企业实名只能由钉钉管理员或子管理员进行，因此实名入口的展示建议做权限判断，只展示给有权限操作的用户，同时进入实名页面后e签宝也会进行鉴权。e签宝应用首页也有实名入口，ISV非必须做实名提醒，按需使用即可。

**说明**

地址有效期为2小时。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 暂不支持 | E签宝数据管理权限 | 暂不支持 |
| 第三方企业应用 | 支持 | E签宝数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=esign_1.0%23GetCorpRealnameUrl) |
| 第三方个人应用 | 暂不支持 | E签宝数据管理权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/esign/corps/realname HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "userId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 当前用户的userId。  **说明**  必须是管理员。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| code | Integer | 返回码。 |
| message | String | 返回码描述。 |
| data | Object | 返回结果。 |
| taskId | String | 任务Id。 |
| pcUrl | String | PC端实名地址。 |
| mobileUrl | String | 移动端实名地址。 |
