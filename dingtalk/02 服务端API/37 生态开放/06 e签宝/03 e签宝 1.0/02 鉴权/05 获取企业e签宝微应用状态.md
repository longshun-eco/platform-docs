更新于 2025-09-23调用本接口获取企业的e签宝微应用的状态。企业开启电子签章时，第一步首先需要确保企业已安装e签宝微应用，需要通过该接口查询是否安装，已安装则直接获取授权地址进授权页，未安装则进行展示渠道码进行安装引导。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 暂不支持 | E签宝数据管理权限 | 暂不支持 |
| 第三方企业应用 | 支持 | E签宝数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=esign_1.0%23GetCropStatus) |
| 第三方个人应用 | 暂不支持 | E签宝数据管理权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/esign/corps/statuses HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| data | Object | 返回结果。 |
| authStatus | String | 企业微应用授权状态   * **AUTH**：已授权e签宝微应用 * **UNAUTH**：未授权已授权e签宝微应用 |
| installStatus | String | 企业微应用安装状态：   * **OPEN**：e签宝微应用已经安装 * **CLOSE**：e签宝微应用停止或已卸载 |
| code | Integer | 返回码。 |
| message | String | 返回码描述。 |
