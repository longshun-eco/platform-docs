更新于 2025-09-23调用本接口在e签宝下转售订单给最终真正使用电子合同的用户。该模式需要提前和商务同学对接，确定每份电子合同的底价，并签订商务合同后，可使用该模式给用户下单。

**说明**

调用接口时直接传入本次给该企业下单多少份电子合同，e签宝将会充值合同份数到用户账号上。必须在企业实名后才能调该接口转售（转售对象必须为实名企业），否则转售会报错。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 暂不支持 | E签宝数据管理权限 | 暂不支持 |
| 第三方企业应用 | 支持 | E签宝数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=esign_1.0%23OrderResale) |
| 第三方个人应用 | 暂不支持 | E签宝数据管理权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/esign/orders/resale HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "serviceStartTime" : Long,
  "serviceStopTime" : Long,
  "orderCreateTime" : Long,
  "orderId" : "String",
  "quantity" : Long
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| serviceStartTime | Long | 是 | 合同生效起始时间，Unix时间戳。 |
| serviceStopTime | Long | 是 | 合同失效截止日期，Unix时间戳，默认有效时间一年。 |
| orderCreateTime | Long | 是 | 下单时间，Unix时间戳。 |
| orderId | String | 是 | ISV方的订单Id，用于幂等，请保证唯一性。 |
| quantity | Long | 是 | 购买数量（电子合同份数）。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| code | Integer | 返回码。 |
| message | String | 返回码描述。 |
| data | Object | 返回结果。 |
| esignOrderId | String | e签宝订单Id。 |
