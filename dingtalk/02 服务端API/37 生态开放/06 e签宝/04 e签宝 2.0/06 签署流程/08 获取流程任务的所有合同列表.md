更新于 2025-09-23获取流程任务的所有合同列表，收到签署完成消息后查询。

文件签署完成后，将推送消息给ISV，此时可通过任务ID获取文件列表，ISV可在应用内展示合同列表给用户，并提供文件查看/下载。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | E签宝数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=esign_2.0%23GetFlowDocs) |
| 第三方企业应用 | 支持 | E签宝数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=esign_2.0%23GetFlowDocs) |
| 第三方个人应用 | 暂不支持 | E签宝数据管理权限 | 暂不支持 |

## 请求方法

```
GET /v2.0/esign/flowTasks/{taskId}/docs HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| serviceGroup | String | 否 | 预留参数，此版本无需此参数。 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)接口获取。 |

## Path参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| taskId | String | 是 | 签署返回的任务ID。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| data | Array | 业务数据。 |
| fileId | String | 文件上传后的文件ID。 |
| fileName | String | 文件名称。 |
| fileUrl | String | 文件地址。 |
