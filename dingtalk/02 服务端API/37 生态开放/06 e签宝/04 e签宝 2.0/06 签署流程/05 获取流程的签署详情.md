更新于 2025-09-23调用本接口根据taskId获取流程签署相关的详细信息，包括用户姓名及对应的签署状态。

![[development-get-the-details-of-process-signing_O1CN01NzVK9b1sDlQdC9S1y_!!6000000005733-2-tps-1112-1318.png]]

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | E签宝数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=esign_2.0%23GetSignDetail) |
| 第三方企业应用 | 支持 | E签宝数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=esign_2.0%23GetSignDetail) |
| 第三方个人应用 | 暂不支持 | E签宝数据管理权限 | 暂不支持 |

## 请求方法

```
GET /v2.0/esign/signTasks/{taskId} HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| serviceGroup | String | 否 | 预留参数，此版本无需此参数。 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证。   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/orgapp/obtain-the-access_token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/isvapp/obtain-the-access_token-of-the-authorized-enterprise)接口获取。 |

## Path参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| taskId | String | 是 | 签署返回的任务ID。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| businessScene | String | 流程业务场景。 |
| flowStatus | Float | 流程状态，取值：   * **0**：草稿 * **1**：签署中 * **2**：完成 * **3**：撤销 * **4**：终止 * **5**：过期 * **6**：删除 * **7**：拒签 |
| signers | Array | 流程任务合同列表。 |
| signStatus | Float | 签署状态，取值：   * **0**：待签 * **1**：未签 * **2**：已签 * **3**：待审批 * **4**：拒签 |
| signerName | String | 签署人姓名。 |
