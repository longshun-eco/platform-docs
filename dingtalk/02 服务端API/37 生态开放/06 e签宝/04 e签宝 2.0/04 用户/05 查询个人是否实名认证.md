更新于 2025-09-23调用本接口查询当前用户是否已在e签宝完成实名认证。已实名则会返回个人实名名称，未实名则可在应用内提醒用户进行实名。

**说明**

完成实名认证后才能使用签署，ISV可根据需要在应用内展示给用户实名状态。

![[development-query-personal-information_O1CN01Cls7QK1lrQPokHzgz_!!6000000004872-2-tps-2580-820.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | E签宝数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=esign_2.0%23GetUserInfo) |
| 第三方企业应用 | 支持 | E签宝数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=esign_2.0%23GetUserInfo) |
| 第三方个人应用 | 暂不支持 | E签宝数据管理权限 | 暂不支持 |

## 请求方法

```
GET /v2.0/esign/users/{userId} HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| serviceGroup | String | 否 | 预留参数，此版本无需此参数。 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 当前用户的userid。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| isRealName | String | 个人是否实名。 |
| userRealName | String | 如果已经实名，会返回个人实名名称。 |
