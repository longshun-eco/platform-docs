更新于 2026-01-29通过此接口查询指定工单的动作记录，支持分页获取操作日志。

### 使用场景

本接口用于查询指定工单的动作记录，适用于客服系统中的操作日志追踪、审计分析、服务过程回溯等业务场景。例如：企业可基于该接口实现工单处理全流程监控，识别关键操作节点，提升服务质量与合规性。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 客服行业的工单读接口权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=customerService_1.0%23PageListAction) |
| 第三方企业应用 | 支持 | 客服行业的工单读接口权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=customerService_1.0%23PageListAction) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/customerService/tickets/{ticketId}/actions?openInstanceId=String&productionType=Long&nextToken=String&maxResults=Long HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| ticketId | String | 是 | 工单ID。 |

## Query参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| openInstanceId | String | 否 | 实例ID。 |
| productionType | Long | 否 | 智能客服产品类型：   * **1**：智能客服 * **2**：服务群 * **1003**：服务台 * **1004**：小蜜客服 |
| nextToken | String | 是 | 查询数据的起始位置，0表示从头开始。 |
| maxResults | Long | 是 | 查询单页查询的最大条目数，最大值为100。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| nextCursor | Long | 是否还有下一页数据，当返回结果里没有nextCursor时，表示分页结束。 |
| total | Long | 总记录数。 |
| list | Array | 查询列表。 |
| operatorId | String | 操作人的userid。 |
| operator | String | 操作人。 |
| operatorRole | String | 操作人角色。 |
| actionCode | String | 动作code。 |
| actionContent | Array | 操作记录。 |
| displayValue | String | 标签的值。 |
| displayName | String | 字段的展示名称。 |
| name | String | 字段的key。 |
| value | String | 字段的值。 |
| valueType | String | 字段的类型：   * **attachments**：附件 * **text**：文本 |
