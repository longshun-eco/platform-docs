更新于 2026-01-29通过此接口可分页查询工单记录，支持按模板、状态、时间范围等条件筛选，适用于客服系统中查看用户提交的工单数据，便于实现工单列表展示、历史记录追溯和分页浏览等功能。

![[development-intelligent-customer-service-paging-query-work-order_O1CN017pgJco1axsAXg4eAj_!!6000000003397-2-tps-936-436.png]]

### 使用场景

本接口适用于以下业务场景：

* 客服后台系统中展示用户提交的工单列表；
* 支持分页加载大量工单数据，提升页面响应性能；
* 根据工单模板、状态、创建时间等条件进行筛选查询；
* 与第三方CRM系统集成时，基于 foreignId 和 sourceId 关联外部用户信息。

| 第三方企业应用 | 支持 | 客服行业的工单读接口权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=customerService_1.0%23PageListTicket) |

## 请求方法

```
GET /v1.0/customerService/tickets?openInstanceId=String&productionType=Integer&templateId=String&ticketId=String&sourceId=String&foreignId=String&ticketStatus=String&startTime=Long&endTime=Long&nextToken=String&maxResults=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| openInstanceId | String | 否 | 实例ID。 |
| productionType | Integer | 否 | 智能客服产品类型：   * **1**：智能客服 * **2**：服务群 * **1003**：服务台 * **1004**：小蜜客服 |
| templateId | String | 是 | 工单模板ID。 |
| ticketId | String | 否 | 工单ID。 |
| sourceId | String | 否 | 会员来源，取`diamond`配置的值。  **说明**   * 必须与`foreignId`参数配合使用。 * 两者需同时为空或同时非空。 |
| foreignId | String | 否 | 第三方用户userid。  **说明**   * 必须与 sourceId 参数配合使用。 * 两者需同时为空或同时非空。 |
| ticketStatus | String | 否 | 工单状态。  **说明**  在钉钉工单类型配置里的工单状态机里自主配置。 |
| startTime | Long | 否 | 开始时间，时间戳，单位毫秒。 |
| endTime | Long | 否 | 结束时间，时间戳，单位毫秒。 |
| nextToken | String | 是 | 查询数据的起始位置，0表示从头开始。 |
| maxResults | Integer | 是 | 查询单页查询的最大条目数，最大值为100。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| nextCursor | Long | 是否还有下一页数据。 |
| total | Long | 总记录数。 |
| list | Array | 查询结果。 |
| foreignId | String | 第三方会员。  **说明**  如果系统中不存在需要在CRM中新建。 |
| sourceId | String | 第三方来源。 |
| foreignName | String | 第三方会员名。 |
| templateId | String | 工单模板ID。 |
| title | String | 工单标题。 |
| ticketId | String | 工单ID。 |
| ticketStatus | String | 工单状态。  **说明**  在钉钉工单类型配置里的工单状态机里自主配置。 |
| openInstanceId | String | 实例ID。 |
| productionType | Integer | 智能客服产品类型：   * **1**：智能客服 * **2**：服务群 * **1003**：服务台 * **1004**：小蜜客服 |
| gmtCreate | String | 创建时间。 |
| gmtModified | String | 最新修改时间。 |
| bizDataMap | Map | 工单自定义业务字段信息。 |
