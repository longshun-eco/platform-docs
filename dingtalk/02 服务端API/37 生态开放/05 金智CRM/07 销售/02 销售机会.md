更新于 2026-01-29通过此接口新增或编辑销售机会，实现CRM系统中商机数据的创建与更新。

## 使用场景

本接口适用于企业内部CRM系统的销售管理模块，典型业务场景包括：

* 销售人员录入新的客户商机信息（当`msgid`为0或不传时）；
* 更新已有销售机会的跟进状态、金额、阶段等关键字段（当`msgid`为已存在记录ID时）；
* 系统间集成同步外部渠道获取的潜在客户数据到钉钉金智CRM；
* 自动化流程中根据业务规则动态创建或修改销售线索。

## 请求方法

```
POST /v1.0/jzcrm/sales HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "datatype" : Long,
  "stamp" : Long,
  "msgid" : Long,
  "data" : {
    "data_userid" : "String",
    "xsh_customerid" : "String",
    "xsh_title" : "String",
    "xsh_date" : "String",
    "xsh_number" : "String",
    "xsh_lxrid" : "String",
    "xsh_lianxi" : "String",
    "xsh_type" : "String",
    "xsh_from" : "String",
    "xsh_preside" : "String",
    "xsh_provider" : "String",
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| datatype | Long | 是 | 数据类型，固定值**158**。 |
| stamp | Long | 是 | 时间戳。 |
| msgid | Long | 否 | 数据ID、  **说明**  值为0或不填时，为新增数据。 |
| data | Object | 否 | 编辑数据。 |
| data\_userid | String | 是 | 创建人。 |
| xsh\_customerid | String | 是 | 对应客户。 |
| xsh\_title | String | 是 | 主题。 |
| xsh\_date | String | 是 | 发现时间。 |
| xsh\_number | String | 否 | 机会编号。 |
| xsh\_lxrid | String | 否 | 联系人。 |
| xsh\_lianxi | String | 否 | 联系方式。 |
| xsh\_type | String | 否 | 类型。 |
| xsh\_from | String | 否 | 来源。 |
| xsh\_preside | String | 否 | 所有者。 |
| xsh\_provider | String | 否 | 提供人。 |
| xsh\_require | String | 否 | 客户需求。 |
| xsh\_expdate | String | 否 | 预计签单日。 |
| xsh\_expmoney | String | 否 | 预期金额。 |
| xsh\_moneynote | String | 否 | 外币备注。 |
| xsh\_phase | String | 否 | 阶段。 |
| xsh\_knx | String | 否 | 可能性。 |
| xsh\_state | String | 否 | 状态。 |
| xsh\_phasenote | String | 否 | 阶段备注。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| time | String | 响应时间。 |
| msgid | Long | 编辑数据的ID。 |
