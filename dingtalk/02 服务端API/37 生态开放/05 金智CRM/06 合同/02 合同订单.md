更新于 2026-01-29通过本接口可新增或编辑合同订单，支持在企业CRM系统中进行合同数据的创建与更新操作。

## 使用场景

该接口适用于企业内部CRM系统的合同管理业务场景，典型用例如下：

* 销售人员在钉钉应用中创建新的客户合同订单；
* 编辑已有合同信息（如金额、交付时间、产品明细等）；
* 与其他业务系统（如OA、进销存）进行合同数据同步。

![[development-add-or-edit-contract-orders_p1049307.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 金智CRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=jzcrm_1.0%23EditOrder) |
| 第三方企业应用 | 支持 | 金智CRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=jzcrm_1.0%23EditOrder) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/jzcrm/orders HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "datatype" : Long,
  "stamp" : Long,
  "msgid" : Long,
  "data" : {
    "data_userid" : "String",
    "ht_customerid" : "String",
    "ht_date" : "String",
    "ht_preside" : "String",
    "ht_state" : "String",
    "ht_summoney" : "String",
    "ht_order" : "String",
    "ht_title" : "String",
    "ht_number" : "String",
    "ht_lxrid" : "String",
    "ht_lxrinfo" : "String",
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |
