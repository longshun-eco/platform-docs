更新于 2026-01-29通过此接口新增或编辑金智CRM系统中的产品信息，支持企业内部应用与第三方企业应用的数据同步。

## 使用场景

本接口适用于以下业务场景：

* 在金智CRM系统中创建新产品记录（当`msgid`为0或未提供时）；
* 实现外部系统（如ERP、WMS）与钉钉内CRM模块的产品数据双向同步；
* 支持企业内部应用和第三方企业应用调用，满足多组织架构下的统一商品管理需求。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 金智CRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=jzcrm_1.0%23EditGoods) |
| 第三方企业应用 | 支持 | 金智CRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=jzcrm_1.0%23EditGoods) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/jzcrm/goods HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "datatype" : Long,
  "stamp" : Long,
  "msgid" : Long,
  "data" : {
    "data_userid" : "String",
    "cpname" : "String",
    "cpunit" : "String",
    "unitrate" : "String",
    "cp_parentid" : "String",
    "cptype" : "String",
    "cpguige" : "String",
    "typeid" : "String",
    "cpno" : "String",
    "isstop" : "String",
    "addedtime" : "String",
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |
