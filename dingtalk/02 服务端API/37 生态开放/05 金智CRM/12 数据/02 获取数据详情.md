更新于 2026-01-29调用本接口获取金智CRM中指定单据类型的字段定义及数据详情。

## 使用场景

该接口适用于需要从金智CRM系统中获取特定类型单据的详细数据和字段元信息的业务场景，例如：

* 同步客户、订单、合同等核心业务数据至企业自有系统。
* 在自建应用或门户页面中展示客户档案、订单详情等结构化信息。
* 构建数据分析看板时，拉取原始业务记录并结合字段中文名称进行可视化呈现。

典型调用时机包括：用户点击查看详情页、定时任务同步数据、审批流程触发数据回写等。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 金智CRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=jzcrm_1.0%23GetDataView) |
| 第三方企业应用 | 支持 | 金智CRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=jzcrm_1.0%23GetDataView) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/jzcrm/dataView?datatype=String&msgid=Long HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| datatype | String | 是 | 数据类型。 |
| msgid | Long | 是 | 数据ID。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| data | Object | 数据详情。 |
| detail | Map<String, String> | 数据详情。 |
| dataname | Map<String, Map> | 字段明细。 |
|  | Map | 字段明细。 |
| time | String | 响应时间。 |
