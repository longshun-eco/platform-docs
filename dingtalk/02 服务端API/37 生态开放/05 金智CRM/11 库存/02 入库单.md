更新于 2026-01-29通过此接口可新增或编辑金智CRM系统中的入库单据。

## 使用场景

该接口适用于以下业务场景：

* 将外部WMS、ERP等系统中的入库单数据同步至金智CRM系统。
* 实现企业内部仓储流程的自动化，减少手工录入错误。
* 在审批流程完成后自动创建或更新入库单状态。
* 支持多系统间的数据联动，提升供应链协同效率。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 金智CRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=jzcrm_1.0%23EditIntostock) |
| 第三方企业应用 | 支持 | 金智CRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=jzcrm_1.0%23EditIntostock) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/jzcrm/intostocks HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "datatype" : Long,
  "stamp" : Long,
  "msgid" : Long,
  "data" : {
    "data_userid" : "String",
    "libiodate" : "String",
    "stocklibid" : "String",
    "libiostate" : "String",
    "billno" : "String",
    "customerid" : "String",
    "empid" : "String",
    "inorout" : "String",
    "libioname" : "String",
    "orderid" : "String",
    "askempid" : "String",
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| datatype | Long | 是 | 数据类型，固定值**189**。 |
| stamp | Long | 是 | 时间戳。 |
| msgid | Long | 否 | 数据ID。  **说明**  值为0或不填时，为新增数据。 |
| data | Object | 否 | 编辑数据。 |
| data\_userid | String | 是 | 创建人。 |
| libiodate | String | 是 | 入库日期。 |
| stocklibid | String | 是 | 入库仓库。 |
| libiostate | String | 是 | 入库状态，取值。   * 未入库 * 已入库 |
| billno | String | 是 | 入库单号。 |
| customerid | String | 否 | 取值。   * 供应商ID或名称 * 客户ID或名称 |
| empid | String | 是 | 入库经办人。 |
| inorout | String | 否 | 单据类型   * 入库 * 销售退货 * 生产退料 * 生产入库 * 维修退货 |
| libioname | String | 否 | 入库主题。 |
| orderid | String | 否 | 对应单据。 |
| askempid | String | 否 | 入库申请人。 |
| remark | String | 否 | 申请备注。 |
| auditreson | String | 否 | 入库备注。 |
| child\_mx | String | 是 | 产品明细，json格式。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| time | String | 响应时间 |
| msgid | Long | 编辑数据的id |
