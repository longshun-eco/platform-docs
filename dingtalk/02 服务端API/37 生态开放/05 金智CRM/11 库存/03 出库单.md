更新于 2026-01-29通过此接口可新增或编辑出库单据，支持根据`msgid`判断操作类型（新增或更新）。

## 使用场景

适用于企业仓储管理系统、ERP系统与钉钉金智CRM对接的业务场景，主要用于：

* 自动创建和更新出库单，实现发货信息同步；
* 与订单系统集成，完成从订单到出库的自动化流转；
* 支持多仓库管理下的出库数据统一维护；
* 为审批流程提供数据支撑，例如：审批通过后自动写入出库记录。

**注意事项**：

* 当`msgid`为 0 或未传时，视为新增出库单；
* 非 0 的`msgid`值表示对已有出库单进行编辑；
* 所有必填字段需完整传递，否则可能导致保存失败。

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 金智CRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=jzcrm_1.0%23EditOutstock) |
| 第三方企业应用 | 支持 | 金智CRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=jzcrm_1.0%23EditOutstock) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/jzcrm/outstocks HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "datatype" : Long,
  "stamp" : Long,
  "msgid" : Long,
  "data" : {
    "data_userid" : "String",
    "libiodate" : "String",
    "stocklibid" : "String",
    "libiostate" : "String",
    "billno" : "String",
    "customerid" : "String",
    "empid" : "String",
    "inorout" : "String",
    "libioname" : "String",
    "orderid" : "String",
    "askempid" : "String",
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| datatype | Long | 是 | 数据类型，固定值**191**。 |
| stamp | Long | 是 | 时间戳。 |
| msgid | Long | 否 | 数据ID。  **说明**  值为0或不填时，为新增数据。 |
| data | Object | 否 | 编辑数据。 |
| data\_userid | String | 是 | 创建人。 |
| libiodate | String | 是 | 出库日期。 |
| stocklibid | String | 是 | 出库仓库。 |
| libiostate | String | 是 | 出库状态。 |
| billno | String | 是 | 出库单号。 |
| customerid | String | 否 | 对应客户 |
| empid | String | 是 | 经办人。 |
| inorout | String | 否 | 单据类型。 |
| libioname | String | 否 | 出库主题。 |
| orderid | String | 否 | 对应订单。 |
| askempid | String | 否 | 申请人。 |
| remark | String | 否 | 申请备注。 |
| auditreson | String | 否 | 出库备注。 |
| child\_mx | String | 是 | 产品明细，json格式。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| time | String | 响应时间。 |
| msgid | Long | 编辑数据的ID。 |
