更新于 2025-10-09调用本接口，更新CRM个人客户或企业客户信息，包括客户的负责人列表、协同人列表和关系类型等信息。

例如，在企业的客户管理应用中有个名为王总的客户。调用本接口可更新客户的信息，接口调用效果与下图操作实现效果类似。

![[development-update-crm-personal-customers_O1CN01oeOFl91aGowzYTQS4_!!6000000003303-2-tps-2418-1458.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 维护CRM主数据的接口写入权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=crm_1.0%23UpdateCrmPersonalCustomer) |
| 第三方企业应用 | 支持 | 维护CRM主数据的接口写入权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=crm_1.0%23UpdateCrmPersonalCustomer) |
| 第三方个人应用 | 暂不支持 | 维护CRM主数据的接口写入权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/crm/personalCustomers HTTP/1.1
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "instanceId" : "String",
  "modifierUserId" : "String",
  "modifierNick" : "String",
  "permission" : {
    "ownerStaffIds" : [ "String" ],
    "participantStaffIds" : [ "String" ]
  },
  "relationType" : "String",
  "skipDuplicateCheck" : Boolean,
  "action" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| instanceId | String | 是 | 客户数据ID，调用[根据指定条件查询个人或企业客户数据](/document/development/obtains-crm-individual-customers-in-batches-based-on-specified-query)接口获取instanceId参数值。 |
| modifierUserId | String | 是 | 操作人的用户userId。 |
| modifierNick | String | 否 | 操作人的用户昵称。 |
| data | Map | 是 | 客户数据内容，JSON格式字符串，格式请参见[新增和更新客户字段格式说明V1](/document/development/add-and-update-customer-field-format-description-v1)。 |
| extendData | Map | 否 | 扩展数据。 |
| permission | Object | 否 | 权限。 |
| ownerStaffIds | Array of String | 否 | 负责人的用户userId。 |
| participantStaffIds | Array of String | 否 | 协同人的用户userId。 |
| relationType | String | 否 | 关系类型。   * **crm\_customer**：企业客户 * **crm\_customer\_personal**：个人客户 |
| skipDuplicateCheck | Boolean | 否 | 是否跳过查重字段。   * **true**：是 * **false**：否   **说明**  企业可在查重设置中设置查重字段，如下图所示。 ![[development-update-crm-personal-customers_O1CN01GLJ3lA1NwsJYjwWNw_!!6000000001635-2-tps-2240-1050.png]] |
| action | String | 否 | 取值。   * **publicDraw**：公海领取客户 * **publicAssign**：公海分配客户 * 其余场景：不用传 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| instanceId | String | 客户数据ID。 |
