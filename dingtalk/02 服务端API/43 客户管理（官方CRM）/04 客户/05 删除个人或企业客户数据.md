更新于 2025-10-09调用本接口，删除CRM个人客户或企业客户数据。

例如，调用本接口从客户管理的客户列表中删除一个客户。接口调用效果与下图操作实现效果一致。

![[development-delete-crm-personal-customer_O1CN01rjZl3Q1FmKGZZFIhE_!!6000000000529-2-tps-2430-1466.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 维护CRM主数据的接口写入权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=crm_1.0%23DeleteCrmPersonalCustomer) |
| 第三方企业应用 | 支持 | 维护CRM主数据的接口写入权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=crm_1.0%23DeleteCrmPersonalCustomer) |
| 第三方个人应用 | 暂不支持 | 维护CRM主数据的接口写入权限 | 暂不支持 |

## 请求方法

```sql
DELETE /v1.0/crm/personalCustomers/{dataId}?relationType=String&currentOperatorUserId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| dataId | String | 是 | 客户数据ID，调用[根据指定条件查询个人或企业客户数据](/document/development/obtains-crm-individual-customers-in-batches-based-on-specified-query)接口获取instanceId参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| relationType | String | 否 | 关系类型。   * **crm\_customer**：企业客户 * **crm\_customer\_personal**：个人客户 |
| currentOperatorUserId | String | 是 | 操作人用户userId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| instanceId | String | 被删除的客户数据ID。 |
