更新于 2025-10-09调用本接口，获取CRM个人或企业客户数据，包括私海个人客户、私海企业客户、公海个人客户、公海企业客户。单个ISV单企业限流1QPS。

例如，在企业的客户管理中的客户列表中，客户信息如下图所示。调用本接口，可获取CRM个人或企业客户数据。

![[development-crm-obtains-all-private-sea-customer-data_O1CN01HQwNXH1buUsWOS2Lh_!!6000000003525-2-tps-2874-1202.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 获取CRM主数据的接口访问权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=crm_1.0%23QueryAllCustomer) |
| 第三方企业应用 | 支持 | 获取CRM主数据的接口访问权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=crm_1.0%23QueryAllCustomer) |
| 第三方个人应用 | 暂不支持 | 获取CRM主数据的接口访问权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/crm/customerInstances HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "operatorUserId" : "String",
  "maxResults" : Long,
  "nextToken" : "String",
  "objectType" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorUserId | String | 否 | 操作人员的userId。 |
| maxResults | Long | 否 | 每页条目数，取值范围1~100。 |
| nextToken | String | 否 | 分页游标。   * 如果是首次调用，该参数传空或者null。 * 如果不是首次调用，该参数传上次调用时返回的nextToken。 |
| objectType | String | 否 | 数据类型，不传或者传null时，默认值为**crm\_customer**，具体参数如下。   * **crm\_customer\_personal**：私海个人客户 * **crm\_customer**：私海企业客户 * **open\_customer\_personal**：公海个人客户 * **open\_customer\_org**：公海企业客户 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 分页结果。 |
| nextToken | String | 下一页的游标，为null则表示无更多数据。 |
| values | Array | 客户数据列表。 |
| creatorNick | String | 创建记录的用户昵称。 |
| modifyTime | String | 记录修改时间。 |
| creatorUserId | String | 创建记录的用户的userId。 |
| instanceId | String | 客户数据ID。 |
| data | Map | 数据内容。 |
| extendData | Map | 扩展数据内容。 |
| createTime | String | 记录创建时间。 |
| objectType | String | 数据类型：   * **crm\_customer\_personal**：私海个人客户 * **crm\_customer**：私海企业客户 * **open\_customer\_personal**：公海个人客户 * **open\_customer\_org**：公海企业客户 |
| permission | Object | 数据权限信息。 |
| participantStaffIds | Array of String | 协同人用户userId。 |
| ownerStaffIds | Array of String | 负责人用户userId。 |
| processOutResult | String | 审批结果。   * **terminate**：撤销 * **agree**：同意 * **refuse**：拒绝   **说明**  本接口返回都是agree。 |
| processInstanceStatus | String | 审批状态。   * **NEW**：新创建 * **RUNNING**：审批中 * **TERMINATED**：撤销 * **COMPLETED**：完成   **说明**  本接口返回都是COMPLETED。 |
| maxResults | Long | 每页条目数量。 |
