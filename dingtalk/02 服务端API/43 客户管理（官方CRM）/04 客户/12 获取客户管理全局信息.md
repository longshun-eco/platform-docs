更新于 2025-10-09调用本接口，获取客户管理全局信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 获取CRM主数据的接口访问权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=crm_1.0%23QueryGlobalInfo) |
| 第三方企业应用 | 支持 | 获取CRM主数据的接口访问权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=crm_1.0%23QueryGlobalInfo) |
| 第三方个人应用 | 暂不支持 | 获取CRM主数据的接口访问权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/crm/globalInfos?userId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 用户userId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果。 |
| oemEnable | Boolean | 营销服oem是否生效：   * **true**：生效，去除三方的 logo、付费墙等 * **false**：不生效 |
