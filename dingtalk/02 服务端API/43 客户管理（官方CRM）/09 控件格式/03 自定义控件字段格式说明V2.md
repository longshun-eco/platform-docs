更新于 2025-09-23本文档介绍客户管理支持操作的自定义控件字段信息，在使用新增或更新客户、联系人、跟进记录、自定义对象相关接口时需了解自定义控件字段信息。

## 电话

|  |  |  |  |
| --- | --- | --- | --- |
| 字段类型 | Key | Value | Value |
| 自定义控件 | 通过调用[获取个人或企业客户的元数据](/document/development/get-metadata-description-of-crm-customer-object-1)接口，获取字段列表fields下的字段名称name。 | value值格式为字符串：   * 国际手机号需加国际区号，否则导致无法解析。 * 国内手机号是否添加+86都符合value格式。 * 固定电话是否添加区号都符合value格式。 | ``` {     "key":"PhoneField-xxxxxx",     "value":"185xxxxxxxx" } ``` |

## 部门

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 字段类型 | Key | Value | extValue | 示例 |
| 自定义控件 | 通过调用[获取个人或企业客户的元数据](/document/development/get-metadata-description-of-crm-customer-object-1)接口，获取字段列表fields下的字段名称name。 | value值格式为字符串。 | value需同时符合以下条件：   * extValue格式必须是数组结构。 * extValue必须是相同的部门名称。 | ``` {     "key":"DepartmentField-xxxxxx",     "value": "部门名称",     "extValue": "[{\"name\":\"部门名称\"}]" } ``` |
