更新于 2025-10-09调用本接口，更新客户群组信息。

**说明**

调用本接口更新客户群组信息，新创建的客户群将按照更新后的信息创建群，已经创建的客户群信息不会更新。

例如，企业有群组名为**核心客户**，已经自动创建了三个客户群，分别为核心客户1群、核心客户2群和核心客户3群。如下图所示。

![[development-crm-update-group_O1CN01t0zKXe1YVohBB2DyX_!!6000000003065-2-tps-2412-1118.png]]

调用本接口将群组名称修改为**修改核心客户**，群信息更新情况如下：

* 已创建的1群、2群、3群信息不变。
* 新创建第四个客户群时，群组名称会显示为**修改核心客户4群**，如下图所示。

  ![[development-crm-update-group_O1CN01Hf6yrL1qcptL8y92t_!!6000000005517-2-tps-2706-1266.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 客户管理客户群写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=crm_1.0%23UpdateGroupSet) |
| 第三方企业应用 | 暂不支持 | 客户管理客户群写权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 客户管理客户群写权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/crm/groupSets/set HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "openGroupSetId" : "String",
  "name" : "String",
  "memberQuota" : Integer,
  "ownerUserId" : "String",
  "managerUserIds" : "String",
  "notice" : "String",
  "noticeToped" : Integer,
  "templateId" : "String",
  "welcome" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| openGroupSetId | String | 是 | 群组openGroupSetId，调用[查询客户群组列表](/document/development/query-groups)接口获取openGroupSetId参数值。 |
| name | String | 否 | 群组名。 |
| memberQuota | Integer | 否 | 单个群的人数上限。 |
| ownerUserId | String | 否 | 群主userId。  **说明**  裂变出的新群会自动设置该userId为群主。 |
| managerUserIds | String | 否 | 群管理员userId列表，多个用逗号隔开，裂变出的新群会自动设置这些userId为群管理员。 |
| notice | String | 否 | 群公告文本。  **说明**  裂变出的新群会自动设置上该群的群公告。 |
| noticeToped | Integer | 否 | 群公告是否置顶。   * **0**：否 * **1**：是   **说明**  裂变出的新群会自动设置上该属性。 |
| templateId | String | 否 | 群模板Id。  **说明**  该参数暂不支持使用。 |
| welcome | String | 否 | 新成员入群后收到的欢迎语。  **说明**  该参数暂不支持使用。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
|  | Boolean | 更新操作是否成功， true表示更新成功。 |
