更新于 2025-10-09调用本接口，根据指定查询条件批量获取联系人数据。

**说明**

最多可一次获取200条数据。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 获取CRM主数据的接口访问权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=crm_1.0%23GetContacts) |
| 第三方企业应用 | 支持 | 获取CRM主数据的接口访问权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=crm_1.0%23GetContacts) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/crm/customObjects/contacts/datas/query HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "currentOperatorUserId" : "String",
  "objectType" : "String",
  "nextToken" : "String",
  "maxResults" : Long,
  "providerCorpId" : "String",
  "queryDsl" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| currentOperatorUserId | String | 否 | 用户 userId。 |
| objectType | String | 否 | 对象类型，固定值`crm_ contact`。 |
| nextToken | String | 否 | 分页游标。 |
| maxResults | Long | 是 | 分页大小。 |
| providerCorpId | String | 否 | 企业 corpId。 |
| queryDsl | String | 否 | 查询条件，格式参考[查询DSL说明](/document/development/inner-query-dsl-description)。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 分页结果。 |
| nextToken | String | 下一页的游标。 |
| hasMore | Boolean | 是否有下一页。 |
| maxResults | Long | 分页大小。 |
| values | Array | 数据列表。 |
| gmtModified | String | 记录修改时间。 |
| creatorUserId | String | 创建记录的用户 userId。 |
| instanceId | String | 数据 ID。 |
| data | Map<String, String> | 数据内容。 |
| extendData | Map<String, String> | 扩展数据内容。 |
| gmtCreate | String | 记录创建时间。 |
| objectType | String | 数据类型。 |
| permission | Object | 数据权限信息。 |
| participantUserIds | Array of String | 协同人用户 userId。 |
| ownerUserIds | Array of String | 负责人用户 userId。 |
