更新于 2025-09-23调用本接口，根据联系人实例id列表批量获取联系人数据，最多可一次获取200条数据。

调用本接口，可按照ID列表批量获取联系人数据，包括创建人的用户userId、数据ID、数据内容等。

![[development-retrieves-contact-data-in-batches-based-on-the-id-list_p373790.png]]

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 是 | 获取CRM主数据的接口访问权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingtalk.oapi.crm.objectdata.contact.list) |
| 第三方企业应用 | 否 | — | — |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/crm/objectdata/contact/list`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | dc73axxxx | 调用该接口的应用凭证，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| current\_operator\_userid | String | 否 | manager1 | 操作人用户userId。 |
| data\_id\_list | String | 是 | inst\_Id1, inst\_Id2 | 数据ID列表，通过[根据指定条件查询联系人数据](/document/development/api-getcontacts)接口获取instance\_id参数值，多个用英文逗号隔开。 |
| provider\_corpid | String | 否 | dingxxxx | 自建应用时传入服务商corpId。 ![[development-retrieves-contact-data-in-batches-based-on-the-id-list_p524342.png]] |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| result\_list | ObjectDataInstanceVo[] |  | 实例数据。 |
| gmt\_modified | String | 2020-09-15 00:00:00 | 修改时间。 |
| creator\_userid | String | user1 | 创建人的用户userId。 |
| instance\_id | String | nst\_Id1 | 数据ID。 |
| data | String | {\"customer\_name\":\"XX有限公司\"} | 数据内容。 |
| extend\_data | String | {\"field\_1\":\"CRM\"} | 扩展数据内容。 |
| gmt\_create | String | 2020-09-15 00:00:00 | 记录创建时间。 |
| object\_type | String | crm\_contact | 数据类型。 |
| permission | DataPermissionVo |  | 数据权限信息。 |
| participant\_userid\_list | String[] | ["user01","user02"] | 协同人用户ID列表。 |
| owner\_userid\_list | String[] | ["user01","user02"] | 负责人用户ID列表 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/crm/objectdata/contact/list?access_token=ACCESS_TOKEN
```

请求正文

```json
{
    "current_operator_userid": "manager1",
    "data_id_list": "inst_Id1, inst_Id2",
    "provider_corpid": "dingxxxx"
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/crm/objectdata/contact/list");
OapiCrmObjectdataContactListRequest req = new OapiCrmObjectdataContactListRequest();
req.setCurrentOperatorUserid("manager1");
req.setDataIdList("nst_Id1, inst_Id2");
req.setProviderCorpid("dingxxxx");
OapiCrmObjectdataContactListResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```
