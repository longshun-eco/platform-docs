![[development-get-video-meeting-details_O1CN018WALI01jMA91W4jao_!!6000000004533-2-tps-2768-1224.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 专属钉钉专属数据读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=exclusive_1.0%23GetConferenceDetail) |
| 第三方企业应用 | 支持 | 专属钉钉专属数据读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=exclusive_1.0%23GetConferenceDetail) |
| 第三方个人应用 | 暂不支持 | 专属钉钉专属数据读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/exclusive/data/conferences/{conferenceId} HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| conferenceId | String | 是 | 会议ID。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| conferenceId | String | 会议ID。 |
| title | String | 会议标题。 |
| confStartTime | Float | 开始时间，毫秒级时间戳。 |
| duration | Float | 持续时间，单位：毫秒。 |
| totalNum | Long | 会议人数。 |
| attendeeNum | Long | 出席会议人数。 |
| attendeePercentage | String | 出席率。 |
| callerId | String | 发起人uid。 |
| callerName | String | 发起人昵称。 |
| memberList | Array | 参会人员列表。 |
| unionId | String | 用户uid。 |
| name | String | 用户昵称。 |
| attendDuration | Float | 参会时长，单位：毫秒。 |
| staffId | String | 员工userid。 |
