更新于 2025-10-22调用本接口新增一篇文章。

**说明**

本接口在互动服务窗内暂无对应产品功能。

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 是 | 企业内部服务号素材权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingtalk.oapi.material.article.add) |
| 第三方企业应用 | 否 | — | — |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/material/article/add`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | 30c9a84136943eaxxxx | 调用该接口的应用凭证，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| unionid | String | 是 | ftO2dFpz1zAiE | 服务号的unionid，可通过[查询服务号列表](/document/development/query-service-number-list)接口获取。 |
| article | ArticleCreateDTO | 是 |  | 文章参数对象。 |
| thumb\_media\_id | String | 否 | @lALPBbCc1XuaP\_rNAljNAlg | 图文消息的封面素材id，可以通过[上传媒体文件](/document/development/upload-media-files)接口上传图片获取mediaId。 |
| content | String | 是 | <html><body><div>123</div></body></html> | 文章内容（html格式）。 |
| title | String | 是 | 标题1 | 文章标题。 |
| uuid | String | 是 | e5836bb2-2421-49fb | 幂等参数，随机生成的UUID，防止文章重复。 |
| digest | String | 否 | 摘要1 | 文章摘要。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |
| success | Boolean | true | 是否成功。 |
| article\_id | Number | 2044004 | 文章id。 |
| request\_id | String | 1056rst63clol | 请求ID。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/material/article/add?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "unionid": "IlrIsTHaiSYXluZP61h0zxgiEiE",
        "article": {
                "thumb_media_id": "@lALPBbCc1XuaP_rNAljNAlg",
                "digest": "摘要1",
                "title": "标题1",
                "uuid": "e5836bb2-2421-49fb",
                "content": "

"
        }
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/material/article/add");
OapiMaterialArticleAddRequest req = new OapiMaterialArticleAddRequest();
ArticleCreateDTO articleCreateDTO = new ArticleCreateDTO();
articleCreateDTO.setThumbMediaId("@lALPBbCc1XuaP_rNAljNAlg");
articleCreateDTO.setContent("

");
articleCreateDTO.setTitle("标题1");
articleCreateDTO.setUuid("e5836bb2-2421-49fb");
articleCreateDTO.setDigest("摘要1");
req.setArticle(articleCreateDTO);
req.setUnionid("IlrIsTHaiSYXluZP61h0zxgiEiE");
OapiMaterialArticleAddResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
    "article_id": 2044004,
    "errcode": 0,
    "success": true,
    "request_id": "1056rst63clol"
}
```
