更新于 2025-09-23调用本接口删除一篇指定的文章。

**重要**

* 文章删除后，内容链接不可访问，且不可再次发布。
* 本接口在互动服务窗内暂无对应产品功能。

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 是 | 企业内部服务号素材权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingtalk.oapi.material.article.delete) |
| 第三方企业应用 | 否 | — | — |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/material/article/delete`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | 30c9a84136943eaxxxx | 调用该接口的应用凭证，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| unionid | String | 是 | lO44Wvqzy7siE | 服务号的unionid，可通过[查询服务号列表](/document/development/query-service-number-list)接口获取。 |
| article\_id | Number | 是 | 129003 | 文章id，可以通过[查询文章列表](/document/development/query-the-article-list)接口获取。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |
| request\_id | String | 4jzld23ajt00 | 请求ID。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/material/article/delete?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "article_id":129003,
        "unionid":"lO44Wvqzy7siE"
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/material/article/delete");
OapiMaterialArticleDeleteRequest req = new OapiMaterialArticleDeleteRequest();
req.setUnionid("lO44Wvqzy7siE");
req.setArticleId(129003L);
OapiMaterialArticleDeleteResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
        "errcode":0,
        "request_id":"4jzld23ajt00"
}
```
