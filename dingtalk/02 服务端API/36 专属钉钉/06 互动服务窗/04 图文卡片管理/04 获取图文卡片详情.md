更新于 2025-09-23调用本接口获取图文卡片详情。

**说明**

本接口在互动服务窗内暂无对应产品功能。

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 是 | 企业内部服务号素材权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingtalk.oapi.material.news.get) |
| 第三方企业应用 | 否 | — | — |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/material/news/get`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | 30c9a84136943eaxxxx | 调用该接口的应用凭证，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| unionid | String | 是 | jYdrJoCmTo0iE | 服务号的unionid，可通过[查询服务号列表](/document/development/query-service-number-list)接口获取。 |
| media\_id | String | 是 | P16mHftLYX8iE | 图文卡片素材id，可以通过[查询图文卡片列表](/document/development/query-message-card-list)接口获取。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| errmsg | String | ok | 返回码描述。 |
| errcode | Number | 0 | 返回码。 |
| request\_id | String | ezsx3d8hrgzx | 请求ID。 |
| media\_id | String | mvFiiRhuwt5IiE | 卡片素材id。 |
| update\_time | Number | 1560137792000 | 图文卡片更新时间。 |
| articles | ArticleDTO[] |  | 文章列表。 |
| article\_id | Number | 114002 | 文章id。 |
| title | String | 新的标题 | 文章标题。 |
| content | String | <p>第一个段落</p> | 文章内容。 |
| thumb\_media\_id | String | @lALPBbCc1XuaP\_rNAljNAlg | 封面图片的素材id。 |
| publish\_status | Number | 1 | 发布状态：   * **0**：未发布 * **1**：已发布   **说明**  文章第一次发布后，状态为1，已发布文章支持修改，修改后此状态保持为1，每次修改文章后需要再次发布内容才会生效。 |
| publish\_time | Number | 1560137792000 | 发布时间。  **说明**  文章成功发布之后才有返回值。 |
| user\_view\_count | Number | 1 | 已读用户数。 |

| update\_time | Number | 1560137792000 | 文章更新时间。 |
| url | String | https://content.dingtalk.com/article?articleId=1234 | 文章链接。  **说明**  文章成功发布之后才有返回值。 |
| digest | String | 关于这篇文章... | 文章摘要。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/material/news/get?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "unionid":"jYdrJoCmTo0iE",
        "media_id":"P16mHftLYX8iE"
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/material/news/get");
OapiMaterialNewsGetRequest req = new OapiMaterialNewsGetRequest();
req.setUnionid("jYdrJoCmTo0iE");
req.setMediaId("P16mHftLYX8iE");
OapiMaterialNewsGetResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```
