更新于 2025-09-23调用本接口查询图文卡片列表。

**说明**

本接口在互动服务窗内暂无对应产品功能。

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 是 | 企业内部服务号素材权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingtalk.oapi.material.news.list) |
| 第三方企业应用 | 否 | — | — |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/material/news/list`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | 30c9a84136943eaxxxx | 调用该接口的应用凭证，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| unionid | String | 是 | jYdrJoCmTo0iE | 服务号的unionid，可通过[查询服务号列表](/document/development/query-service-number-list)接口获取。 |
| page\_start | Number | 否 | 1 | 页码，从1开始。 |
| page\_size | Number | 否 | 5 | 每页条数。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/material/news/list?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "unionid":"jYdrJoCmTo0iE",
        "page_start":1,
        "page_size":5
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/material/news/list");
OapiMaterialNewsListRequest req = new OapiMaterialNewsListRequest();
req.setUnionid("jYdrJoCmTo0iE");
req.setPageStart(1L);
req.setPageSize(5L);
OapiMaterialNewsListResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```
