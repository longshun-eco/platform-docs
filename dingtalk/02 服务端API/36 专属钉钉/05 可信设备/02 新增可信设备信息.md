更新于 2025-10-13调用本接口新增一个可信设备。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 专属钉钉可信设备信息读写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=exclusive_1.0%23CreateTrustedDevice) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/exclusive/trustedDevices HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "userId" : "String",
  "platform" : "String",
  "macAddress" : "String",
  "did" : "String",
  "status" : Integer
}
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，可调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| userId | String | 是 | 员工userid，为0时表示这个设备为公共设备。 |
| platform | String | 是 | 平台类型，目前仅支持Mac和Win两种类型。 |
| macAddress | String | 否 | 可信设备mac地址。 |
| did | String | 否 | 设备唯一标识。 |
| status | Integer | 否 | 设备状态，取值：   * **1**：待审批 * **2**：可信 * **3**：挂失 * **4**：拒绝 |
| title | String | 否 | 设备标题。 |
| serialNumber | String | 否 | 设备序列号。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| success | Boolean | 查询结果。  **说明**  添加是否成功由http状态码表示，状态码**200**表示添加成功。 |
