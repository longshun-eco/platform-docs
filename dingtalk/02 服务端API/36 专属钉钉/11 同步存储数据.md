更新于 2025-09-23为应用同步数据到专属存储。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 专属钉钉基础数据读写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=exclusive_1.0%23DataSync) |
| 第三方企业应用 | 支持 | 专属钉钉基础数据读写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=exclusive_1.0%23DataSync) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/exclusive/datas/sync HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "sql" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| sql | String | 是 | sql 语句。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| rowsAffected | Integer | 影响行数。 |
| dataList | Array of Object | 代表数据的 map。 |

## 示例

**请求示例**

HTTP

```sql
POST /v1.0/exclusive/datas/sync HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:7fdb7cf6b1aa340186809206a55a4a0d
Content-Type:application/json

{
  "sql" : "select id from connect_test"
}
```

**返回示例**

```
HTTP/1.1 200 OK
Content-Type:application/json

{
  "rowsAffected" : 1,
  "dataList" : [ {
    "id" : 101
  } ]
}
```

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | function.not.open | 企业未开通相应功能 | 企业未开通相应功能 |
| 400 | sql.format.error | sql格式错误 | sql格式错误 |
| 500 | system.busy | 系统繁忙 | 内部服务发生的异常情况 |
