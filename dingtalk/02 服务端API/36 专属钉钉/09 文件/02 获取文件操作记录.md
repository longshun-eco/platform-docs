更新于 2025-09-23获取专属钉钉内成员所有所在组织的操作文件或文档的记录。

### 使用说明

**说明**

获取文件操作记录接口，需同时满足以下条件才可调用：

* 调用的组织类型是专属钉钉组织
* 创建的应用类型是企业内部应用

例如，员工小钉共在两个组织，分别是测试专属钉钉组织、测试普通组织。小钉在测试普通组织知识库内删除了一篇文档，在专属钉钉组织知识库新建了一篇文档。调用本接口可以获取到小钉在所有组织的文档操作记录，即在普通组织内删除文档的操作信息，在专属钉钉组织新建文档的操作信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 专属钉钉文件审计日志读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=exclusive_1.0%23ListAuditLog) |
| 第三方企业应用 | 暂不支持 | 专属钉钉文件审计日志读权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 专属钉钉文件审计日志读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/exclusive/fileAuditLogs?startDate=Long&endDate=Long&pageSize=Integer&nextGmtCreate=Long&nextBizId=Long HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| startDate | Long | 是 | 操作日志起始时间，UNIX时间戳，单位毫秒。 |
| endDate | Long | 是 | 操作日志截止时间，UNIX时间戳，单位毫秒。 |
| pageSize | Integer | 是 | 每页最大条目数，最大值500。 |
| nextGmtCreate | Long | 否 | 操作记录生成时间，UNIX时间戳，单位毫秒，作为分页偏移量。   * 如果是首次调用，该参数不传入。 * 如果是非首次调用，该参数传上次调用时返回的最后一条记录的gmtCreate。 |
| nextBizId | Long | 否 | 操作记录文件id，作为分页偏移量。   * 如果是首次调用，该参数不传入。 * 如果是非首次调用，该参数传上次调用时返回的最后一条记录的bizId。 |
