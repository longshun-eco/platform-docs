更新于 2025-09-23调用本接口，获取人脸对比接口调用记录。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 专属钉钉人脸识别权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=exclusive_1.0%23GetRecognizeRecords) |
| 第三方企业应用 | 暂不支持 | 专属钉钉人脸识别权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 专属钉钉人脸识别权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/exclusive/faces/recognizeRecords/query HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "fromTime" : Long,
  "toTime" : Long,
  "agentId" : Long,
  "userIds" : [ "String" ],
  "nextToken" : Long,
  "maxResults" : Integer,
  "faceCompareResult" : Integer
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| fromTime | Long | 否 | 记录开始时间戳，单位毫秒。 |
| toTime | Long | 否 | 记录结束时间戳，单位毫秒。 |
| agentId | Long | 否 | 应用唯一标识。 |
| userIds | Array of String | 否 | 员工userId列表。 |
| nextToken | Long | 是 | 分页游标，从0开始。 |
| maxResults | Integer | 是 | 每页最大条目数，最大50。 |
| faceCompareResult | Integer | 否 | 人脸对比结果。   * 1：成功 * 2：失败 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| nextToken | Long | 分页游标。 |
| total | Integer | 总数据数。 |
| data | Array | 数据信息列表。 |
| agentId | Long | 应用的agentId。 |
| userId | String | 用户userId。 |
| invokeTime | Long | 接口调用时间戳，单位毫秒。 |
| faceCompareResult | Integer | 人脸对比结果。   * 1：成功    + 2：失败 |
| platform | Integer | 平台。   * 0：Android * 1：iOS |

## 错误码

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | app.not.exist | app不存在 | app不存在 |
| 400 | batch.exceed | 每批获取量超最大值 | 每批获取量超最大值 |
| 400 | time.should.close | 时间段需要闭合(fromTime和toTime需都传或都不传) | 时间段需要闭合(fromTime和toTime需都传或都不传) |
