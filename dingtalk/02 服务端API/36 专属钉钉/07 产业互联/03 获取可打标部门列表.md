更新于 2025-09-23调用本接口获取可打标部门的信息，包括部门人数、ID、伙伴编码和伙伴类型等。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 伙伴钉部门信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=exclusive_1.0%23GetAllLabelableDepts) |
| 第三方企业应用 | 支持 | 伙伴钉部门信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=exclusive_1.0%23GetAllLabelableDepts) |
| 第三方个人应用 | 暂不支持 | 伙伴钉部门信息读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/exclusive/partnerDepartments HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。   **说明**  调用本接口，需要使用上下游组织的访问凭证，不能使用所属组织的访问凭证。 |
