更新于 2025-09-23调用本接口，获取防截屏操作记录。

### 接口功能介绍

获取防截屏操作记录，需同时满足以下条件才可调用：

1. 调用的组织类型是**专属钉钉**组织，如果是非专属钉钉组织，可点击[开通专属钉钉](https://oa.dingtalk.com/register_new.htm?spm=ding_open_doc.document.0.0.10423c33KgZ0UA&source=50061&useMt2=1#/)咨询。
2. 创建的应用类型是企业内部应用。

**说明**

如果用户是普通账号且没有签署协议，则只能获取用户操作行为，不能获取图片的url。员工签署协议入口参考以下截图。

![[development-obtain-anti-screen-capture-operation-records_O1CN01mxRr0J26BO1dDIppV_!!6000000007623-2-tps-1757-656.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 专属钉钉防截屏读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=exclusive_1.0%23QueryUserBehavior) |
| 第三方企业应用 | 暂不支持 | 专属钉钉防截屏读权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 专属钉钉防截屏读权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/exclusive/enterpriseSecurities/userBehaviors/screenshots/query HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "type" : Integer,
  "platform" : Integer,
  "startTime" : Long,
  "endTime" : Long,
  "pageSize" : Integer,
  "pageNumber" : Long,
  "userId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| type | Integer | 是 | 用户行为：   * **0**：全部 * **1**：截屏 * **2**：录屏 |
| platform | Integer | 是 | 端类型：   * **0**：全部 * **1**：iOS * **2**：Android * **3**：Mac * **4**：Windows |
| startTime | Long | 否 | 开始时间，时间戳，单位毫秒。  **说明**  默认当前时间前7天。 |
| endTime | Long | 否 | 结束时间，时间戳，单位毫秒。  **说明**  默认当前时间。 |
| pageSize | Integer | 是 | 分页大小。  **说明**  最大值100。 |
| pageNumber | Long | 是 | 起始页。  **说明**  默认从1开始。 |
| userId | String | 否 | 用户userId信息，可调用[获取部门用户userid列表](/document/development/query-the-list-of-department-userids)接口获取userId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| data | Array | 数据列表。 |
| userName | String | 用户昵称。 |
| time | Long | 用户操作时间，时间戳，单位毫秒。 |
| type | Integer | 用户操作类型：   * **1**：截屏 * **2**：录屏 |
| pictureUrl | String | 用户截屏图片url。  **说明**  当用户签署响应条款后，才能获取到图片URL。 |
| platform | Integer | 端类型：   * **1**：iOS * **2**：Android * **3**：Mac * **4**：Windows |
| scene | String | 场景。 |
| userId | String | 用户userId。 |
| totalCnt | Integer | 数据总量。 |
| dataCnt | Integer | 当前页数据量。 |
