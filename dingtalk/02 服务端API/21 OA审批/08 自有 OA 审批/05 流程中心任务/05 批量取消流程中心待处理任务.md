更新于 2025-11-24调用本接口，批量取消流程中心待处理任务。

### 接口功能介绍

例如，钉钉客户端**进行中**的待处理事项如下图所示。

![[development-cancel-multiple-oa-approval-tasks_O1CN01ZK9WEU1f1m41KWK3q_!!6000000003947-2-tps-2056-1286.png]]

调用本接口，批量将正在进行中的待处理事项设置为**CANCELED**后，对应的卡片会在钉钉待办里面消失，移到**已完成**列表中。接口调用成功后，效果如下图所示。

![[development-cancel-multiple-oa-approval-tasks_O1CN01wW8KKM1b2uaIJMS5K_!!6000000003408-2-tps-2056-1286.png]]

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 工作流实例写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=workflow_1.0%23CancelIntegratedTask) |
| 第三方企业应用 | 支持 | 工作流实例写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=workflow_1.0%23CancelIntegratedTask) |
| 第三方个人应用 | 暂不支持 | 工作流实例写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/workflow/processCentres/tasks/cancel HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "processInstanceId" : "String",
  "activityId" : "String",
  "activityIds" : [ "String" ]
}
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| processInstanceId | String | 是 | OA审批流程实例ID，调用[创建实例](https://open.dingtalk.com/document/development/create-a-ticket-approval-instance)接口获取processInstanceId参数值。 |
| activityId | String | 是 | 待办组ID，调用[创建流程中心待处理任务](https://open.dingtalk.com/document/orgapp/create-pending-tasks-in-process-center)接口获取，最大长度512字符。  **说明**  需要在调用创建流程中心待处理任务接口时，主动设置该值。 |
| activityIds | Array of String | 否 | 待办组ID列表，调用[创建流程中心待处理任务](https://open.dingtalk.com/document/development/create-pending-tasks-in-process-center)接口获取。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| success | Boolean | 操作是否成功。 true表示成功。 |
