更新于 2025-09-11调用本接口，清理审批相关数据。

### 接口功能介绍

企业在某种情况下不再使用第三方企业应用，比如服务到期或主动解除授权（非停用），产品方案商可以调用本接口，删除企业的审批模板、实例、任务等数据。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 暂不支持 | 工作流数据清理专用权限 | 暂不支持 |
| 第三方企业应用 | 支持 | 工作流数据清理专用权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=workflow_1.0%23CleanProcessData) |
| 第三方个人应用 | 暂不支持 | 工作流数据清理专用权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/workflow/processes/clean HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "processCode" : "String",
  "corpId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用本接口的访问凭证，通过调用[获取第三方企业应用的suiteAccessToken](/document/development/obtains-the-suite-acess-token-of-third-party-enterprise-applications)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| processCode | String | 是 | 模板唯一码，通过[创建或更新审批模板](/document/development/create-or-update-approval-templates-new)接口获取。 |
| corpId | String | 是 | 授权企业的corpId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 清除操作是否成功。 |
