更新于 2025-12-08调用本接口，删除为企业创建的审批模板，同时删除该模板下创建的实例和待办任务。

### 接口功能介绍

调用本接口，实现删除自有OA审批模板。接口调用效果如下图所示。

![[development-self-owned-approval-deletion-template_流程中心测试.png]]

**说明**

* 企业内部应用，只删除模板，不删除通过模板创建的实例和待办任务。
* 第三方企业应用，删除模板，同时删除使用该模板创建的实例和待办任务。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 工作流模板写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=workflow_1.0%23DeleteProcess) |
| 第三方企业应用 | 支持 | 工作流模板写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=workflow_1.0%23DeleteProcess) |
| 第三方个人应用 | 暂不支持 | 工作流模板写权限 | 暂不支持 |

## 请求方法

```sql
DELETE /v1.0/workflow/processCentres/schemas?processCode=String&cleanRunningTask=Boolean HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| processCode | String | 是 | 模板code。   * 企业内部应用，通过调用[获取模板code](/document/development/obtain-the-template-code)接口获取。 |
| cleanRunningTask | Boolean | 否 | 是否清理正在运行的任务。   * **true**：是 * **false**：否 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 结果。 |
| processCode | String | 模板code。 |
