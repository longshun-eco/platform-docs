更新于 2025-09-11调用本接口，可根据员工的userId分页获取该用户可见的审批表单列表，每次最多获取100个表单。

调用本接口，根据员工的userId，分页获取该员工可见的表单列表。例如，员工可见的表单为**审批模板测试**和**isv测试模板**，如下图所示。调用本接口，可获取到**审批模板测试**和**isv测试模板**的信息。

![[development-obtains-a-list-of-approval-forms-visible-to-the-specified_p352513.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 工作流模板读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=workflow_1.0%23ListUserVisibleBpmsProcesses) |
| 第三方企业应用 | 支持 | 工作流模板读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=workflow_1.0%23ListUserVisibleBpmsProcesses) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/workflow/processes/userVisibilities/templates?userId=String&maxResults=Long&nextToken=Long HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 否 | 要查询的员工的userId。  **说明**  不传表示查询企业下所有审批表单。 |
| maxResults | Long | 是 | 分页大小，最大值100。 |
| nextToken | Long | 是 | 分页游标。   * 如果是首次调用，该参数传0。 * 如果是非首次调用，该参数传上次调用时返回的nextToken。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果。 |
| processList | Array | 可见表单列表。 |
| name | String | 表单名称。 |
| url | String | 表单URL。 |
| iconUrl | String | 图标URL。 |
| processCode | String | 表单唯一标识。 |
| dirId | String | 模板所在分组 ID。 |
| dirName | String | 模板所在分组名称。 |
| nextToken | Long | 分页游标。  **说明**  不为空表示有更多数据。 |
