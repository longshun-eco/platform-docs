更新于 2025-11-24调用本接口，创建或更新一个OA审批的流程表单模板，可指定表单控件列表并生成默认审批流程。

创建过程类似于在**钉钉管理后台** > **OA审批** > **表单管理**中编辑并发布审批表单。

![[development-create-an-approval-form-template_O1CN01fn9cMn1fMqDkV3HcG_!!6000000003993-0-tps-1235-967.jpg]]

**说明**

* 每个企业最多创建200个官方审批模板，超过最大数量后调用接口会报错。
* 官方OA审批模板仅支持文档下方所展示的审批组件，其他组件均不支持。
* 官方OA审批模板仅支持文档下方所展示的审批组件，其他组件均不支持。
* 更新审批模板时更新的组件在流程设计中设置为分支条件，则该模板表单不支持修改。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 工作流模板写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=workflow_1.0%23FormCreate) |
| 第三方企业应用 | 支持 | 工作流模板写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=workflow_1.0%23FormCreate) |
| 第三方个人应用 | 暂不支持 | 工作流模板写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/workflow/forms HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "processCode" : "String",
  "name" : "String",
  "description" : "String",
  "formComponents" : [ {
    "componentType" : "String",
    "props" : {
      "componentId" : "String",
      "label" : "String",
      "asyncCondition" : Boolean,
      "required" : Boolean,
      "content" : "String",
      "format" : "String",
      "upper" : "String",
      "unit" : "String",
      "placeholder" : "String",
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| processCode | String | 否 | 表单ProcessCode，更新表单模板时需指定ProcessCode。  **说明**  如果传递**ProcessCode**进行表单组件修改和更新，不允许删除或修改已作为流程设计中条件分支的控件数据。 例如原表单中存在**单选控件**【员工类型】作为分支条件，则更新此表单时，不允许修改【员工类型】字段的控件类型，也不允许删除。   * 未填写该参数，表示新建一个模板。 * 填写该参数，表示更新所传值对应的审批模板。   **如何获取process\_code**：在钉钉管理后台-审批模板查看。  **说明**  新旧版钉钉管理后台，获取方式不同：可参见[OA审批概述-名词解释-processCode](https://open.dingtalk.com/document/development/workflow-overview)。  **新版钉钉管理后台**：在审批模板编辑页-基础设置-页面底部查看。 ![[development-create-an-approval-form-template_O1CN01IITbZO1cr7amdr9pA_!!6000000003653-2-tps-2574-1378.png]] **旧版钉钉管理后台**：在审批模板编辑页的URL中查看。 ![[development-create-an-approval-form-template_O1CN01a584Ro1SU5WGSstfm_!!6000000002249-2-tps-790-431.png]] |
| name | String | 是 | 表单模板名称，最大长度200字符。 |
| description | String | 否 | 表单模板描述，最大长度300字符。 |
| formComponents | Array | 是 | 表单控件列表，单一表单最大组件个数不超过200。 |
| FormComponent | FormComponent | 是 | 表单控件列表，详情请参考[FormComponent参数说明](https://open.dingtalk.com/document/development/oa-formcomponent-message#900adc515fxr6)参数补充说明。 |
| templateConfig | Object | 否 | 表单全局属性配置。 |
| disableStopProcessButton | Boolean | 否 | 管理列表页是否禁用**停用**按钮。 |
| hidden | Boolean | 否 | 是否全局隐藏流程模板入口。 |
| disableDeleteProcess | Boolean | 否 | 是否禁用模板**删除**按钮。 |
| disableFormEdit | Boolean | 否 | 是否禁止表单**编辑**功能。 |
| disableResubmit | Boolean | 否 | 是否禁用详情页**再次发起**按钮。 |
| disableHomepage | Boolean | 否 | 是否在首页隐藏模板。 |
| dirId | String | 否 | 模板的目录ID，创建模板目录时指定。 |
| originDirId | String | 否 | 原目录ID，更新模板目录时，需指定源目录ID。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| result | Object | 返回的表单模板信息。 |
| processCode | String | 表单模板Code，企业内唯一 。  **说明**  生成企业内唯一的审批模板编码，可使用此**processCode**发起表单实例。 |
