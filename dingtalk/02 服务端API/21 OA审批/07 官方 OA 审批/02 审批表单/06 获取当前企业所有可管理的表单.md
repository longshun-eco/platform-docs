更新于 2025-09-11调用本接口，获取用户在当前企业所有可管理的审批表单。

### 接口调用说明

**说明**

调用本接口，userId对应的人员必须拥有该企业OA审批的权限。

调用本接口，获取在当前企业，该用户可以管理的表单。例如，用户可以管理的表单为**审批模板测试**，如下图所示。调用本接口，只能获取到**审批模板测试**表单的信息。

![[development-get-all-manageable-forms-for-the-current-enterprise_p352513.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 工作流模板读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=workflow_1.0%23GetManageProcessByStaffId) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/workflow/processes/managements/templates?userId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 用户的userId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Array | 模板列表。 |
| iconName | String | 模板图标名。 |
| flowTitle | String | 模板名称。 |
| processCode | String | 模板code。 |
| newProcess | Boolean | 是否为新模板。   * **true**：是 * **false**：不是 |
| gmtModified | String | 修改时间。 |
| attendanceType | Integer | 关联考勤类型，取值。   * **0**：无 * **1**：补卡申请 * **2**：请假 |
| iconUrl | String | 图标URL地址。 |
| success | Boolean | 接口调用是否成功。   * **true**：成功 * **false**：失败 |
