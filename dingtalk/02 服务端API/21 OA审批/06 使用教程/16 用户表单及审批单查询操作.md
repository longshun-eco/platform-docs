更新于 2025-09-10本文介绍了如何调用官方OA审批接口创建审批模板、发起审批、查询获取当前企业所有可管理的审批表单、获取指定用户可见的审批表单、用户待审批数量等流程。

## 接入流程简介

本文档展示了，创建一个企业内部应用，使用官方OA审批提供的API，实现创建审批模板、发起审批、查询获取当前企业所有可管理的审批表单、获取指定用户可见的审批表单、用户待审批数量等流程：

前提条件：完成[创建应用](/document/dingstart/create-application)的流程。

步骤一：获取应用凭证信息，获取应用 Client ID 和 Client Secret。

步骤二：申请接口权限，申请 OA 审批相关接口权限。

步骤三：获取应用访问凭证[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)。调用接口时，通过accessToken鉴权调用者身份。

步骤四：调用OA审批相关API：

1. 调用新版服务端API-[创建或更新审批表单模板](/document/development/create-an-approval-form-template)接口，获取模板的唯一编码`processCode`。
2. 创建多个审批表单模板后，可调用新版服务端API-[获取当前企业所有可管理的表单](/document/development/get-all-manageable-forms-for-the-current-enterprise)接口，获取用户在当前企业所有可管理的审批表单列表。调用本接口，userId对应的人员必须拥有该企业OA审批的权限。
3. 创建多个审批表单模板后，可调用新版服务端API-[获取指定用户可见的审批表单列表](/document/development/obtains-a-list-of-approval-forms-visible-to-the-specified)接口，获取该用户可见的审批表单列表。
4. 根据模板编码`processCode`，调用新版服务端API-[发起审批实例](/document/development/create-an-approval-instance)接口发起审批实例，获取审批实例`instanceId`。
5. 发起多个审批实例后，可根据模板编码`processCode`，调用新版服务端API-[获取审批实例ID列表](/document/development/obtain-an-approval-list-of-instance-ids)接口，获取权限范围内的相关部门审批实例ID列表。
6. 发起多个审批实例后，也可根据用户userId，调用新版服务端API-[获取用户待审批数量](/document/development/queries-the-number-of-requests-to-be-approved-by-users)接口，获取该用户待处理的审批数量。

## **前提条件**

完成[创建应用](/document/dingstart/create-application)的流程。

## **步骤一：获取应用凭证**

1. 选择目标应用，进入应用详情页，单击**基础信息** > **凭证与基础信息**。
2. 获取应用 Client ID 和 Client Secret。

## 步骤二：添加接口权限

单击**开发配置** > **权限管理**，在权限搜索框中分别输入`Workflow.Form.Write`、 `Workflow.Form.Read`、`Workflow.Instance.Write`和`Workflow.Instance.Read`，并申请权限。

## 步骤三：获取应用访问凭证accessToken

**重要**

服务端API差异详情参见[旧版服务端API、新版服务端API的区别](/document/development/how-to-call-apis#section-8lr-id4-rbz)。以下接口均使用服务端API接口，SDK下载详情参见[服务端SDK下载](/document/development/download-the-server-side-sdk)。

根据步骤一中的 Client ID 和 Client Secret，获取应用访问凭证[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)。

```java
 public void getAccessToken() throws Exception {

        GetAccessTokenResponse accessToken = client.getAccessToken(accessTokenRequest);
        System.out.println(JSON.toJSONString(accessToken.getBody()));
    }
```
