更新于 2025-09-26调用本接口，可以将三方系统内的审批节点信息同步到钉钉OA审批，并生成对应的钉钉待办任务。同时支持三方系统配置审批任务的回调接口，可实现在钉钉待办中心和OA审批消息卡片中直接进行快捷审批操作。

### 特别提醒

**说明**

当前接口为OA高级版专享接口，升级OA高级版可用，可满足更高级的开发需求，响应个性化的业务场景。[查看全部专享OpenAPI](https://open.dingtalk.com/document/development/description-of-new-oa-approval-premium-exclusive-openapi-and-solutions)

### 接口功能介绍

调用本接口，可以将三方系统内的审批节点信息同步到钉钉OA审批，并生成对应的钉钉待办任务。同时三方系统可以配置审批任务的回调接口，这样审批人可以在钉钉待办中心和OA审批消息卡片中直接进行审批操作，审批待办中心和OA审批消息卡片会回调三方系统，三方系统收到回调后更新任务信息，并将新的任务信息同步回审批中心，形成闭环。接口调用成功后，实现效果如下图所示。

* 可支持在钉钉待办中心快捷审批

  ![[development-api-premiumsaveintegratedtask_O1CN01GvjY7q1Rn2VaR8kHK_!!6000000002155-0-tps-1320-449.jpg]]
* 可支持在OA审批消息卡片快捷审批

  ![[development-api-premiumsaveintegratedtask_O1CN01tWFoYo1wI7rYEuC1m_!!6000000006284-0-tps-900-698.jpg]]

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | OA审批工作流读写权限（OA高级版专享） | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=workflow_1.0%23PremiumSaveIntegratedTask) |
| 第三方企业应用 | 支持 | OA审批工作流读写权限（OA高级版专享） | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=workflow_1.0%23PremiumSaveIntegratedTask) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/workflow/premium/processCentres/tasks HTTP/1.1
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "processInstanceId" : "String",
  "activityId" : "String",
  "tasks" : [ {
    "userId" : "String",
    "url" : "String",
    "customData" : "String",
    "dueTimestamp" : Long
  } ],
  "featureConfig" : {
    "features" : [ {
      "name" : "String",
      "pcUrl" : "String",
      "mobileUrl" : "String",
      "runType" : "String",
      "callback" : {
        "appUuid" : "String",
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：  企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。  第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| processInstanceId | String | 是 | OA审批流程实例ID，调用[创建实例](https://open.dingtalk.com/document/development/create-a-ticket-approval-instance)接口获取`processInstanceId`参数值。 |
| activityId | String | 否 | 自定义审批节点ID，自定义参数。支持以审批节点ID维度幂等。 |
| tasks | Array | 是 | 任务列表。 |
| userId | String | 否 | OA审批任务执行人的用户ID。 |
| url | String | 否 | 待办事项跳转URL。  **说明**   * 创建审批实例里的url，实现的是钉钉审批应用里的审批单跳转。 * 本接口的url，实现的是钉钉待办页面，对应的待办卡片的跳转。 * 钉钉的待办页面，同时支持移动端和PC端，所以本接口传的url参数，它所对应的页面需要适配移动端和PC端。 |
| customData | String | 否 | 用户自定义数据，页面跳转时将通过 url 查询参数回传，格式为 customData=xx。 |
| dueTimestamp | Long | 否 | 任务截止时间，Unix时间戳，单位毫秒。设置该参数后，将在钉钉待办中心展示审批任务截止时间信息，支持按截止时间排序，今日截止、已逾期等标签筛选等。 |
| featureConfig | Object | 否 | 流程中心集成配置。 |
| features | Array | 否 | 配置列表。 |
| name | String | 否 | 支持三方进行自定义配置的功能模块名称，本接口当前支持：   * **CUSTOM\_SHORTCUT**：自定义待办、卡片通知中的快捷操作按钮 |
| pcUrl | String | 否 | 三方自定义的pc端跳转链接。  **说明**   * 当前CUSTOM\_SHORTCUT场景下无需传该值。 |
| mobileUrl | String | 否 | 三方自定义的手机端跳转链接。  **说明**   * 当前CUSTOM\_SHORTCUT场景下无需传该值。 |
| runType | String | 否 | 运行方式。  当features.name为`CUSTOM_SHORTCUT`时，支持   * **REDIRECT**：外部跳转运行，打开待办详情页时，将会跳转到三方业务系统详情页地址。在待办列表执行同意/拒绝快捷审批操作时，将会回调callback中配置的回调接口。 |
| callback | Object | 否 | 网关回调配置，需支持快捷操作按钮时该参数必填。  网关回调钉钉外数据接口需要统一在“数据源管理”中注册成网关，详细的使用说明请参考[创建数据源](https://open.dingtalk.com/document/dingstart/create-a-data-source)。 |
| appUuid | String | 否 | 网关appUuid，需支持快捷操作按钮时该参数必填。  传[创建数据源](https://open.dingtalk.com/document/dingstart/create-a-data-source)时所属企业corpId值。 |
| apiKey | String | 否 | 网关apiKey，若需支持快捷操作按钮入口时该参数必填。通过[创建数据源](https://open.dingtalk.com/document/dingstart/create-a-data-source)内容获取`apiKey`。  **说明**   * 在网关回调外部接口时，钉钉侧会根据不同业务场景，回传一些业务处理所需的参数给到ISV，ISV在收到回调请求后，若需要解析获取对应参数信息，需要在[创建数据源](https://open.dingtalk.com/document/dingstart/create-a-data-source)时配置对应的参数key。   例如，CUSTOM\_SHORTCUT自定义快捷审批模块，即待办、卡片通知中的快捷操作按钮配置回调时，钉钉侧回传的固定参数如下：`{"outResult":"agree","processInstanceId":"xxx","activityId":"xxx","corpId":"dingxxx","data":[],"remark":"同意","title":"xxx提交的资产领用申请","taskId":111,"operator":"manager0001"}`。 * ISV在创建数据源时，对应的参数配置需按业务需要填对应的key进行解析：outResult,processInstanceId,activityId,corpId,remark,title,taskId,operator ![[development-api-premiumsaveintegratedtask_p232907.png]] |
| version | String | 否 | 网关接口版本  **说明**   * 若需支持快捷操作按钮入口时该参数必填。 * 默认传1。 |
| config | String | 否 | 三方进行自定义配置的功能模块对应的配置信息。  **说明**   * 当前CUSTOM\_SHORTCUT场景下无需传该值。 |
| taskConfig | Object | 否 | 流程中心任务维度属性配置。 |
| disableSendCard | Boolean | 否 | 创建流程中心待处理任务时是否禁用消息卡片通知：   * true：禁用，将不发送消息卡片通知 * false：默认值，不禁用，将发送消息卡片通知 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| result | Array | 返回结果列表。 |
| taskId | Long | OA审批任务ID。 |
| userId | String | OA审批任务执行人用户ID。 |
| success | Boolean | 是否创建成功。true表示成功。 |
