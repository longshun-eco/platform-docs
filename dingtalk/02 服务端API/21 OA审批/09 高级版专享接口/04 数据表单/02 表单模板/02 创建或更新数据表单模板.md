更新于 2025-09-11调用本接口，创建或更新数据表单模板。

### 特别提醒

**说明**

当前接口为OA高级版专享接口，升级OA高级版可用，可满足更高级的开发需求，响应个性化的业务场景，查看全部[专享OpenAPI](/document/development/description-of-new-oa-approval-premium-exclusive-openapi-and-solutions)。

### 接口调用说明

* 调用该接口创建的数据表单模板，使用的模板名称和接口返回的processCode值，请务必注意保存，方便后续调用其他接口使用。
* 数据表单模板仅支持文档下方所展示的表单组件，其他组件均不支持。

更新表单模板时需指定ProcessCode。

* 未填写该参数，表示新建一个模板。
* 填写该参数，表示更新所传值对应的审批模板。

**如何获取processCode**：在钉钉管理后台-审批模板查看。

**说明**

新旧版钉钉管理后台，获取方式不同：可参见[名词解释-processCode](/document/development/workflow-overview)。

**新版钉钉管理后台**：在审批模板编辑页-基础设置-页面底部查看。

| --- | --- | --- | --- |

## 请求方法

```
POST /v1.0/workflow/premium/dataForms/templates HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "processCode" : "String",
  "name" : "String",
  "description" : "String",
  "formComponents" : [ {
    "componentType" : "String",
    "props" : {
      "componentId" : "String",
      "label" : "String",
      "asyncCondition" : Boolean,
      "required" : Boolean,
      "content" : "String",
      "format" : "String",
      "upper" : "String",
      "unit" : "String",
      "placeholder" : "String",
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| processCode | String | 否 | 模板code，更新表单模板时需指定ProcessCode。   * 未填写该参数，表示新建一个模板。 * 填写该参数，表示更新所传值对应的审批模板。   **如何获取processCode**：在钉钉管理后台-审批模板查看。  **说明**  新旧版钉钉管理后台，获取方式不同：可参见[名词解释-processCode](/document/development/workflow-overview)。  **新版钉钉管理后台**：在审批模板编辑页-基础设置-页面底部查看。 ![[development-api-premiumsaveform_O1CN01IITbZO1cr7amdr9pA_!!6000000003653-2-tps-2574-1378.png]] **旧版钉钉管理后台**：在审批模板编辑页的URL中查看。 ![[development-api-premiumsaveform_O1CN01a584Ro1SU5WGSstfm_!!6000000002249-2-tps-790-431.png]] |
| name | String | 是 | 表单模板名称。 |
| description | String | 否 | 表单模板描述。 |
| formComponents | Array | 是 | 表单控件列表，单一表单最大组件个数不超过200。 |
| FormComponent | FormComponent | 是 | 表单控件列表。详情请参考**FormComponent参数补充说明**。 |
| userId | String | 是 | 操作人userId，需为管理员。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回的表单模板信息。 |
| processCode | String | 表单模板Code，企业内唯一 。  **说明**  生成企业内唯一的数据表单模板编码，可使用此**processCode**发起数据表单实例。 |
