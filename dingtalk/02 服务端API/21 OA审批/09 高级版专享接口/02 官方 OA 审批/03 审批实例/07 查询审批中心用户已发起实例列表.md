更新于 2025-09-11调用本接口，可查询企业内指定用户已发起的实例列表（包含官方OA审批实例和三方业务系统同步到钉钉的外部集成审批实例），帮助企业自建审批中心。

### 特别提醒

**说明**

当前接口为OA高级版专享接口，升级OA高级版可用，可满足更高级的开发需求，响应个性化的业务场景查看全部[专享OpenAPI](/document/development/description-of-new-oa-approval-premium-exclusive-openapi-and-solutions)。

### 接口功能介绍

调用本接口，可查询企业内指定用户已发起的实例列表（包含官方OA审批实例和三方业务系统同步到钉钉的外部集成审批实例），帮助企业自建审批中心。接口调用成功后，实现效果如下图所示。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | OA审批工作流读写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=workflow_1.0%23PremiumGetSubmittedInstances) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/workflow/premium/processCentres/submittedInstances?userId=String&pageSize=Integer&pageNumber=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 实例发起人的用户userId。 |
| pageSize | Integer | 是 | 分页大小，从1开始，最大值20。 |
| pageNumber | Integer | 是 | 页码，从1开始, 最大限制10。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 返回结果。 |
| list | Array | 已发起实例列表。 |
| processInstanceId | String | 实例ID。 |
| result | String | 实例处理结果。   * **agree**：同意 * **refuse**：拒绝 |
| status | String | 实例状态。   * **RUNNING**：审批中 * **TERMINATED**：已撤销 * **COMPLETED**：审批完成 |
| processCreateTime | String | 实例发起时间，iso8601格式。 |
| processEndTime | String | 实例完成时间，iso8601格式。 |
| originatorName | String | 发起人名称。 |
| originatorId | String | 发起人ID。 |
| originatorPhoto | String | 发起人头像。 |
| formMassage | String | 摘要。 |
| url | String | 实例详情页链接。 |
| title | String | 标题。 |
| processType | Integer | 审批类型。   * **0**：官方OA审批 * **1**：自有OA审批 |
| appType | Integer | 表单类型。   * **0**：流程表单 * **1**：数据表单 * **2**：办事流程 |
| hasMore | Boolean | 是否还有下一页。 |
| success | Boolean | 接口是否调用成功。 |
