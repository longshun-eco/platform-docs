更新于 2025-09-11调用本接口，使用指定的userId和任务result及评论remark，对一批具有不同审批实例ID、任务节点ID的审批任务，进行批量处理。

### 特别提醒

**说明**

当前接口为OA高级版专享接口，升级OA高级版可用，可满足更高级的开发需求，响应个性化的业务场景，查看全部[专享OpenAPI](/document/development/description-of-new-oa-approval-premium-exclusive-openapi-and-solutions)。

### 接口功能介绍

调用本接口，可以批量使用同一个评审人及评审意见，批量执行不同的审批节点。如图所示，可以使用一个评审意见批量执行该员工待处理列表下的多个任务。

![[development-api-premiumbatchexecuteprocessinstances_批量执行审批.png]]

### 接口调用说明

审批流程可以包含多个审批节点，单个审批节点可能包含一个或多个审批任务。操作单个审批任务，不代表审批流程结束。

* 审批流程只有一个审批人，对单个审批任务操作同意或拒绝，审批流程结束。

  ![[development-api-premiumbatchexecuteprocessinstances_p349526.png]]
* 审批流程有多个审批人，整个审批流程受多个任务影响。

  + 对单个审批任务操作拒绝，审批流程结束。
  + 对单个审批任务操作同意，审批流程转到下一个审批人。

![[development-api-premiumbatchexecuteprocessinstances_p349521.png]]

**说明**

批量同意或拒绝审批任务暂不支持上传附件。

## 请求方法

```
POST /v1.0/workflow/premium/processInstances/batchExecute HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "taskInfoList" : [ {
    "taskId" : Long,
    "processInstanceId" : "String"
  } ],
  "actionerUserId" : "String",
  "result" : "String",
  "remark" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| taskInfoList | Array | 是 | 任务详情列表。  最多一次可以提交10个 |
| taskId | Long | 是 | 任务ID，可调用[获取单个审批实例详情](/document/development/obtains-the-details-of-a-single-approval-instance-pop)接口获取`taskId`参数值。 |
| processInstanceId | String | 是 | 审批实例ID。   * 调用[发起审批实例](/document/development/create-an-approval-instance)接口获取`InstanceId`参数值。 * 调用[获取审批实例ID列表](/document/development/obtain-an-approval-list-of-instance-ids)接口获取`list`参数值。 |
| actionerUserId | String | 是 | 实际执行人用户ID，可通过调用[获取单个审批实例详情](/document/development/obtains-the-details-of-a-single-approval-instance-pop)接口获取。 |
| result | String | 是 | 审批操作，取值：   * **agree**：同意 * **refuse**：拒绝 |
| remark | String | 否 | 审批意见，可为空。  **说明**  最长1024个字符。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 请求是否成功。 |
| result | Map<String, Object> | 任务结果。  **说明**  数据格式为map，key是字符串形式taskId。 |
|  | Object | 单个taskId的执行结果。 |
| result | Boolean | 审批结果。 |
| message | String | 审批结果信息。  **说明**  ok表示无异常，其余会提示异常原因。 |
