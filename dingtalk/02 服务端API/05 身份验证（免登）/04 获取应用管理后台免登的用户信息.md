更新于 2025-09-10在应用管理后台免登场景中，需要本接口通过获取到的免登授权码code和获取到的应用后台免登的access\_token来换取应用管理员的身份信息。

![[development-obtains-the-identity-of-an-application-administrator_O1CN01MNP2xV1JOHvdf2xoo_!!6000000001018-2-tps-994-260.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业微应用后台免登接口的访问权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=oauth2_1.0%23GetSsoUserInfo) |
| 第三方企业应用 | 支持 | 企业微应用后台免登接口的访问权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=oauth2_1.0%23GetSsoUserInfo) |
| 第三方个人应用 | 暂不支持 | 企业微应用后台免登接口的访问权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/oauth2/ssoUserInfo?code=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，这里需要使用微应用后台免登的access\_token，可以通过调用[获取微应用后台免登的accessToken](/document/development/obtain-the-access-token-of-the-micro-application-background-without-log-on)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| code | String | 是 | 临时授权码，管理员在钉钉管理后台，跳转到应用管理页面时，该授权码会附带在URL中。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| corpId | String | 微应用免登用户所在企业的corpId。 |
| corpName | String | 微应用免登用户所在企业名称。 |
| userId | String | 用户userid。 |
| email | String | 用户邮箱。 |
| userName | String | 用户名称。 |
| avatar | String | 用户头像链接。 |
| isAdmin | Boolean | 是否为企业管理员。 |
