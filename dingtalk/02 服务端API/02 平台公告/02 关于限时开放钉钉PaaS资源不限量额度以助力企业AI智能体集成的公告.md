亲爱的钉钉开发者：

近期，越来越多的企业将钉钉机器人作为 AI Agent（智能体）、工作流及工具的交互通道。通过与钉钉机器人对话，企业已成功实现多种智能化场景，包括但不限于：

* 竞品动态监控与市场情报采集
* 跨系统工作编排与日常办公自动化
* 企业内部高效沟通
* 邮件自动化整理与回复
* 智能运维监控与自动告警

钉钉正逐渐成为企业“人机协同”的关键基础设施。

**钉钉机器人作为钉钉企业应用的核心能力，会消耗企业的 API 和 Stream 资源。**

为支持此类高集成度场景，助力企业探索 AI 落地，钉钉上的企业/个人开发者在使用OpenClaw调用API/Webhook/Stream时，额度免费“不限量”，有效期至 2026 年 3 月 31 日。

钉钉同时支持限时开放 API 及 Stream 不限量调用额度的申请通道，点击👉 [立即申请](https://alidocs.dingtalk.com/notable/share/form/v01ybEnB5jNyw2gPlP1_KPbo2Hw_2PDJC9o)填写申请表单，申请豁免的权益有效期截止至 2026年 4 月30 日。

**注意事项：**

* 提交申请后，系统将自动核验并发放额度，请耐心等待。发放成功后，企业管理员可前往「钉钉开发者后台 - 资源管理」查看相关详情。
* 钉钉机器人与智能体快速构建：[将 🦞OpenClaw 接入钉钉，创建你的 AI 助理员工](/document/dingstart/build-dingtalk-ai-employees)

我们致力于让每一个连接更稳定、每一次流转更高效，与企业携手共创更智能的工作未来。
