更新于 2025-09-11调用本接口，获取上下游组织的邀请链接。可以将此链接发送给合作的企业管理员，企业管理员可以申请加入。

### 接口功能介绍

例如，调用本接口返回上下游组织的邀请链接，企业管理员访问该链接可以申请加入该上下游组织，如图。

![[development-obtain-the-invitation-information-of-a-cooperation-space_O1CN01N3cxfU1l6FaBXuRGA_!!6000000004769-2-tps-978-782.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录上下游组织读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23GetCooperateOrgInviteInfo) |
| 第三方企业应用 | 支持 | 通讯录上下游组织读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=contact_1.0%23GetCooperateOrgInviteInfo) |
| 第三方个人应用 | 暂不支持 | 通讯录上下游组织读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/contact/cooperateCorps/{cooperateCorpId}/inviteInfos HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。   **说明**  调用本接口，访问凭证需要使用上下游组织所属企业的凭证，不能使用上下游组织的访问凭证。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| cooperateCorpId | String | 否 | 上下游组织的corpId，详情参见[CorpId](/document/development/basic-concepts-beta#section-bbk-mv0-oxd)。  ![[development-obtain-the-invitation-information-of-a-cooperation-space_O1CN01X0RGo41GFBFOwwpZD_!!6000000000592-2-tps-2198-389.png]] |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| inviteUrl | String | 邀请伙伴组织加入上下游组织的链接。 |
