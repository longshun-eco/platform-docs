## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录上下游组织写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23CreateCooperateOrg) |
| 第三方企业应用 | 支持 | 通讯录上下游组织写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=contact_1.0%23CreateCooperateOrg) |
| 第三方个人应用 | 暂不支持 | 通讯录上下游组织写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/contact/cooperateCorps HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "orgName" : "String",
  "logoMediaId" : "String",
  "industryCode" : Long
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| orgName | String | 是 | 上下游组织名称。 |
| logoMediaId | String | 否 | 上下游组织的logo，调用[上传媒体文件](/document/development/upload-media-files)接口获取图片的media\_id参数值。 |
| industryCode | Long | 否 | 行业code。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| cooperateCorpId | String | 上下游组织的CorpId。 |
