更新于 2026-01-22在发送API请求后，可以通过返回的HTTP状态码和错误信息进行问题排查。本文档适用于使用钉钉开放平台**企业内部应用**、**第三方企业应用（ISV）或第三方个人应用** 的开发者，帮助您快速定位调用服务端API时遇到的问题。

## 使用前准备

1. **确认应用类型**：请明确您的应用为**企业内部应用**、**第三方企业应用（ISV）或第三方个人应用**，不同应用类型的权限申请和限流策略有所不同。
2. **获取 access\_token**：有效期一般为7200秒，建议缓存并定期刷新。
3. **配置接口权限**：

   * 确保已在钉钉开发者后台为应用添加所需接口的调用权限。
   * 配置路径：**基础信息 > 凭证与基础信息** → **权限管理**
   * 参考文档：[添加接口调用权限](https://open.dingtalk.com/document/development/add-api-permission)
4. **最小可运行示例**

   以下是一个使用`curl`调用通用接口的示例，验证身份凭证和网络连通性：

   ```bash
   curl 'https://oapi.dingtalk.com/topapi/v2/user/get?access_token=YOUR_ACCESS_TOKEN&userid=zhangsan'
   ```

   成功响应将返回用户信息，如：

   ```json
   {
     "errcode": 0,
     "errmsg": "ok",
     "result": { ... }
   }
   ```

## 返回数据格式

DingTalk OpenAPI的返回数据包括以下信息：

* **HTTP状态码**：表示请求的通用处理结果（如200表示成功，4xx表示客户端错误等）。
* **requestId**：用于唯一标识一次API请求，当遇到异常时可提供此ID协助钉钉技术支持定位问题。

  ```
  HTTP/1.1 200 OK
  Content-type: application/json
  {
    "requestId": "21B0BD8E-744B-45D2-A3E4-C28382860B38",
    "hostId": "api.dingtalk.com"
  }
  ```

## 常见的HTTP状态码

|  |  |  |
| --- | --- | --- |
| HTTP状态码 | 信息 | 解释 |
| 200 | OK | 请求成功。 |
| 400 | Bad Request | 请求参数错误，服务器无法理解。 |
| 401 | Unauthorized | 未授权，access\_token缺失或无效。 |
| 403 | Forbidden | 请求被拒绝，通常由于权限不足或触发限流。 |
| 404 | Not Found | 请求的资源不存在。 |
| 405 | Method Not Allowed | 请求方法不被允许。 |
| 500 | Internal Server Error | 服务器内部错误。 |
| 502 | Bad Gateway | 网关错误，上游服务异常。 |
| 503 | Service Unavailable | 服务不可用，系统过载。 |
| 504 | Gateway Timeout | 网关超时。 |

## 通用错误码

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | InvalidAuthentication | 不合法的access\_token | access\_token过期或者不存在，需要重新获取一下。 |
| 403 | Forbidden.AccessDenied.AccessTokenPermissionDenied | 没有调用该接口的权限 | 需要申请下接口权限，请参考[添加接口调用权限](/document/development/add-api-permission)。 |
| 403 | Forbidden.AccessDenied.IpNotInWhiteList | 访问ip不在白名单之中 | * 企业应用：检查配置的服务器出口IP地址是否和请求IP地址一致。 * ISV应用：检查套IP白名单和请求是否一致。 |
| 403 | Forbidden.AccessDenied.QpsLimitForApi | 您的服务器调用钉钉开放平台当前接口的所有请求都被暂时禁用了 | 触发了当前开放接口的全局QPS限流。 |
| 403 | Forbidden.AccessDenied.QpsLimitForAppkeyAndApi | 您的企业应用调用钉钉开放平台当前接口的所有请求都被暂时禁用了 | 当前企业应用调用开放接口触发了QPS限流。 |
| 403 | Forbidden.AccessDenied.QpmLimitForAppkeyAndApi | 您的企业应用调用钉钉开放平台当前接口的所有请求都被暂时禁用了 | 当前企业应用调用开放接口触发了QPM限流。 |
| 403 | Forbidden.AccessDenied.QpsLimitForSuitekeyAndOrgAndApi | 您的ISV应用调用钉钉开放平台当前接口的所有请求都被暂时禁用了 | 当前ISV应用调用开放接口触发了QPS限流，仅针对一个客户企业。 |
| 403 | Forbidden.AccessDenied.QpmLimitForSuitekeyAndOrgAndApi | 您的ISV应用调用钉钉开放平台当前接口的所有请求都被暂时禁用了 | 当前ISV应用调用开放接口触发了QPM限流，仅针对一个客户企业。 |
