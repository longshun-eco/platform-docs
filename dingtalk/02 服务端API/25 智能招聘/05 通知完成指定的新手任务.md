更新于 2025-09-12产品服务商开发了智能招聘插件的应用，用户在插件中完成了新手任务，应用可调用本接口通知智能招聘产品该用户完成了指定的新手任务。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方企业应用 | 支持 | 智能招聘插件管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=ats_1.0%23FinishBeginnerTask) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/ats/beginnerTasks/{taskCode}/finish?userId=String&scope=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| taskCode | String | 是 | 任务编码。  **说明**  该参数需线下提供，请通过[技术支持](/document/services/ngliko)咨询。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 员工userId。 |
| scope | String | 否 | 任务范围。  **说明**  该参数需线下提供，请通过[技术支持](/document/services/ngliko)咨询。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 调用是否成功。   * true：成功 * false：失败 |
