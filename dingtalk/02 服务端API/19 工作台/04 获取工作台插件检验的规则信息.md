更新于 2025-09-11调用本接口获取工作台插件检验的规则信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 工作台组件信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=workbench_1.0%23GetPluginRuleCheckInfo) |
| 第三方企业应用 | 支持 | 工作台组件信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=workbench_1.0%23GetPluginRuleCheckInfo) |
| 第三方个人应用 | 支持 | 工作台组件信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=workbench_1.0%23GetPluginRuleCheckInfo) |

## 请求方法

```
GET /v1.0/workbench/plugins/validationRules?miniAppId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，请参考[调用工作台 API](/document/dingstart/workbench)。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| miniAppId | String | 否 | 插件的appId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| packCode | String | 权限包code。 |
| pluginRuleCheckDetail | String | 校验规则。 |
