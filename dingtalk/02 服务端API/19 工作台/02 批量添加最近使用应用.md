更新于 2025-09-11批量添加最近使用应用，企业可以把员工点击过的应用批量写入到工作台员工”最近使用“类型中。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 工作台组件信息写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=workbench_1.0%23AddRecentUserAppList) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/workbench/components/recentUsed/batch HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "corpId" : "String",
  "usedAppDetailList" : [ {
    "agentId" : "String"
  } ],
  "userId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| corpId | String | 是 | 组织corpId。 |
| usedAppDetailList | Array | 是 | 最近使用应用列表。 |
| agentId | String | 是 | 组织开通的应用Id，可通过调用[获取企业所有应用列表](/document/development/obtains-a-list-of-all-enterprise-applications)接口获取返回参数`agentId`字段。 |
| userId | String | 是 | 员工userId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 返回结果 |
