更新于 2026-02-27应用服务商需要登录开发者后台创建内购商品。本文档适用于希望为第三方企业应用配置周期型或消耗型内购商品的开发者，重点介绍操作流程及关键配置说明。

## 适用对象

本文档适用于**第三方企业应用**的应用服务商（ISV），用于在钉钉开放平台自运营平台中创建和管理内购商品。

**说明**

* 如果第三方企业应用上架之前，需要测试内购商品流程，可以创建**测试商品**。
* 测试商品默认支付0.01元，并且只能ISV本企业授权开通该第三方企业应用测试购买。
* 钉钉第三方企业应用授权本企业授权开通的方式：

  + H5微应用：在版本管理与发布中，授权体验组织时选择本企业即可开通。
  + 小程序应用：需要发布线上版本，审核通过后，使用线上版本二维码扫码授权开通。

## **创建周期型内购商品**

**周期型商品**：一次订购，一段时间有效，在有效期内不允许继续订购，如增值模板使用一年。目前仅支持企业管理员购买。

1. 在钉钉工作台进入**自运营平台**，依次单击**应用管理 > 应用内购**。

   ![[development-byb8fg_p492618.png]]
2. 在**应用内购配置**页面，单击**新建商品**。

   ![[development-byb8fg_p492621.png]]
3. 在**新建商品-配置商品信息**页面，依照下列要求填写商品基础信息后，单击**下一步**继续。

   * **商品名称：**输入商品名称。
   * **商品描述：**输入商品描述。
   * **发布类型：**选择发布类型。
   * **关联应用：**选择关联应用。
   * **收费模式：**选择**周期型。**

     **重要**

     创建周期型内购商品，收费模式必须选择**周期型**。
   * **规格关系：**选择规格关系。

     + **各规格独立**：各个规格间不存在升降级关系。
     + **规格有升级关系**：商品支持多规格, 且可以从低规格升级到高规格, 低规格剩余费用会抵扣高规格价格, 但不可降级。

   ![[development-byb8fg_p492140.png]]
4. 在**新建商品-设置商品规格**页面，编辑商品规格后，单击**提交审核**。

   ![[development-byb8fg_p492959.png]]

## **创建消耗型内购商品**

**消耗型**：一次订购可订购多个，允许多次订购，如企业福利体检一次。可配置支持企业普通员工购买，配置方式请参考创建内购商品。

1. 在钉钉工作台进入**自运营平台**，依次单击**应用管理 > 应用内购**。

   ![[development-byb8fg_p492618.png]]
2. 在**应用内购配置**页面，单击**新建商品**。

   ![[development-byb8fg_p492621.png]]
3. 在**新建商品-配置商品信息**页面，依照下列要求填写商品基础信息后，单击**下一步**继续。

   * **商品名称：**输入商品名称。
   * **商品描述：**输入商品描述。
   * **发布类型：**选择发布类型。
   * **关联应用：**选择关联应用。
   * **收费模式：**选择**消耗型。**

     **重要**

     创建消耗型内购商品，收费模式必须选择**消耗型**。
   * **支持个人购买：**选择是否支持个人购买。

     + **不支持个人购买**：只有企业管理员才可以在应用市场购买开通该第三方企业应用。
     + **支持个人购买**：企业管理员和非管理员都可以在应用市场购买并开通该第三方企业应用。

       **重要**

       非企业管理员购买开通该第三方企业应用时的效果如下图所示，可选择个人购买。

       ![[development-byb8fg_p492483.png]]

       **个人购买**分为以下两种方式：

       - **纯个人购买：**订单归属于个人，与个人所属的企业无关，企业管理员看不到这类订单。比如以个人名义买一张机票或者打车。
       - **企业内非管理员购买：**订单归属于企业，企业管理员可以看到这类订单，可以理解为非管理员以公司名义下单，比如团结出行或者为公司采购一批办公物品。

![[development-byb8fg_p492973.png]]

4. 在**新建商品-设置商品规格**页面，依照下列要求编辑商品规格。

   * 选择**固定规格，**编辑商品规格信息后，单击**提交审核**。

     ![[development-byb8fg_p493009.png]]
   * 选择**非固定规格，**单击**提交审核**。

     **说明**

     内购商品审核上架后，调用[获取内购商品SKU页面地址](/document/development/obtain-the-address-of-the-product-sku-details-page)接口，使用参数extend\_param指定商品价格，详情请参考[非固定规格实现流程](/document/development/application-purchase-overview#8406275059y4m)。

     ![[development-byb8fg_p493013.png]]
