更新于 2026-01-06调用本接口，获取工作通知消息的发送进度。

本接口适用于企业需监控消息推送状态、排查消息发送异常等场景。

**说明**

调用本接口，只能获取7天内工作通知消息的发送进度。

### **接口调用步骤**

步骤一：调用[发送工作通知](/document/development/asynchronous-sending-of-enterprise-session-messages)接口获取工作通知任务`task_id`。

步骤二：调用本接口获取工作通知消息的发送进度。

| 第三方个人应用 | 暂不支持 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/message/corpconversation/getsendprogress`

| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | 6d1bxxxx | 调用该API的应用凭证。   * 企业内部应用，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 * 第三方企业应用，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| agent\_id | Number | 是 | 836390886 | 发送消息时使用的微应用的AgentID。   * 企业内部应用，可在[开发者后台](https://open-dev.dingtalk.com/#/appMgr/inner/h5/836390886/1)的应用详情页面查看。 ![[development-obtain-the-sending-progress-of-asynchronous-sending-of-enterprise-session_p520676.png]] * 第三方企业应用可调用[获取企业授权信息](/document/development/obtains-the-basic-information-of-an-enterprise)接口获取。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/message/corpconversation/getsendprogress?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "agent_id":123,
        "task_id":456
}
```

**请求示例（JAVA SDK）**

```java
import com.dingtalk.api.DefaultDingTalkClient;
import com.dingtalk.api.DingTalkClient;
import com.dingtalk.api.request.OapiMessageCorpconversationGetsendprogressRequest;
import com.dingtalk.api.response.OapiMessageCorpconversationGetsendprogressResponse;

public class GetSendProgress {
    public static void main(String[] args) throws Exception {

        String accessToken = "你的access_token";
        Long agentId = 123L; // 请替换为你的agentId
        Long taskId = 456L; // 请替换为你的taskId

        DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/message/corpconversation/getsendprogress");
        OapiMessageCorpconversationGetsendprogressRequest req = new OapiMessageCorpconversationGetsendprogressRequest();
        req.setAgentId(agentId);
        req.setTaskId(taskId);
        OapiMessageCorpconversationGetsendprogressResponse rsp = client.execute(req, accessToken);
        System.out.println(rsp.getBody());
    }
}
```

**返回示例**

```json
{
    "errcode": 0,
    "errmsg":"ok",
    "progress": {
        "progress_in_percent": 100,
        "status": 2
    },
    "request_id": "5y2znla8tw3d"
}
```

|  |  |  |
| --- | --- | --- |
| 错误码（errorcode） | 错误信息描述（errmsg） | 解决方案 |
| 40035 | 不合法的参数 | 确认task\_id或者agent\_id是否正确 |
| 500 | 系统异常 | 出现未知的系统异常 |
