更新于 2025-09-12调用本接口查询人与人会话中机器人消息已读列表。

### 接口功能介绍

调用本接口，根据群ID、机器人编码和消息ID，查询人与人会话中机器人消息的已读列表信息，适用于企业人员单聊沟通中需要确认机器人消息是否被阅读的场景。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业内机器人发送消息权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=robot_1.0%23PrivateChatQuery) |
| 第三方企业应用 | 支持 | 企业内机器人发送消息权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=robot_1.0%23PrivateChatQuery) |
| 第三方个人应用 | 暂不支持 | 企业内机器人发送消息权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/robot/privateChatMessages/query HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "openConversationId" : "String",
  "robotCode" : "String",
  "token" : "String",
  "processQueryKey" : "String",
  "maxResults" : Long,
  "nextToken" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| openConversationId | String | 否 | 人与人单聊开放会话ID，可通过[批量安装酷应用到单聊会话](/document/development/batch-chat-session)并监听[会话管理](/document/development/org-event-overview#260a4e7084gs2)获取OpenConversationId参数值。 |
| robotCode | String | 否 | 机器人的编码，详情参考[机器人 ID](/document/development/robot-overview)。 |
| processQueryKey | String | 是 | 消息id，可通过[创建并投放卡片](/document/development/create-and-deliver-cards)接口或[人与人会话中机器人发送普通消息](/document/development/the-robot-sends-ordinary-messages-in-a-person-to-person-conversation)接口，获取`processQueryKey`参数值。 |
| maxResults | Long | 否 | 分页查询每页的数量。 |
| nextToken | String | 否 | 一次查询后返回的加密的分页凭证，首次查询不填。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| sendStatus | String | 消息发送状态。   * SUCCESS：成功 * RECALLED：已撤回 * PROCESSING：处理中 |
| readUserIds | Array of String | 消息已读人的userId列表。 |
| nextToken | String | 下次分页查询的加密凭证。 |
| hasMore | Boolean | 分页查询是否还有更多信息。 |
