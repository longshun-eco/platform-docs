更新于 2025-10-21本文档介绍了通过应用机器人发送群聊消息。

**说明**

本接口支持企业内部应用机器人，详情参考[创建企业内部应用机器人](/document/dingstart/configure-the-robot-application)。

### 接口功能介绍

调用本接口，可以使用机器人发送群聊消息，适用于企业需向群聊会话定时推送系统通知、发送公告消息等场景具体示例如下图所示：

> 接口暂不支持 @ 功能。

![[development-the-robot-sends-a-group-message_群聊消息.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业内机器人发送消息权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=robot_1.0%23OrgGroupSend) |
| 第三方企业应用 | 支持 | 企业内机器人发送消息权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=robot_1.0%23OrgGroupSend) |
| 第三方个人应用 | 暂不支持 | 企业内机器人发送消息权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/robot/groupMessages/send HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "msgParam" : "String",
  "msgKey" : "String",
  "openConversationId" : "String",
  "robotCode" : "String",
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| msgParam | String | 是 | 消息模板参数，详情参考[企业机器人发送消息的消息类型](/document/orgapp/types-of-messages-sent-by-robots)。长度限制 15000 字节以内。 |
| msgKey | String | 是 | 消息模板key，详情参考[企业机器人发送消息的消息类型](/document/dingstart/types-of-messages-sent-by-robots)。 |
| openConversationId | String | 是 | 会话ID：   * 如果是企业内部群    + 新创建企业内部群，企业内部应用，可调用[创建企业内部群](/document/development/create-group-session)接口获取。已存在的企业内部群，可调用[chooseChat](/document/development/jsapi-choose-chat)获取。 * 如果是场景群    + 企业内部应用，可调用[创建场景群](/document/development/create-a-scene-group-v2)接口获取。已存在的场景群，可调用 [chooseChat](/document/orgapp/jsapi-choose-chat) 选择会话 JSAPI 获取。 |
| robotCode | String | 是 | 机器人的编码，详情参考[机器人 ID](/document/development/development-robot-overview)。 |
| coolAppCode | String | 否 | 群聊酷应用编码，详情参考[群聊酷应用](/document/dingstart/configuration-group-chat-quick-entry)。 ![[development-the-robot-sends-a-group-message_群聊酷应用编码.jpg]]  **说明**  当使用群聊酷应用的方式安装机器人时，**必须**传入此参数。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| processQueryKey | String | 加密消息id，根据此id可查询消息已读状态和撤回消息，调用[查询人与人会话中机器人消息已读列表](/document/development/query-the-read-list-of-robot-messages-in-person-to-person-conversations)和[企业机器人撤回内部群消息](/document/development/enterprise-chatbot-withdraws-internal-group-messages)接口。 |
