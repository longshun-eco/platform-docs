更新于 2025-09-12调用本接口获取群内机器人列表。

### 接口功能介绍

调用本接口，根据群会话ID查询群内机器人列表信息。

![[development-obtain-the-list-of-robots-in-the-group_群内机器人列表.png]]

**说明**

目前只支持获取企业内部机器人、第三方企业应用机器人，场景群机器人和公共模板机器人四种类型的机器人，其他类型的机器人均无法查询，如小钉。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉群基础信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=robot_1.0%23GetBotListInGroup) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/robot/getBotListInGroup HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "openConversationId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| openConversationId | String | 是 | 群ID：基于群模板创建的群，调用[创建群](/document/development/create-a-scene-group-v2)接口获取`open_conversation_id`参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| chatbotInstanceVOList | Array | 返回的机器人列表。 |
| robotCode | String | 机器人的编码，参见[机器人名词表-robotCode](/document/development/robot-overview)内容，获取`robotCode`。 |
| name | String | 机器人的名称。 |
| downloadIconURL | String | 机器人头像的下载链接。 |
| openRobotType | Integer | 机器人的类型：   * **1**：公共模板机器人，如自定义机器人，心知天气等等。 * **2**：企业内部应用机器人。 * **3**：第三方企业应用机器人。 * **4**：场景群机器人。 |
