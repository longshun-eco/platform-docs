### 接口功能介绍

调用本接口清空单聊机器人的快捷入口设置，适用于企业需要批量清理历史机器人入口配置的场景，具体展示如下图：

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 单聊机器人使用管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=robot_1.0%23ClearRobotPlugin) |
| 第三方企业应用 | 暂不支持 | 单聊机器人使用管理权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 单聊机器人使用管理权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/robot/plugins/clear HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "robotCode" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| robotCode | String | 是 | 机器人的编码，参见[机器人名词表-robotCode](/document/development/robot-overview)内容，获取`robotCode`。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 是否成功清空机器人快捷入口。 |
