更新于 2025-09-12调用本接口查询单聊机器人的快捷入口。

### 接口功能介绍

调用本接口查询单聊机器人的快捷入口，包含快捷入口名称、快捷入口图标、PC端快捷入口跳转链接和移动端快捷入口链接，适用于企业管理员在配置机器人功能时快速获取快捷入口配置信息。

![[development-quick-entrance-of-inquiry-single-chat-robot_单聊机器人快捷入口.PNG]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 单聊机器人使用管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=robot_1.0%23QueryRobotPlugin) |
| 第三方企业应用 | 暂不支持 | 单聊机器人使用管理权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 单聊机器人使用管理权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/robot/plugins/query HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "robotCode" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| robotCode | String | 是 | 机器人的编码，参见[机器人名词表-robotCode](/document/development/robot-overview)内容，获取`robotCode`。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| pluginInfoList | Array | 快捷入口列表。 |
| name | String | 快捷入口的名称，支持国际化设置，如`{"en_US":"test123","zh_CN":"测试123"}`。 |
| icon | String | 快捷入口的图标id。 |
| pcUrl | String | pc端会话快捷入口跳转链接。 |
| mobileUrl | String | 手机端快捷入口跳转链接。 |
