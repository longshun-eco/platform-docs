## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉群基础信息管理权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=im_1.0%23ChatSubAdminUpdate) |
| 第三方企业应用 | 否 | — | — |
| 第三方个人应用 | 否 | — | — |

## 请求方法

```
POST /v1.0/im/subAdministrators HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "openConversationId" : "String",
  "userIds" : [ "String" ],
  "role" : Integer
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的应用凭证，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| openConversationId | String | 是 | 开放群ID。可以调用[创建群](/document/development/create-group-session)接口获取openConversationId参数值。 |
| userIds | Array of String | 是 | 企业员工userid列表。可以调用[获取部门用户userid列表](/document/development/query-the-list-of-department-userids)接口获取userid\_list参数值。 |
| role | Integer | 是 | 设置类型，取值：   * **2**：添加为管理员 * **3**：删除该管理员 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | String | 调用是否成功。true表示调用成功。 |
