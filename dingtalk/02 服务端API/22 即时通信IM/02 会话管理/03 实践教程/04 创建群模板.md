更新于 2025-09-25如果你需要创建场景群，需要先进行创建群模板，你可以参考本文档操作步骤完成群模板创建。

## **前提条件**

完成[成为钉钉开发者](/document/dingstart/dingtalk-developer)的流程。

## **操作步骤**

1. 登录[开发者后台](https://open-dev.dingtalk.com/#/)，单击顶部导航栏**开放能力** > **场景群。**选择侧边导航栏**群模板，**并单击**创建群模板**。
2. 进入群模板创建页面：

   1. 配置 ① 选择模板：

      |  |  | | --- | --- | | **配置项** | **说明** | | 企业类型 | 企业类型分为：  * 企业三方（第三方企业应用） * 企业内部（企业内部应用） | | 可选应用 | 选择已经创建完成的应用。 |

      配置完成后，单击**下一步**。
   2. 配置 ② 填写模板信息：

      |  |  |  | | --- | --- | --- | | **配置项** | **必填** | **说明** | | 模板名称 | 是 | 填写模板名称信息。 | | 图标 | 是 | 上传模板图标图片，要求：尺寸大小 32\*32 px，图片大小不超过 3 M，支持jpeg、jpg、png格式。 | | 描述 | 是 | 填写模板说明信息。 | | 模板图文介绍 | 是 | * 文案介绍：填写模板介绍标题 * 模板描述：填写模板详细介绍 * 图片介绍：上传图片介绍，要求：尺寸大小 343\*195 px，图片大小不超过 3 M，支持jpeg、jpg、png格式 | | 模板可见范围 | 否 | 选择模板可见的范围。 | | 群设置 | 否 | 选择该模板创建群后的基础设置。 | | 选择模板 | 否 | 默认为空白模板。 | | 选择插件 | 否 | 选择已创建的群插件/新建插件。  群插件内容，请参考[创建群插件](/document/development/create-a-group-plug-in-1)。 | | 选择机器人 | 否 | 选择前提条件中已审核通过的机器人。  群模板机器人内容，请参考[创建群模板机器人](/document/dingstart/creation-and-installation-of-swarm-template-robots)。 |

      配置完成后，单击**保存编辑**。
3. 单击左侧导航栏群模板，选择上述已保存的群模板。并在模板信息页面，单击提交审核。
4. 审核通过后，单击发布。
