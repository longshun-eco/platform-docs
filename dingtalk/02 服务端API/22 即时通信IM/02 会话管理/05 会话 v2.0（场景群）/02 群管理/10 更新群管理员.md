更新于 2025-09-12调用本接口更新群的群管理员。

本接口适用于群创建者或已有管理员需要调整群管理员设置，如添加新管理员、移除现有管理员等场景，具体示例如下图所示：

![[development-update-group-administrators_群管理员.png]]

**说明**

支持以下场景使用：基于群模板创建的群，详情参见[创建群](/document/development/create-a-scene-group-v2)。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉群基础信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=im_1.0%23UpdateGroupSubAdmin) |
| 第三方企业应用 | 支持 | 钉钉群基础信息管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=im_1.0%23UpdateGroupSubAdmin) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
PUT /v1.0/im/sceneGroups/subAdmins HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "openConversationId" : "String",
  "userIds" : [ "String" ],
  "role" : Long
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| openConversationId | String | 是 | 群ID，可调用[创建群](/document/development/create-a-scene-group-v2)接口获取`open_conversation_id`参数值。 |
| userIds | Array of String | 是 | 用户userid列表。  **说明**  最多传12个。 |
| role | Long | 是 | 群成员类型：   * **2**：群管理员 * **3**：普通群成员 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 更新是否成功，true表示执行成功，否则表示失败。 |
