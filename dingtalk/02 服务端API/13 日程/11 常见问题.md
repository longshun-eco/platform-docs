## **全天日程的返回示例**

例如，2023年1月1日的全天日程，返回的时间示例如下（结束时间为 T+1 ）：

```json
{
  "start" : {
    "date" : "2023-01-01"
  },
  "end" : {
    "date" : "2023-01-02"
  }
}
```
