更新于 2025-09-11调用本接口，删除指定日程。

**说明**

* 当组织者删除日程时（url中使用组织者uninoId），所有参与者会收到日程取消通知，同时日程将从所有参与者的日历中删除。
* 当参与者删除日程时（url中使用参与者uninoId），仅将日程从自己的日历中删除，其他日程参与者不受影响。

调用本接口，可删除指定日程。接口调用成功后，与下图操作实现效果一致。

![[development-delete-event_O1CN012VpDn11kYoinvpxsW_!!6000000004696-2-tps-1808-1390.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 日历应用中日程写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=calendar_1.0%23DeleteEvent) |
| 第三方企业应用 | 支持 | 日历应用中日程写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=calendar_1.0%23DeleteEvent) |
| 第三方个人应用 | 支持 | 日历应用中日程写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=calendar_1.0%23DeleteEvent) |

## 请求方法

```sql
DELETE /v1.0/calendar/users/{userId}/calendars/{calendarId}/events/{eventId}?pushNotification=Boolean HTTP/1.1
Host:api.dingtalk.com
x-client-token:String
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-client-token | String | 否 | 幂等校验。  **说明**   * 相同的`x-client-token`表示同一次请求。 * 过期失效，1天。 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 * 第三方个人应用，调用[获取用户token](/document/development/obtain-user-token)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 期望删除的日程所属的用户uninoId。   * 企业内部应用和第三方企业应用，调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 * 第三方个人应用，调用[获取用户通讯录个人信息](/document/development/dingtalk-retrieve-user-information)接口获取unionId参数值。 |
| calendarId | String | 是 | 日程所属的日历ID，统一为primary，表示用户的主日历。 |
| eventId | String | 是 | 日程ID，可调用[创建日程](/document/development/create-event)接口或[查询日程列表](/document/development/query-an-event-list)接口获取id参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| pushNotification | Boolean | 否 | 是否弹窗提醒：   * true：是 * false：否 |
