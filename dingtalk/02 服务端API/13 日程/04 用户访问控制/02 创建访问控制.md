**说明**

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 日历应用访问控制写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=calendar_1.0%23CreateAcls) |
| 第三方企业应用 | 暂不支持 | 日历应用访问控制写权限 | 暂不支持 |
| 第三方个人应用 | 支持 | 日历应用访问控制写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=calendar_1.0%23CreateAcls) |

## 请求方法

```
POST /v1.0/calendar/users/{userId}/calendars/{calendarId}/acls HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "privilege" : "String",
  "sendMsg" : Boolean,
  "scope" : {
    "scopeType" : "String",
    "userId" : "String"
  }
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方个人应用，调用[获取用户token](/document/development/obtain-user-token)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 日程组织者的unionId。   * 企业内部应用，调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 * 第三方个人应用，通过[获取用户通讯录个人信息](/document/development/dingtalk-retrieve-user-information)接口获取unionId参数值。 |
| calendarId | String | 是 | 日程所属的日历ID，统一为primary，表示用户的主日历。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| privilege | String | 是 | 权限信息，取值：   * **free\_busy\_reader**：查看忙闲 * **title\_reader**：查看标题 * **reader**：查看详情 * **writer**：创建和编辑 |
| sendMsg | Boolean | 是 | 是否向授权人发消息。   * **true**：发 * **false**：不发 |
| scope | Object | 是 | 权限范围。 |
| scopeType | String | 是 | 权限类型，目前只支持**user**表示用户。 |
| userId | String | 是 | 用户ID。  当**scopeType**取值为**user**时，传入用户的**unionId**。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| privilege | String | 权限信息，取值：   * **free\_busy\_reader**：查看忙闲 * **title\_reader**：查看标题 * **reader**：查看详情 * **writer**：创建和编辑 |
| aclId | String | 权限资源ID。 |
| scope | Object | 权限范围。 |
| scopeType | String | 权限类型，目前只支持**user**表示用户。 |
| userId | String | 用户ID。  当**scopeType**取值为**user**时，值为用户的**unionId**。 |
