## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 日历应用中日程写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=calendar_1.0%23RespondEvent) |
| 第三方企业应用 | 支持 | 日历应用中日程写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=calendar_1.0%23RespondEvent) |
| 第三方个人应用 | 支持 | 日历应用中日程写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=calendar_1.0%23RespondEvent) |

## 请求方法

```
POST /v1.0/calendar/users/{userId}/calendars/{calendarId}/events/{eventId}/respond HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "responseStatus" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 * 第三方个人应用，调用[获取用户token](/document/development/obtain-user-token)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 日程参与者unionId。   * 企业内部应用和第三方企业应用，调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 * 第三方个人应用，调用[获取用户通讯录个人信息](/document/development/dingtalk-retrieve-user-information)接口获取unionId参数值。 |
| calendarId | String | 是 | 日程所属的日历id，统一为primary，表示用户的主日历。 |
| eventId | String | 是 | 日程ID，可调用[创建日程](/document/development/create-event)接口或[查询日程列表](/document/development/query-an-event-list)接口获取id参数值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| responseStatus | String | 是 | 响应状态。   * **needsAction**（默认）：未操作 * **accepted**：已接受 * **declined**：已拒绝 * **tentative**：暂定接受 |
