## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 日历应用中日程写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=calendar_1.0%23RemoveMeetingRooms) |
| 第三方企业应用 | 支持 | 日历应用中日程写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=calendar_1.0%23RemoveMeetingRooms) |
| 第三方个人应用 | 暂不支持 | 日历应用中日程写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/calendar/users/{userId}/calendars/{calendarId}/events/{eventId}/meetingRooms/batchRemove HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "meetingRoomsToRemove" : [ {
    "roomId" : "String"
  } ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 日程组织者的unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |
| calendarId | String | 是 | 日程所属的日历ID，统一为**primary**，表示用户的主日历。 |
| eventId | String | 是 | 日程ID，可调用[查询日程列表](/document/development/query-an-event-list)接口获取id参数值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| meetingRoomsToRemove | Array | 否 | 需要取消预定的会议室信息列表。 |
| roomId | String | 否 | 需要取消预定的会议室roomId，可调用[查询单个日程详情](/document/development/query-details-about-an-event)接口获取的roomId参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 请求是否成功，true表示成功。 |
