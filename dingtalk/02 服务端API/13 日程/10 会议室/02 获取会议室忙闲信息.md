更新于 2025-09-11调用本接口，获取指定会议室忙闲信息。

### 接口调用步骤

步骤1：调用本接口，查询企业下会议室在某段时间内的忙闲信息。

步骤2：如果会议室没有被占用，可以调用[预定会议室](/document/development/add-a-meeting-room)接口将该会议室添加到指定的日程中。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 日历应用中日程读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=calendar_1.0%23GetMeetingRoomsSchedule) |
| 第三方企业应用 | 支持 | 日历应用中日程读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=calendar_1.0%23GetMeetingRoomsSchedule) |
| 第三方个人应用 | 暂不支持 | 日历应用中日程读权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/calendar/users/{userId}/meetingRooms/schedules/query HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "roomIds" : [ "String" ],
  "startTime" : "String",
  "endTime" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 本次请求的资源所归属的用户unionId，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| roomIds | Array of String | 是 | 待查询的会议室roomId列表，可调用[查询会议室列表](/document/development/check-the-meeting-room-list)接口获取，建议不超过5个。 |
| startTime | String | 是 | 查询开始时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| endTime | String | 是 | 查询结束时间，iso8601格式，例如：2022-07-29T14:55Z。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| scheduleInformation | Array | 会议室的忙闲信息列表。 |
| roomId | String | 会议室roomId。 |
| error | String | 异常信息。 |
| scheduleItems | Array | 忙闲信息列表。  **说明**  如果该字段返回为空，表示该会议室在指定时间段内是闲置状态。 |
| status | String | 忙闲状态。   * **BUSY**：繁忙 * **TENTATIVE**：审批中 |
| eventId | String | 日程id。 |
| organizer | Object | 日程组织者信息。 |
| id | String | 组织者unionId。 |
| start | Object | 日程开始时间相关信息。 |
| dateTime | String | 日程开始时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| timeZone | String | 时区名称，使用**IANA Time Zone Database**标准，例如：`Asia/Shanghai`。  **说明**  全天日程时区固定为UTC，非全天日程时区默认为`Asia/Shanghai`。 |
| end | Object | 日程结束时间相关信息。 |
| dateTime | String | 日程结束时间，iso8601格式，例如：2022-07-29T14:55Z。 |
| timeZone | String | 时区名称，使用**IANA Time Zone Database**标准，例如：`Asia/Shanghai`。  **说明**  全天日程时区固定为UTC，非全天日程时区默认为`Asia/Shanghai`。 |
