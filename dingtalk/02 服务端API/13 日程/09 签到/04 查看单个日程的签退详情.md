更新于 2025-09-11调用本接口，根据日程ID查询单个日程签退与未签退人员列表。

调用本接口，可获取单个日程的签退人员列表和未签退人员列表，列表信息包含人员unionid、用户名等信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 日历应用中日程读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=calendar_1.0%23GetSignOutList) |
| 第三方企业应用 | 支持 | 日历应用中日程读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=calendar_1.0%23GetSignOutList) |
| 第三方个人应用 | 支持 | 日历应用中日程读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=calendar_1.0%23GetSignOutList) |

## 请求方法

```
GET /v1.0/calendar/users/{userId}/calendars/{calendarId}/events/{eventId}/signOut?maxResults=Integer&nextToken=String&type=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 * 第三方个人应用，调用[获取用户token](/document/development/obtain-user-token)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 本次请求的资源所归属的用户unionId，该用户需要为请求日程的组织者和日程的参与者。   * 企业内部应用和第三方企业应用，调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 * 第三方个人应用，调用[获取用户通讯录个人信息](/document/development/dingtalk-retrieve-user-information)接口获取unionId参数值。 |
| calendarId | String | 是 | 日程所属的日历ID，参数填写为primary，表示用户的主日历。 |
| eventId | String | 是 | 日程ID，可调用[查询日程列表](/document/development/query-an-event-list)接口获取id参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| maxResults | Integer | 是 | 查询返回结果数，最大值500。 |
| nextToken | String | 否 | 分页游标。  **说明**  如果一个查询条件一次无法全部返回结果，会返回分页token，下次查询带上该token后会返回后续数据，直到分页token为null表示数据已经全部查询完毕。 |
| type | String | 是 | 签退信息类型。   * **sign\_out**：已签退 * **not\_yet\_sign\_out**：未签退 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| nextToken | String | 分页游标。  **说明**  如果一个查询条件一次无法全部返回结果，会返回分页token，下次查询带上该token后会返回后续数据，直到分页token为null表示数据已经全部查询完毕。 |
| users | Array | 签退信息。 |
| userId | String | 日程所属用户的unionId。 |
| displayName | String | 用户名。 |
| checkOutTime | Long | 签退时间，单位毫秒。 |
