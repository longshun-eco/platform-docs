更新于 2025-09-11调用本接口，查询订阅日历的详情信息。

例如，订阅日历的详情信息页面如下图所示。

调用本接口，可获取该订阅日历的日历标题为**测试订阅1**，描述为**测试描述**，共同编辑人**小明的unionId**，可订阅人群中**测试群的OpenConversationId**。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 日历应用中日历读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=calendar_1.0%23GetSubscribedCalendar) |
| 第三方企业应用 | 支持 | 日历应用中日历读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=calendar_1.0%23GetSubscribedCalendar) |
| 第三方个人应用 | 支持 | 日历应用中日历读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=calendar_1.0%23GetSubscribedCalendar) |

## 请求方法

```
GET /v1.0/calendar/users/{userId}/subscribedCalendars/{calendarId} HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 * 第三方个人应用，调用[获取用户token](/document/development/obtain-user-token)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 日历作者或者共同编辑人的unionId。   * 企业内部应用和第三方企业应用，调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 * 第三方个人应用，调用[获取用户通讯录个人信息](/document/development/dingtalk-retrieve-user-information)接口获取unionId参数值。 |
| calendarId | String | 是 | 订阅日历id，可调用[创建订阅日历](/document/development/create-subscription-calendar)接口获取calendarId参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| calendarId | String | 订阅日历id。 |
| name | String | 日历标题。 |
| description | String | 日历描述。 |
| author | String | 日历创建者。 |
| managers | Array of String | 可编辑人员的unionId。 |
| subscribeScope | Object | 可订阅范围。 |
| unionIds | Array of String | 可订阅人员的unionId列表。 |
| openConversationIds | Array of String | 可订阅群组的OpenConversationId列表。 |
| corpIds | Array of String | 可订阅组织的corpId列表。 |
