更新于 2025-09-11调用本接口，查询用户日历。

例如，某个员工的日历功能如下图所示。调用本接口，可以获取到我的日历（钉钉）和一个淘票票电影日历，对应的类型分别为primary和subscribed。

![[development-query-a-calendar_O1CN01LMkcdT25LAnYV2ipK_!!6000000007509-2-tps-1332-918.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 日历应用中日历读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=calendar_1.0%23ListCalendars) |
| 第三方企业应用 | 支持 | 日历应用中日历读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=calendar_1.0%23ListCalendars) |
| 第三方个人应用 | 支持 | 日历应用中日历读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=calendar_1.0%23ListCalendars) |

## 请求方法

```
GET /v1.0/calendar/users/{userId}/calendars HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 * 第三方个人应用，调用[获取用户token](/document/development/obtain-user-token)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 查询目标用户的unionId。   * 企业内部应用和第三方企业应用，调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 * 第三方个人应用，调用[获取用户通讯录个人信息](/document/development/dingtalk-retrieve-user-information)接口获取unionId参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| response | Object | 日历信息。 |
| calendars | Array | 日历信息列表。 |
| id | String | 日历ID。 |
| summary | String | 日历标题。 |
| description | String | 日历描述。 |
| timeZone | String | 时区名称，使用**IANA Time Zone Database**标准，例如：`Asia/Shanghai`。  **说明**  全天日程时区固定为UTC，非全天日程时区默认为`Asia/Shanghai`。 |
| eTag | String | 日程资源的eTag，用于检测该日程以及内部的Event是否有被更新。 |
| type | String | 日历类型。   * **unknown**：未知类型 * **primary**：用户或应用的主日历 * **subscribed**：由用户或应用创建的订阅日历（无参与人） * **shared**：由用户或应用创建的普通共享日历 |
| privilege | String | 权限信息。   * **free\_busy\_reader**：查看闲忙 * **title\_reader**：查看标题 * **reader**：查看详情 * **writer**：可编辑 * **owner**：可管理和可设置共享关系 |
