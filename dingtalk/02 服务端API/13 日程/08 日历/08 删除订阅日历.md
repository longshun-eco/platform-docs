更新于 2025-09-11调用本接口，删除订阅日历。

例如，订阅日历创建者查看**我管理的订阅日历**列表，如下图所示，有**测试订阅**和**订阅日历**。

![[development-delete-subscription-calendar_O1CN0133TRLs1ZdtItAmqUm_!!6000000003218-2-tps-2168-1414.png]]

调用本接口可删除**测试订阅**日历。接口调用成功后，实现效果如下图所示。

![[development-delete-subscription-calendar_O1CN01wEFlJh1N9PYYwr3tx_!!6000000001527-2-tps-2150-1410.png]]

**说明**

* 目前仅允许日历创建者删除日历，因此Path参数中的userId字段必须是日历创建者的unionId。
* 订阅日历被删除后，已订阅该日历的用户会自动解除订阅。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 日历应用中日历写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=calendar_1.0%23DeleteSubscribedCalendar) |
| 第三方企业应用 | 支持 | 日历应用中日历写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=calendar_1.0%23DeleteSubscribedCalendar) |
| 第三方个人应用 | 支持 | 日历应用中日历写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=calendar_1.0%23DeleteSubscribedCalendar) |

## 请求方法

```sql
DELETE /v1.0/calendar/users/{userId}/subscribedCalendars/{calendarId} HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 * 第三方个人应用，调用[获取用户token](/document/development/obtain-user-token)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 日历创建者的unionId。   * 企业内部应用和第三方企业应用，调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 * 第三方个人应用，调用[获取用户通讯录个人信息](/document/development/dingtalk-retrieve-user-information)接口获取unionId参数值。 |
| calendarId | String | 是 | 订阅日历id，可调用[创建订阅日历](/document/development/create-subscription-calendar)接口获取calendarId参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 接口是否调用成功。true表示成功。 |
