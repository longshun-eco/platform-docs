## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 日历应用中日程闲忙状态读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=calendar_1.0%23GetSchedule) |
| 第三方企业应用 | 支持 | 日历应用中日程闲忙状态读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=calendar_1.0%23GetSchedule) |
| 第三方个人应用 | 支持 | 日历应用中日程闲忙状态读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=calendar_1.0%23GetSchedule) |

## 请求方法

```
POST /v1.0/calendar/users/{userId}/querySchedule HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "userIds" : [ "String" ],
  "startTime" : "String",
  "endTime" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 * 第三方个人应用，调用[获取用户token](/document/development/obtain-user-token)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 本次请求的资源所归属的用户unionId。   * 企业内部应用和第三方企业应用，调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 * 第三方个人应用，调用[获取用户通讯录个人信息](/document/development/dingtalk-retrieve-user-information)接口获取unionId参数值。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userIds | Array of String | 是 | 查询目标用户的unionId。   * 企业内部应用和第三方企业应用，调用[查询用户详情](/document/development/query-user-details)接口获取unionid参数值。 * 第三方个人应用，调用[获取用户通讯录个人信息](/document/development/dingtalk-retrieve-user-information)接口获取unionId参数值。   **说明**  用户列表最大长度 20。 |
| startTime | String | 是 | 查询的开始时间。 |
| endTime | String | 是 | 查询的结束时间。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| scheduleInformation | Array | 用户忙闲信息。 |
| userId | String | 查询目标用户的unionId。 |
| error | String | 异常信息。 |
| scheduleItems | Array | 忙闲信息列表。 |
| status | String | 状态。   * **BUSY**：繁忙 * **TENTATIVE**：暂定繁忙 |
| start | Object | 日程开始时间。 |
| date | String | 日程开始日期，格式：`yyyy-MM-dd`。 |
| dateTime | String | 日程开始时间，格式为`ISO-8601`的`date-time`格式。 |
| timeZone | String | 时区名称，使用**IANA Time Zone Database**标准，例如：`Asia/Shanghai`。  **说明**  全天日程时区固定为UTC，非全天日程时区默认为`Asia/Shanghai`。 |
| end | Object | 日程结束时间。 |
| date | String | 日程结束日期，格式：`yyyy-MM-dd`。 |
| dateTime | String | 日程结束时间，格式为`ISO-8601`的`date-time`格式。 |
| timeZone | String | 时区名称，使用**IANA Time Zone Database**标准，例如：`Asia/Shanghai`。  **说明**  全天日程时区固定为UTC，非全天日程时区默认为`Asia/Shanghai`。 |
