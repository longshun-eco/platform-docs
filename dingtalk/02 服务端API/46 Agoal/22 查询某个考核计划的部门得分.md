更新于 2025-11-28调用该接口通过组织计划Id，查询某个考核计划下的部门得分。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | Agoal组织绩效文档读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=agoal_1.0%23AgoalOrgPerfDocQuery) |
| 第三方企业应用 | 支持 | Agoal组织绩效文档读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=agoal_1.0%23AgoalOrgPerfDocQuery) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/agoal/org_perf/documents/query?planId=String&pageNumber=Integer&pageSize=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| planId | String | 是 | 组织计划Id。 |
| pageNumber | Integer | 是 | 分页页码。 |
| pageSize | Integer | 否 | 分页大小。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| requestId | String | 请求Id。 |
| success | Boolean | 请求是否成功。 |
| content | Object | 返回分页结果。 |
| pageNumber | Integer | 分页页码。 |
| pageSize | Integer | 分页大小。 |
| totalCount | Long | 总数量。 |
| result | Array | 考核文档对象数组。 |
| OpenOrgPerfDocDTO | OpenOrgPerfDocDTO | 考核文档对象。 |
