更新于 2025-09-12调用本接口，根据填表的Code获取填表实例列表。

例如，依次单击**工作台 > 智能填表 > 统计 > 我创建的**，其中**测试一次性填表**已经收集了2份回复。如下图所示。

调用本接口，可获取**测试一次性填表**模板下填写的2份回复的详情信息，包括填表提交时间、表单实例ID、提交人姓名等信息。

![[development-obtain-the-table-filling-instance-list-data_O1CN01pD3DGs294SqEb8xoI_!!6000000008014-2-tps-2768-642.png]]

**说明**

* 调用本接口的QPS不能超过20，超过会返回限流错误。
* 目前智能填表目前仅支持查询旧版界面创建的表单数据，新版界面创建的表单无法查询。

  ![[development-obtain-the-table-filling-instance-list-data_p957609.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 智能填表数据读取权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=swform_1.0%23ListFormInstances) |
| 第三方企业应用 | 支持 | 智能填表数据读取权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=swform_1.0%23ListFormInstances) |
| 第三方个人应用 | 暂不支持 | 智能填表数据读取权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/swform/forms/{formCode}/instances?bizType=Integer&actionDate=String&nextToken=Integer&maxResults=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| formCode | String | 是 | 填表code，可调用[获取用户创建的填表模板列表](/document/development/new-obtains-the-template-that-a-user-creates)接口获取formCode参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| bizType | Integer | 否 | 填表类型。   * **0**：表示通用填表 * **1**：表示教育版填表 |
| actionDate | String | 否 | 时间，格式要求为yyyy-MM-dd。  **说明**  周期性填表才需要传该参数，一次性填表不需要传该参数。 |
| nextToken | Integer | 是 | 分页游标。   * 如果是首次查询，该参数值传0。 * 如果是非首次查询，该参数传上次调用时返回的nextToken。 |
| maxResults | Integer | 是 | 每页最大条目数，最大值100。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 接口调用是否成功。   * **true**：成功 * **false**：失败 |
| result | Object | 返回结果。 |
| hasMore | Boolean | 是否还有下一页数据。   * **true**：是 * **false**：否 |
| nextToken | Long | 分页游标。 |
| list | Array | 填表实例列表。 |
| createTime | String | 填表提交时间，iso8601格式，例如2022-07-29T15:06Z。 |
| modifyTime | String | 填表更新时间，iso8601格式，例如2022-07-29T15:06Z。 |
| formCode | String | 表单模板code。 |
| title | String | 表单名称。 |
| submitterUserId | String | 提交人的userId。 |
| submitterUserName | String | 提交人姓名。 |
| forms | Array | 表单内容列表。 |
| label | String | 表单控件的名称。 |
| key | String | 表单控件的key。 |
| value | String | 表单控件的值。 |
| formInstanceId | String | 表单实例ID。 |
| studentClassName | String | 学生班级名称。  **说明**  教育版填表返回该字段。 |
| studentClassId | String | 学生班级ID。  **说明**  教育版填表返回该字段。 |
| studentUserId | String | 学生ID。  **说明**  教育版填表返回该字段。 |
| studentName | String | 学生名称。  **说明**  教育版填表返回该字段。 |
