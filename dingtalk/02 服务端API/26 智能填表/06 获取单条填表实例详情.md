更新于 2025-09-12调用本接口，获取单条填表实例详情。

例如，**测试一次性填表**表单共有3条回复数据，如下图所示。

调用本接口可以根据某条回复数据的实例ID，获取该条回复实例内的表单详情信息，包括填表实例提交时间、表单实例提交者userId等。

![[development-obtains-the-instance-details-of-a-single-fill-table_O1CN01Zboj491Tu1ZArOzv5_!!6000000002441-2-tps-2772-1064.png]]

**说明**

目前智能填表目前仅支持查询旧版界面创建的表单数据，新版界面创建的表单无法查询。

![[development-obtains-the-instance-details-of-a-single-fill-table_p957609.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 智能填表数据读取权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=swform_1.0%23GetFormInstance) |
| 第三方企业应用 | 支持 | 智能填表数据读取权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=swform_1.0%23GetFormInstance) |
| 第三方个人应用 | 暂不支持 | 智能填表数据读取权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/swform/instances/{formInstanceId}?bizType=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| formInstanceId | String | 是 | 填表实例ID，调用[获取填表实例列表](/document/development/obtain-the-table-filling-instance-list-data)接口获取formInstanceId参数值。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| bizType | Integer | 否 | 填表类型。   * **0**：表示通用填表 * **1**：表示教育版填表 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| success | Boolean | 接口调用是否成功。   * **true**：成功 * **false**：失败 |
| result | Object | 返回结果。 |
| createTime | String | 填表实例提交时间，iso8601格式，例如2022-07-29T15:07Z。 |
| modifyTime | String | 填表更新时间，iso8601格式，例如2022-07-29T15:07Z。 |
| formCode | String | 表单模板code。 |
| title | String | 表单名称。 |
| creator | String | 表单实例提交者userId。 |
| forms | Array | 表单内容列表。 |
| label | String | 表单控件名称。 |
| key | String | 表单控件key。 |
| value | String | 表单控件的值。 |
