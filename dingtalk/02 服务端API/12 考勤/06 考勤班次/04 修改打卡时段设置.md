更新于 2025-09-10调用本接口，修改考勤班次打卡时间的设置信息。

**说明**

本接口目前仅支持设置卡点情况下是否打卡。

调用本接口，可修改考勤班次打卡时间设置。接口调用效果与下图操作实现效果一致。

![[development-modify-card-settings_p961046.png]]

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 是 | 考勤组管理权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingtalk.oapi.attendance.shift.updatepunches) |
| 第三方企业应用 | 否 | — | — |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/attendance/shift/updatepunches`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | af21axxx | 调用该接口的应用凭证，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| op\_user\_id | String | 是 | user456 | 操作者的userId。 |
| punches | TopPunchVO[] | 否 |  | 卡点信息。 |
| id | Number | 是 | 123 | 卡点ID， 可通过[获取班次详情](/document/development/shift-query)接口获取id参数值。 |
| free\_check | Boolean | 是 | true | 是否无需打卡。   * **true**：开启无需打卡。 * **false**：关闭无需打卡。 |
| shift\_id | Number | 是 | 456 | 班次ID， 可通过[获取班次摘要信息](/document/development/enterprise-shift-query-in-batches)接口获取id参数值。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| success | Boolean | true | 调用是否成功。   * **true**：成功 * **false**：失败 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | 系统错误 | 调用失败时返回的错误信息。 |
| request\_id | String | 5moh5icna65x | 请求ID。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/attendance/shift/updatepunches?access_token=ACCESS_TOKEN
```

请求正文

```json
{
    "punches": [
        {
            "free_check": true,
            "id": 373271823
        }
    ],
    "op_user_id": "user456",
    "shift_id": "712550377"
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/attendance/shift/updatepunches");
OapiAttendanceShiftUpdatepunchesRequest req = new OapiAttendanceShiftUpdatepunchesRequest();
List punchVOS = new ArrayList<>();
TopPunchVO punchVO = new TopPunchVO();
punchVO.setFreeCheck(true);
punchVO.setId(373271823L);
req.setPunches(punchVOS);
req.setOpUserId("user456");
req.setShiftId(712550377L);
OapiAttendanceShiftUpdatepunchesResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
    "errcode": 0,
    "success": true,
    "request_id": "ed67bcymo0bw"
}
```
