更新于 2025-09-11调用本接口，根据考勤机设备ID查询这台考勤机的相关信息。

调用本接口，可根据考勤机设备ID查询考勤机信息，包括考勤机设备ID、设备名称、设备类型名称、固件版本等。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 考勤组查询权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=attendance_1.0%23GetMachine) |
| 第三方企业应用 | 支持 | 考勤组查询权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=attendance_1.0%23GetMachine) |
| 第三方个人应用 | 暂不支持 | 考勤组查询权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/attendance/machines/{devId} HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| devId | Long | 是 | 考勤机设备ID，可调用[查询设备列表](/document/development/intelligent-hardware-list-query)接口获取device\_id参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Object | 考勤机的相关信息。 |
| deviceId | String | 加密后的考勤机设备ID。 |
| devId | Long | 考勤机设备ID。 |
| deviceName | String | 设备名称。 |
| productName | String | 设备类型名称。 |
| netStatus | String | 网络状态，取值。   * **1**：激活 * **2**：未激活 * **3**：已连接 * **4**：已断开 |
| productVersion | String | 固件版本。 |
| deviceSn | String | 设备序列号。 |
| maxFace | Integer | 人脸容量。 |
| voiceMode | Integer | 音量模式。 |
| atmManagerList | Array of String | 设备管理员列表。 |
| machineBluetoothVO | Object | 考勤机蓝牙相关设置信息。 |
| bluetoothValue | Boolean | 蓝牙是否打开打卡。   * **true**：打开 * **false**：未打开 |
| bluetoothCheckWithFace | Boolean | 蓝牙是否打开打卡人脸识别。   * **true**：打开 * **false**：未打开 |
| bluetoothDistanceMode | String | 蓝牙打卡范围。 |
| bluetoothDistanceModeDesc | String | 蓝牙打卡范围描述。 |
| monitorLocationAbnormal | Boolean | 是否打开位置异常监控。   * **true**：打开 * **false**：未打开 |
| address | String | 地址位置描述。 |
| longitude | double | 经度。 |
| latitude | double | 纬度。 |
| limitUserDeviceCount | Boolean | 是否限制员工常用手机。   * **true**：限制 * **false**：不限制 |
| userDeviceCount | Integer | 员工常用手机数量。 |
