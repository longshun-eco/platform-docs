更新于 2025-09-10调用本接口，查询企业员工在考勤组内的操作权限。

例如，**测试考勤组**的考勤组子负责人是小明。调用本接口，根据子负责人权限resourceKey和张三的userId，查询是否有对应权限。接口调用效果与下图操作实现效果一致。

![[development-attendance-writing-operation-is-brand-new-query_p960340.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 考勤授权信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=attendance_1.0%23CheckWritePermission) |
| 第三方企业应用 | 支持 | 考勤授权信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=attendance_1.0%23CheckWritePermission) |
| 第三方个人应用 | 暂不支持 | 考勤授权信息读权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/attendance/writePermissions/query HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "opUserId" : "String",
  "category" : "String",
  "resourceKey" : "String",
  "entityIds" : [ Long ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该API的应用凭证。   * 企业内部应用，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 * 第三方企业应用，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| opUserId | String | 是 | 员工userId。 |
| category | String | 是 | 资源类型：   * **GROUP**：考勤组，目前仅支持该值。 |
| resourceKey | String | 是 | 权限点：   * **GROUP\_MEMBER**：设置参与考勤人员 * **GROUP\_NAME**：修改考勤组名称 * **GROUP\_TYPE**：设置考勤组类型 * **CHECK\_TIME**：设置考勤时间 * **SCHEDULE**：员工排班 * **CHECK\_POSITION\_TYPE**：设置打卡方式 * **OVER\_TIME\_RULE**：设置加班规则 * **CAMERA\_CHECK**：拍照验证规则 * **OUT\_SIDE\_CHECK**：设置外勤打卡 * **MANAGE**：考勤组子负责人 * **OWNER**：考勤组主负责人 * **DELETE\_GROUP**：删除考勤组 |
| entityIds | Array of Long | 是 | 资源ID，如果category参数值为GROUP，该参数值传考勤组ID，可调用[获取用户考勤组](/document/development/queries-a-user-attendance-group)接口获取group\_id参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| entityPermissionMap | Map<String, Boolean> | 权限结果。  返回示例：{"entityPermissionMap":{"key":value}}。   * key，指资源ID，即请求参数entityIds的值。 * value，指是否有权限。    + true：有权限。   + false：无权限。 |
