更新于 2025-09-12调用本接口，给智能人事员工调岗，支持以下内容调整，如员工部门列表、主部门、职务、职位和职级。

### 接口功能介绍

例如，员工小钉3的花名册数据如下图所示，部门和主部门为测试部门1、职务为职务1、职位为产品经理、职级为职级1。

![[development-intelligent-personnel-staff-transfer_O1CN01PEum6h1ycqn1nvbl5_!!6000000006600-2-tps-2398-1210.png]]

调用本接口对员工小钉3进行调岗，调岗结果如下图所示，部门和主部门为测试部门2、职务为职务2、职位为经理、职级为职级2。

![[development-intelligent-personnel-staff-transfer_O1CN01i0NydY1kFa2qptQYo_!!6000000004654-2-tps-2380-1334.png]]

### 接口调用说明

**说明**

调用本接口给员工调岗，可能会涉及到智能人事升级职位管理功能，升级职位管理功能后需注意以下：

调用通讯录[更新用户信息](/document/development/user-information-update)接口更新员工部门或者员工职位时，接口会出现报错，报错信息如下图所示。如果需要更新员工部门或者员工职位，请调用本接口实现。

![[development-intelligent-personnel-staff-transfer_O1CN01aL7Jss1qVxuTak12v_!!6000000005502-2-tps-2044-246.png]]

![[development-intelligent-personnel-staff-transfer_O1CN01Ic29JG1Jx63CWW0dV_!!6000000001094-2-tps-2288-1452.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 智能人事流程读写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=hrm_1.0%23HrmProcessTransfer) |
| 第三方企业应用 | 暂不支持 | 智能人事流程读写权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 智能人事流程读写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/hrm/processes/transfer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "userId" : "String",
  "deptIdsAfterTransfer" : [ Long ],
  "mainDeptIdAfterTransfer" : Long,
  "positionNameAfterTransfer" : "String",
  "positionLevelAfterTransfer" : "String",
  "jobIdAfterTransfer" : "String",
  "positionIdAfterTransfer" : "String",
  "rankIdAfterTransfer" : "String",
  "operateUserId" : "String"
}
```
