更新于 2025-09-12调用本接口，新增或删除智能人事花名册选项类型字段的选项。

### 接口功能介绍

调用本接口，可新增或删除花名册中**选项类型**字段内的选项。接口调用效果与下图操作实现效果一致。

![[development-intelligent-personnel-roster-field-option-modification_O1CN01BRA5Xb1eQDaBes0sW_!!6000000003865-2-tps-2482-1476.png]]

**说明**

本接口不支持操作系统字段，例如：性别。

![[development-intelligent-personnel-roster-field-option-modification_O1CN011G1vWQ1Z9Cnr1MfXT_!!6000000003151-2-tps-2732-1232.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 钉钉HRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=hrm_1.0%23RosterMetaFieldOptionsUpdate) |
| 第三方企业应用 | 支持 | 钉钉HRM数据管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=hrm_1.0%23RosterMetaFieldOptionsUpdate) |
| 第三方个人应用 | 暂不支持 | 钉钉HRM数据管理权限 | 暂不支持 |

## 请求方法

```
PUT /v1.0/hrm/rosters/meta/fields/options?appAgentId=Long HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "groupId" : "String",
  "fieldCode" : "String",
  "labels" : [ "String" ],
  "modifyType" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| appAgentId | Long | 否 | 对应应用的agentId值，请参考[基础概念-AgentId](/document/development/basic-concepts-beta#884d363067bnq)。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| groupId | String | 是 | 花名册分组ID，可调用[获取花名册元数据](/document/development/intelligent-personnel-roster-metadata-query)接口获取group\_id参数值。 |
| fieldCode | String | 是 | 花名册字段标识。   * 企业内部应用，可调用[获取花名册元数据](/document/development/intelligent-personnel-roster-metadata-query)接口获取。 * 第三方企业应用，可调用[查询花名册中有权限的字段列表](/document/development/query-the-list-of-fields-with-permissions-in-the-roster)接口获取。 |
| labels | Array of String | 是 | 需要修改的选项值列表，最大值20。   * 如果modifyType值为**OPTIONS\_ADD**，该参数值为自定义值。 * 如果modifyType值为**OPTIONS\_DELETE**，该参数值可调用[获取员工花名册字段信息](/document/development/api-getemployeerosterbyfield)接口获取label参数值。 |
| modifyType | String | 是 | 修改类型。   * **OPTIONS\_ADD**：添加选项 * **OPTIONS\_DELETE**：删除选项 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 是否操作成功，返回true表示成功。 |
