更新于 2025-10-21调用本接口，获取指定公告的已读未读人员列表。

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 钉钉公告管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=blackboard_1.0%23QueryBlackboardReadUnRead) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/blackboard/readers?operationUserId=String&maxResults=Integer&nextToken=String&blackboardId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| operationUserId | String | 是 | 操作人userId。 |
| maxResults | Integer | 是 | 每页条目数，最大500。 |
| nextToken | String | 否 | 分页游标。  **说明**   * 如果是首次调用，该参数不传。 * 如果是非首次调用，该参数传上次调用时返回的nextToken。 |
| blackboardId | String | 是 | 公告id，可通过调用[获取公告ID列表](https://open.dingtalk.com/document/development/obtains-the-id-list-of-announcements-that-are-not-deleted)接口获取。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| nextToken | String | 分页游标。 |
| users | Array | 已读未读用户列表。 |
| userId | String | 员工userId。 |
| read | String | 是否已读：   * **true**：已读 * **false**：未读 |
| readTimestamp | Long | 员工已读的时间戳，单位毫秒。 |

## 错误码

| HttpCode | 错误码 | 错误信息 | 说明 |
| --- | --- | --- | --- |

| 403 | accessDenied | %s | 请求被拒绝，请确认操作人是企业主管理员或者是公告的发送人，并且公告归属于当前组织。 |
