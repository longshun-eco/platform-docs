## 什么是公告

管理员可以通过公告发布公司或单位的规章制度、节假日信息等通知，快速通知到全体员工。更多功能详情可参考[钉钉使用手册-公告](https://alidocs.dingtalk.com/i/p/Y7kmbokZp3pgGLq2/docs/mM3zoYAw1Rr8Dan5xEk6WnZ07y9NpXxD)。

![[development-blackboard-announcement-overview_p522596.png]]

## 使用公告产品客户案例

钉钉的启用使传统石油公司迈入云和移动时代。在信息化浪潮的变革中，作为连锁油气站专业化管理公司的中油公交，使用公告进一步实现了管理成本的降低和管理效率的提升。

|  |  |  |
| --- | --- | --- |
| 痛点 | 解决方案 | 效果 |
| 连锁型企业的员工分布较为分散，在传达内容较多的文件资料时比较费劲。 | 公告 | 使用钉钉公告发送长篇幅的文件资料，不但可以轻松实现图文混排，并且可以实现文件资料分人群的精准传送，为管理人员和员工都节省了大量时间和精力。同时，公告的“保密”功能也非常实用，有些文件只在企业内部传阅，有效保证公告不会被转发，而保密公告会自带水印，防止截屏转发，对于企业信息安全起到了强有力的保护作用。 |

## 如何开通公告

公告是钉钉默认安装的官方应用。员工可以在钉钉工作台打开应用并使用。

* 手机端：钉钉手机客户端-工作台

  ![[development-blackboard-announcement-overview_p522577.png]]
* 电脑端：钉钉电脑客户端-工作台

  ![[development-blackboard-announcement-overview_p522579.png]]

## 名词解释

* **公告分类**

  公告分类的唯一标识字段是category\_id。

  例如，公告分类“分类1”的category\_id是3b583d630587021beaa3b32d9fxxxxxx。

  ![[development-blackboard-announcement-overview_p522606.png]]
* **公告ID**

  公告的唯一标识字段是blackboard\_id。

  例如公告“正式公告1”的blackboard\_id是72b4f87d27e815f6fecxxxxxx。

  ![[development-blackboard-announcement-overview_p522603.png]]
* **公告的保密等级**

  公告保密等级的唯一标识是private\_level。

  + **0**：表示普通公告。
  + **1**：表示保密公告。

  ![[development-blackboard-announcement-overview_p423161.png]]
