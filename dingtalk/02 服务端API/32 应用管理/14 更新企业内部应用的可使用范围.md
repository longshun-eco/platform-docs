更新于 2026-01-22通过此接口，可更新企业内部应用的可使用范围，实现对用户、部门和角色的灵活权限管理。

### 使用场景

本接口用于调整企业内部微应用的可使用范围。 适用于需要动态增减访问权限的管理需求，实现精细化权限控制与安全保障。
例如：

* 为应用添加或移除可访问的用户；
* 为应用添加或移除可访问的部门；
* 为应用添加或移除可访问的角色；
* 设置应用仅限管理员可见。

### 接口功能介绍

该接口支持对企业内部微应用的使用范围进行增量更新，包括添加/删除指定用户、部门、角色，以及设置“仅管理员可见”模式。
例如，在开发者后台存在一个名为**测试微应用**的应用，通过该接口对可使用范围内的用户实现更新操作（如下图所示）：

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 管理微应用的权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=microApp_1.0%23SetMicroAppScope) |
| 第三方企业应用 | 暂不支持 | 管理微应用的权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 管理微应用的权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/microApp/apps/{agentId}/scopes HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "addUserIds" : [ "String" ],
  "delUserIds" : [ "String" ],
  "addDeptIds" : [ Long ],
  "delDeptIds" : [ Long ],
  "addRoleIds" : [ Long ],
  "delRoleIds" : [ Long ],
  "onlyAdminVisible" : Boolean
}
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| agentId | Long | 是 | 应用agentId，参考[基础概念-AgentId](https://open.dingtalk.com/document/development/development-basic-concepts#813cbd7067yn0)。 |

## Body参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| addUserIds | Array of String | 否 | 待添加的用户userId列表，最大长度100。  **说明**  添加后总用户数不得超过2000，否则将返回错误。 |
| delUserIds | Array of String | 否 | 删除的可使用用户userId列表，最大长度100。 |
| addDeptIds | Array of Long | 否 | 待添加的部门ID列表，最大长度100。  **说明**  添加后总部门数不得超过2000，否则将返回错误。 |
| delDeptIds | Array of Long | 否 | 待删除的部门ID列表，最大长度100。 |
| addRoleIds | Array of Long | 否 | 待添加的角色ID列表，最大长度100。可通过[获取角色列表](https://open.dingtalk.com/document/development/obtains-a-list-of-enterprise-roles)接口获取具体ID值。  **说明**  添加后总角色数不得超过2000，否则接口会报错。 |
| delRoleIds | Array of Long | 否 | 删除的可使用角色列表，最大长度100。可通过[获取角色列表](https://open.dingtalk.com/document/orgapp/obtains-a-list-of-enterprise-roles)接口获取id参数值。 |
| onlyAdminVisible | Boolean | 否 | 是否仅管理员可使用。   * **true**： 是 * **false**：否 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| result | Boolean | 是否更新成功，true表示更新成功。 |
