更新于 2026-01-22通过此接口获取企业内部应用的可使用范围信息，包括允许访问该应用的用户、部门和角色列表。

### 使用场景

本接口适用于企业内部应用权限范围管理的场景，例如：

* 当需要查看某个企业内部应用当前配置的权限范围时，可通过本接口查询其可使用范围设置。
* 在进行应用权限审计或安全检查时，可用于验证实际开放的用户、部门是否符合预期策略。
* 系统集成中，当需根据应用的使用范围动态调整功能可见性时，可调用本接口获取最新配置。
* 应用管理员在调整权限前，可通过该接口获取当前设置作为参考依据，确保变更操作准确无误。

### 接口功能介绍

## 权限

要调用此API，需要以下权限之一。

| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| --- | --- | --- | --- |
| 企业内部应用 | 支持 | 管理微应用的权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=microApp_1.0%23GetMicroAppScope) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/microApp/apps/{agentId}/scopes HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](https://open.dingtalk.com/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| agentId | Long | 是 | 应用agentId，参考[基础概念-AgentId](https://open.dingtalk.com/document/development/basic-concepts-beta#813cbd7067yn0)。 |

## 返回参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| result | Object | 可使用范围结果。 |
| userIds | Array of String | 用户userId。 |
| deptIds | Array of Long | 部门ID。 |
| roleIds | Array of Long | 角色ID。 |
| onlyAdminVisible | Boolean | 是否仅管理员可使用。   * **true**：userIds、deptIds和roleIds字段值为空。 * **false**：userIds、deptIds和roleIds字段有对应值。 |
