更新于 2025-09-23调用本接口，获取用户可使用的企业应用列表及应用信息，包括应用名称、应用图标、应用访问地址、应用的状态等。

### 使用场景

本接口适用于企业用户应用权限管理的场景。例如：

* 需要查看特定用户可见和可使用的企业应用列表时；
* 需要了解用户的应用访问权限范围时；
* 需要验证用户的应用权限配置是否正确时；
* 需要为用户应用权限调整提供数据支持时。

通过本接口，管理员可获取指定用户可见的企业应用信息，包括应用名称、应用图标、应用访问地址、应用状态等，为用户的权限管理和应用分发提供数据支持。

### 如何设置应用的使用范围

应用类型不同，设置可使用范围的方法不同。详情可查看[获取企业内部应用的可使用范围](/document/development/obtains-the-application-visible-range)。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 企业已安装的应用列表查询权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=microApp_1.0%23ListUserVilebleApp) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/microApp/users/{userId}/apps HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 否 | 用户的userId值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| appList | Array | 应用列表。 |
| agentId | Long | 应用AgentId。 |
| name | String | 应用名称。 |
| desc | String | 应用描述。 |
| icon | String | 应用图标。 |
| homepageLink | String | 应用移动端首页地址。 |
| pcHomepageLink | String | 应用PC端首页地址。 |
| ompLink | String | 应用管理后台地址。 |
| appId | Long | 应用ID。   * 基础应用，返回应用对应的AppId值。 * 企业自建应用，AppId值是0。 * 第三方企业应用，返回应用对应的AppId值。 |
| appStatus | Integer | 应用状态，取值：   * **0**：停用 * **1**：启用 * **3**：过期 |
| developType | Integer | 应用类型。   * **0**：表示H5微应用。 * **1**：表示小程序。 |
| unifiedAppId | String | 统一应用ID，详情参考[Unified App ID](/document/development/basic-concepts-beta#f20a795ad844u)。 |
