更新于 2025-09-23调用本接口，可以对企业内部小程序进行线上和体验版的发布。

### 使用场景

本接口适用于企业内部小程序版本发布管理的场景。例如：

* 小程序开发完成后需要发布线上版本供用户正式使用时；
* 需要发布体验版本进行内部测试和功能验证时；
* 小程序功能迭代更新后需要发布新版本时；
* 需要控制小程序在移动端和PC端的发布范围时。

通过本接口，管理员可灵活发布小程序的不同版本类型，支持线上版本和体验版本的发布，并可选择是否支持PC端访问，实现小程序版本的全生命周期管理。

### 接口功能介绍

例如，调用本接口可以获取[开发者后台](%5Bhttps%3A//open-dev.dingtalk.com/)-企业内部应用-应用详情-版本管理与发布页，在小程序开发版本列表可以点击小程序的发布或体验版发布。

**说明**

只有企业内部小程序才能调用该接口获取数据，且当前小程序存在对应的版本信息

调用该接口和如下图触发小程序发布按钮的效果一致，调用本接口需要调用者具备应用管理权限或应用开发权限。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 管理微应用的权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=microApp_1.0%23PublishInnerAppVersion) |
| 第三方企业应用 | 暂不支持 | 管理微应用的权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 管理微应用的权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/microApp/innerMiniApps/{agentId}/versions/publish HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "versionId" : Long,
  "opUnionId" : "String",
  "publishType" : "String",
  "miniAppOnPc" : Boolean
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| agentId | Long | 是 | 应用的AgentId，请参考[基础概念-AgentId](/document/development/basic-concepts-beta#884d363067bnq)。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| appVersionId | Long | 是 | 小程序版本id，用于唯一标识小程序版本信息，可调用[获取企业内部小程序的版本列表](/document/development/get-the-version-list-of-the-enterprise-internal-applet)接口，获取返回参数中 `appVersionId` 字段值。 |
| opUnionId | String | 是 | 操作人的unionId，该用户必须是拥有**应用管理权限**的管理员，可调用[查询用户详情](/document/development/query-user-details)接口获取。 |
| publishType | String | 否 | 小程序发布类型，取值：   * **online**：发布线上版本 * **experience**：发布体验版本 |
| miniAppOnPc | Boolean | 否 | 是否支持PC端打开小程序，取值：   * **false**：只发布移动端 * **true**：既发布移动端又发布PC端   **说明**  体验版本发布目前默认只支持移动端发布 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 小程序发布结果。 |
