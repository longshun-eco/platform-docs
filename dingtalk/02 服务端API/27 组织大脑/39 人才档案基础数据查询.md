更新于 2025-09-12调用本接口查询人员档案基础信息，支持批量查询。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 组织大脑数据查询权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=hrbrain_1.0%23HrbrainTalentProfileBasicQuery) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/hrbrain/profiles/basicData/query?dingCorpId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

[ "String" ]
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| dingCorpId | String | 否 | 组织编码。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
|  | Array of String | 否 | 钉钉用户UserId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| requestId | String | 请求ID。 |
| success | Boolean | 接口调用是否成功。 |
| result | Boolean | 查询是否成功。 |
| content | Object | 返回结果。 |
| profileBaseInfoList | Array | 人员基础信息列表。 |
| workNo | String | 钉钉用户UserId。 |
| name | String | 用户姓名。 |
| age | String | 年龄。 |
| deptName | String | 部门名称。 |
| jobcode | String | 职务。 |
| position | String | 职位。 |
| jobLevel | String | 职级。 |
| superName | String | 主管名。 |
| workPlace | String | 工作城市。 |
| gender | String | 性别。 |
| birthday | String | 生日。 |
| seniorityYears | String | 司龄（年）。 |
| superWorkNo | String | 主管的钉钉UserId。 |
| deptNo | String | 部门编码。 |
