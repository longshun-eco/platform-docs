更新于 2025-09-12调用本接口，自定义模型信息同步至组织大脑，支持批量同步。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 组织大脑数据集成写入权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=hrbrain_1.0%23HrbrainImportCustom) |
| 第三方企业应用 | 支持 | 组织大脑数据集成写入权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=hrbrain_1.0%23HrbrainImportCustom) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/hrbrain/datas/customModels/import?corpId=String&modelCode=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

[ null ]
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| corpId | String | 是 | 组织编码。 |
| modelCode | String | 是 | 自定义模型编码。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
|  | Array of Object | 是 | 自定义模型数据。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| requestId | String | 请求 ID。 |
| result | Boolean | 更新是否成功。 |
| success | Boolean | 接口调用是否成功。 |
