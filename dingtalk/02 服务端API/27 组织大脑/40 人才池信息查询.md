更新于 2025-09-12调用本接口，分页查询人才池信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 组织大脑数据查询权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=hrbrain_1.0%23HrbrainEmpPoolQuery) |
| 第三方企业应用 | 暂不支持 | 暂不支持 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
POST /v1.0/hrbrain/datas/empPools/infos/query HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "nextToken" : Integer,
  "maxResults" : Integer,
  "keyword" : "String",
  "labels" : [ "String" ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| nextToken | Integer | 否 | 分页起始位置。 |
| maxResults | Integer | 否 | 分页条数。 |
| keyword | String | 否 | 搜索关键字。 |
| labels | Array of String | 否 | 人才池标签值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| requestId | String | 请求ID。 |
| success | Boolean | 接口调用是否成功。 |
| result | Boolean | 查询是否成功。 |
| content | Object | 返回结果。 |
| totalCount | Integer | 总记录数。 |
| maxResults | Integer | 每页条数。 |
| nextToken | Integer | 分页起始位置。 |
| poolInfos | Array | 人才池列表。 |
| poolCode | String | 人才池code。 |
| poolName | String | 人才池名称。 |
| poolDesc | String | 人才池描述。 |
| poolTags | Array | 人才池标签列表。 |
| label | String | 标签名称。 |
| value | String | 标签值。 |
