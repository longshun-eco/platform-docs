## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | AI 表格应用读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=notable_1.0%23ListRecords) |
| 第三方企业应用 | 暂不支持 | AI 表格应用读权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | AI 表格应用读权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/notable/bases/{baseId}/sheets/{sheetIdOrName}/records/list?operatorId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "maxResults" : Integer,
  "nextToken" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| baseId | String | 是 | AI表格ID，获取方式请参考[数据结构](/document/development/data-structure)。 |
| sheetIdOrName | String | 是 | 数据表ID或数据表名称。数据表ID可以通过以下两种方式获取：   * 调用[获取所有数据表](/document/development/api-notable-getallsheets)接口获取。 * 调用[获取数据表](/document/development/api-notable-getsheet)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| operatorId | String | 是 | 操作人的unionId，可通过以下两种方式获取：   * 调用[查询用户详情](/document/development/query-user-details)接口获取。 * 调用[通过免登码获取用户信息](/document/development/obtain-the-userid-of-a-user-by-using-the-log-free)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| filter | Object | 否 | 筛选条件，设置后将只返回满足条件的记录。 |
| combination | String | 否 | 条件组合方式，可选值：   * **and**：同时满足所有条件 * **or**：满足任一条件 |
| conditions | Array | 是 | 条件。 |
| field | String | 是 | 字段ID或字段名。 |
| operator | String | 是 | 条件类型：equal | notEqual | greater | greaterEqual | less | lessEqual | contain | notContain | empty | notEmpty。 |
| value | Array of Any | 否 | 条件值。 |
| maxResults | Integer | 否 | 每页获取的数据量，默认值为100，最小值为1，最大值为100。 |
| nextToken | String | 否 | 上一次查询返回的游标，首次查询时不需要传。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| hasMore | Boolean | 是否有更多数据。 |
| nextToken | String | 分页游标。 |
| records | Array | 记录数据列表。 |
| id | String | 记录ID。 |
| fields | Map<String, Any> | 字段值。 |
|  | Any | 字段值。 |
| createdBy | Object | 创建人。 |
| unionId | String | 创建人的unionId。 |
| lastModifiedBy | Object | 上次修改人。 |
| unionId | String | 上次修改人的unionId。 |
| createdTime | Long | 创建时间。 |
| lastModifiedTime | Long | 上次修改时间。 |
