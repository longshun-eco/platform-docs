更新于 2025-09-11调用本接口，批量获取通讯录隐藏设置信息列表。

### 接口功能介绍

企业在调用[新增或更新通讯录隐藏设置](/document/development/update-address-book-hide-settings)接口后，可调用本接口查询企业下的通讯录隐藏设置列表。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录数据可见范围管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23ListContactHideSettings) |
| 第三方企业应用 | 暂不支持 | 通讯录数据可见范围管理权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 通讯录数据可见范围管理权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/contact/contactHideSettings?nextToken=Long&maxResults=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| nextToken | Long | 否 | 分页游标。   * 首次调用，该参数传0。 * 非首次调用，该参数传上次调用时返回的nextToken。 |
| maxResults | Integer | 否 | 分页大小，最大值100。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| hasMore | Boolean | 是否还有更多数据。   * true：是 * false：否 |
| nextToken | Long | 下一页的游标，为空字符串则表示分页结束。 |
| list | Array | 通讯录隐藏的设置列表。 |
| name | String | 设置名称。 |
| description | String | 设置描述。 |
| objectStaffIds | Array of String | 要隐藏的员工userId列表。 |
| objectDeptIds | Array of Long | 要隐藏的部门ID列表。 |
| objectTagIds | Array of Long | 要影藏的角色Id列表。 |
| excludeStaffIds | Array of String | 不受本次隐藏设置影响的用户userId列表。 |
| excludeDeptIds | Array of Long | 不受本次隐藏影响的部门ID列表。 |
| excludeTagIds | Array of Long | 不受本次隐藏设置影响的角色Id列表。 |
| active | Boolean | 规则是否生效。   * true：生效 * false：不生效 |
| id | Long | 设置ID。 |
