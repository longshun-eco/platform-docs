## зӨәдҫӢ

**иҜ·жұӮзӨәдҫӢпјҲHTTPпјү**

```http
POST https://oapi.dingtalk.com/role/add_role?access_token=ACCESS_TOKEN
```

иҜ·жұӮжӯЈж–Ү

```json
{
        "roleName":"жңҚеҠЎз»ҸзҗҶ",
        "groupId":1507113595
}
```

**иҜ·жұӮзӨәдҫӢпјҲJAVA SDKпјү**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/role/add_role");
OapiRoleAddRoleRequest req = new OapiRoleAddRoleRequest();
req.setRoleName("жөӢиҜ•1");
req.setGroupId(1507113595L);
OapiRoleAddRoleResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**иҝ”еӣһзӨәдҫӢ**

```json
{
        "errcode":0,
        "roleId":1688611118,
        "errmsg":"ok"
}
```
