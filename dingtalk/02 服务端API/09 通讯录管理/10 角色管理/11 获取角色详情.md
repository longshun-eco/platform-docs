## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 是 | 通讯录部门成员读权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingtalk.oapi.role.getrole) |
| 第三方企业应用 | 是 | 通讯录部门成员读权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.role.getrole) |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/role/getrole`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | 6ed1xxxx | 调用该API的应用凭证。   * 企业内部应用，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 * 第三方企业应用，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| roleId | Number | 是 | 1581321999 | 角色ID，可调用[获取角色列表](/document/development/obtains-a-list-of-enterprise-roles)接口获取id参数值。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| role | OpenRole |  | 角色信息。 |
| name | String | 负责人 | 角色名称。 |
| groupId | Number | 1507113577 | 所属的角色组ID。 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |
| request\_id | String | 4cmr6app8ujf | 请求ID。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/role/getrole?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "roleId":1507113578
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/role/getrole");
OapiRoleGetroleRequest req = new OapiRoleGetroleRequest();
req.setRoleId(1507113578L);
OapiRoleGetroleResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
        "errcode":0,
        "role":{
                "groupId":1507113577,
                "name":"负责人"
        },
        "request_id":"4cmr6app8ujf"
}
```

## 错误码

|  |  |  |
| --- | --- | --- |
| 错误码（errcode） | 错误码描述（errmsg） | 解决方案 |
| 33011 | 无效的roleId | 请校验roleId是否正确 |
| 43007 | 需要操作权限 | 请确认access\_token是否有权限 |
| 400001 | 系统错误 | 请稍后再试 |
