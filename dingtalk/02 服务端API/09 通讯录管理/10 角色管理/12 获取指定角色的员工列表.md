## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 是 | 通讯录部门成员读权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingtalk.oapi.role.simplelist) |
| 第三方企业应用 | 是 | 通讯录部门成员读权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.role.simplelist) |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/role/simplelist`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | 6ed1xxxx | 调用该API的应用凭证。   * 企业内部应用，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 * 第三方企业应用，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| role\_id | Number | 是 | 1203141 | 角色roleId，可通过调用[获取角色列表](/document/development/obtains-a-list-of-enterprise-roles)接口获取id参数值。 |
| size | Number | 否 | 20 | 分页大小。  **说明**  与offset参数同时设置时才生效，此参数代表分页大小，默认值20，最大100。 |
| offset | Number | 否 | 0 | 分页偏移量。  **说明**  与size参数同时设置时才生效，此参数代表偏移量，偏移量从0开始。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| request\_id | String | f6249uw1kuus | 请求ID。 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | ok | 返回码描述。 |
| result | PageVo |  | 查询结果。 |
| hasMore | Boolean | true | 是否还有更多数据。 |
| list | OpenEmpSimple[] |  | 角色下的员工列表。 |
| userid | String | manager4220 | 员工的userid。 |
| name | String | 小钉 | 员工姓名。 |
| manageScopes | OrgDeptVo[] |  | 管理范围。 |
| dept\_id | Number | 11 | 部门ID。 |
| name | String | 纪检部 | 部门名称。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/role/simplelist?access_token=ACCESS_TOKEN
```

请求正文

```json
{
        "size":20,
        "offset":0,
        "role_id":1203141
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/role/simplelist");
OapiRoleSimplelistRequest req = new OapiRoleSimplelistRequest();
req.setRoleId(1203141L);
req.setSize(20L);
req.setOffset(0L);
OapiRoleSimplelistResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
    "errcode": 0,
    "result": {
        "hasMore": false,
        "list": [
            {
                "name": "小钉",
                "userid": "manager4220",
                 "manageScopes":[
                        {
                                "dept_id":11,
                                "name":"纪检部"
                        }
                    ]
            }
        ]
    },
    "request_id": "f6249uw1kuus"
}
```

| --- | --- | --- |

| 40069 | 无效的size | 请调整分页大小 |
| 40068 | 无效的偏移量 | 请调整偏移量 |
| 400001 | 系统错误 | 请稍后重试 |
