更新于 2025-09-10调用本接口获取企业用户通讯录中的个人信息。

**说明**

企业内部应用和第三方企业应用在调用该接口前，需要先获取个人用户的accessToken，获取方式请参考[获取登录用户的访问凭证](/document/development/obtain-identity-credentials)。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录个人信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23GetUser) |
| 第三方企业应用 | 支持 | 通讯录个人信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=contact_1.0%23GetUser) |
| 第三方个人应用 | 支持 | 通讯录个人信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=personal&api=contact_1.0%23GetUser) |

## 请求方法

```
GET /v1.0/contact/users/{unionId} HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用服务端接口的授权凭证。使用个人用户的accessToken，请参考[获取登录用户的访问凭证](/document/development/obtain-identity-credentials)。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| unionId | String | 是 | 用户的unionId。  **说明**  如需获取当前授权人的信息，unionId参数可以传me。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| nick | String | 用户的钉钉昵称。 |
| avatarUrl | String | 头像URL。 |
| mobile | String | 用户的手机号。  **说明**   * 第三方企业应用无法获取用户完整手机号，本接口对手机号进行脱敏处理，第三方企业应用获取手机号示例：`155****3240`。 * 如果要获取用户手机号，需要在[钉钉开发者后台](https://open-dev.dingtalk.com/)申请**个人手机号信息**权限，如下图。 ![[development-dingtalk-retrieve-user-information_O1CN01hqI7gN1q6JepNXzJa_!!6000000005446-2-tps-520-274.png]] |
| openId | String | 用户的openId。 |
| unionId | String | 用户的unionId。 |
| email | String | 用户的个人邮箱。  **说明**  添加个人邮箱。 ![[development-dingtalk-retrieve-user-information_O1CN01UtYbLG1Dz21do15qR_!!6000000000286-2-tps-1314-698.png]] |
| stateCode | String | 手机号对应的国家号。 |
