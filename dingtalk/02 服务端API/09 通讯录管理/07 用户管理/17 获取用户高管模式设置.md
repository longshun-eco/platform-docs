更新于 2025-09-11调用本接口获取用户高管模式设置。

### 接口功能介绍

企业在调用[设置高管模式](https://open.dingtalk.com/document/orgapp/update-executive-settings)接口后，可调用本接口查询企业下的高管模式设置信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 成员信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23ListSeniorSettings) |
| 第三方企业应用 | 暂不支持 | 成员信息读权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 成员信息读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/contact/seniorSettings?seniorStaffId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| seniorStaffId | String | 是 | 用户userId，可通过[通过免登码获取用户信息](/document/development/obtain-the-userid-of-a-user-by-using-the-log-free)获得userId。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| seniorStaffId | String | 用户userId。 |
| protectScenes | Array of String | 拦截场景列表。 |
| seniorWhiteList | Array | 高管白名单列表。 |
| id | String | 高管白名单对象ID，取值为：   * type=1：表示userId。 * type=2：表示部门deptId。 * type=3：表示角色roleId。 |
| type | Integer | 高管白名单类型，取值：   * **1**：表示为用户 * **2**：表示为部门 * **3**：表示为角色 |
| name | String | 高管白名单对象名称，取值为：   * type=1：表示员工名称。 * type=2：表示部门名称。 * type=3：表示角色名称。 |
