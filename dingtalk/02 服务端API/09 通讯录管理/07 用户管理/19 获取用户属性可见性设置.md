更新于 2025-09-11调用本接口获取用户属性可见性设置。

### 接口功能介绍

企业调用[设置用户属性可见性](/document/development/add-or-update-the-hidden-settings-of-the-employee-property)接口后，可调用本接口获取企业员工属性字段的隐藏设置信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录数据可见性的管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23ListEmpAttributeVisibility) |
| 第三方企业应用 | 支持 | 通讯录数据可见性的管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=contact_1.0%23ListEmpAttributeVisibility) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```
GET /v1.0/contact/staffAttributes/visibilitySettings?nextToken=Long&maxResults=Integer HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| nextToken | Long | 否 | 分页游标。   * 首次调用，该参数传-1。 * 非首次调用，该参数传上次调用时返回的nextToken。 |
| maxResults | Integer | 否 | 分页大小，最大支持100。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| hasMore | Boolean | 是否还有更多数据。   * **true**：有 * **false**：没有 |
| nextCursor | Long | 下一次拉取时的token。 |
| list | Array | 设置列表。 |
| id | Long | 设置的ID。 |
| gmtCreate | String | 设置创建的时间。 |
| gmtModified | String | 设置最后修改的时间。 |
| name | String | 设置的名称。 |
| description | String | 设置的描述信息。 |
| objectStaffIds | Array of String | 需要隐藏的字段的用户userId。 |
| objectDeptIds | Array of Long | 需要隐藏的字段的部门deptId。 |
| objectTagIds | Array of Long | 需要隐藏的字段的角色roleId。 |
| hideFields | Array of String | 隐藏的字段ID。 |
| excludeStaffIds | Array of String | 白名单用户userId。 |
| excludeDeptIds | Array of Long | 白名单部门deptId。 |
| excludeTagIds | Array of Long | 白名单角色roleId。 |
| active | Boolean | 是否生效。   * **true**：生效 * **false**：未生效 |
