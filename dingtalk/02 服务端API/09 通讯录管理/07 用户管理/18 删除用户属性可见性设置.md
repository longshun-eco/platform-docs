更新于 2025-09-11调用本接口删除企业员工属性字段可见性设置。

### 接口功能介绍

企业调用[设置用户属性可见性](/document/development/add-or-update-the-hidden-settings-of-the-employee-property)接口后，可调用本接口删除企业员工属性字段的隐藏设置信息。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录数据可见性的管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23DeleteEmpAttributeVisibility) |
| 第三方企业应用 | 支持 | 通讯录数据可见性的管理权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=contact_1.0%23DeleteEmpAttributeVisibility) |
| 第三方个人应用 | 暂不支持 | 暂不支持 | 暂不支持 |

## 请求方法

```sql
DELETE /v1.0/contact/staffAttributes/visibilitySettings/{settingId} HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Path参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| settingId | Long | 是 | 设置的ID，可通过[设置用户属性可见性](/document/development/add-or-update-the-hidden-settings-of-the-employee-property)接口获取id参数值。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 是否删除成功，true表示成功。 |
