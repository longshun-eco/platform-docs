更新于 2025-09-11调用本接口设置员工的高管模式。

**说明**

当设置员工为高管，则针对设置的拦截场景，同企业内只有其他高管、高管白名单内用户以及高管的好友可以联系。其他人不可联系。
例如：设置userA为高管，拦截场景为VIDEO\_CALL，白名单用户设置了userB。userB、userC为同一企业的普通员工，且跟userA没有好友关系，则userB可以对userA发起视频通话，userC不可以对userA发起视频通话。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 维护通讯录的接口访问权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23UpdateSeniorSetting) |
| 第三方企业应用 | 暂不支持 | 维护通讯录的接口访问权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 维护通讯录的接口访问权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/contact/seniorSettings HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "seniorStaffId" : "String",
  "open" : Boolean,
  "permitStaffIds" : [ "String" ],
  "permitDeptIds" : [ Long ],
  "permitTagIds" : [ Long ],
  "protectScenes" : [ "String" ]
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| seniorStaffId | String | 是 | 需要设置的员工userid。 |
| open | Boolean | 是 | 是否开启高管模式，取值：   * **true**：开启高管模式 * **false**：关闭高管模式 |
| permitStaffIds | Array of String | 否 | 高管白名单员工userid列表。  **说明**  参数**permitStaffIds**、**permitDeptIds**、**permitTagIds**列表内元素之和最大为200。 |
| permitDeptIds | Array of Long | 否 | 高管白名单部门列表。  **说明**  参数**permitStaffIds**、**permitDeptIds**、**permitTagIds**列表内元素之和最大为200。 |
| permitTagIds | Array of Long | 否 | 高管白名单角色列表。  **说明**  参数**permitStaffIds**、**permitDeptIds**、**permitTagIds**列表内元素之和最大为200。 |
| protectScenes | Array of String | 否 | 高管拦截场景，取值：   * **DING\_SMS**： 短信Ding消息 * **DING\_MSG**： 应用内Ding消息 * **CHAT**： IM聊天（单聊、群聊） * **CONFERENCE\_VIDEO**：视频会议 * **CONFERENCE**：语音会议 * **VIDEO\_CALL**：视频通话 * **CONFERENCE\_VOIP**：语音通话 * **CALL**：电话会议、办公电话 * **SEARCH**： 电话会议里搜索功能 * **CALL\_CENTER**：呼叫中心 |
