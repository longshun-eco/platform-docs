更新于 2025-09-11调用本接口，获取企业认证信息。

### 接口功能介绍

例如，调用本接口，根据目标企业的corpId，获取企业企业认证信息，包含企业通讯录名称等信息。

![[development-obtain-enterprise-authentication-information_O1CN01hy8EOy1r7ynp3gbV3_!!6000000005585-2-tps-617-459.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录组织基础信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23GetOrgAuthInfo) |
| 第三方企业应用 | 支持 | 通讯录组织基础信息读权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=contact_1.0%23GetOrgAuthInfo) |
| 第三方个人应用 | 暂不支持 | 通讯录组织基础信息读权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/contact/organizations/authInfos?targetCorpId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过以下获取：   * 企业内部应用，调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 * 第三方企业应用，调用[获取第三方应用授权企业的accessToken](/document/development/obtain-the-access-token-of-the-authorized-enterprise-1)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| targetCorpId | String | 否 | 需要获取的企业认证信息的企业corpId，详情参见[基础概念-CorpId](/document/development/basic-concepts-beta#section-bbk-mv0-oxd)。  ![[development-obtain-enterprise-authentication-information_O1CN01X0RGo41GFBFOwwpZD_!!6000000000592-2-tps-2198-389.png]] |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| orgName | String | 企业在钉钉通讯录的名称。 |
| licenseOrgName | String | 提交企业认证时营业执照上面的企业名称。 |
| registrationNum | String | 营业执照注册号（一般15位）。 |
| unifiedSocialCredit | String | 社会统一信用代码（固定18位）。 |
| organizationCode | String | 组织机构代码证号（格式11111111-1）。 |
| legalPerson | String | 法人。 |
| licenseUrl | String | 营业执照url。 |
| authLevel | Long | 认证等级，取值：   * 0：无认证，返回参数仅返回 `authLevel` 和 `orgName` 字段 * 1：高级认证 * 2：中级认证 |

## 错误码

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | invalidParameter.system.param | 无效的企业targetCorpId | 需要查询的企业不存在时抛错 |
| 400 | permission.denied | 没有权限获取企业认证信息 | 企业没有向自建应用授权时报错 |
| 400 | invalidParameter.system.param | 系统参数为空 | 系统参数为空的时候返回 |
| 400 | invalidParameter.appKey.blank | appKey为空 | 自建应用的appKey为空 |
| 500 | service.system.error | 系统错误 | 依赖服务调用失败时抛这个错误 |
