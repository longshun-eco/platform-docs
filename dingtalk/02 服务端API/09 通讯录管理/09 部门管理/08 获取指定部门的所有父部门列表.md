更新于 2025-09-10调用本接口，获取指定部门的所有父部门ID列表。

**重要**

该接口不受通讯录权限范围限制。

例如，部门A的部门ID为1000，获取A部门的所有父部门列表，得到的结果是[1000,789,456,123,1]。

![[development-query-the-list-of-all-parent-departments-of-a-department_p358764.png]]

## 权限

服务端API是以应用维度授权的，在调用接口前，确保已经为应用添加了接口权限。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持调用 | 权限申请方式 | API Explorer调试 |
| 企业内部应用 | 是 | 通讯录部门信息读权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=dingtalk.oapi.v2.department.listparentbydept) |
| 第三方企业应用 | 是 | 通讯录部门信息读权限 | [调试](https://open-dev.dingtalk.com/apiExplorer#/?devType=isv&api=dingtalk.oapi.v2.department.listparentbydept) |
| 第三方个人应用 | 否 | — | — |

## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/v2/department/listparentbydept`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | bE74xxxx | 调用该API的应用凭证。   * 企业内部应用，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 * 第三方企业应用，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

## Body参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| dept\_id | Number | 是 | 42xxxx36 | 要查询的部门ID，可调用[获取部门列表](/document/development/obtain-the-department-list-v2)接口获取dept\_id参数值。 |

## 返回参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 示例值 | 描述 |
| request\_id | String | 10u9dj0853fwe | 请求ID。 |
| errcode | Number | 0 | 返回码。 |
| errmsg | String | 不合法的部门id | 调用失败时返回的错误信息。 |
| result | DeptListParentByDeptIdResponse |  | 父部门ID列表。 |
| parent\_id\_list | Number[] | [789, 456, 123, 1] | 该部门的所有父部门ID列表。  **说明**  获取的list元素第一个是当前部门ID，最后一个是父部门ID，从左至右部门层级递增。 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/v2/department/listparentbydept?access_token=ACCESS_TOKEN
```

请求正文

```json
{
 "dept_id":42xxxx36
}
```

**请求示例（JAVA SDK）**

```java
DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/v2/department/listparentbydept");
OapiV2DepartmentListparentbydeptRequest req = new OapiV2DepartmentListparentbydeptRequest();
req.setDeptId(100L);
OapiV2DepartmentListparentbydeptResponse rsp = client.execute(req, access_token);
System.out.println(rsp.getBody());
```

**返回示例**

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "result": {
        "parent_id_list": [
            42xxxx36,
            1
        ]
    },
    "request_id": "ov7bc964j9pe"
}
```

## 错误码

|  |  |  |
| --- | --- | --- |
| 错误码（errcode） | 错误码描述（errmsg） | 解决方案 |
| 43007 | 需要授权 | 请确认access\_token具有操作权限 |
| 60003 | 未找到对应部门 | 请确认dept\_id是否正确 |
| 400002 | 无效的参数 | 请确认参数是否按要求输入 |
| -1 | 系统繁忙 | 请稍后再试 |
