## 基本信息

**请求方式**：POST

**请求地址**：`https://oapi.dingtalk.com/topapi/v2/department/get`

## Query参数

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| access\_token | String | 是 | bE74xxxx | 调用该API的应用凭证。   * 企业内部应用，通过[获取企业内部应用的access\_token](/document/development/obtain-orgapp-token)接口获取。 * 第三方企业应用，通过[获取第三方企业的access\_token](/document/development/obtain-isvapp-token)接口获取。 |

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 示例值 | 描述 |
| dept\_id | Number | 是 | 100 | 部门ID，根部门ID为1，可调用[获取部门列表](/document/development/obtain-the-department-list-v2)接口获取dept\_id参数值。 |
| language | String | 否 | zh\_CN | 通讯录语言：   * **zh\_CN**（默认）：中文 * **en\_US**：英文 |

## 示例

**请求示例（HTTP）**

```http
POST https://oapi.dingtalk.com/topapi/v2/department/get?access_token=ACCESS_TOKEN
```

请求正文

```json
{
    "language":"zh_CN",
    "dept_id":100
}
```

**请求示例（JAVA SDK）**

```java
public class Main {
    public static void main(String[] args) {
        try {
            DingTalkClient client = new DefaultDingTalkClient("https://oapi.dingtalk.com/topapi/v2/department/get");

            System.out.println(rsp.getBody());
        } catch (ApiException e) {
            e.printStackTrace();
        }
    }
}
```
