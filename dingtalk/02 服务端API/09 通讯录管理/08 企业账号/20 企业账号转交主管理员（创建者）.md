更新于 2025-09-11调用本接口，将本组织内某企业账号有所有权的组织，转交给另一企业账号，如果接收的账号不在该组织内则自动加入。

例如，钉钉组织A下有两个企业账号，分别为企业账号A1和企业账号A2，其中企业账号A1是另一企业名称为钉钉组织B的创建者身份（当前测试企业OA管理后台-安全与权限-主管理员列表可查看）。调用本接口可用于把钉钉组织B的创建者身份变更为企业账号A2。

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录企业账号安全数据读写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23ChangeMainAdmin) |
| 第三方企业应用 | 暂不支持 | 通讯录企业账号安全数据读写权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 通讯录企业账号安全数据读写权限 | 暂不支持 |

## 请求方法

```
POST /v1.0/contact/orgAccounts/mainAdministrators/change HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json

{
  "sourceUserId" : "String",
  "targetUserId" : "String",
  "effectCorpId" : "String"
}
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Body参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| sourceUserId | String | 是 | 原企业账号userid，可通过以下四种方式获得：   * [根据手机号查询企业账号用户](/document/development/obtain-the-userid-of-your-mobile-phone-number) * [创建SSO企业账号](/document/development/create-an-sso-account) * [创建钉钉自建企业账号](/document/development/create-dingtalk-user-created-dedicated-account) * [邀请其他组织企业账号加入](/document/development/invite-other-organization-specific-accounts-to-join) |
| targetUserId | String | 是 | 接收专属账号userid，可通过以下四种方式获得：   * [根据手机号查询企业账号用户](/document/development/obtain-the-userid-of-your-mobile-phone-number) * [创建SSO企业账号](/document/development/create-an-sso-account) * [创建钉钉自建企业账号](/document/development/create-dingtalk-user-created-dedicated-account) * [邀请其他组织企业账号加入](/document/development/invite-other-organization-specific-accounts-to-join) |
| effectCorpId | String | 是 | 被转交的组织corpId。详情参见基础概念-[CorpId](/document/development/basic-concepts-beta#section-bbk-mv0-oxd)。 |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| result | Boolean | 转接操作结果。 |

## 错误码

|  |  |  |  |
| --- | --- | --- | --- |
| HttpCode | 错误码 | 错误信息 | 说明 |
| 400 | not.main.administrator | 不是主管理员(创建者) | 不是主管理员(创建者) |
| 400 | exclusive.account.limit | 只能操作归属该组织的专属账号 | 只能操作归属该组织的专属账号 |
| 400 | profile.not.exist | 账号不存在 | 账号不存在 |
