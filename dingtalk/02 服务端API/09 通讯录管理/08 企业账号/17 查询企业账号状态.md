更新于 2025-09-11调用本接口，查询某企业账号的启用状态。

如下图所示，企业账号可分为**正常**和**已停用**两种状态。

![[development-query-dedicated-account-status-1_O1CN01DqG1Pr1GrCDlPN9e6_!!6000000000675-2-tps-932-189.png]]

## 权限

要调用此API，需要以下权限之一。

|  |  |  |  |
| --- | --- | --- | --- |
| 应用类型 | 是否支持 | 权限 | API Explorer调试 |
| 企业内部应用 | 支持 | 通讯录企业账号安全数据读写权限 | [API Explorer](https://open-dev.dingtalk.com/apiExplorer#/?devType=org&api=contact_1.0%23QueryStatus) |
| 第三方企业应用 | 暂不支持 | 通讯录企业账号安全数据读写权限 | 暂不支持 |
| 第三方个人应用 | 暂不支持 | 通讯录企业账号安全数据读写权限 | 暂不支持 |

## 请求方法

```
GET /v1.0/contact/orgAccounts/status?userId=String HTTP/1.1
Host:api.dingtalk.com
x-acs-dingtalk-access-token:String
Content-Type:application/json
```

## Header参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| x-acs-dingtalk-access-token | String | 是 | 调用该接口的访问凭证，通过调用[获取企业内部应用的accessToken](/document/development/obtain-the-access-token-of-an-internal-app)接口获取。 |

## Query参数

|  |  |  |  |
| --- | --- | --- | --- |
| 名称 | 类型 | 是否必填 | 描述 |
| userId | String | 是 | 企业账号的userid，可通过以下四种方式获得：   * [根据手机号查询企业账号用户](/document/development/obtain-the-userid-of-your-mobile-phone-number) * [创建SSO企业账号](/document/development/create-an-sso-account) * [创建钉钉自建企业账号](/document/development/create-dingtalk-user-created-dedicated-account) * [邀请其他组织企业账号加入](/document/development/invite-other-organization-specific-accounts-to-join) |

## 返回参数

|  |  |  |
| --- | --- | --- |
| 名称 | 类型 | 描述 |
| disable | Boolean | 企业账号状态，取值：   * **true**：停用 * **false**：可用 |
