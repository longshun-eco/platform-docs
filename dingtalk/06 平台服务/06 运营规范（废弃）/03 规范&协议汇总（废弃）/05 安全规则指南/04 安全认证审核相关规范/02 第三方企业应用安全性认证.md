更新于 2025-09-23本文介绍第三方企业应用安全性认证的说明。

## **一、目的**

钉钉和钉钉的应用服务商合作，提供关于钉钉上第三方企业应用安全性认证评估的相关信息，旨在帮助企业客户从安全角度评估、决策、使用和管理钉钉上的第三方企业应用。

## **二、认证说明**

### **1、部署环境可信**

第三方企业应用遵循钉钉开放平台的部署规范要求，将应用部署在计算巢、聚石塔环境，且应用的部署状态校验通过，满足钉钉安全管控的要求，未按照要求部署在计算巢、聚石塔的应用无安全认证勋章。

### **2、安全评级**

安全评级是由钉钉安全根据[《第三方企业应用安全评分标准》](https://open.dingtalk.com/document/isvapp/dingtalk-isv-application-security-score)进行安全评分后认证的安全级别，目前分为优、良、中三个认证通过的级别，分数未满足要求的应用无安全认证勋章。

安全评分对应的安全评级如下：

|  |  |
| --- | --- |
| 分数范围 | 安全评级 |
| 80分≤总分≤100分 | 优 |
| 70分≤总分＜80分 | 良 |
| 60分≤总分＜70分 | 中 |
| 总分<60分 | 无认证（未通过） |

**注：安全评级是基于第三方企业应用的开发者提供的安全审核材料和钉钉平台的安全检测数据，由钉钉安全同学综合审核评估，相对客观体现应用的安全性，但不能保证所有已评估项的完全准确性。**
