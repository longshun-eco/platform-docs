更新于 2025-09-03本文主要介绍应用开发和运营过程中，通过安全运营中心的功能模块检查应用安全状态。

## 一、功能描述

为上架应用的开发者更方便清晰的了解上架应用的安全风险状态，钉钉提供安全运营中心功能，可以通过查看**安全基本信息**、**安全自检**、**安全审核**、**应用风险列表**查看应用的安全检测状态。

**说明**

安全运营中心功能只支持开发者后台的新版本功能，不支持旧版本。

## **二、操作步骤**

1. 登录[开发者后台](https://open-dev.dingtalk.com/)，单击目标应用，进入应用详情页。
2. 在应用详情页，单击**应用发布 > 安全运营中心**。
3. 查看使用功能的各个子模块。

   1. 安全基本信息：只做展示，根据不同的部署方式和应用状态，展示项有所差异。

      ![[dingstart-safe-operation_p873059.png]]

      安全项说明

      |  |  | | --- | --- | | 安全项 | 说明 | | 上架状态 | 应用的上架状态，包括“未上架”、“上架到自有渠道”、“上架到应用市场”等状态 | | 部署方式 | 应用选择的部署方案，包括“计算巢+数据安全中心”、“计算巢+通讯录加密”、“计算巢”等方式 | | 部署状态 | 应用选择的部署方式检测状态是否通过，包括“已通过”、“未通过”等状态 | | 安全保护层级 | 安全审核通过后，根据应用调用的接口、类目、用户量级确定安全保护级别（安全审核未通过情况下不展示） | | 安全分 | 安全审核通过后，根据应用提交的安全审核材料及安全巡检的结果，计算出的安全得分，分数范围：60分 ≤ 安全分 ≤ 100分（安全审核未通过情况下不展示） | | 安全评级 | 安全审核通过后，根据应用取得的安全分映射的安全级别，包含“优”、“良”、“中”三个级别（安全审核未通过情况下不展示） |
   2. 安全自检：点击**开始检测，**未通过的安全检查项可以参考应用风险列表提供的建议进行处理。

      ![[dingstart-safe-operation_p873061.png]]

      自检项说明

      |  |  |  |  | | --- | --- | --- | --- | | 自检项 | 说明 | 上架检测 | 版本迭代检测 | | 是否已经签署钉钉的《安全责任共担及安全承诺》 | 是否已经签署钉钉的《安全责任共担及安全承诺》 | 否 | 是 | | 是否已设置服务器出口IP | 需要在应用的开发配置中完成服务器出口IP设置，且IP地址是部署方案中要求的可信环境。 | 是 | 是 | | 应用首页或管理后台地址是否https协议 | 需要在应用的开发配置中把应用首页或管理后台地址设置为https协议。 | 是 | 是 | | 是否已完成安全网关域名申请并在应用中使用或自有域名接入商业waf防护 | 需要在钉钉安全域名中申请域名并且在应用中（如应用首页地址、PC端首页地址）使用该安全域名，或自有域名接入商业waf防护。 | 是 | 是 | | 是否通过安全审核评估 | 上广场的应用必须通过应用自检的安全审核。 | 是 | 是 | | 服务端是否已部署在阿里云 | 上架应用必须部署到阿里云，并且使用计算巢管理云资源。 | 是 | 是 | | 是否已完成数据安全中心云产品接入 | 选择“计算巢+数据安全中心”部署方案时生效，要求在阿里云上购买一年的数据安全中心云产品，并且对业务数据库授权进行数据防泄漏保护。 | 是 | 是 | | 应用已申请权限的使用率是否达标 | 应用已申请权限的使用率应达到平台规定的最低标准（80%），未使用权限点请在【开发配置】的【权限管理】移除。 | 否 | 是 | | 是否完成应用自采集的用户信息使用说明填写 | 上架应用需要在【开发配置】的【安全设置】，填写应用自采集的用户信息及使用场景说明景。 | 否 | 否 |
   3. 安全审核：只做展示，只有“审核未通过”和“审核状态已人工重置”情况下支持重新提交。

      ![[dingstart-safe-operation_p873060.png]]

      展示项说明

      |  |  | | --- | --- | | 展示项 | 说明 | | 审核状态 | 应用的安全审核状态显示，包括未提交、审核中、审核通过、审核未通过、审核状态已人工重置等。 | | 审核时间 | 最近一次审核状态的变更时间。 | | 审核结论 | 最近一次审核结论的展示或说明，可能为空。 |
   4. 应用风险列表：需要计算巢操作的风险类型请参考[计算巢风险列表处理指引](https://alidocs.dingtalk.com/i/p/JDb6Vz6xJd3mZ9NArwmPZZyKnQ0Bdz79)，其他风险类型参考解决方案建议。
