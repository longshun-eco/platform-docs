更新于 2026-01-22本文介绍了OA审批表单数据同步至宜搭。

## **简介**

OA数据表单同步到宜搭可以为很多行业带来重大意义，特别是那些需要频繁交换数据的行业。提高了工作效率、数据质量和安全性，优化行业的数据管理和协作。这项能力的运用促进企业的数字化转型和创新发展。

## **准备工作**

1. 拥有所在钉钉组织开发者后台的[开发者权限](/document/dingstart/get-developer-permissions)。
2. 已开通钉钉专业版（OA审批中使用连接器必备）。
3. 拥有一个所在组织的[宜搭应用](/document/development/yida-faq#7b1f6070ebwcc)。

## **预期效果**

OA审批数据表单提交后，同步至宜搭数据表单。

![[dingstart-approval-form-appropriate_p697458.png]]

## **步骤一：创建宜搭数据表单**

* 如果无宜搭数据表单，详情参见[如何创建流程表单？](/document/development/yida-faq#fc88b0c0ebdtm)。

  **说明**

  本示例使用**单行文本**和**数值**两个组件。
* 如果已有宜搭数据表单，可直接使用。

## **步骤二：创建连接流**

1. [创建连接流-子流程](/document/connection/configure-sub-processes-1)，并完善连接流基本信息。
2. 配置触发事件：

   1. 单击**设置入参**，在**编码模式**下粘贴下方JSON示例。

      ```
      {
        "title": "OAPI业务参数",
        "type": "object",
        "properties": {
          "number": {
            "type": "number",
            "title": "宜搭表单数字输入框",
            "description": "宜搭表单数字输入框"
          },
          "text": {
            "type": "string",
            "title": "宜搭表单文本输入框",
            "description": "宜搭表单文本输入框"
          },
          "userId": {
            "title": "用户的userid",
            "type": "string",
            "description": "用户的userid"
          }
        },
        "required": [
      ```

      ![[dingstart-approval-form-appropriate_p697479.png]]
   2. 完成配置。
3. 配置执行动作：

   1. 选择官方连接器。

      ![[dingstart-approval-form-appropriate_p697532.png]]
   2. 选择执行动作 > **保存表单数据。**

      ![[dingstart-approval-form-appropriate_p697534.png]]
   3. 配置参数。
   4. 测试并预览：

      1. 输入测试值。

         ![[dingstart-approval-form-appropriate_p697541.png]]
      2. 完成测试。

         ![[dingstart-approval-form-appropriate_p697545.png]]

         此时，测试数据已经同步至宜搭数据表单。
   5. 设置出参。

      1. 单击**设置出参**，在**编码模式**下粘贴下方JSON示例。

         ```json
         {
           "title": "OneConsole返回结果",
           "type": "object",
           "properties": {
             "success": {
               "title": "是否成功",
               "type": "boolean"
             },
             "errorMsg": {
               "title": "错误详情",
               "type": "string"
             }
           }
         }
         ```
      2. 单击**点击进行配置，**设置参数出参信息。
      3. 测试、预览并完成**保存**。

         ![[dingstart-approval-form-appropriate_p697548.png]]
   6. 发布连接流。

      ![[dingstart-approval-form-appropriate_p697549.png]]

## **步骤三：配置OA数据表单**

1. 登录钉钉客户端，单击**工作台** > **审批**。

![[dingstart-approval-form-appropriate_p696429.png]]

2. 单击管理后台。进入后台管理页面。

**重要**

进入OA审批管理后台，必须拥有OA审批应用管理权限，否则该按钮图标不显示。

![[dingstart-approval-form-appropriate_p697488.png]]

3. 创建新表单，选择**数据表单**。

![[dingstart-approval-form-appropriate_p697489.png]]

4. 表单设计。

   1. 选择表单组件，选择**单行输入框**和**数字输入框**。

      ![[dingstart-approval-form-appropriate_p697583.png]]
   2. 配置连接器。

      1. 添加连接器。

         ![[dingstart-approval-form-appropriate_p697587.png]]
      2. 设置触发条件。

         ![[dingstart-approval-form-appropriate_p697601.png]]
      3. 选择连接器。

         ![[dingstart-approval-form-appropriate_p697606.png]]
      4. 配置执行动作并保存。

         ![[dingstart-approval-form-appropriate_p697613.png]]
      5. 发布流程表单。

         ![[dingstart-approval-form-appropriate_p697617.png]]

## 恭喜，你已完成全部配置！

你已完成本教程的全部内容，可以通过以下方式进行体验。

1. 进入钉钉客户端**工作台** > **审批**。

![[dingstart-approval-form-appropriate_p696480.png]]

2. 选择上述发布的表单，填写信息并提交表单。

![[dingstart-approval-form-appropriate_p697626.png]]
