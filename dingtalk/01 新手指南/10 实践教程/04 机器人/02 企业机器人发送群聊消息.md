更新于 2025-10-21如果你需要通过企业机器人在群内发送消息，可以参考本文档操作步骤。

## 明确需求

在正式开发之前，你可以提前了解，本文档将帮助你在 Stream 模式下，你可以在群内 @ 企业机器人，接着通过调用 [机器人发送群聊消息](/document/development/the-robot-sends-a-group-message) 接口，使机器人回复你的消息。你可以尝试运行“快速体验”提供的示例 demo。这将帮助你对企业机器人的使用有一个直观的体验。

## 前提条件

1. 完成[配置企业机器人](/document/dingstart/configure-the-robot-application)流程（消息接收模式选择 Stream 模式）。
2. 机器人创建完成后，你需要[添加机器人入群](/document/dingstart/add-robot-to-group)。
3. 完成 OpenAPI 接口的[调用权限申请](/document/development/add-api-permission)，需申请`企业内机器人发送消息权限`。
4. 开发环境准备：

   |  |  | | --- | --- | | **开发环境** | **说明** | | Java | * 已安装 JDK 1.8 及以上 * 已安装 Maven 3 | | Python | * Python 3 |

## **快速体验**

1. 你可以下载示例 demo：

   |  |  | | --- | --- | | **示例** | **说明** | | Java | [stream-robot-group-message-quick-start-java.zip](https://help-static-aliyun-doc.aliyuncs.com/file-manage-files/zh-CN/20250519/vbadtr/stream-robot-group-message-quick-start-java.zip) | | Python | [stream-robot-group-message-quick-start-python.zip](https://help-static-aliyun-doc.aliyuncs.com/file-manage-files/zh-CN/20250519/xifjds/stream-robot-group-message-quick-start-python.zip) |
2. 启动示例 demo：

   |  |  | | --- | --- | | **类型** | **说明** | | Java | 1. 修改 application.properties 配置文件的参数：  * app.appKey：应用的 Client ID，详情参考 [Client ID](/document/dingstart/basic-concepts-beta#section-pje-9wf-l7c)。 * app.appSecret：应用的 Client Secret，详情参考[Client Secret](/document/dingstart/basic-concepts-beta#section-pje-9wf-l7c)。 * robot.code：机器人的编码，详情参考[机器人ID](/document/development/development-robot-overview)。 2. 启动 Application.java 文件。  3. 群内 @ 机器人发送消息。例如@机器人发送“你好”。  ![[dingstart-the-application-robot-in-the-enterprise-sends-group-chat-messages_p768539.png]]  此时，你可以看到机器人发送的消息内容。​ | | Python | 1. 引入依赖：`pip install alibabacloud-dingtalk dingtalk-stream`。  2. 在当前文件目录下输入启动命令：`python3 send_robot_group_message.py --client_id="your app client id" --client_secret="your app client secret" --robot_code="your robot code"`。  注意替换 client\_id、client\_secret 和 robot\_code，参数如何获取请参考 Java 中的说明。  3. 群内 @ 机器人发送消息。例如@机器人发送“你好”。 |
