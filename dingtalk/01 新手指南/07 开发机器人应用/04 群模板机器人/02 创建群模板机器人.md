更新于 2025-10-21如果你需要创建群聊会话时，同时将机器人添加至群内，可以使用群模板机器人，具体创建流程参考本文档操作步骤。

**重要**

群模板机器人只支持在场景群内发送群聊消息，不支持发送单聊消息。

## **前提条件**

已经完成[获取开发者权限](/document/dingstart/get-developer-permissions)流程。

## **操作步骤**

1. 登录[开发者后台](https://open-dev.dingtalk.com/)，单击**开放能力** > **场景群。**
2. 在左侧导航栏选择**机器人**，进入群机器人页面，单击**创建群机器人**。
3. 配置群机器人信息：

   |  |  | | --- | --- | | 配置项 | 说明 | | 机器人名称 | 填写群机器人名称信息。 | | 机器人头像 | 上传机器人头像图片，仅支持 JPG/PNG 文件。 | | 简介 | 简要说明机器人能力，限制 10 字以内。 | | 消息预览图 | 上传消息预览图片，仅支持 JPG/PNG 文件。 | | 详细描述 | 详细说明机器人的使用场景和详细能力，限制 256 字以内。 | | 消息回调地址 | 当群内用户 @ 群机器人并发送消息时，该消息可以通过消息回调地址发送到开发者的服务器，必须是公网可访问地址。  例如：  https://example.com/callback  机器人接收消息类型和数据格式，详情参考[机器人接收消息](/document/dingstart/dingstart-robot-receive-message)。 | | 回调消息token | 当开发者在收到回调地址对应的服务器上收到请求时，需要根据此 token 数据验证是否为合法请求。 | | 信息来源网站 | 用户单击群机器人个人信息页展示的对应开发者网站信息。 |

   配置完成后，单击**创建**。
4. 单击创建后，需等待审批，审批会在 1 ～ 2 个工作日通过。

   ![[dingstart-creation-and-installation-of-swarm-template-robots_p768463.png]]

## **后续步骤**

创建完成后，你需要将群模板机器人关联到群模板中，详情参考[关联群模板机器人到群模板](/document/dingstart/associate-group-template-robot-to-group-template)。
