更新于 2025-09-03如果你需要添加机器人入群，可以参考本文档操作步骤。

## **前提条件**

1. 完成[配置企业机器人](/document/dingstart/configure-the-robot-application)流程。
2. 完成[发布应用](/document/dingstart/publish-dingtalk-application)流程。

## **操作步骤**

1. 登录钉钉客户端，并创建群聊会话：

   * 如果你已经拥有群聊会话，可忽略此步骤。
   * 如果你没有群聊会话，你需要：

     1. 单击右上角“➕”标识，选择发起群聊。

        ![[dingstart-add-robot-to-group_p752518.png]]
     2. 单击**选择联系人**，并配置群信息。

        |  |  | | --- | --- | | **配置项** | **描述** | | 群名称 | 填写群名称信息。 | | 群场景 | 选择内部群。  **说明**  群场景必须为内部群。 | | 群归属 | 需与[配置企业机器人](/document/dingstart/configure-the-robot-application)的企业保持一致。 |
     3. 单击确认，完成群聊会话创建。
2. 进入群聊会话，单击右上角**群设置**标识。

   ![[dingstart-add-robot-to-group_p752522.png]]
3. 单击**群管理** > **机器人** > **添加机器人**。
4. 在企业机器人中选择前提条件中配置发布的机器人。

   ![[dingstart-add-robot-to-group_p752526.png]]
5. 选择需要添加的机器人后，单击添加 > 完成。
