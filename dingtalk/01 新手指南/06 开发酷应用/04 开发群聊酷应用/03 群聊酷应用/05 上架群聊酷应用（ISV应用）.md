更新于 2025-09-03如果你需要上架酷应用或变更已上架的酷应用，你可以依据本文档操作步骤进行操作。

## **前提条件**

需要完成接入[接入群聊酷应用](/document/dingstart/configuration-group-chat-quick-entry)流程。

## **操作步骤**

### **群聊酷应用上架**

提交审核后，需要在[钉钉合作伙伴组织](https://alidocs.dingtalk.com/i/p/KrwmPQ5g4OEAG79n/docs/6wPdlBDrQk4JYBBMgPR28XKx72oEGeL5?dontjump=true)-OA审批内提交**酷应用上架申请**审批单，并提供以下信息，审批通过后1-2个工作日将完成上架。

|  |  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| **群应用code** | **群应用名称** | **一句话简介** | **群应用图标（图）** | **适合行业** | **适合角色** | **缩略图详情3张（图<100k）** | **客户案例1-2张（图<100k）** | **功能说明** | **员工邀请管理员开通引导语** |
| 例如：COOLAPP-1-1018DAB422412127xxxxxxx | 控制在6字以内 | 展示在应用详情页，控制在12字内 | / | 最多可选3个行业 | 最多可选3个角色 | / | / | 应用详细介绍100字以内 | 请以员工向管理员说明价值的视角介绍该群应用30以内 |

图片相关设计规范参考：

### **群聊酷应用变更**

修改群聊酷应用信息，可在[钉钉合作伙伴组织](https://alidocs.dingtalk.com/i/p/KrwmPQ5g4OEAG79n/docs/6wPdlBDrQk4JYBBMgPR28XKx72oEGeL5?dontjump=true)-OA审批提交**酷应用版本更新**审批单，预计1-2个工作日完成版本更新。
